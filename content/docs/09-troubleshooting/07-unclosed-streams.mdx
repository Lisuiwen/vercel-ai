# 未关闭的流

有时流未正确关闭，会导致意外行为。以下是流未关闭时常见的一些问题。

## 问题

可流式 UI 更新很慢。

## 解决方案

这通常是因为使用 [`createStreamableUI`](/docs/reference/ai-sdk-rsc/create-streamable-ui) 创建可流式 UI 后没有关闭流。
修复方法是务必调用 [`.done()`](/docs/reference/ai-sdk-rsc/create-streamable-ui#done) 关闭流，
这样流才会正常结束。

```tsx file='app/actions.tsx'
import { createStreamableUI } from '@ai-sdk/rsc';

const submitMessage = async () => {
  'use server';

  const stream = createStreamableUI('1');

  stream.update('2');
  stream.append('3');
  stream.done('4'); // [!code ++]

  return stream.value;
};
```
