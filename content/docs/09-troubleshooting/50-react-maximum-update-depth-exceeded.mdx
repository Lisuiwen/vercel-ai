# React 报错「Maximum update depth exceeded」

## 问题

在 React 项目中配合 `useChat` 或 `useCompletion` 使用 AI SDK，在 AI 回复流式输出时出现：`Maximum update depth exceeded`。

## 背景

默认情况下，每个到达的数据块都会触发界面重渲染。
在较慢设备或需要更新复杂组件（例如 Markdown）时，可能造成渲染压力过大。节流可以缓解。

## 解决方案

使用 `experimental_throttle` 选项限制界面更新频率：

### `useChat`

```tsx filename="page.tsx" highlight="2-3"
const { messages, ... } = useChat({
  // Throttle the messages and data updates to 50ms:
  experimental_throttle: 50
})
```

### `useCompletion`

```tsx filename="page.tsx" highlight="2-3"
const { completion, ... } = useCompletion({
  // Throttle the completion and data updates to 50ms:
  experimental_throttle: 50
})
```
