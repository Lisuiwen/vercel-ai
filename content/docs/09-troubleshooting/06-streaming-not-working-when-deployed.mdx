# 部署后流式不工作

## 问题

在本地开发环境中，使用 AI SDK 的流式传输正常。
但部署后，线上应用里流式不工作。
不会逐段输出，而是过一会儿才一次性返回完整响应。

## 原因

原因因部署环境而异。

## 解决方案

可以尝试：

- 添加 `'Transfer-Encoding': 'chunked'` 和/或 `Connection: 'keep-alive'` 响应头

  ```tsx
  return result.toUIMessageStreamResponse({
    headers: {
      'Transfer-Encoding': 'chunked',
      Connection: 'keep-alive',
    },
  });
  ```
