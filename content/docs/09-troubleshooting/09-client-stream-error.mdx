# 「Only plain objects can be passed from client components」Server Action 错误

## 问题

我在 Server Actions 中使用 [`streamText`](/docs/reference/ai-sdk-core/stream-text)，出现 `"only plain objects and a few built ins can be passed from client components"` 错误。

## 背景

当你试图从 Server Action 向客户端组件返回不可序列化的对象时会出现该错误。`streamText` 返回的对象可能包含方法或复杂结构，无法直接序列化传给客户端。

## 解决方案

需要确保 Server Action 只返回可序列化的数据。可以这样调整：

1. 不要返回整个 `streamText` 结果对象，只提取需要的可序列化字段。
2. 使用 [`createStreamableValue`](/docs/reference/ai-sdk-rsc/create-streamable-value) 创建可安全传给客户端的可流式值。

示例实现见：[Streaming Text Generation](/examples/next-app/basics/streaming-text-generation)。

这样只有可序列化的数据（例如文本）会传到客户端，可避免「only plain objects」错误。
