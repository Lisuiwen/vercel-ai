# Azure OpenAI 流式输出缓慢

## 问题

在使用托管在 Azure 上的 OpenAI 时，流式输出很慢，且往往是大块到达。

## 原因

这是 Microsoft Azure 侧的问题。部分用户反馈可通过以下方式缓解：

- **更新内容过滤设置**：
  在 [Azure AI Studio](https://ai.azure.com/) 中，进入「共享资源」>「内容过滤器」，新建内容过滤器，
  并在「输出过滤器」下将「流式处理模式（预览）」从「默认」
  改为「异步筛选」。

## 解决方案

可以使用 [`smoothStream` 转换](/docs/ai-sdk-core/generating-text#smoothing-streams)，按词逐个流式输出。

```tsx highlight="6"
import { smoothStream, streamText } from 'ai';

const result = streamText({
  model,
  prompt,
  experimental_transform: smoothStream(),
});
```
