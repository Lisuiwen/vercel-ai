# `useChat`「Failed to Parse Stream String」错误

## 问题

我在使用 [`useChat`](/docs/reference/ai-sdk-ui/use-chat) 或 [`useCompletion`](/docs/reference/ai-sdk-ui/use-completion)，出现 `"Failed to parse stream string. Invalid code"` 错误。AI SDK 版本为 `3.0.20` 或更新。

## 背景

AI SDK 自 `3.0.20` 起切换到流数据协议。
[`useChat`](/docs/reference/ai-sdk-ui/use-chat) 与 [`useCompletion`](/docs/reference/ai-sdk-ui/use-completion) 期望接收支持数据、工具调用等的流片段。
该错误表示无法解析流。
可能原因包括：后端仍使用较旧版 AI SDK、自定义 provider 提供的是文本流，或使用了原始 LangChain 流结果。

## 解决方案

可将 [`useChat`](/docs/reference/ai-sdk-ui/use-chat) 与 [`useCompletion`](/docs/reference/ai-sdk-ui/use-completion) 切换为原始文本流处理，使用 [`streamProtocol`](/docs/reference/ai-sdk-ui/use-completion#stream-protocol) 参数。
设为 `text` 即可：

```tsx
const { messages, append } = useChat({ streamProtocol: 'text' });
```
