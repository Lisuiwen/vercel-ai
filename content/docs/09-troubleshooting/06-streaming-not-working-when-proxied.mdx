# 经代理时流式不工作

## 问题

在本地开发或某些经代理部署的环境中，使用 AI SDK 时流式不工作。
不会逐段输出，而是过一会儿才一次性返回完整响应。

## 原因

问题来自代理中间件。

若中间件对响应做了压缩配置，会导致流式失败。

## 解决方案

可以尝试以下做法（仅影响流式 API）：

- 添加 `'Content-Encoding': 'none'` 响应头

  ```tsx
  return result.toUIMessageStreamResponse({
    headers: {
      'Content-Encoding': 'none',
    },
  });
  ```
