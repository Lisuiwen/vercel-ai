# `streamText` 不工作

## 问题

我在使用 [`streamText`](/docs/reference/ai-sdk-core/stream-text)，但它不工作。
不会抛出错误，流里只有错误片段。

## 背景

`streamText` 会立即开始流式输出，以便无需等待模型即可发送数据。
错误会成为流的一部分而不会抛出，从而避免例如导致服务端进程崩溃。

## 解决方案

可提供在出错时触发的 `onError` 回调来记录错误。

```tsx highlight="6-8"
import { streamText } from 'ai';
__PROVIDER_IMPORT__;

const result = streamText({
  model: __MODEL__,
  prompt: 'Invent a new holiday and describe its traditions.',
  onError({ error }) {
    console.error(error); // your error logging logic here
  },
});
```
