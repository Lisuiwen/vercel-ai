# 客户端组件中的 Server Actions

你可以在客户端组件中使用 Server Actions，但有时会遇到以下问题。

## 问题

不允许在客户端组件中内联定义带 `"use server"` 注解的 Server Actions。

## 解决方案

若要在客户端组件中使用 Server Actions，可以：

- 在单独文件的顶部写 `"use server"` 并从中导出。
- 通过 Server Component 的 props 向下传递。
- 组合使用 [`createAI`](/docs/reference/ai-sdk-rsc/create-ai) 与 [`useActions`](/docs/reference/ai-sdk-rsc/use-actions) 等钩子来访问它们。

进一步了解 [Server Actions 与变更](https://nextjs.org/docs/app/api-reference/functions/server-actions#with-client-components)。

```ts file='actions.ts'
'use server';

import { generateText } from 'ai';
__PROVIDER_IMPORT__;

export async function getAnswer(question: string) {
  'use server';

  const { text } = await generateText({
    model: __MODEL__,
    prompt: question,
  });

  return { answer: text };
}
```
