# TypeScript 报错「Cannot find namespace 'JSX'」

## 问题

在没有 React 的项目中使用 AI SDK（例如 Hono 服务端），出现：
`error TS2503: Cannot find namespace 'JSX'.`

## 背景

AI SDK 依赖 `@types/react`，由其定义 `JSX` 命名空间。
在 AI SDK 的下一次大版本中计划移除该依赖。

## 解决方案

安装 `@types/react` 作为依赖即可消除错误。

```bash
npm install @types/react
```
