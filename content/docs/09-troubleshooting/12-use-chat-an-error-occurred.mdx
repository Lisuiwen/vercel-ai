# `useChat`「An error occurred」

## 问题

我在使用 [`useChat`](/docs/reference/ai-sdk-ui/use-chat)，报错「An error occurred」。

## 背景

使用 `toDataStreamResponse` 时，出于安全考虑（secure-by-default），来自 `streamText` 的错误信息默认会被遮蔽。
这样可避免向客户端泄露敏感信息。

## 解决方案

若要将错误详情转发到客户端或记录日志，可在调用 `toUIMessageStreamResponse` 时使用 `getErrorMessage` 函数。

```tsx
export function errorHandler(error: unknown) {
  if (error == null) {
    return 'unknown error';
  }

  if (typeof error === 'string') {
    return error;
  }

  if (error instanceof Error) {
    return error.message;
  }

  return JSON.stringify(error);
}
```

```tsx
const result = streamText({
  // ...
});

return result.toUIMessageStreamResponse({
  getErrorMessage: errorHandler,
});
```

若使用 `createDataStreamResponse`，可在调用 `toDataStreamResponse` 时使用 `onError`：

```tsx
const response = createDataStreamResponse({
  // ...
  async execute(dataStream) {
    // ...
  },
  onError: errorHandler,
});
```
