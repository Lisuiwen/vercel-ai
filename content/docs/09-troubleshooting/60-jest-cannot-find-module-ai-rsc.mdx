# Jest: cannot find module '@ai-sdk/rsc'

## 问题

我在使用 AI SDK RSC，并用 Jest 为 RSC 组件编写测试。

出现错误：`Cannot find module '@ai-sdk/rsc'`。

## 解决方案

在 Jest 配置的 `moduleNameMapper` 中配置模块解析：

```json filename="jest.config.js"
"moduleNameMapper": {
  "^@ai-sdk/rsc$": "<rootDir>/node_modules/@ai-sdk/rsc/dist"
}
```
