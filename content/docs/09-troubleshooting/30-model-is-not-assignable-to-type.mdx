# 模型不能赋值给类型「LanguageModelV1」

## 问题

升级 AI SDK 后出现：`Type 'SomeModel' is not assignable to type 'LanguageModelV1'.`

<Note>类似错误也可能出现在 `EmbeddingModelV4` 上。</Note>

## 背景

模型规范会不断加入新特性，
可能与旧版 provider 不兼容。

## 解决方案

将各 provider 包与 AI SDK 升级到最新版本。
