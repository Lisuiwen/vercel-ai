# AI_RetryError

此错误在重试操作失败时发生。

## 属性

- `reason`：重试失败的原因
- `lastError`：重试期间最近发生的错误
- `errors`：重试尝试期间发生的所有错误的数组
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_RetryError` 的实例：

```typescript
import { RetryError } from 'ai';

if (RetryError.isInstance(error)) {
  // Handle the error
}
```
