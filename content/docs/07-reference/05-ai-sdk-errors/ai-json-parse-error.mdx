# AI_JSONParseError

此错误在 JSON 解析失败时发生。

## 属性

- `text`：无法解析的文本值
- `cause`：底层解析错误（构造函数中必填）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_JSONParseError` 的实例：

```typescript
import { JSONParseError } from 'ai';

if (JSONParseError.isInstance(error)) {
  // Handle the error
}
```
