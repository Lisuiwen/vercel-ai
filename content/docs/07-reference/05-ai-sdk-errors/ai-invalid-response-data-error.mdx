# AI_InvalidResponseDataError

此错误在服务器返回的响应包含无效数据内容时发生。

## 属性

- `data`：无效的响应数据值
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_InvalidResponseDataError` 的实例：

```typescript
import { InvalidResponseDataError } from 'ai';

if (InvalidResponseDataError.isInstance(error)) {
  // Handle the error
}
```
