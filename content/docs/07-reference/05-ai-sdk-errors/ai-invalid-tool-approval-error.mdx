# AI_InvalidToolApprovalError

此错误在工具审批响应引用了未知的 `approvalId` 时发生。消息历史中没有匹配的 `tool-approval-request`。

## 属性

- `approvalId`：未找到的审批 ID

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_InvalidToolApprovalError` 的实例：

```typescript
import { InvalidToolApprovalError } from 'ai';

if (InvalidToolApprovalError.isInstance(error)) {
  // Handle the error
}
```
