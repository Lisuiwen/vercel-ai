# AI_MessageConversionError

此错误在消息转换失败时发生。

## 属性

- `originalMessage`：转换失败的原始消息
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_MessageConversionError` 的实例：

```typescript
import { MessageConversionError } from 'ai';

if (MessageConversionError.isInstance(error)) {
  // Handle the error
}
```
