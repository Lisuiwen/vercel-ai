# AI_TooManyEmbeddingValuesForCallError

此错误在单次嵌入调用中提供的值过多时发生。

## 属性

- `provider`：AI 提供方名称
- `modelId`：嵌入模型的 ID
- `maxEmbeddingsPerCall`：每次调用允许的最大嵌入数量
- `values`：所提供的值的数组

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_TooManyEmbeddingValuesForCallError` 的实例：

```typescript
import { TooManyEmbeddingValuesForCallError } from 'ai';

if (TooManyEmbeddingValuesForCallError.isInstance(error)) {
  // Handle the error
}
```
