# AI_DownloadError

此错误在下载失败时发生。

## 属性

- `url`：下载失败的 URL
- `statusCode`：服务器返回的 HTTP 状态码（可选）
- `statusText`：服务器返回的 HTTP 状态文本（可选）
- `cause`：导致下载失败的底层错误（可选）
- `message`：包含下载失败详情的错误消息（可选，自动生成）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_DownloadError` 的实例：

```typescript
import { DownloadError } from 'ai';

if (DownloadError.isInstance(error)) {
  // Handle the error
}
```
