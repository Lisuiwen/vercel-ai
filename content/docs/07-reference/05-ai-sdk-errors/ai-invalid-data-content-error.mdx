# AI_InvalidDataContentError

此错误在多模态消息片段中提供的数据内容无效时发生。请参阅[多模态消息的提示示例](/docs/foundations/prompts#message-prompts)。

## 属性

- `content`：无效的内容值
- `cause`：导致此错误的底层错误（可选）
- `message`：描述期望与收到内容类型的错误消息（可选，自动生成）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_InvalidDataContentError` 的实例：

```typescript
import { InvalidDataContentError } from 'ai';

if (InvalidDataContentError.isInstance(error)) {
  // Handle the error
}
```
