# AI_NoOutputGeneratedError

此错误在未生成任何 LLM 输出时抛出，例如因发生错误。

## 属性

- `message`：错误消息（可选，默认为 `'No output generated.'`）
- `cause`：导致未生成输出的底层错误（可选）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoOutputGeneratedError` 的实例：

```typescript
import { NoOutputGeneratedError } from 'ai';

if (NoOutputGeneratedError.isInstance(error)) {
  // Handle the error
}
```
