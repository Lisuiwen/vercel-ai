# AI_APICallError

此错误在 API 调用失败时发生。

## 属性

- `url`：导致失败的 API 请求的 URL
- `requestBodyValues`：发送到 API 的请求体字段值
- `statusCode`：API 返回的 HTTP 状态码（可选）
- `responseHeaders`：API 返回的响应头（可选）
- `responseBody`：API 返回的响应体（可选）
- `isRetryable`：根据状态码判断请求是否可重试
- `data`：与该错误关联的任意额外数据（可选）
- `cause`：导致 API 调用失败的底层错误（可选）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_APICallError` 的实例：

```typescript
import { APICallError } from 'ai';

if (APICallError.isInstance(error)) {
  // Handle the error
}
```
