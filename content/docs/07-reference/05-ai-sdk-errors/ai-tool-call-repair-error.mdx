# ToolCallRepairError

此错误在尝试修复无效工具调用失败时发生。
通常在 AI 尝试修复
`NoSuchToolError` 或 `InvalidToolInputError` 时会出现。

## 属性

- `originalError`：触发修复尝试的原始错误（`NoSuchToolError` 或 `InvalidToolInputError`）
- `message`：错误消息
- `cause`：导致修复失败的底层错误

## 如何检查此错误

你可以通过以下方式检查错误是否为 `ToolCallRepairError` 的实例：

```typescript
import { ToolCallRepairError } from 'ai';

if (ToolCallRepairError.isInstance(error)) {
  // Handle the error
}
```
