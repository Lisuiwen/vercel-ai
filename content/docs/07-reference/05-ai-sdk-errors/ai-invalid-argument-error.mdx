# AI_InvalidArgumentError

此错误在提供了无效参数时发生。

## 属性

- `parameter`：无效参数的名称
- `value`：无效的值
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_InvalidArgumentError` 的实例：

```typescript
import { InvalidArgumentError } from 'ai';

if (InvalidArgumentError.isInstance(error)) {
  // Handle the error
}
```
