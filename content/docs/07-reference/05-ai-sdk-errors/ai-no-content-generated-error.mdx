# AI_NoContentGeneratedError

此错误在 AI 提供方未能生成内容时发生。

## 属性

- `message`：错误消息（可选，默认为 `'No content generated.'`）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoContentGeneratedError` 的实例：

```typescript
import { NoContentGeneratedError } from 'ai';

if (NoContentGeneratedError.isInstance(error)) {
  // Handle the error
}
```
