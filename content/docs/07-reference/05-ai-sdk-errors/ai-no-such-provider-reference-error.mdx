# AI_NoSuchProviderReferenceError

此错误在无法解析提供方引用时发生，因为
在提供方引用映射中找不到指定的提供方。

## 属性

- `provider`：未找到的提供方
- `reference`：被检索的完整提供方引用映射
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoSuchProviderReferenceError` 的实例：

```typescript
import { NoSuchProviderReferenceError } from 'ai';

if (NoSuchProviderReferenceError.isInstance(error)) {
  // Handle the error
}
```
