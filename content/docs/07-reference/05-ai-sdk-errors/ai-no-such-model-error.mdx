# AI_NoSuchModelError

此错误在找不到模型 ID 时发生。

## 属性

- `modelId`：未找到的模型 ID
- `modelType`：模型类型（`'languageModel'`、`'embeddingModel'`、`'imageModel'`、`'transcriptionModel'`、`'speechModel'` 或 `'rerankingModel'`）
- `message`：错误消息（可选，根据 `modelId` 和 `modelType` 自动生成）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoSuchModelError` 的实例：

```typescript
import { NoSuchModelError } from 'ai';

if (NoSuchModelError.isInstance(error)) {
  // Handle the error
}
```
