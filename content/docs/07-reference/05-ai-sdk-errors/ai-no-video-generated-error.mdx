# AI_NoVideoGeneratedError

此错误在 AI 提供方未能生成视频时发生。
可能由以下原因引起：

- 模型未能生成响应。
- 模型生成了无效响应。

## 属性

- `message`：错误消息（可选，默认为 `'No video generated.'`）。
- `responses`：关于视频模型响应的元数据，包括时间戳、模型和响应头（可选）。
- `cause`：错误原因。可用于更详细的错误处理（可选）。

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoVideoGeneratedError` 的实例：

```typescript
import {
  experimental_generateVideo as generateVideo,
  NoVideoGeneratedError,
} from 'ai';

try {
  await generateVideo({ model, prompt });
} catch (error) {
  if (NoVideoGeneratedError.isInstance(error)) {
    console.log('NoVideoGeneratedError');
    console.log('Cause:', error.cause);
    console.log('Responses:', error.responses);
  }
}
```
