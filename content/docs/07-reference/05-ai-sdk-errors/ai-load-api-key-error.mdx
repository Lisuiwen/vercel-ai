# AI_LoadAPIKeyError

此错误在 API 密钥未能成功加载时发生。

## 属性

- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_LoadAPIKeyError` 的实例：

```typescript
import { LoadAPIKeyError } from 'ai';

if (LoadAPIKeyError.isInstance(error)) {
  // Handle the error
}
```
