# AI_InvalidToolInputError

此错误在提供了无效的工具输入时发生。

## 属性

- `toolName`：输入无效的工具名称
- `toolInput`：无效的工具输入
- `message`：错误消息
- `cause`：错误原因

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_InvalidToolInputError` 的实例：

```typescript
import { InvalidToolInputError } from 'ai';

if (InvalidToolInputError.isInstance(error)) {
  // Handle the error
}
```
