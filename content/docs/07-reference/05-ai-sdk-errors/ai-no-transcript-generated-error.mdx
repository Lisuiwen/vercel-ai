# AI_NoTranscriptGeneratedError

此错误在无法从输入生成转写文本时发生。

## 属性

- `responses`：转写模型响应元数据数组（构造函数中必填）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoTranscriptGeneratedError` 的实例：

```typescript
import { NoTranscriptGeneratedError } from 'ai';

if (NoTranscriptGeneratedError.isInstance(error)) {
  // Handle the error
}
```
