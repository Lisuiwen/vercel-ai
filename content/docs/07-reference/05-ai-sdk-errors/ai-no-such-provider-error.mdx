# AI_NoSuchProviderError

此错误在找不到提供方 ID 时发生。

## 属性

- `providerId`：未找到的提供方 ID
- `availableProviders`：可用提供方 ID 的数组
- `modelId`：模型 ID
- `modelType`：模型类型
- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoSuchProviderError` 的实例：

```typescript
import { NoSuchProviderError } from 'ai';

if (NoSuchProviderError.isInstance(error)) {
  // Handle the error
}
```
