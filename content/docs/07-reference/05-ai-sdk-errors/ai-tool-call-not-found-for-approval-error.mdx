# AI_ToolCallNotFoundForApprovalError

此错误在工具审批请求引用了未找到的工具调用时发生。在处理提供方发出的审批请求（例如 MCP 流程）且引用的工具调用 ID 不存在时可能会出现。

## 属性

- `toolCallId`：未找到的工具调用 ID
- `approvalId`：审批请求 ID

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_ToolCallNotFoundForApprovalError` 的实例：

```typescript
import { ToolCallNotFoundForApprovalError } from 'ai';

if (ToolCallNotFoundForApprovalError.isInstance(error)) {
  // Handle the error
}
```
