# AI_NoSuchToolError

此错误在模型尝试调用不可用工具时发生。

## 属性

- `toolName`：未找到的工具名称
- `availableTools`：可用工具名称的数组（可选）
- `message`：错误消息（可选，根据 `toolName` 和 `availableTools` 自动生成）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_NoSuchToolError` 的实例：

```typescript
import { NoSuchToolError } from 'ai';

if (NoSuchToolError.isInstance(error)) {
  // Handle the error
}
```
