# AI_TypeValidationError

此错误在类型校验失败时发生。

## 属性

- `value`：未通过校验的值
- `cause`：底层校验错误（构造函数中必填）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_TypeValidationError` 的实例：

```typescript
import { TypeValidationError } from 'ai';

if (TypeValidationError.isInstance(error)) {
  // Handle the error
}
```
