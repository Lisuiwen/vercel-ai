# AI_UnsupportedFunctionalityError

此错误在功能不受支持时发生。

## 属性

- `functionality`：不受支持的功能名称
- `message`：错误消息（可选，根据 `functionality` 自动生成）

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_UnsupportedFunctionalityError` 的实例：

```typescript
import { UnsupportedFunctionalityError } from 'ai';

if (UnsupportedFunctionalityError.isInstance(error)) {
  // Handle the error
}
```
