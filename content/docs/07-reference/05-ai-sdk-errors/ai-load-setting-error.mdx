# AI_LoadSettingError

此错误在设置未能成功加载时发生。

## 属性

- `message`：错误消息

## 如何检查此错误

你可以通过以下方式检查错误是否为 `AI_LoadSettingError` 的实例：

```typescript
import { LoadSettingError } from 'ai';

if (LoadSettingError.isInstance(error)) {
  // Handle the error
}
```
