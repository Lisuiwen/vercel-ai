# `pipeUIMessageStreamToResponse`

`pipeUIMessageStreamToResponse` 函数将流式数据传输到 Node.js ServerResponse 对象（参见[流式数据](/docs/ai-sdk-ui/streaming-data)）。

## 导入

<Snippet
  text={`import { pipeUIMessageStreamToResponse } from "ai"`}
  prompt={false}
/>

## 示例

```tsx
pipeUIMessageStreamToResponse({
  response: serverResponse,
  status: 200,
  statusText: 'OK',
  headers: {
    'Custom-Header': 'value',
  },
  stream: myUIMessageStream,
  consumeSseStream: ({ stream }) => {
    // Optional: consume the SSE stream independently
    console.log('Consuming SSE stream:', stream);
  },
});
```
