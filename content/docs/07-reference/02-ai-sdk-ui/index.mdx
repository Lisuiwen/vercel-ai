# AI SDK UI

[AI SDK UI](/docs/ai-sdk-ui) 旨在帮助你轻松构建交互式聊天、补全与助手类应用。
它是框架无关的工具包，可简化将高级 AI 能力集成到应用中的流程。

AI SDK UI 包含以下 Hooks：

## UI 框架支持

AI SDK UI 支持以下框架：[React](https://react.dev/)、[Svelte](https://svelte.dev/)、[Vue.js](https://vuejs.org/)、
[Angular](https://angular.dev/) 与 [SolidJS](https://www.solidjs.com/)。

各框架所支持功能的对比：

|                                                                 | [useChat](/docs/reference/ai-sdk-ui/use-chat) | [useCompletion](/docs/reference/ai-sdk-ui/use-completion) | [useObject](/docs/reference/ai-sdk-ui/use-object) | [MCP Apps](/docs/ai-sdk-core/mcp-apps) |
