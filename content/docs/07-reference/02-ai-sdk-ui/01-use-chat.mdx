# `useChat()`

让你可以轻松为聊天机器人应用创建对话式用户界面。它支持从 AI provider 流式传输聊天消息、管理聊天状态，并在收到新消息时自动更新 UI。

<Note>
  `useChat` API 在 AI SDK 5.0 中已有重大更新。它现在使用基于 transport
  的架构，并且不再在内部管理输入状态。详情请参阅[迁移指南](/docs/migration-guides/migration-guide-5-0#usechat-changes)。
</Note>
