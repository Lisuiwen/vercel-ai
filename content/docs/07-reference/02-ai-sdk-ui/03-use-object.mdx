# `experimental_useObject()`

<Note>
  `useObject` 是实验性功能，仅在 React、Svelte 和 Vue 中可用。
</Note>

允许你消费表示 JSON 对象的文本流，并基于 schema 将其解析为完整对象。
你可以在后端将它与 [`streamText`](/docs/reference/ai-sdk-core/stream-text) 和 [`Output.object()`](/docs/reference/ai-sdk-core/output#output-object) 一起使用。

```tsx
'use client';

import { experimental_useObject as useObject } from '@ai-sdk/react';

export default function Page() {
  const { object, submit } = useObject({
    api: '/api/use-object',
    schema: z.object({ content: z.string() }),
  });

  return (
    <div>
      <button onClick={() => submit('example input')}>Generate</button>
      {object?.content && <p>{object.content}</p>}
    </div>
  );
}
```
