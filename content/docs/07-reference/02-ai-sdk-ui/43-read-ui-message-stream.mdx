# readUIMessageStream

将 `UIMessageChunk` 流转换为 `UIMessage` 的 `AsyncIterableStream`。

UI message stream 也适用于 Chat 之外的场景，例如终端 UI、客户端上的自定义流消费，或 RSC（React Server Components）。

## 导入

```tsx
import { readUIMessageStream } from 'ai';
```

### 返回值

`UIMessage` 的 `AsyncIterableStream`。每个流部分表示同一条消息在完成过程中的不同状态。

完整示例和使用场景请参阅[读取 UI Message Streams](/docs/ai-sdk-ui/reading-ui-message-streams)。
