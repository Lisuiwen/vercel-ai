# AI SDK Workflow

[`@ai-sdk/workflow`](/docs/agents/workflow-agent) 提供 `WorkflowAgent` 类，用于在工作流内构建持久化、可恢复的 AI 智能体。它处理工具 schema 序列化、工作流步骤边界以及内置的工具审批流程。
