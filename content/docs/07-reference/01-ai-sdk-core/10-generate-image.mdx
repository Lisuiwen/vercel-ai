# `generateImage()`

使用图像模型根据给定提示生成图像。

它非常适合需要以编程方式生成图像的使用场景，
例如创建视觉内容或为数据增强生成图像。

```ts
import { generateImage } from 'ai';
__PROVIDER_IMPORT__;

const { images } = await generateImage({
  model: __IMAGE_MODEL__,
  prompt: 'A futuristic cityscape at sunset',
  n: 3,
  size: '1024x1024',
});

console.log(images);
```
