# `filterActiveTools()`

<Note type="warning">`filterActiveTools` 为实验性功能。</Note>

`filterActiveTools` 将工具集过滤为仅包含 `activeTools` 中列出的字符串工具名。
若 `activeTools` 为 `undefined`，返回原始工具集。
若 `tools` 为 `undefined`，返回 `undefined`。

`filterActiveTools` 适用于在特定步骤限制发给模型的工具集合。

```ts
import { experimental_filterActiveTools as filterActiveTools, tool } from 'ai';
import { z } from 'zod';

const tools = {
  weather: tool({
    description: 'Get the weather for a city',
    inputSchema: z.object({ city: z.string() }),
  }),
  time: tool({
    description: 'Get the current time for a city',
    inputSchema: z.object({ city: z.string() }),
  }),
};

const activeTools = ['weather'] as const;

const filteredTools = filterActiveTools({
  tools,
  activeTools,
});
```

### 返回值

过滤后的工具集；当 `tools` 为 `undefined` 时返回 `undefined`。

当 `activeTools` 为字面量数组（如 `["weather"] as const`）时，
TypeScript 会将返回类型收窄为该工具子集。

## 类型

### `ActiveTools`

```ts
type ActiveTools<TOOLS extends ToolSet> =
  | ReadonlyArray<keyof TOOLS & string>
  | undefined;
```

`ActiveTools` 仅接受工具集中字符串形式的键，因为运行时工具名为字符串。
