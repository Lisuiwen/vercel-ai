# `LanguageModelV4Middleware`

<Note type="warning">
  语言模型中间件为实验性功能。
</Note>

语言模型中间件通过拦截并修改对语言模型的调用来增强行为，可用于以与具体模型无关的方式加入护栏、RAG、缓存、日志等。

详见[语言模型中间件](/docs/ai-sdk-core/middleware)。

## 导入

<Snippet
  text={`import { LanguageModelV4Middleware } from "ai"`}
  prompt={false}
/>
