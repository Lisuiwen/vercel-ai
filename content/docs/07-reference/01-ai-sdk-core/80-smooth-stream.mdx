# `smoothStream()`

`smoothStream` 创建用于 `streamText` 的 `transform` 选项的 `TransformStream`，通过缓冲并在可配置延迟后释放完整分块来平滑文本与推理的流式输出，使流式阅读体验更自然。

```ts highlight={"6-9"}
import { smoothStream, streamText } from 'ai';

const result = streamText({
  model,
  prompt,
  experimental_transform: smoothStream({
    delayInMs: 20, // 可选：默认 10ms
    chunking: 'line', // 可选：默认 'word'
  }),
});
```

#### 非拉丁语言下按词分块的注意事项

对**不用空格分词**的语言，基于词的分块**效果较差**，包括：

- 中文
- 日文
- 韩文
- 越南文
- 泰文

#### 使用 Intl.Segmenter（推荐）

对上述语言，建议使用 `Intl.Segmenter` 进行符合区域设置的词边界识别。
这是在 CJK 及其他语言上较准确的做法。

<Note>
  `Intl.Segmenter` 在 Node.js 16+ 与现代浏览器（Chrome
  87+、Firefox 125+、Safari 14.1+）中可用。
</Note>

```tsx filename="Japanese example with Intl.Segmenter"
import { smoothStream, streamText } from 'ai';
__PROVIDER_IMPORT__;

const segmenter = new Intl.Segmenter('ja', { granularity: 'word' });

const result = streamText({
  model: __MODEL__,
  prompt: 'Your prompt here',
  experimental_transform: smoothStream({
    chunking: segmenter,
  }),
});
```

```tsx filename="Chinese example with Intl.Segmenter"
import { smoothStream, streamText } from 'ai';
__PROVIDER_IMPORT__;

const segmenter = new Intl.Segmenter('zh', { granularity: 'word' });

const result = streamText({
  model: __MODEL__,
  prompt: 'Your prompt here',
  experimental_transform: smoothStream({
    chunking: segmenter,
  }),
});
```

#### 基于正则的分块

将 `RegExp` 传给 `chunking` 即可使用正则分块。

```ts
// 按下划线切分：
smoothStream({
  chunking: /_+/,
});

// 也可这样写，行为相同
smoothStream({
  chunking: /[^_]*_/,
});
```

#### 自定义回调分块

向 `chunking` 传入函数即可自定义分块逻辑。

```ts
smoothStream({
  chunking: text => {
    const findString = 'some string';
    const index = text.indexOf(findString);

    if (index === -1) {
      return null;
    }

    return text.slice(0, index) + findString;
  },
});
```

### 返回值

返回 `TransformStream`，其行为如下：

- 缓冲传入的文本与推理分块
- 在命中分块规则时释放内容
- 在分块之间加入可配置延迟以实现平滑输出
- 非文本/推理分块（如工具调用、step-finish 事件）立即透传
