# `addToolInputExamplesMiddleware`

`addToolInputExamplesMiddleware` 将输入示例追加到工具描述中。对于**原生不支持 `inputExamples` 属性**的语言模型提供商特别有用——中间件会将示例序列化并注入工具的 `description`，供模型学习。

## 导入

<Snippet
  text={`import { addToolInputExamplesMiddleware } from "ai"`}
  prompt={false}
/>

## API

### 签名

```ts
function addToolInputExamplesMiddleware(options?: {
  prefix?: string;
  format?: (example: { input: JSONObject }, index: number) => string;
  remove?: boolean;
}): LanguageModelMiddleware;
```

### 参数

<PropertiesTable
  content={[
    {
      name: 'prefix',
      type: 'string',
      isOptional: true,
      description:
        "置于输入示例区块前的字符串前缀。默认为 `'Input Examples:'`。",
    },
    {
      name: 'format',
      type: '(example: { input: JSONObject }, index: number) => string',
      isOptional: true,
      description:
        '可选的每条示例自定义格式化函数。接收示例对象及其下标。默认：JSON.stringify(example.input)。',
    },
    {
      name: 'remove',
      type: 'boolean',
      isOptional: true,
      description:
        '是否在将示例加入描述后从工具上移除 `inputExamples` 属性。默认：true。',
    },
  ]}
/>

### 返回值

[LanguageModelMiddleware](/docs/ai-sdk-core/middleware)，会：

- 查找带 `inputExamples` 属性的函数工具。
- 序列化每个输入示例（默认 JSON，或使用自定义 `format`）。
- 在工具描述末尾以前缀 `prefix` 为标题追加包含所有格式化示例的区块。
- 从工具上移除 `inputExamples` 属性（除非 `remove: false`）。
- 其他工具（含无示例者）原样透传。

## 使用示例

```ts
import {
  generateText,
  tool,
  wrapLanguageModel,
  addToolInputExamplesMiddleware,
} from 'ai';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';

const model = wrapLanguageModel({
  model: __MODEL__,
  middleware: addToolInputExamplesMiddleware({
    prefix: 'Input Examples:',
    format: (example, index) =>
      `${index + 1}. ${JSON.stringify(example.input)}`,
  }),
});

const result = await generateText({
  model,
  tools: {
    weather: tool({
      description: 'Get the weather in a location',
      inputSchema: z.object({ location: z.string() }),
      inputExamples: [
        { input: { location: 'San Francisco' } },
        { input: { location: 'London' } },
      ],
    }),
  },
  prompt: 'What is the weather in Tokyo?',
});
```

## 工作原理

1. 对每个定义了 `inputExamples` 的函数工具，中间件会：
   - 使用 `format`（默认 JSON.stringify）格式化每条示例。
   - 构建类似以下区块：

     ```
     Input Examples:
     {"location":"San Francisco"}
     {"location":"London"}
     ```

   - 将其追加到工具 `description` 末尾。

2. 默认在追加后移除 `inputExamples` 以防重复（可用 `remove: false` 关闭）。
3. 无输入示例的工具或非函数工具保持不变。

> **提示：** 该中间件对 OpenAI、Anthropic 等尚不支持原生 `inputExamples` 的提供商尤其有用。

## 示例效果

若原始工具定义为：

```ts
{
  type: 'function',
  name: 'weather',
  description: 'Get the weather in a location',
  inputSchema: { ... },
  inputExamples: [
    { input: { location: 'San Francisco' } },
    { input: { location: 'London' } }
  ]
}
```

应用中间件（默认设置）后，传给模型的工具形如：

```ts
{
  type: 'function',
  name: 'weather',
  description: `Get the weather in a location

Input Examples:
{"location":"San Francisco"}
{"location":"London"}`,
  inputSchema: { ... }
  // 默认会移除 inputExamples
}
```
