# `ToolLoopAgent`

创建一个可复用的 AI agent，能够生成文本、流式返回响应，并在多个步骤中使用工具（推理与行动循环）。`ToolLoopAgent` 非常适合构建自主的多步骤 agent，它可以执行操作、调用工具，并基于结果进行推理，直到达到停止条件。

不同于 `generateText()` 这类单步调用，agent 可以迭代地调用工具、收集工具结果，并决定下一步操作，直到完成或需要用户批准。

```ts
import { ToolLoopAgent } from 'ai';
__PROVIDER_IMPORT__;

const agent = new ToolLoopAgent({
  model: __MODEL__,
  instructions: 'You are a helpful assistant.',
  tools: {
    weather: weatherTool,
    calculator: calculatorTool,
  },
});

const result = await agent.generate({
  prompt: 'What is the weather in NYC?',
});

console.log(result.text);
```

对于 agent，`runtimeContext` 是贯穿整个循环流转的共享运行时状态。有关 `runtimeContext`、`toolsContext`、工具 `context` 和敏感上下文过滤的指南，请参阅 [Runtime and Tool Context](/docs/ai-sdk-core/runtime-and-tool-context)。
当工具需要访问命令或代码执行环境时，请将 `experimental_sandbox` 传递给 `generate()` 或 `stream()`。

要查看 `ToolLoopAgent` 的实际用法，请查看[这些示例](#examples)。

## 类型

### `ActiveTools`

```ts
type ActiveTools<TOOLS extends ToolSet> =
  | ReadonlyArray<keyof TOOLS & string>
  | undefined;
```

将 agent 步骤限制为列出的工具名称。`undefined` 表示不应用工具限制。

### `InferAgentUIMessage`

推断给定 agent 实例的 UI message 类型。适用于类型安全的 UI 和消息交换。

#### 基本示例

```ts
import { ToolLoopAgent, InferAgentUIMessage } from 'ai';

const weatherAgent = new ToolLoopAgent({
  model: __MODEL__,
  tools: { weather: weatherTool },
});

type WeatherAgentUIMessage = InferAgentUIMessage<typeof weatherAgent>;
```

#### 带消息元数据的示例

你可以提供第二个类型参数，为每条消息自定义元数据。这适用于跟踪 agent 返回的丰富元数据（例如 createdAt、tokens、finish reason 等）。

```ts
import { ToolLoopAgent, InferAgentUIMessage } from 'ai';
import { z } from 'zod';

// Example schema for message metadata
const exampleMetadataSchema = z.object({
  createdAt: z.number().optional(),
  model: z.string().optional(),
  totalTokens: z.number().optional(),
  finishReason: z.string().optional(),
});
type ExampleMetadata = z.infer<typeof exampleMetadataSchema>;

// Define agent as usual
const metadataAgent = new ToolLoopAgent({
  model: __MODEL__,
  // ...other options
});

// Type-safe UI message type with custom metadata
type MetadataAgentUIMessage = InferAgentUIMessage<
  typeof metadataAgent,
  ExampleMetadata
>;
```

## 示例

### 带工具的基本 Agent

```ts
import { ToolLoopAgent, isStepCount } from 'ai';
import { weatherTool, calculatorTool } from './tools';

const assistant = new ToolLoopAgent({
  model: __MODEL__,
  instructions: 'You are a helpful assistant.',
  tools: {
    weather: weatherTool,
    calculator: calculatorTool,
  },
  stopWhen: isStepCount(3),
});

const result = await assistant.generate({
  prompt: 'What is the weather in NYC and what is 100 * 25?',
});

console.log(result.text);
console.log(result.steps); // Array of all steps taken by the agent
```

### 流式 Agent 响应

const agent = new ToolLoopAgent({

const stream = agent.stream({

### 带输出解析的 Agent

```ts
import { z } from 'zod';

const analysisAgent = new ToolLoopAgent({
  model: __MODEL__,
  output: {
    schema: z.object({
      sentiment: z.enum(['positive', 'negative', 'neutral']),
      score: z.number(),
      summary: z.string(),

  },

```

### 示例：已批准的工具执行

```ts
import { ToolLoopAgent, ModelMessage, ToolApprovalResponse, tool } from 'ai';
import { z } from 'zod';

const agent = new ToolLoopAgent({
  model: __MODEL__,
  instructions: 'You are an agent with access to a weather API.',
  tools: {
    weather: tool({
      description: 'Get the weather in a location',
      inputSchema: z.object({
        location: z.string(),
      }),
      execute: async ({ location }) => ({
        location,
        temperature: 72,
      }),
    }),
  },
  toolApproval: {
    weather: 'user-approval',
  },
});

const messages: ModelMessage[] = [
  { role: 'user', content: 'Is it raining in Paris today?' },
];

const result = await agent.generate({ messages });
const approvals: ToolApprovalResponse[] = [];

for (const part of result.content) {
  if (part.type === 'tool-approval-request') {
    approvals.push({
      type: 'tool-approval-response',
      approvalId: part.approvalId,
      approved: true,
    });
  }
}

messages.push(...result.responseMessages);
messages.push({ role: 'tool', content: approvals });

const approvedResult = await agent.generate({ messages });
console.log(approvedResult.text);
```
