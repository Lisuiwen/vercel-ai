# AI SDK Core

[AI SDK Core](/docs/ai-sdk-core) 提供一组与语言模型及其他 AI 模型交互的函数。
这些函数设计为易用且灵活，可用于从语言模型及其他模型生成文本、结构化数据与嵌入向量。
