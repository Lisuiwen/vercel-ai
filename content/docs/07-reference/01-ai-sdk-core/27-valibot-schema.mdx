# `valibotSchema()`

`valibotSchema` 将 Valibot schema 转换为与 AI SDK 兼容的 JSON schema 对象。
输入为 Valibot schema，返回带类型的 schema。

可用于[生成结构化数据](/docs/ai-sdk-core/generating-structured-data)与[工具](/docs/ai-sdk-core/tools-and-tool-calling)。

## 示例

```ts
import { valibotSchema } from '@ai-sdk/valibot';
import { object, string, array } from 'valibot';

const recipeSchema = valibotSchema(
  object({
    name: string(),
    ingredients: array(
      object({
        name: string(),
        amount: string(),
      }),
    ),
    steps: array(string()),
  }),
);
```
