# `generateText()`

使用语言模型根据给定提示生成文本并调用工具。

它非常适合非交互式用例，例如需要撰写文本的自动化任务（如起草电子邮件或总结网页），以及使用工具的 agent。

```ts
import { generateText } from 'ai';
__PROVIDER_IMPORT__;

const { text } = await generateText({
  model: __MODEL__,
  prompt: 'Invent a new holiday and describe its traditions.',
});

console.log(text);
```

有关 `runtimeContext`、`toolsContext`、工具 `context` 以及敏感上下文过滤的指南，请参阅[运行时和工具上下文](/docs/ai-sdk-core/runtime-and-tool-context)。

要查看 `generateText` 的实际用法，请参阅[这些示例](#examples)。

## 类型

### `ActiveTools`

```ts
type ActiveTools<TOOLS extends ToolSet> =
  | ReadonlyArray<keyof TOOLS & string>
  | undefined;
```

将生成步骤限制为列出的工具名称。`undefined` 表示不应用工具限制。
