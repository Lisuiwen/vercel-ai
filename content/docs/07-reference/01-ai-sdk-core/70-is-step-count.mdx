# `isStepCount()`

创建在已完成步数等于指定计数时触发的停止条件。

该函数与 `generateText`、`streamText` 中的 `stopWhen` 配合使用，按已执行步数控制工具调用循环何时停止。

```ts
import { generateText, isStepCount } from 'ai';
__PROVIDER_IMPORT__;

const result = await generateText({
  model: __MODEL__,
  tools: {
    // your tools
  },
  // 5 步后停止
  stopWhen: isStepCount(5),
});
```

### 返回值

`StopCondition` 函数：当已完成步数等于指定值时返回 `true`。可与 `generateText`、`streamText` 的 `stopWhen` 参数一起使用。

## 示例

### 基本用法

3 步后停止：

```ts
import { generateText, isStepCount } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: yourTools,
  stopWhen: isStepCount(3),
});
```

### 与其他条件组合

可将多个停止条件放入数组：

```ts
import { generateText, isStepCount, hasToolCall } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: yourTools,
  // 10 步后停止，或调用 finalAnswer 工具时停止
  stopWhen: [isStepCount(10), hasToolCall('finalAnswer')],
});
```
