# `hasToolCall()`

创建在最近一步中调用了任一指定工具时触发的停止条件。

该函数与 `generateText`、`streamText` 中的 `stopWhen` 配合使用，按是否调用指定工具控制工具调用循环何时停止。

```ts
import { generateText, hasToolCall } from 'ai';
__PROVIDER_IMPORT__;

const result = await generateText({
  model: __MODEL__,
  tools: {
    weather: weatherTool,
    finalAnswer: finalAnswerTool,
  },
  // 调用 finalAnswer 工具时停止
  stopWhen: hasToolCall('finalAnswer'),
});
```

## API 签名

### 参数

<PropertiesTable
  content={[
    {
      name: '...toolNames',
      type: 'string[]',
      description:
        '一个或多个工具名。最近一步中调用其中任一时触发停止条件。',
    },
  ]}
/>

### 返回值

`StopCondition` 函数：当最近一步中调用了任一指定工具时返回 `true`。可与 `generateText`、`streamText` 的 `stopWhen` 参数一起使用。

## 示例

### 基本用法

在调用特定工具时停止：

```ts
import { generateText, hasToolCall } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: {
    submitAnswer: submitAnswerTool,
    search: searchTool,
  },
  stopWhen: hasToolCall('submitAnswer'),
});
```

### 匹配多个工具

最近一步中调用以下任一时停止：

```ts
import { generateText, hasToolCall } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: {
    weather: weatherTool,
    search: searchTool,
    finalAnswer: finalAnswerTool,
  },
  stopWhen: hasToolCall('weather', 'finalAnswer'),
});
```

### 与其他条件组合

可将多个停止条件放入数组：

```ts
import { generateText, hasToolCall, isStepCount } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: {
    weather: weatherTool,
    search: searchTool,
    finalAnswer: finalAnswerTool,
  },
  // 调用 weather、调用 finalAnswer 或满 5 步时停止
  stopWhen: [
    hasToolCall('weather'),
    hasToolCall('finalAnswer'),
    isStepCount(5),
  ],
});
```

### 智能体模式

常见模式：运行直至给出最终答案：

```ts
import { generateText, hasToolCall } from 'ai';

const result = await generateText({
  model: yourModel,
  tools: {
    search: searchTool,
    calculate: calculateTool,
    finalAnswer: {
      description: 'Provide the final answer to the user',
      parameters: z.object({
        answer: z.string(),
      }),
      execute: async ({ answer }) => answer,
    },
  },
  stopWhen: hasToolCall('finalAnswer'),
});
```
