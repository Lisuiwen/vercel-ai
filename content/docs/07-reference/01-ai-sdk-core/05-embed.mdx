# `embed()`

使用嵌入模型为单个值生成嵌入向量。

这非常适合需要嵌入单个值的场景，例如检索相似项，或在下游任务中使用该嵌入向量。

```ts
import { embed } from 'ai';

const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
});
```
