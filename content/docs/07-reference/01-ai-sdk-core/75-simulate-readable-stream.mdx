# `simulateReadableStream()`

`simulateReadableStream` 用于创建 `ReadableStream`，按顺序、可配置延迟地发出给定值。适用于测试流式行为或模拟带时间间隔的数据流。

```ts
import { simulateReadableStream } from 'ai';

const stream = simulateReadableStream({
  chunks: ['Hello', ' ', 'World'],
  initialDelayInMs: 100,
  chunkDelayInMs: 50,
});
```

### 返回值

返回 `ReadableStream<T>`，其行为如下：

- 按顺序发出 `chunks` 数组中的每个值
- 在发出第一个值前等待 `initialDelayInMs`（若非 `null`）
- 在后续值之间等待 `chunkDelayInMs`（若非 `null`）
- 所有分块发出后自动关闭

### 类型参数

- `T`：分块数组所含类型，亦为流发出值的类型

## 示例

### 基本用法

```ts
const stream = simulateReadableStream({
  chunks: ['Hello', ' ', 'World'],
});
```

### 带延迟

```ts
const stream = simulateReadableStream({
  chunks: ['Hello', ' ', 'World'],
  initialDelayInMs: 1000, // 首块前等待 1 秒
  chunkDelayInMs: 500, // 分块之间等待 0.5 秒
});
```

### 无延迟

```ts
const stream = simulateReadableStream({
  chunks: ['Hello', ' ', 'World'],
  initialDelayInMs: null, // 无初始延迟
  chunkDelayInMs: null, // 分块间无延迟
});
```
