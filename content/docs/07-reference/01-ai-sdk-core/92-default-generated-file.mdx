# `DefaultGeneratedFile`

`GeneratedFile` 接口的具体实现，在 base64 与 `Uint8Array` 格式之间提供惰性转换。

```ts
import { DefaultGeneratedFile } from 'ai';

const file = new DefaultGeneratedFile({
  data: uint8ArrayData,
  mediaType: 'image/png',
});

console.log(file.base64); // 自动转换为 base64
console.log(file.uint8Array); // 原始 Uint8Array
```
