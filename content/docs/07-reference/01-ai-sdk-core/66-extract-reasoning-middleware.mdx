# `extractReasoningMiddleware()`

`extractReasoningMiddleware` 从生成文本中提取带 XML 标签的推理段落，并将其与主文本内容分开。适用于需要将模型推理过程与最终输出分离的场景。

```ts
import { extractReasoningMiddleware } from 'ai';

const middleware = extractReasoningMiddleware({
  tagName: 'reasoning',
  separator: '\n',
});
```

### 返回值

返回中间件对象，其行为如下：

- 同时处理流式与非流式响应
- 将指定 XML 标签之间的内容作为推理提取
- 从主文本中移除 XML 标签与推理内容
- 在结果上添加包含所提取内容的 `reasoning` 属性
- 使用指定分隔符在文本段之间保持正确分隔

### 类型参数

该中间件对流式响应使用 `LanguageModelV4StreamPart` 类型。
