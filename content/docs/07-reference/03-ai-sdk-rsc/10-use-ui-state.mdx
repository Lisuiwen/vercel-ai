# `useUIState`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

这是一个 hook，可让你读取并更新 UI State。该状态位于客户端，可以包含函数、React 节点和其他数据。UIState 是 AI 状态的视觉表示。

## 导入

<Snippet text={`import { useUIState } from "@ai-sdk/rsc"`} prompt={false} />

## API 签名

### 返回值

与 useState 类似，它是一个数组，第一个元素是当前 UI 状态，第二个元素是更新该状态的函数。
