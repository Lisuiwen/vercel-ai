# `useActions`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

这是一个 hook，可帮助你从客户端访问 Server Actions。它对于构建需要用户与服务器交互的界面尤其有用。

必须通过此 hook 访问这些 server actions，因为它们在通过 context 传递时会被修补。直接访问它们可能会导致 [Cannot find Client Component 错误](/docs/troubleshooting/server-actions-in-client-components)。

## 导入

<Snippet text={`import { useActions } from "@ai-sdk/rsc"`} prompt={false} />

## API 签名

### 返回值

`Record<string, Action>`，server actions 的字典。
