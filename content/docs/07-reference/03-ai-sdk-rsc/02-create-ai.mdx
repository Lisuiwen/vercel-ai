# `createAI`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

创建一个客户端-服务器 context provider，可用于包裹应用树的一部分，从而轻松管理应用的 UI 状态和 AI 状态。

## 导入

<Snippet text={`import { createAI } from "@ai-sdk/rsc"`} prompt={false} />

### 返回值

它会返回一个 `<AI/>` context provider。
