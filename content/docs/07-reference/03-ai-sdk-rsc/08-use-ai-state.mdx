# `useAIState`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

这是一个 hook，可让你读取并更新 AI 状态。在同一个 `<AI/>` provider 下，AI 状态会在所有 `useAIState` hook 之间全局共享。

AI 状态用于包含与 AI 模型共享的上下文和信息，例如系统消息、函数响应以及其他相关数据。

## 导入

<Snippet text={`import { useAIState } from "@ai-sdk/rsc"`} prompt={false} />

## API 签名

### 返回值

与 useState 类似，它是一个数组，第一个元素是当前 AI 状态，第二个元素是用于更新该状态的函数。
