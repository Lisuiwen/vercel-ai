# `render`（已移除）

<Note type="warning">"render" 已在 AI SDK 4.0 中移除。</Note>

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

一个辅助函数，用于从 LLM providers 创建 streamable UI。此函数类似于 AI SDK Core API，并支持相同的模型接口。

> **注意**：`render` 已弃用，请改用 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)。迁移期间，请确保 `messages` 参数遵循更新后的[规范](/docs/reference/ai-sdk-rsc/stream-ui#messages)。

## 导入（不再可用）

由于 `render` 已被移除，以下导入将不再生效：

<Snippet text={`import { render } from "@ai-sdk/rsc"`} prompt={false} />

请改用 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)。
