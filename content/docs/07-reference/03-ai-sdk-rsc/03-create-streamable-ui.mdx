# `createStreamableUI`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

创建一个从服务器向客户端发送 UI 的流。在客户端，它可以像普通 React 节点一样渲染。

## 导入

<Snippet
  text={`import { createStreamableUI } from "@ai-sdk/rsc"`}
  prompt={false}
/>
