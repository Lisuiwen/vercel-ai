# `useStreamableValue`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

这是一个 React hook，接收使用 [`createStreamableValue`](/docs/reference/ai-sdk-rsc/create-streamable-value) 创建的 streamable value，并返回当前值、错误和 pending 状态。

## 导入

<Snippet
  text={`import { useStreamableValue } from "@ai-sdk/rsc"`}
  prompt={false}
/>

## 示例

这对于消费从组件 props 接收到的 streamable values 很有用。

```tsx
function MyComponent({ streamableValue }) {
  const [data, error, pending] = useStreamableValue(streamableValue);

  if (pending) return <div>Loading...</div>;
  if (error) return <div>Error: {error.message}</div>;

  return <div>Data: {data}</div>;
}
```

## API 签名

### 参数

它接收使用 `createStreamableValue` 创建的 streamable value。

### 返回值

它是一个数组：第一个元素包含数据，第二个元素包含流期间任意时刻抛出的错误（如果有），第三个元素是一个布尔值，表示该值是否处于 pending 状态。
