# `streamUI`

<Note type="warning">
  AI SDK RSC 目前处于实验阶段。我们建议在生产环境中使用 [AI SDK
  UI](/docs/ai-sdk-ui/overview)。关于从 RSC 迁移到 UI 的指导，请参阅我们的[迁移指南](/docs/ai-sdk-rsc/migrating-to-ui)。
</Note>

一个辅助函数，用于从 LLM providers 创建 streamable UI。此函数类似于 AI SDK Core API，并支持相同的模型接口。

要查看 `streamUI` 的实际用法，请参阅[这些示例](#examples)。

## 导入

<Snippet text={`import { streamUI } from "@ai-sdk/rsc"`} prompt={false} />
