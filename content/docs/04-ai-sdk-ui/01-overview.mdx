# AI SDK UI

AI SDK UI 旨在帮助你轻松构建交互式聊天、补全和助手应用。它是一个**与框架无关的工具包**，可简化将高级 AI 功能集成到应用中的流程。

AI SDK UI 提供强大的抽象，简化前端管理聊天流和 UI 更新的复杂任务，使你能够更高效地开发由 AI 驱动的动态界面。通过三个主要 hook — **`useChat`**、**`useCompletion`** 和 **`useObject`** — 你可以将实时聊天能力、文本补全、流式 JSON 和交互式助手功能融入应用。

- **[`useChat`](/docs/ai-sdk-ui/chatbot)** 提供聊天消息的实时流式传输，抽象了输入、消息、加载和错误等状态管理，可无缝集成到任何 UI 设计中。
- **[`useCompletion`](/docs/ai-sdk-ui/completion)** 使你可以在应用中处理文本补全，管理提示输入，并在新补全流式返回时自动更新 UI。
- **[`useObject`](/docs/ai-sdk-ui/object-generation)** 是一个允许你消费流式 JSON 对象的 hook，为在应用中处理和显示结构化数据提供简单方式。

这些 hook 旨在降低实现 AI 交互的复杂度和所需时间，让你专注于创造出色的用户体验。

## UI 框架支持

AI SDK UI 支持以下框架：[React](https://react.dev/)、[Svelte](https://svelte.dev/)、[Vue.js](https://vuejs.org/)、
[Angular](https://angular.dev/) 和 [SolidJS](https://www.solidjs.com/)。
