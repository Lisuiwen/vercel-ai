# 消息元数据

消息元数据允许你在消息级别附加自定义信息。这对于跟踪时间戳、模型信息、token 用量、用户上下文及其他消息级数据很有用。

## 概述

消息元数据与[数据部分](/docs/ai-sdk-ui/streaming-data)不同：它附加在消息级别，而不是消息内容的一部分。数据部分适合作为消息内容的动态内容，而元数据适合关于消息本身的信息。

## 入门

以下示例展示如何使用消息元数据跟踪时间戳和模型信息：

### 定义元数据类型

首先定义元数据类型以确保类型安全：

```tsx filename="app/types.ts"
import { UIMessage } from 'ai';
import { z } from 'zod';

// Define your metadata schema
export const messageMetadataSchema = z.object({
  createdAt: z.number().optional(),
  model: z.string().optional(),
  totalTokens: z.number().optional(),
});

export type MessageMetadata = z.infer<typeof messageMetadataSchema>;

// Create a typed UIMessage
export type MyUIMessage = UIMessage<MessageMetadata>;
```

### 从服务端发送元数据

在 `toUIMessageStreamResponse` 中使用 `messageMetadata` 回调，在不同流式阶段发送元数据：

```ts filename="app/api/chat/route.ts" highlight="11-20"
import { convertToModelMessages, streamText } from 'ai';
__PROVIDER_IMPORT__;
import type { MyUIMessage } from '@/types';

export async function POST(req: Request) {
  const { messages }: { messages: MyUIMessage[] } = await req.json();

  const result = streamText({
    model: __MODEL__,
    messages: await convertToModelMessages(messages),
  });

  return result.toUIMessageStreamResponse({
    originalMessages: messages, // pass this in for type-safe return objects
    messageMetadata: ({ part }) => {
      // Send metadata when streaming starts
      if (part.type === 'start') {
        return {
          createdAt: Date.now(),
          model: 'your-model-id',
        };
      }

      // Send additional metadata when streaming completes
      if (part.type === 'finish') {
        return {
          totalTokens: part.totalUsage.totalTokens,
        };
      }
    },
  });
}
```

<Note>
  若要在 `messageMetadata` 中启用类型安全的元数据返回对象，请传入以你的 UIMessage 类型标注的 `originalMessages` 参数。
</Note>

### 在客户端访问元数据

通过 `message.metadata` 属性访问元数据：

```tsx filename="app/page.tsx" highlight="8,18-23"
'use client';

import { useChat } from '@ai-sdk/react';
import { DefaultChatTransport } from 'ai';
import type { MyUIMessage } from '@/types';

export default function Chat() {
  const { messages } = useChat<MyUIMessage>({
    transport: new DefaultChatTransport({
      api: '/api/chat',
    }),
  });

  return (
    <div>
      {messages.map(message => (
        <div key={message.id}>
          <div>
            {message.role === 'user' ? 'User: ' : 'AI: '}
            {message.metadata?.createdAt && (
              <span className="text-sm text-gray-500">
                {new Date(message.metadata.createdAt).toLocaleTimeString()}
              </span>
            )}
          </div>

          {/* Render message content */}
          {message.parts.map((part, index) =>
            part.type === 'text' ? <div key={index}>{part.text}</div> : null,
          )}

          {/* Display additional metadata */}
          {message.metadata?.totalTokens && (
            <div className="text-xs text-gray-400">
              {message.metadata.totalTokens} tokens
            </div>
          )}
        </div>
      ))}
    </div>
  );
}
```

<Note>
  对于生成过程中变化的任意流式数据，请考虑改用[数据部分](/docs/ai-sdk-ui/streaming-data)。
</Note>

## 常见用例

消息元数据适用于：

- **时间戳**：消息创建或完成的时间
- **模型信息**：使用的 AI 模型
- **Token 用量**：跟踪成本和用量限制
- **用户上下文**：用户 ID、会话信息
- **性能指标**：生成时间、首 token 时间
- **质量指标**：完成原因、置信度分数
