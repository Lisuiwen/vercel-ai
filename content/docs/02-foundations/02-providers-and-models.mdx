# 提供商与模型

OpenAI、Anthropic 等公司（提供商）通过各自的 API 提供多种具有不同优势与能力的大语言模型（LLM）访问。

各提供商通常有各自独特的模型接入方式，这使切换提供商的过程变得复杂，并增加了供应商锁定的风险。

为解决这些挑战，AI SDK Core 通过[语言模型规范](https://github.com/vercel/ai/tree/main/packages/provider/src/language-model/v3)提供与 LLM 交互的标准化方式，抽象了提供商之间的差异。这一统一接口让你可以在所有提供商上使用相同 API 的同时轻松切换提供商。

## 自托管模型

可通过以下提供商访问自托管模型：

- [Ollama Provider](/providers/community-providers/ollama)
- [LM Studio](/providers/openai-compatible-providers/lmstudio)
- [Baseten](/providers/ai-sdk-providers/baseten)
- [Browser AI](/providers/community-providers/browser-ai)

此外，任何支持 OpenAI 规范的自托管提供商均可与 [OpenAI Compatible Provider](/providers/openai-compatible-providers) 一起使用。
