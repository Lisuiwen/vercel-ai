要有效使用 AI SDK，建议先熟悉以下概念：

## 生成式人工智能

**生成式人工智能**指根据统计上最可能的结果，预测并生成各类输出（如文本、图像或音频）的模型，其依据来自训练数据中学到的模式。例如：

- 给定一张照片，生成式模型可生成标题。
- 给定音频文件，生成式模型可生成转录文本。
- 给定文本描述，生成式模型可生成图像。

## 大语言模型

**大语言模型（LLM）**是生成式模型的一个子集，主要面向**文本**。LLM 接收一系列词作为输入，并尝试预测最可能出现的后续序列。它为潜在的下一段序列分配概率，然后选择其一。模型会持续生成序列，直到满足指定的停止条件。

LLM 通过在海量文本上训练来学习，这意味着它们对某些用例会比其他用例更合适。例如，在 GitHub 数据上训练的模型会特别擅长理解源代码中的序列概率。

然而，理解 LLM 的局限性至关重要。当被问及较少见或训练数据中不存在的信息（例如某位亲属的生日）时，LLM 可能会「幻觉」或编造信息。你需要考虑所需信息在模型中的覆盖程度。

## 嵌入模型

**嵌入模型**用于将复杂数据（如词语或图像）转换为稠密向量（数字列表）表示，即嵌入。与生成式模型不同，嵌入模型不会生成新的文本或数据，而是提供实体之间语义与句法关系的表示，可作为其他模型或其他自然语言处理任务的输入。
