# AI SDK

AI SDK 是专为帮助开发者在 React、Next.js、Vue、Svelte、Node.js 等环境中构建 AI 驱动应用与 Agent 而设计的 TypeScript 工具包。

## 为何使用 AI SDK？

将大语言模型（LLM）集成到应用中既复杂，又高度依赖你所使用的具体模型提供商。

AI SDK 在[各受支持提供商](/docs/foundations/providers-and-models)之间标准化了人工智能（AI）模型的集成方式，让开发者可以专注于构建出色的 AI 应用，而不必在琐碎的技术细节上耗费时间。

例如，使用 AI SDK 可以通过多种模型生成文本：

<PreviewSwitchProviders />

AI SDK 包含两个主要库：

- **[AI SDK Core](/docs/ai-sdk-core)：** 用于生成文本、结构化对象、工具调用以及使用 LLM 构建 Agent 的统一 API。
- **[AI SDK UI](/docs/ai-sdk-ui)：** 一组与框架无关的 Hook，可快速构建聊天与生成式用户界面。

## `llms.txt`（适用于 Cursor、Windsurf、Copilot、Claude 等）

你可以在 [ai-sdk.dev/llms.txt](/llms.txt) 以 Markdown 格式访问完整的 AI SDK 文档。你可以将其用于向任意 LLM（前提是上下文窗口足够大）基于最新文档提问 AI SDK 相关问题。

### 使用示例

例如，向 LLM 提问 AI SDK 相关问题：

1. 从 [ai-sdk.dev/llms.txt](/llms.txt) 复制文档内容
2. 使用以下提示格式：

```prompt
Documentation:
{paste documentation here}
---
Based on the above documentation, answer the following:
{your question}
```
