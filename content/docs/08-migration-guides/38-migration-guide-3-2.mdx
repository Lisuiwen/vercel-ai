# 将 AI SDK 3.1 迁移到 3.2

<Note>
  更多信息请参阅 [AI SDK 3.2 发布公告](https://vercel.com/blog/introducing-vercel-ai-sdk-3-2)。
</Note>

本指南帮助你升级到 AI SDK 3.2：

- 实验性的 `StreamingReactResponse` 功能已移除
- 若干能力已被标记为弃用
- UI 框架集成迁移到各自的 Node 模块

## 升级

### AI SDK

要使用你喜欢的包管理器升级到 AI SDK 3.2，请运行：

<Snippet text="pnpm add ai@latest" />

## 已移除功能

实验性的 `StreamingReactResponse` 已移除。你可以使用 [AI SDK RSC](/docs/ai-sdk-rsc/overview) 构建流式 UI。

## 已弃用功能

`nanoid` 导出已弃用。请改用 [`generateId`](/docs/reference/ai-sdk-core/generate-id)。

## UI 包拆分

AI SDK UI 支持多种框架：[React](https://react.dev/)、[Svelte](https://svelte.dev/)、[Vue.js](https://vuejs.org/) 与 [SolidJS](https://www.solidjs.com/)。

除 React 与 RSC 之外的集成已迁移到独立的 Node 模块。你需要按如下方式更新 import 与 require：

- `ai/svelte` → `@ai-sdk/svelte`
- `ai/vue` → `@ai-sdk/vue`
- `ai/solid` → `@ai-sdk/solid`

旧导出仍然可用，但将在未来版本中移除。
