# 将 AI SDK 3.0 迁移到 3.1

<Note>
  更多信息请参阅 [AI SDK 3.1 发布公告](https://vercel.com/blog/vercel-ai-sdk-3-1-modelfusion-joins-the-team)。
</Note>

本指南帮助你：

- 升级到 AI SDK 3.1
- 从旧版 Provider 迁移到 AI SDK Core
- 从 [`render`](/docs/reference/ai-sdk-rsc/render) 迁移到 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)

升级到 AI SDK 3.1 **不要求**你立刻改用新发布的 AI SDK Core API 或 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)。

## 升级

### AI SDK

要使用你喜欢的包管理器升级到 AI SDK 3.1，请运行：

<Snippet text="pnpm add ai@3.1" />

## 下一步

AI SDK 3.1 引入了多项新功能，帮助你更高效地使用 SDK 构建 AI 应用：

- **AI SDK Core**：全新的统一 API，用于与大语言模型（LLM）交互。
- [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)：基于 AI SDK Core 构建的新抽象，简化流式 UI 的开发。

## 从旧版 Provider 迁移到 AI SDK Core

在 AI SDK Core 之前，你必须使用各模型提供商的 SDK 来调用他们的模型。

在下面的 Route Handler 中，你使用 OpenAI SDK 调用模型，然后将响应通过 `OpenAIStream` 函数转换为 [`ReadableStream`](https://developer.mozilla.org/en-US/docs/Web/API/ReadableStream)，再用 `StreamingTextResponse` 返回给客户端。

```tsx
import OpenAI from 'openai';
import { OpenAIStream, StreamingTextResponse } from 'ai';

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY!,
});

export async function POST(req: Request) {
  const { messages } = await req.json();

  const response = await openai.chat.completions.create({
    model: 'gpt-4.1',
    stream: true,
    messages,
  });

  const stream = OpenAIStream(response);

  return new StreamingTextResponse(stream);
}
```

借助 AI SDK Core，你可以使用统一 API 调用任意实现了 [AI SDK Language Model Specification](/providers/community-providers/custom-providers) 的提供商。

下面是将上文示例重构为 AI SDK Core 与 AI SDK OpenAI provider 的版本：你从 `ai` 包导入要使用的 LLM 函数，从 `@ai-sdk/openai` 导入 OpenAI provider，然后调用模型并通过 `toDataStreamResponse()` 辅助函数返回响应。

```tsx
import { streamText } from 'ai';
import { openai } from '@ai-sdk/openai';
__PROVIDER_IMPORT__;

export async function POST(req: Request) {
  const { messages } = await req.json();

  const result = await streamText({
    model: __MODEL__,
    messages,
  });

  return result.toUIMessageStreamResponse();
}
```

## 从 `render` 迁移到 `streamUI`

AI SDK RSC API 随 3.0 发布，其中包含 [`render`](/docs/reference/ai-sdk-rsc/render) 函数，用于借助 OpenAI 模型创建可流式更新的 UI。借助新的 AI SDK Core API，可以在任意兼容的提供商上实现流式 UI。

下面的 Server Action 示例直接使用 OpenAI 的模型提供商调用 `render`。你先用 OpenAI SDK 创建 OpenAI provider 实例，再把它传给 `render` 的 provider 字段，并声明一个在工具的 `render` 字段中返回 React Server Component 的工具。

```tsx
import { render } from '@ai-sdk/rsc';
import OpenAI from 'openai';
import { z } from 'zod';
import { Spinner, Weather } from '@/components';
import { getWeather } from '@/utils';

const openai = new OpenAI();

async function submitMessage(userInput = 'What is the weather in SF?') {
  'use server';

  return render({
    provider: openai,
    model: 'gpt-4.1',
    messages: [
      { role: 'system', content: 'You are a helpful assistant' },
      { role: 'user', content: userInput },
    ],
    text: ({ content }) => <p>{content}</p>,
    tools: {
      get_city_weather: {
        description: 'Get the current weather for a city',
        parameters: z
          .object({
            city: z.string().describe('the city'),
          })
          .required(),
        render: async function* ({ city }) {
          yield <Spinner />;
          const weather = await getWeather(city);
          return <Weather info={weather} />;
        },
      },
    },
  });
}
```

借助新的 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui)，你可以使用任意兼容的 AI SDK provider。此例导入 AI SDK OpenAI provider，将其传给 [`streamUI`](/docs/reference/ai-sdk-rsc/stream-ui) 的 [`model`](/docs/reference/ai-sdk-rsc/stream-ui#model) 字段，最后在工具的 [`generate`](/docs/reference/ai-sdk-rsc/stream-ui#tools-generate) 字段中返回 React Server Component。

```tsx
import { streamUI } from '@ai-sdk/rsc';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';
import { Spinner, Weather } from '@/components';
import { getWeather } from '@/utils';

async function submitMessage(userInput = 'What is the weather in SF?') {
  'use server';

  const result = await streamUI({
    model: __MODEL__,
    system: 'You are a helpful assistant',
    messages: [{ role: 'user', content: userInput }],
    text: ({ content }) => <p>{content}</p>,
    tools: {
      get_city_weather: {
        description: 'Get the current weather for a city',
        parameters: z
          .object({
            city: z.string().describe('Name of the city'),
          })
          .required(),
        generate: async function* ({ city }) {
          yield <Spinner />;
          const weather = await getWeather(city);
          return <Weather info={weather} />;
        },
      },
    },
  });

  return result.value;
}
```
