# DevTools

<Note type="warning">
  AI SDK DevTools 为实验性功能，仅用于本地开发。
  请勿在生产环境使用。
</Note>

AI SDK DevTools 为通过 [`generateText`](/docs/reference/ai-sdk-core/generate-text)、[`streamText`](/docs/reference/ai-sdk-core/stream-text) 与 [`ToolLoopAgent`](/docs/reference/ai-sdk-core/tool-loop-agent) 发起的 AI SDK 调用提供完整可见性，帮助你通过 Web UI 调试并检查 LLM 请求、响应、tool 调用与多步交互。

DevTools 由两部分组成：

1. **遥测集成**：通过[遥测](/docs/ai-sdk-core/telemetry)系统捕获 AI SDK 调用中的 run 与 step
2. **查看器**：用于检查已捕获数据的 Web UI

## 安装

安装 DevTools 包：

```bash
pnpm add @ai-sdk/devtools
```

## 要求

- AI SDK v6 beta（`ai@^6.0.0-beta.0`）
- 兼容 Node.js 的运行时

## 使用 DevTools

### 注册集成

全局注册 `DevToolsTelemetry`，以捕获所有 AI SDK 调用：

```ts
import { registerTelemetry } from 'ai';
import { DevToolsTelemetry } from '@ai-sdk/devtools';

registerTelemetry(DevToolsTelemetry());
```

注册集成后，遥测会自动启用——无需为每次调用单独配置：

```ts
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What cities are in the United States?',
});
```

也可将集成传入单次调用，而非全局注册：

```ts highlight="7-9"
import { streamText } from 'ai';
import { DevToolsTelemetry } from '@ai-sdk/devtools';

const result = streamText({
  model: openai('gpt-4o'),
  prompt: 'Hello!',
  telemetry: {
    integrations: [DevToolsTelemetry()],
  },
});
```

### 启动查看器

启动 DevTools 查看器：

```bash
npx @ai-sdk/devtools
```

在浏览器中打开 [http://localhost:4983](http://localhost:4983) 查看 AI SDK 交互。

### Monorepo 用法

若使用 monorepo（如 Turborepo、Nx），请在运行 AI SDK 代码的同一 workspace 中启动 DevTools。

例如 API 位于 `apps/api` 时：

```bash
cd apps/api
npx @ai-sdk/devtools
```

## 捕获的数据

DevTools 会从 AI SDK 调用中捕获以下信息：

- **输入参数与提示**：查看发送给 LLM 的完整输入
- **输出内容与 tool 调用**：检查生成的文本与 tool 调用
- **Token 用量与耗时**：监控资源消耗与性能
- **原始 provider 数据**：访问完整请求与响应载荷

### Run 与 Step

DevTools 将捕获的数据组织为 run 与 step：

- **Run**：一次完整的多步 AI 交互，按初始提示分组
- **Step**：run 内的单次 LLM 调用（例如一次 `generateText` 或 `streamText`）

由 tool 调用或 agent 循环产生的多步交互会作为单个 run、包含多个 step 进行分组。嵌套的子 agent 调用会关联到父 run，便于追踪完整执行树。

## 工作原理

`DevToolsTelemetry` 集成接入 AI SDK [遥测](/docs/ai-sdk-core/telemetry)生命周期，捕获所有 `generateText`、`streamText`、`generateObject` 与 `streamObject` 调用。数据本地存储在 JSON 文件（`.devtools/generations.json`）中，并通过基于 Hono 与 React 的 Web UI 展示。

<Note type="warning">
  集成会自动将 `.devtools` 加入 `.gitignore`。
  请确认 `.gitignore` 中包含 `.devtools`，避免将敏感的 AI 交互数据提交到仓库。
</Note>

## 安全注意事项

DevTools 以明文本地文件存储所有 AI 交互，包括：

- 用户提示与消息
- LLM 响应
- Tool 调用参数与结果
- API 请求与响应数据

**仅在本地开发环境使用 DevTools。** 不要在生产环境启用，也不要在处理敏感数据时使用。
