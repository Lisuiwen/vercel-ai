# 嵌入

嵌入是将词、短语或图像表示为高维空间向量的方式。
在该空间中，相似词彼此接近，词之间的距离可用于衡量相似度。

## 嵌入单个值

AI SDK 提供 [`embed`](/docs/reference/ai-sdk-core/embed) 函数用于嵌入单个值，适用于查找相似词或短语、文本聚类等任务。
可与嵌入模型一起使用，例如 `openai.embeddingModel('text-embedding-3-large')` 或 `mistral.embeddingModel('mistral-embed')`。

```tsx
import { embed } from 'ai';

// 'embedding' is a single embedding object (number[])
const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
});
```

## 嵌入多个值

加载数据时（例如为检索增强生成（RAG）准备数据存储），
通常需要一次嵌入多个值（批量嵌入）。

AI SDK 为此提供 [`embedMany`](/docs/reference/ai-sdk-core/embed-many) 函数。
与 `embed` 类似，可与嵌入模型一起使用，
例如 `openai.embeddingModel('text-embedding-3-large')` 或 `mistral.embeddingModel('mistral-embed')`。

```tsx
import { embedMany } from 'ai';

// 'embeddings' is an array of embedding objects (number[][]).
// It is sorted in the same order as the input values.
const { embeddings } = await embedMany({
  model: 'openai/text-embedding-3-small',
  values: [
    'sunny day at the beach',
    'rainy afternoon in the city',
    'snowy night in the mountains',
  ],
});
```

## 嵌入相似度

嵌入值后，可使用 [`cosineSimilarity`](/docs/reference/ai-sdk-core/cosine-similarity) 函数计算相似度。
适用于在数据集中查找相似词或短语。
也可根据相似度对相关项排序和筛选。

```ts highlight={"1,9"}
import { cosineSimilarity, embedMany } from 'ai';

const { embeddings } = await embedMany({
  model: 'openai/text-embedding-3-small',
  values: ['sunny day at the beach', 'rainy afternoon in the city'],
});

console.log(
  `cosine similarity: ${cosineSimilarity(embeddings[0], embeddings[1])}`,
);
```

## Token 用量

许多提供商按生成嵌入所用的 token 数计费。
`embed` 和 `embedMany` 均在结果对象的 `usage` 属性中提供 token 用量信息：

```ts highlight={"3,8"}
import { embed } from 'ai';

const { embedding, usage } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
});

console.log(usage); // { tokens: 10 }
```

## 设置

### 提供商选项

可使用 `providerOptions` 配置嵌入模型的提供商特定参数：

```ts highlight={"4-8"}
import { embed } from 'ai';

const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
  providerOptions: {
    openai: {
      dimensions: 512, // Reduce embedding dimensions
    },
  },
});
```

Google 的 `gemini-embedding-2-preview` 模型也支持通过 `providerOptions.google.content` 进行多模态嵌入内容。每个条目对应相同索引处的值，可包含 `{ text: string }`、`{ inlineData: { mimeType: string; data: string } }` 或 `{ fileData: { fileUri: string; mimeType: string } }` 部分。`fileUri` 可引用远程内容，例如 HTTP URL 或 Google Cloud Storage URI（`gs://...`）。

### 并行请求

`embedMany` 现支持通过可配置的 `maxParallelCalls` 进行并行处理以优化性能：

```ts highlight={"3"}
import { embedMany } from 'ai';

const { embeddings, usage } = await embedMany({
  maxParallelCalls: 2, // Limit parallel requests
  model: 'openai/text-embedding-3-small',
  values: [
    'sunny day at the beach',
    'rainy afternoon in the city',
    'snowy night in the mountains',
  ],
});
```

### 重试

`embed` 和 `embedMany` 均接受可选的 `maxRetries` 参数（类型为 `number`），
用于设置嵌入过程的最大重试次数。
默认为 `2` 次重试（共 3 次尝试）。设为 `0` 可禁用重试。

```ts highlight={"6"}
import { embed } from 'ai';

const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
  maxRetries: 0, // Disable retries
});
```

### 中止信号与超时

`embed` 和 `embedMany` 均接受可选的 `abortSignal` 参数（类型为 [`AbortSignal`](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal)），
用于中止嵌入过程或设置超时。

```ts highlight={"6"}
import { embed } from 'ai';

const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
  abortSignal: AbortSignal.timeout(1000), // Abort after 1 second
});
```

### 自定义头

`embed` 和 `embedMany` 均接受可选的 `headers` 参数（类型为 `Record<string, string>`），
用于向嵌入请求添加自定义头。

```ts highlight={"6"}
import { embed } from 'ai';

const { embedding } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
  headers: { 'X-Custom-Header': 'custom-value' },
});
```

## 响应信息

`embed` 和 `embedMany` 均返回包含原始提供商响应的响应信息：

```ts highlight={"3,8"}
import { embed } from 'ai';

const { embedding, response } = await embed({
  model: 'openai/text-embedding-3-small',
  value: 'sunny day at the beach',
});

console.log(response); // Raw provider response
```

## 嵌入中间件

可使用 `wrapEmbeddingModel` 和 `EmbeddingModelMiddleware` 增强嵌入模型，例如设置默认值。

以下示例使用内置的 `defaultEmbeddingSettingsMiddleware`：

```ts
import {
  defaultEmbeddingSettingsMiddleware,
  embed,
  wrapEmbeddingModel,
  gateway,
} from 'ai';

const embeddingModelWithDefaults = wrapEmbeddingModel({
  model: gateway.embeddingModel('google/gemini-embedding-001'),
  middleware: defaultEmbeddingSettingsMiddleware({
    settings: {
      providerOptions: {
        google: {
          outputDimensionality: 256,
          taskType: 'CLASSIFICATION',
        },
      },
    },
  }),
});
```
