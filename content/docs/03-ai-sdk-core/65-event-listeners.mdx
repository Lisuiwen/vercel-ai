# 事件回调

AI SDK 提供可按次传入 `generateText`、`streamText`、`embed`、`embedMany` 与 `rerank` 的事件回调，用于观察生命周期事件。适用于构建可观测性工具、日志系统、分析与调试工具。

## 基本用法

将回调直接传给 `generateText`、`streamText`、`embed`、`embedMany` 或 `rerank`：

```ts
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What is the weather in San Francisco?',
  experimental_onStart: event => {
    console.log('Generation started:', event.modelId);
  },
  onFinish: event => {
    console.log('Generation finished:', event.totalUsage);
  },
});
```

使用 tool 的多步文本生成中，回调顺序通常为
`experimental_onStepStart` -> `experimental_onLanguageModelCallStart` ->
`experimental_onLanguageModelCallEnd` -> tool 执行回调 -> `onStepFinish`。

#### `onToolExecutionEnd`

在 tool 的 `execute` 函数完成或出错后调用。`toolOutput` 字段为判别联合：检查 `toolOutput.type` 可判断是 `'tool-result'`（成功）还是 `'tool-error'`（失败）。

```ts
const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What is the weather?',
  tools: { getWeather },
  onToolExecutionEnd: event => {
    console.log('Tool:', event.toolCall.toolName);
    console.log('Duration:', event.toolExecutionMs, 'ms');

    if (event.toolOutput.type === 'tool-result') {
      console.log('Output:', event.toolOutput.output);
    } else {
      console.error('Error:', event.toolOutput.error);
    }
  },
});
```

    console.log('Step:', event.stepNumber);
    console.log('Finish reason:', event.finishReason);

## 使用场景

### 日志与调试

```ts
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Hello!',
  experimental_onStart: event => {
    console.log(`[${new Date().toISOString()}] Generation started`, {
      model: event.modelId,
      provider: event.provider,
    });
  },
  onStepFinish: event => {
    console.log(
      `[${new Date().toISOString()}] Step ${event.stepNumber} finished`,
      {
        finishReason: event.finishReason,
        tokens: event.usage.totalTokens,
      },
    );
  },
  onFinish: event => {
    console.log(`[${new Date().toISOString()}] Generation complete`, {
      totalSteps: event.steps.length,
      totalTokens: event.totalUsage.totalTokens,
    });
  },
});
```

### Tool 执行监控

```ts
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What is the weather?',
  tools: { getWeather },
  onToolExecutionStart: event => {
    console.log(`Tool "${event.toolCall.toolName}" starting...`);
  },
  onToolExecutionEnd: event => {
    if (event.toolOutput.type === 'tool-result') {
      console.log(
        `Tool "${event.toolCall.toolName}" completed in ${event.toolExecutionMs}ms`,
      );
    } else {
      console.error(
        `Tool "${event.toolCall.toolName}" failed:`,
        event.toolOutput.error,
      );
    }
  },
});
```

### 嵌入可观测性

```ts
import { embedMany } from 'ai';

const result = await embedMany({
  model: openai.embedding('text-embedding-3-small'),
  values: ['sunny day at the beach', 'rainy afternoon in the city'],
  experimental_onStart: event => {
    console.log(`Embedding started (${event.operationId})`, {
      model: event.model.modelId,
      valueCount: Array.isArray(event.value) ? event.value.length : 1,
    });
  },
  experimental_onEnd: event => {
    console.log(`Embedding complete (${event.operationId})`, {
      tokens: event.usage.tokens,
    });
  },
});
```

## 错误处理

回调内抛出的错误会被捕获，不会中断生成、嵌入或重排序流程。这可确保监控代码不会破坏应用：

```ts
const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Hello!',
  experimental_onStart: () => {
    throw new Error('This error is caught internally');
    // Generation continues normally
  },
});
```
