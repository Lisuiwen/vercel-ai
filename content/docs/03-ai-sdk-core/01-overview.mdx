# AI SDK Core

大语言模型（LLM）是能够理解、创造并与人类语言进行大规模交互的高级程序。
它们在海量文本上训练，以识别语言中的模式，并预测给定文本片段之后可能出现的内容。

AI SDK Core **通过提供与 LLM 集成的标准化方式，简化你在应用中使用 LLM 的流程**——这样你可以专注于为用户构建出色的 AI 应用，而不必在实现细节上耗费时间。

## AI SDK Core 函数

AI SDK Core 提供多种函数，分别面向[文本生成](./generating-text)、[结构化数据生成](./generating-structured-data)和 [tool 使用](./tools-and-tool-calling)。
这些函数对[提示](./prompts)与[设置](./settings)采用统一方式，使你在不同模型之间切换时更容易协作。

- [`generateText`](/docs/ai-sdk-core/generating-text)：生成文本并执行 [tool 调用](./tools-and-tool-calling)。
  该函数适合非交互式场景，例如需要生成文本的自动化任务（起草邮件、摘要网页等），以及使用 tool 的 agent。
- [`streamText`](/docs/ai-sdk-core/generating-text)：流式输出文本与 tool 调用。
  可将 `streamText` 用于[聊天机器人](/docs/ai-sdk-ui/chatbot)、[内容流式输出](/docs/ai-sdk-ui/completion)等交互式场景。

`generateText` 与 `streamText` 均支持通过 `output` 属性实现[结构化输出](/docs/ai-sdk-core/generating-structured-data)（例如 `Output.object()`、`Output.array()`），从而生成经过 schema 校验的类型化数据，用于信息抽取、合成数据生成、分类任务以及[流式生成 UI](/docs/ai-sdk-ui/object-generation)。
