# 选择 Provider

AI SDK 通过[官方](/providers/ai-sdk-providers)、[OpenAI 兼容](/providers/openai-compatible-providers)和[社区](/providers/community-providers)包支持数十家模型 provider。

```ts
import { generateText } from 'ai';
__PROVIDER_IMPORT__;

const { text } = await generateText({
  model: __MODEL__,
  prompt: 'What is love?',
});
```

## AI Gateway

[Vercel AI Gateway](/providers/ai-sdk-providers/ai-gateway) 是快速上手 AI SDK 的最快方式。可访问来自 OpenAI、Anthropic、Google 等 provider 的模型。使用 [OIDC](https://ai-sdk.dev/providers/ai-sdk-providers/ai-gateway#oidc-authentication-vercel-deployments) 或 AI Gateway API key 进行认证。

将 API key 添加到环境变量：

```env filename=".env.local"
AI_GATEWAY_API_KEY=your_api_key_here
```

AI Gateway 是默认的[全局 provider](/docs/ai-sdk-core/provider-management#global-provider-configuration)，因此你可以使用简单字符串访问模型：

```ts
import { generateText } from 'ai';

const { text } = await generateText({
  model: 'anthropic/claude-sonnet-4.5',
  prompt: 'What is love?',
});
```

你也可以显式导入并使用 gateway provider：

```ts
// Option 1: Import from 'ai' package (included by default)
import { gateway } from 'ai';
model: gateway('anthropic/claude-sonnet-4.5');

// Option 2: Install and import from '@ai-sdk/gateway' package
import { gateway } from '@ai-sdk/gateway';
model: gateway('anthropic/claude-sonnet-4.5');
```

## 使用专用 Provider

你也可以直接使用[官方](/providers/ai-sdk-providers)、[OpenAI 兼容](/providers/openai-compatible-providers)和[社区](/providers/community-providers) provider 包。安装包并创建 provider 实例。例如，使用 Anthropic：

<div className="my-4">
  <Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>

      <Snippet text="pnpm add @ai-sdk/anthropic" dark />
    </Tab>
    <Tab>
      <Snippet text="npm install @ai-sdk/anthropic" dark />
    </Tab>
    <Tab>
      <Snippet text="yarn add @ai-sdk/anthropic" dark />
    </Tab>
    <Tab>
      <Snippet text="bun add @ai-sdk/anthropic" dark />
    </Tab>
  </Tabs>

```ts
import { anthropic } from '@ai-sdk/anthropic';

model: anthropic('claude-sonnet-4-5');
```

你可以更改默认全局 provider，使字符串形式的模型引用在整个应用中使用你偏好的 provider。了解更多关于 [provider 管理](/docs/ai-sdk-core/provider-management#global-provider-configuration) 的信息。

各 provider 的设置说明请参见[可用 provider](/providers/ai-sdk-providers)。

## 自定义 Provider

你可以构建自己的 provider，将任意服务与 AI SDK 集成。AI SDK 提供了 [Language Model Specification](https://github.com/vercel/ai/tree/main/packages/provider/src/language-model/v3)，确保各 provider 之间的兼容性。

```ts
import { generateText } from 'ai';
import { yourProvider } from 'your-custom-provider';

const { text } = await generateText({
  model: yourProvider('your-model-id'),
  prompt: 'What is love?',
});
```

完整指南请参见[编写自定义 Provider](/providers/community-providers/custom-providers)。
