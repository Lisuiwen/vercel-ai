# 编程 Agent 入门

本页说明如何在编程 agent（如 Claude Code、Codex、OpenCode、Cursor 或其他 AI 辅助开发环境）中充分利用 AI SDK。

## 安装 AI SDK Skill

让编程 agent 深入了解 AI SDK 的最快方式是安装官方 AI SDK skill。Skills 是轻量级 markdown 文件，可按需将专门指令加载到 agent 上下文中——这样 agent 无需你额外说明即可正确使用 SDK。

使用 `npx skills add` 安装 AI SDK skill：

```bash
npx skills add vercel/ai
```

这会将 skill 安装到 agent 专属的 skills 目录（如 `.claude/skills`、`.codex/skills`）。若选择多个 agent，CLI 会创建符号链接以便各 agent 发现该 skill。使用 `-a` 直接指定 agent，例如 `-a amp` 会安装到通用 `.agents/skills` 目录。使用 `-y` 进行非交互式安装。

安装后，支持 [Agent Skills](https://agentskills.io) 格式的任意 agent 在处理 AI SDK 任务时会自动发现并加载该 skill。

<Note>
  Agent Skills 采用**渐进式披露**：agent 在启动时仅加载 skill 的名称和描述。完整指令仅在任务需要时才拉入上下文，保持 agent 快速且专注。
</Note>

## `node_modules` 中的文档与源代码

安装 `ai` 包后，完整的 AI SDK 文档和源代码已本地存在于 `node_modules` 中。编程 agent 可直接读取——无需联网。

若尚未安装，请安装 `ai` 包：

## 安装 DevTools

AI SDK DevTools 可在开发期间完整查看 AI SDK 调用。它捕获 LLM 请求、响应、tool 调用、token 用量和多步交互，并在本地 Web UI 中展示。

<Note type="warning">
  AI SDK DevTools 处于实验阶段，仅用于本地开发。请勿在生产环境使用。
</Note>

安装 DevTools 包：

### 注册集成

全局注册 `DevToolsTelemetry`，以捕获所有 AI SDK 调用：

```ts
import { registerTelemetry } from 'ai';
import { DevToolsTelemetry } from '@ai-sdk/devtools';

registerTelemetry(DevToolsTelemetry());
```

注册集成后，所有 AI SDK 调用会自动启用[遥测](/docs/ai-sdk-core/telemetry)：

```ts
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What cities are in the United States?',
});
```

### 启动查看器

在单独终端中启动 DevTools 查看器：

```bash
npx @ai-sdk/devtools
```

打开 [http://localhost:4983](http://localhost:4983) 实时检查 AI SDK 交互。

## 检查 Tool 调用与输出

DevTools 为每次调用捕获并展示以下内容：

- **输入参数与 prompt** — 发送给 LLM 的完整输入
- **输出内容与 tool 调用** — 生成的文本和 tool 调用
- **Token 用量与耗时** — 每步的资源消耗与延迟
- **原始 provider 数据** — 完整的请求与响应负载

对于多步 agent 交互，DevTools 将所有内容分组为**运行**（完整交互）和**步骤**（其中的每次 LLM 调用），便于追踪 agent 的行为与原因。

你也可以在开发期间直接在代码中记录 tool 结果：

```ts
import { streamText, tool, isStepCount } from 'ai';
import { z } from 'zod';

const result = streamText({
  model,
  prompt: "What's the weather in New York in celsius?",
  tools: {
    weather: tool({
      description: 'Get the weather in a location (fahrenheit)',
      inputSchema: z.object({
        location: z.string().describe('The location to get the weather for'),
      }),
      execute: async ({ location }) => ({
        location,
        temperature: Math.round(Math.random() * (90 - 32) + 32),
      }),
    }),
  },
  stopWhen: isStepCount(5),
  onStepFinish: async ({ toolResults }) => {
    if (toolResults.length) {
      console.log(JSON.stringify(toolResults, null, 2));
    }
  },
});
```

`onStepFinish` 回调在每个 LLM 步骤完成后触发，并将 tool 结果打印到终端——无需打开 DevTools UI 即可快速调试。

<Note>
  DevTools 将所有 AI 交互存储在本地 `.devtools/generations.json`
  文件中。它会自动将 `.devtools` 添加到 `.gitignore`，防止提交敏感的交互数据。
</Note>
