# Clarifai 提供商

[Clarifai](https://docs.clarifai.com/getting-started/quickstart) 是用于构建、部署和扩展 AI 驱动应用的平台。它提供面向计算机视觉、自然语言处理和生成式 AI 的一套工具与 API。Clarifai 通过其全栈 AI 开发平台提供与 OpenAI 兼容的 API，因此可以便捷地使用 AI SDK 集成强大的 AI 能力。

## 安装

Clarifai provider 通过 `@ai-sdk/openai-compatible` 模块提供，因为它与 OpenAI API 兼容。可按以下方式安装：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/openai-compatible" dark />
  </Tab>
</Tabs>

## Provider 实例

要使用 Clarifai，可使用 `@ai-sdk/openai-compatible` 中的 `createOpenAICompatible` 创建自定义 provider 实例：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const clarifai = createOpenAICompatible({
  name: 'clarifai',
  baseURL: 'https://api.clarifai.com/v2/ext/openai/v1',
  apiKey: process.env.CLARIFAI_PAT,
});
```

<Note>
  可在 Clarifai [账户设置](https://clarifai.com/settings/security)中创建个人访问令牌（PAT）以获取 API 密钥。请务必将 `CLARIFAI_PAT` 环境变量设置为你的 PAT。

新用户可在 [Clarifai](https://clarifai.com/signup) 注册免费账户以开始使用。

</Note>

## 语言模型

你可以使用 provider 实例与 Clarifai 上提供的多种大语言模型（LLM）交互。例如，使用 [DeepSeek-R1](https://clarifai.com/deepseek-ai/deepseek-chat/models/DeepSeek-R1-0528-Qwen3-8B) 这款强大的开源语言模型：

```ts
const model = clarifai.chatModel(
  'https://clarifai.com/deepseek-ai/deepseek-chat/models/DeepSeek-R1-0528-Qwen3-8B',
);
```

### 示例 - 生成文本

可使用 Clarifai 语言模型配合 `generateText` 生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const clarifai = createOpenAICompatible({
  name: 'clarifai',
  baseURL: 'https://api.clarifai.com/v2/ext/openai/v1',
  apiKey: process.env.CLARIFAI_PAT,
});

const model = clarifai.chatModel(
  'https://clarifai.com/deepseek-ai/deepseek-chat/models/DeepSeek-R1-0528-Qwen3-8B',
);

const { text, usage, finishReason } = await generateText({
  model,
  prompt: 'What is photosynthesis?',
});

console.log(text);
console.log('Token usage:', usage);
console.log('Finish reason:', finishReason);
```

### 示例 - 流式文本

也可使用 `streamText` 从 Clarifai 模型流式获取文本响应：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { streamText } from 'ai';

const clarifai = createOpenAICompatible({
  name: 'clarifai',
  baseURL: 'https://api.clarifai.com/v2/ext/openai/v1',
  apiKey: process.env.CLARIFAI_PAT,
});

const model = clarifai.chatModel(
  'https://clarifai.com/deepseek-ai/deepseek-chat/models/DeepSeek-R1-0528-Qwen3-8B',
);

const result = streamText({
  model,
  prompt: 'What is photosynthesis?',
});

for await (const message of result.textStream) {
  console.log(message);
}
```

完整可用模型列表见 [Clarifai Model Gallery](https://clarifai.com/explore)。
