# Heroku 提供商

[Heroku](https://heroku.com/) 是云平台，可用于部署和运行应用，包括对 OpenAI API 兼容的 AI 模型。
你可以部署与 OpenAI API 兼容的模型，并将其与 AI SDK 一起使用。

## 安装

Heroku provider 通过 `@ai-sdk/openai-compatible` 模块提供，因为它与 OpenAI API 兼容。
可按以下方式安装：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/openai-compatible" dark />
  </Tab>
</Tabs>

### Heroku 配置

1. 在 Heroku 中创建测试应用：

```bash
heroku create
```

2. 使用 claude-3-5-haiku 进行推理：

```bash
heroku ai:models:create -a $APP_NAME claude-3-5-haiku
```

3. 导出变量：

```bash
export INFERENCE_KEY=$(heroku config:get INFERENCE_KEY -a $APP_NAME)
export INFERENCE_MODEL_ID=$(heroku config:get INFERENCE_MODEL_ID -a $APP_NAME)
export INFERENCE_URL=$(heroku config:get INFERENCE_URL -a $APP_NAME)
```

## Provider 实例

要使用 Heroku，可使用 `@ai-sdk/openai-compatible` 中的 `createOpenAICompatible` 创建自定义 provider 实例：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const heroku = createOpenAICompatible({
  name: 'heroku',
  baseURL: process.env.INFERENCE_URL + '/v1',
  apiKey: process.env.INFERENCE_KEY,
});
```

请确保在环境变量中设置 `INFERENCE_KEY`、`INFERENCE_MODEL_ID` 和 `INFERENCE_URL`。

## 语言模型

你可以使用 provider 实例创建 Heroku 模型。
第一个参数为已提供服务的模型名称，例如 `claude-3-5-haiku`。

```ts
const model = heroku('claude-3-5-haiku');
```

### 示例

可使用 Heroku 语言模型配合 `generateText` 生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const heroku = createOpenAICompatible({
  name: 'heroku',
  baseURL: process.env.INFERENCE_URL + '/v1',
  apiKey: process.env.INFERENCE_KEY,
});

const { text } = await generateText({
  model: heroku('claude-3-5-haiku'),
  prompt: 'Tell me about yourself in one sentence',
});

console.log(text);
```

Heroku 语言模型也可配合 `streamText` 以流式方式生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { streamText } from 'ai';

const heroku = createOpenAICompatible({
  name: 'heroku',
  baseURL: process.env.INFERENCE_URL + '/v1',
  apiKey: process.env.INFERENCE_KEY,
});

const result = streamText({
  model: heroku('claude-3-5-haiku'),
  prompt: 'Tell me about yourself in one sentence',
});

for await (const message of result.textStream) {
  console.log(message);
}
```

Heroku 语言模型也支持通过 [`Output`](/docs/reference/ai-sdk-core/output) 进行结构化数据生成。
