# LM Studio Provider

[LM Studio](https://lmstudio.ai/) 是用于在本地运行模型的用户界面。

它内置与 OpenAI 兼容的 API 服务器，可与 AI SDK 一起使用。
你可以在 LM Studio 界面中的 [Local Server 标签页](https://lmstudio.ai/docs/basics/server)启动本地服务器（「Start Server」按钮）。

## 安装

LM Studio provider 通过 `@ai-sdk/openai-compatible` 模块提供，因为它与 OpenAI API 兼容。
可按以下方式安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>

  </Tab>

  </Tab>

## Provider 实例

要使用 LM Studio，可使用 `@ai-sdk/openai-compatible` 中的 `createOpenAICompatible` 创建自定义 provider 实例：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const lmstudio = createOpenAICompatible({
  name: 'lmstudio',
  baseURL: 'http://localhost:1234/v1',
});
```

<Note>
  LM Studio 默认使用端口 `1234`，但你可以在应用 [Local
  Server 标签页](https://lmstudio.ai/docs/basics/server)中更改。
</Note>

## 语言模型

你可以使用 provider 实例在 [LM Studio](https://lmstudio.ai/docs/basics/server#endpoints-overview) 中与本地 LLM 交互。
第一个参数为模型 id，例如 `llama-3.2-1b`。

```ts
const model = lmstudio('llama-3.2-1b');
```

###### 要使用某个模型，需要先[下载模型](https://lmstudio.ai/docs/basics/download-model)。

### 示例

可使用 LM Studio 语言模型配合 `generateText` 生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const lmstudio = createOpenAICompatible({
  name: 'lmstudio',
  baseURL: 'https://localhost:1234/v1',
});

const { text } = await generateText({
  model: lmstudio('llama-3.2-1b'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
  maxRetries: 1, // immediately error if the server is not running
});
```

LM Studio 语言模型也可配合 `streamText` 使用。

## 嵌入模型

可使用 `.embeddingModel()` 工厂方法创建调用 [LM Studio embeddings API](https://lmstudio.ai/docs/basics/server#endpoints-overview)
的模型。

```ts
const model = lmstudio.embeddingModel('text-embedding-nomic-embed-text-v1.5');
```

### 示例 - 嵌入单个值

```tsx
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { embed } from 'ai';

const lmstudio = createOpenAICompatible({
  name: 'lmstudio',
  baseURL: 'https://localhost:1234/v1',
});

// 'embedding' is a single embedding object (number[])
const { embedding } = await embed({
  model: lmstudio.embeddingModel('text-embedding-nomic-embed-text-v1.5'),
  value: 'sunny day at the beach',
});
```

### 示例 - 嵌入多个值

在加载数据时（例如为检索增强生成（RAG）准备数据存储），
通常需要一次嵌入多个值（批量嵌入）。

AI SDK 为此提供 [`embedMany`](/docs/reference/ai-sdk-core/embed-many) 函数。
与 `embed` 类似，可将其用于嵌入模型，
例如 `lmstudio.embeddingModel('text-embedding-nomic-embed-text-v1.5')` 或 `lmstudio.embeddingModel('text-embedding-bge-small-en-v1.5')`。

```tsx
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { embedMany } from 'ai';

const lmstudio = createOpenAICompatible({
  name: 'lmstudio',
  baseURL: 'https://localhost:1234/v1',
});

// 'embeddings' is an array of embedding objects (number[][]).
// It is sorted in the same order as the input values.
const { embeddings } = await embedMany({
  model: lmstudio.embeddingModel('text-embedding-nomic-embed-text-v1.5'),
  values: [
    'sunny day at the beach',
    'rainy afternoon in the city',
    'snowy night in the mountains',
  ],
});
```
