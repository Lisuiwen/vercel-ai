# NVIDIA NIM Provider

[NVIDIA NIM](https://www.nvidia.com/en-us/ai/) 提供优化的推理微服务，用于部署基础模型。它提供可与 AI SDK 一起使用的 OpenAI 兼容 API。

## 安装

NVIDIA NIM provider 通过 `@ai-sdk/openai-compatible` 模块提供，因为它与 OpenAI API 兼容。
可按以下方式安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/openai-compatible" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/openai-compatible" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/openai-compatible" dark />
  </Tab>
</Tabs>

## Provider 实例

要使用 NVIDIA NIM，可使用 `@ai-sdk/openai-compatible` 中的 `createOpenAICompatible` 创建自定义 provider 实例：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';

const nim = createOpenAICompatible({
  name: 'nim',
  baseURL: 'https://integrate.api.nvidia.com/v1',
  headers: {
    Authorization: `Bearer ${process.env.NIM_API_KEY}`,
  },
});
```

<Note>
  可在 [NVIDIA
  Build](https://build.nvidia.com/explore/discover) 注册以获取 API 密钥和免费额度。新用户可获得 1,000
  推理额度用于上手。
</Note>

## 语言模型

你可以使用 provider 实例与 NIM 模型交互。例如，使用 [DeepSeek-R1](https://build.nvidia.com/deepseek-ai/deepseek-r1) 这款强大的开源语言模型：

```ts
const model = nim.chatModel('deepseek-ai/deepseek-r1');
```

### 示例 - 生成文本

可使用 NIM 语言模型配合 `generateText` 生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const nim = createOpenAICompatible({
  name: 'nim',
  baseURL: 'https://integrate.api.nvidia.com/v1',
  headers: {
    Authorization: `Bearer ${process.env.NIM_API_KEY}`,
  },
});

const { text, usage, finishReason } = await generateText({
  model: nim.chatModel('deepseek-ai/deepseek-r1'),
  prompt: 'Tell me the history of the San Francisco Mission-style burrito.',
});

console.log(text);
console.log('Token usage:', usage);
console.log('Finish reason:', finishReason);
```

### 示例 - 流式文本

NIM 语言模型也可配合 `streamText` 以流式方式生成文本：

```ts
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { streamText } from 'ai';

const nim = createOpenAICompatible({
  name: 'nim',
  baseURL: 'https://integrate.api.nvidia.com/v1',
  headers: {
    Authorization: `Bearer ${process.env.NIM_API_KEY}`,
  },
});

const result = streamText({
  model: nim.chatModel('deepseek-ai/deepseek-r1'),
  prompt: 'Tell me the history of the Northern White Rhino.',
});

for await (const textPart of result.textStream) {
  process.stdout.write(textPart);
}

console.log();
console.log('Token usage:', await result.usage);
console.log('Finish reason:', await result.finishReason);
```

NIM 语言模型也支持通过 [`Output`](/docs/reference/ai-sdk-core/output) 进行结构化数据生成。

<Note>
  不同模型对 tool 调用和结构化输出的支持程度不同。例如，
  [`meta/llama-3.3-70b-instruct`](https://build.nvidia.com/meta/llama-3_3-70b-instruct)
  模型支持结构化输出能力。具体支持的功能请参阅 NVIDIA Build 上各模型的
  文档。
</Note>
