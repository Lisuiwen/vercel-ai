# MiniMax 提供商

[vercel-minimax-ai-provider](https://github.com/MiniMax-AI/vercel-minimax-ai-provider) 是社区提供商，可访问 [MiniMax](https://www.minimax.io/) 最新的 [MiniMax-M2 模型](https://platform.minimax.io/docs/guides/text-generation)。

API 密钥可在 [MiniMax 平台](https://platform.minimax.io/user-center/basic-information/interface-key) 获取。

## 安装与配置

MiniMax 提供商通过 `vercel-minimax-ai-provider` 模块提供。可使用以下命令安装：

## 提供商实例

MiniMax 提供商支持两种 API 兼容模式：

### Anthropic 兼容 API（默认）

可从 `vercel-minimax-ai-provider` 导入默认提供商实例 `minimax`：

```ts
import { minimax } from 'vercel-minimax-ai-provider';
```

或显式使用 Anthropic 兼容实例：

```ts
import { minimaxAnthropic } from 'vercel-minimax-ai-provider';
```

或 OpenAI 兼容 API 格式：

```ts
import { minimaxOpenAI } from 'vercel-minimax-ai-provider';
```

## 自定义配置

如需自定义配置，可使用 `createMinimax`（Anthropic 兼容）或 `createMinimaxOpenAI`（OpenAI 兼容）：

### Anthropic 兼容配置（默认）

```ts
import { createMinimax } from 'vercel-minimax-ai-provider';

const minimax = createMinimax({
  apiKey: process.env.MINIMAX_API_KEY,
});
```

### 配置选项

可使用以下可选设置自定义 MiniMax 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀。
  - Anthropic 兼容默认：`https://api.minimax.io/anthropic/v1`
  - OpenAI 兼容默认：`https://api.minimax.io/v1`

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。默认读取
  `MINIMAX_API_KEY` 环境变量。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## API 兼容性

MiniMax 提供两种 API 格式，本包均包含。

主要区别在于 API 的请求/响应格式：

- **Anthropic 格式**（默认）：使用带 `anthropic-version` 标头的 Anthropic Messages API 格式
- **OpenAI 格式**：使用标准 OpenAI 聊天补全格式

两种格式均可访问相同的 MiniMax 模型与能力。

## 模型能力

| 模型                | 文本生成            | 结构化对象生成      | 图像输入            | 工具使用            | 工具流式            |
| ------------------- | ------------------- | ------------------- | ------------------- | ------------------- | ------------------- |
| `MiniMax-M2`        | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Check size={18} /> | <Check size={18} /> |
| `MiniMax-M2-Stable` | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Check size={18} /> | <Check size={18} /> |

<Note>
  完整可用模型及能力列表请参阅 [MiniMax

</Note>

## 示例用法

### 基础文本生成

```ts
import { minimax } from 'vercel-minimax-ai-provider';
import { generateText } from 'ai';

const result = await generateText({
  model: minimax('MiniMax-M2'),
  prompt: 'Explain quantum computing in simple terms.',
});

console.log(result.text);
```

### 流式输出

```ts
import { minimax } from 'vercel-minimax-ai-provider';
import { streamText } from 'ai';

const result = streamText({
  model: minimax('MiniMax-M2'),
  prompt: 'Write a short story about a robot learning to paint.',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```
