# Voyage AI 提供商

[patelvivekdev/voyage-ai-provider](https://github.com/patelvivekdev/voyageai-ai-provider) 是社区提供商，通过 [Voyage AI](https://www.voyageai.com) 为 AI SDK 提供嵌入支持。

## 安装与配置

Voyage 提供商位于 `voyage-ai-provider` 模块。可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add voyage-ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install voyage-ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add voyage-ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="bun add voyage-ai-provider" dark />
  </Tab>
</Tabs>

## 提供商实例

可从 `voyage-ai-provider` 导入默认提供商实例 `voyage`：

```ts
import { voyage } from 'voyage-ai-provider';
```

如需自定义配置，可导入 `createVoyage` 并创建实例：

```ts
import { createVoyage } from 'voyage-ai-provider';

const voyage = createVoyage({
  // custom settings
});
```

可使用以下可选设置自定义 Voyage 提供商实例：

- **baseURL** _string_

  Voyage API 的基础 URL。
  默认前缀为 `https://api.voyageai.com/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。
  默认读取 `VOYAGE_API_KEY` 环境变量。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作中间件拦截请求，
  或为测试等场景提供自定义 fetch。

## 文本嵌入模型

使用 `.embeddingModel()` 工厂方法可创建调用 [Voyage embeddings API](https://docs.voyageai.com/reference/embeddings-api) 的模型。

```ts
import { voyage } from 'voyage-ai-provider';

const embeddingModel = voyage.embeddingModel('voyage-3.5-lite');
```

可使用 Voyage 嵌入模型配合 `embed` 或 `embedMany` 生成向量：

```ts
import { voyage } from 'voyage-ai-provider';
import { embed } from 'ai';

const { embedding } = await embed({
  model: voyage.embeddingModel('voyage-3.5-lite'),
  value: 'sunny day at the beach',
  providerOptions: {
    voyage: {
      inputType: 'document',
    },
  },
});
```

Voyage 嵌入模型支持通过 `providerOptions.voyage` 传入额外选项：

```ts
import { voyage } from 'voyage-ai-provider';
import { embed } from 'ai';

const { embedding } = await embed({
  model: voyage.embeddingModel('voyage-3.5-lite'),
  value: 'sunny day at the beach',
  providerOptions: {
    voyage: {
      inputType: 'query',
      outputDimension: 512,
    },
  },
});
```

以下 [提供商选项](https://docs.voyageai.com/reference/embeddings-api) 可用：

- **inputType** _'query' | 'document' | 'null'_

  指定传入模型的输入类型。默认为 `'null'`。
  - `'null'`：`inputType` 为 `'null'` 时，嵌入模型直接将输入转为数值向量。

  用于检索/搜索时建议使用 `'query'` 或 `'document'`。
  - `'query'`：输入为检索查询，例如「Represent the query for retrieving supporting documents: ...」。
  - `'document'`：输入为存入向量库的文档，例如「Represent the document for retrieval: ...」。

- **outputDimension** _number_

  输出嵌入向量的维度数。默认为 `'null'`。
  - 例如 `voyage-code-3` 与 `voyage-3-large` 支持：2048、1024（默认）、512、256。
  - 支持取值请参阅 [模型文档](https://docs.voyageai.com/docs/embeddings)。

- **outputDtype** _'float' | 'int8' | 'uint8' | 'binary' | 'ubinary'_

  输出嵌入的数据类型。默认为 `'float'`。
  - `'float'`：32 位浮点（所有模型均支持）。
  - `'int8'`、`'uint8'`：8 位整数（`voyage-3-large`、`voyage-3.5`、`voyage-3.5-lite`、`voyage-code-3` 支持）。
  - `'binary'`、`'ubinary'`：位打包的单比特量化嵌入（上述模型支持）。返回列表长度为 `outputDimension` 的 1/8。`'binary'` 使用偏移二进制编码。

  详见 [FAQ：输出数据类型](https://docs.voyageai.com/docs/faq#what-is-quantization-and-output-data-types)。

- **truncation** _boolean_

  是否截断输入文本以适配模型上下文长度。未指定时默认为 true。

更多模型见 [Voyage Library](https://docs.voyageai.com/docs/embeddings) 主页。

## 图像嵌入

### 示例 1：将单张图像嵌入为单向量

```ts
import { voyage, ImageEmbeddingInput } from 'voyage-ai-provider';
import { embedMany } from 'ai';

const imageModel = voyage.imageEmbeddingModel('voyage-multimodal-3');

const { embeddings } = await embedMany<ImageEmbeddingInput>({
  model: imageModel,
  values: [
    {
      image:
        'https://raw.githubusercontent.com/voyage-ai/voyage-multimodal-3/refs/heads/main/images/banana_200_x_200.jpg',
    },
    {
      image: 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...',
    },
  ],
  // or you can pass the array of images url and base64 string directly
  // values: [
  //   'https://raw.githubusercontent.com/voyage-ai/voyage-multimodal-3/refs/heads/main/images/banana_200_x_200.jpg',
  //   'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...',
  // ],
});
```

### 示例 2：将多张图像嵌入为单向量

```ts
import { voyage, ImageEmbeddingInput } from 'voyage-ai-provider';
import { embedMany } from 'ai';

const imageModel = voyage.imageEmbeddingModel('voyage-multimodal-3');

const { embeddings } = await embedMany<ImageEmbeddingInput>({
  model: imageModel,
  values: [
    {
      image: [
        'https://raw.githubusercontent.com/voyage-ai/voyage-multimodal-3/refs/heads/main/images/banana_200_x_200.jpg',
        'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...',
      ],
    },
  ],
});
```

<Note>
  若出现图像 URL 未找到错误，可将图像转为 base64，并以 data URL 形式传入图像数组。值应为 `data:[mediatype];base64,<data>` 格式的 Base64 图像。支持的媒体类型：`image/png`、`image/jpeg`、`image/webp`、`image/gif`。
</Note>

## 多模态嵌入

### 示例 1：将多段文本与多张图像嵌入为单向量

```ts
import { voyage, MultimodalEmbeddingInput } from 'voyage-ai-provider';
import { embedMany } from 'ai';

const multimodalModel = voyage.multimodalEmbeddingModel('voyage-multimodal-3');

const { embeddings } = await embedMany<MultimodalEmbeddingInput>({
  model: multimodalModel,
  values: [
    {
      text: ['Hello, world!', 'This is a banana'],
      image: [
        'https://raw.githubusercontent.com/voyage-ai/voyage-multimodal-3/refs/heads/main/images/banana_200_x_200.jpg',
      ],
    },
    {
      text: ['Hello, coders!', 'This is a coding test'],
      image: ['data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...'],
    },
  ],
});
```

对 `values` 列表的约束如下：

- 列表不得超过 1,000 个值。
- 每张图像不得超过 1,600 万像素或 20 MB。
- 图像每 560 像素计为 1 个 token，列表中每个输入不得超过 32,000 token，所有输入合计不得超过 320,000 token。

Voyage 多模态嵌入模型支持通过 `providerOptions.voyage` 传入额外选项：

```ts
import { voyage, MultimodalEmbeddingInput } from 'voyage-ai-provider';
import { embedMany } from 'ai';

const multimodalModel = voyage.multimodalEmbeddingModel('voyage-multimodal-3');

const { embeddings } = await embedMany<MultimodalEmbeddingInput>({
  model: multimodalModel,
  values: [
    {
      text: ['Hello, world!'],
      image: ['data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA...'],
    },
  ],
  providerOptions: {
    voyage: {
      inputType: 'query',
      outputEncoding: 'base64',
      truncation: true,
    },
  },
});
```

可用提供商选项：

- **inputType** _'query' | 'document'_

  指定传入模型的输入类型。默认为 `'query'`。

  当 `inputType` 为 `'query'` 或 `'document'` 时，Voyage 会在向量化前自动为输入添加前缀提示，生成适用于检索/搜索任务的向量：
  - `'query'`：前缀为 "Represent the query for retrieving supporting documents: "
  - `'document'`：前缀为 "Represent the document for retrieval: "

- **outputEncoding** _'base64'_

  输出嵌入的数据编码。默认为 `null`（32 位浮点列表）。
  - `null` 时，嵌入为浮点数列表（float32）。
  - `'base64'` 时，嵌入为 Base64 编码的单精度浮点 NumPy 数组。

  详见 [FAQ：输出数据类型](https://docs.voyageai.com/docs/faq#what-is-quantization-and-output-data-types)。

- **truncation** _boolean_

  是否截断输入以适配模型上下文长度。未指定时默认为 `true`。

| 模型                 | 上下文长度（token） | 嵌入维度 |
| -------------------- | ------------------- | -------- |
| `voyage-multimodal-3`| 32,000              | 1024     |
