# OpenCode 提供商

[ai-sdk-provider-opencode-sdk](https://github.com/ben-vargas/ai-sdk-provider-opencode-sdk) 社区提供商可通过 [OpenCode SDK](https://www.npmjs.com/package/@opencode-ai/sdk) 使用多家 AI 提供商（Anthropic、OpenAI、Google）的模型。OpenCode 是基于终端的 AI 编程助手，为各类 AI 模型提供统一接口。

## 版本兼容性

| 提供商版本 | AI SDK 版本 | NPM 标签      | 状态     |
| ---------- | ----------- | ------------- | -------- |
| 1.x        | v6          | `latest`      | 稳定     |
| 0.x        | v5          | `ai-sdk-v5`   | 维护     |

```bash
# AI SDK v6 (default)
npm install ai-sdk-provider-opencode-sdk ai

# AI SDK v5
npm install ai-sdk-provider-opencode-sdk@ai-sdk-v5 ai@^5.0.0
```

## 提供商实例

可从 `ai-sdk-provider-opencode-sdk` 导入默认提供商实例 `opencode`：

```ts
import { opencode } from 'ai-sdk-provider-opencode-sdk';
```

如需自定义配置，可导入 `createOpencode` 并创建实例：

```ts
import { createOpencode } from 'ai-sdk-provider-opencode-sdk';

const opencode = createOpencode({
  autoStartServer: true,
  serverTimeout: 10000,
  defaultSettings: {
    agent: 'build',
  },
});
```

提供商设置：

- **hostname** _string_ — 服务器主机名（默认：`127.0.0.1`）。
- **port** _number_ — 服务器端口（默认：`4096`）。
- **autoStartServer** _boolean_ — 是否自动启动 OpenCode 服务器（默认：`true`）。
- **serverTimeout** _number_ — 服务器启动超时（毫秒，默认：`10000`）。
- **defaultSettings** _object_ — 应用于所有模型调用的默认设置。

## 语言模型

模型使用 `providerID/modelID` 格式指定：

```ts
const model = opencode('anthropic/claude-sonnet-4-5-20250929');
```

**模型快捷方式**（以 `OpencodeModels` 导出）：

```ts
import { OpencodeModels } from 'ai-sdk-provider-opencode-sdk';

// Anthropic Claude
opencode(OpencodeModels['claude-opus-4-5']); // anthropic/claude-opus-4-5-20251101
opencode(OpencodeModels['claude-sonnet-4-5']); // anthropic/claude-sonnet-4-5-20250929
opencode(OpencodeModels['claude-haiku-4-5']); // anthropic/claude-haiku-4-5-20251001

// OpenAI GPT
opencode(OpencodeModels['gpt-4o']); // openai/gpt-4o
opencode(OpencodeModels['gpt-4o-mini']); // openai/gpt-4o-mini

// Google Gemini
opencode(OpencodeModels['gemini-3.1-pro-preview']); // google/gemini-3.1-pro-preview
opencode(OpencodeModels['gemini-2.5-pro']); // google/gemini-2.5-pro
opencode(OpencodeModels['gemini-2.5-flash']); // google/gemini-2.5-flash
opencode(OpencodeModels['gemini-2.0-flash']); // google/gemini-2.0-flash
```

也可使用完整模型标识符：

```ts
opencode('openai/gpt-5.1-codex');
opencode('openai/gpt-5.1-codex-max');
opencode('google/gemini-3.1-pro-preview');
```

### 示例

```ts
import { opencode } from 'ai-sdk-provider-opencode-sdk';
import { generateText } from 'ai';

const { text } = await generateText({
  model: opencode('anthropic/claude-sonnet-4-5-20250929'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

### 模型设置

```ts
const model = opencode('anthropic/claude-opus-4-5-20251101', {
  agent: 'build', // 'build' | 'plan' | 'general' | 'explore'
  sessionTitle: 'My Task',
  systemPrompt: 'You are a helpful assistant.',
});
```

### 模型能力

| 提供商    | 图像输入            | 结构化对象生成      | 工具使用            | 工具流式            |
| --------- | ------------------- | ------------------- | ------------------- | ------------------- |
| Anthropic | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |
| OpenAI    | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |
| Google    | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |

<Note>
  工具使用与工具流式显示为 ❌，是因为该提供商不支持
  AI SDK 自定义工具（传给 `generateText`/`streamText` 的 Zod schema）。
  自定义工具定义会被显式忽略。OpenCode 在服务端执行工具，
  可通过流式事件观察。图像输入仅支持 data URL 与 base64。结构化对象生成采用基于提示的 JSON 模式。
</Note>

## 服务器管理

OpenCode 以托管服务器方式运行。用完后请释放提供商：

```ts
import { opencode } from 'ai-sdk-provider-opencode-sdk';

// After you're done
await opencode.dispose();

// Or if you need direct access to the client manager:
// await opencode.getClientManager().dispose();
```

客户端管理器会在进程退出时自动清理（SIGINT、SIGTERM）。

## 环境要求

- Node.js 18 及以上
- 已安装 OpenCode CLI（`npm install -g opencode`）
- 在 OpenCode 中配置提供商凭证（Anthropic、OpenAI 或 Google API 密钥）

更多说明请参阅 [提供商文档](https://github.com/ben-vargas/ai-sdk-provider-opencode-sdk)。
