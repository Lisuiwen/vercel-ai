# OpenRouter

[OpenRouter](https://openrouter.ai/) 是统一 API 网关，可访问来自 Anthropic、Google、Meta、Mistral 等厂商的数百种 AI 模型。面向 AI SDK 的 OpenRouter 提供商可无缝接入这些模型，并具有以下优势：

- **通用模型访问**：一把 API 密钥访问多家提供商的数百种模型
- **成本友好**：按量付费，无月费或长期合约
- **定价透明**：各模型按 token 明码标价
- **高可用**：企业级基础设施与自动故障转移
- **接入简单**：跨模型统一 API
- **紧跟新模型**：新模型上线即可使用

更多能力说明见 [OpenRouter 文档](https://openrouter.ai/docs)。

## 提供商实例

使用 `createOpenRouter` 创建 OpenRouter 提供商实例：

```typescript
import { createOpenRouter } from '@openrouter/ai-sdk-provider';

const openrouter = createOpenRouter({
  apiKey: 'YOUR_OPENROUTER_API_KEY',
});
```

可在 [OpenRouter 控制台](https://openrouter.ai/keys) 获取 API 密钥。

## 语言模型

OpenRouter 支持聊天模型与补全模型。聊天模型使用 `openrouter.chat()`，补全模型使用 `openrouter.completion()`：

```typescript
// Chat models (recommended)
const chatModel = openrouter.chat('anthropic/claude-3.5-sonnet');

// Completion models
const completionModel = openrouter.completion(
  'meta-llama/llama-3.1-405b-instruct',
);
```

完整模型列表见 [OpenRouter 模型文档](https://openrouter.ai/docs#models)。

## 示例

以下示例展示如何将 OpenRouter 与 AI SDK 配合使用：

### `generateText`

```javascript
import { createOpenRouter } from '@openrouter/ai-sdk-provider';
import { generateText } from 'ai';

const openrouter = createOpenRouter({
  apiKey: 'YOUR_OPENROUTER_API_KEY',
});

const { text } = await generateText({
  model: openrouter.chat('anthropic/claude-3.5-sonnet'),
  prompt: 'What is OpenRouter?',
});

console.log(text);
```

### `streamText`

```javascript
import { createOpenRouter } from '@openrouter/ai-sdk-provider';
import { streamText } from 'ai';

const openrouter = createOpenRouter({
  apiKey: 'YOUR_OPENROUTER_API_KEY',
});

const result = streamText({
  model: openrouter.chat('meta-llama/llama-3.1-405b-instruct'),
  prompt: 'Write a short story about AI.',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

## 高级功能

OpenRouter 还提供多项高级能力以增强 AI 应用：

1. **模型灵活**：无需改代码或管理多把密钥即可切换数百种模型。

2. **成本管理**：在控制台实时按模型查看用量与费用。

3. **企业支持**：面向高用量用户提供自定义 SLA 与专属支持。

4. **跨提供商兼容**：不同模型提供商可使用相同的代码结构。

5. **持续更新**：新模型与功能可用时自动接入。

更多信息与高级配置见 [OpenRouter 文档](https://openrouter.ai/docs)。
