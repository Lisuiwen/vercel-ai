# Mem0 提供商

[Mem0 Provider](https://github.com/mem0ai/mem0/tree/main/vercel-ai-sdk) 是由 [**Mem0**](https://mem0.ai)
开发的库，用于与 AI SDK 集成。
该库通过引入持久记忆功能，为你的应用带来更强的 AI 交互能力。

<Note type="info">
  🎉 好消息：Mem0 AI SDK 现已支持 <strong>工具调用（Tools Call）</strong>。
</Note>

## 安装与配置

Mem0 提供商位于 `@mem0/vercel-ai-provider` 模块。可使用以下命令安装：

## 提供商实例

首先，在 [Mem0 控制台](https://app.mem0.ai/dashboard/api-keys) 获取 **Mem0 API Key**。

然后在应用中初始化 `Mem0 Client`：

```ts
import { createMem0 } from '@mem0/vercel-ai-provider';

const mem0 = createMem0({
  provider: 'openai',
  mem0ApiKey: 'm0-xxx',
  apiKey: 'provider-api-key',
  config: {
    // Configure the LLM Provider here
  },
  // Optional Mem0 Global Config
  mem0Config: {
    user_id: 'mem0-user-id',
    enable_graph: true,
  },
});
```

<Note>
  默认使用 `openai` 提供商。为安全起见，建议使用环境变量 `MEM0_API_KEY` 和
  `OPENAI_API_KEY`。
</Note>

<Note>
  `mem0Config` 为可选，用于为 Mem0 Client 设置全局配置（例如 `user_id`、`agent_id`、`app_id`、`run_id`、`org_id`、`project_id`
  等）。
</Note>

- 添加记忆以增强上下文：

```ts
import { LanguageModelV4Prompt } from '@ai-sdk/provider';
import { addMemories } from '@mem0/vercel-ai-provider';

const messages: LanguageModelV4Prompt = [
  { role: 'user', content: [{ type: 'text', text: 'I love red cars.' }] },
];

await addMemories(messages, { user_id: 'borat' });
```

## 功能

### 添加与检索记忆

- `retrieveMemories()`：为提示检索记忆上下文。
- `getMemories()`：以数组格式从个人资料中获取记忆。
- `addMemories()`：添加用户记忆以增强上下文相关的回复。

```ts
await addMemories(messages, {
  user_id: 'borat',
  mem0ApiKey: 'm0-xxx',
});
await retrieveMemories(prompt, {
  user_id: 'borat',
  mem0ApiKey: 'm0-xxx',
});
await getMemories(prompt, {
  user_id: 'borat',
  mem0ApiKey: 'm0-xxx',
});
```

<Note>
  对于独立功能（如 `addMemories`、`retrieveMemories`、`getMemories`），你必须将 `MEM0_API_KEY` 设为环境变量，
  或在函数调用中直接传入。
</Note>

<Note>
  `getMemories` 会以对象数组形式返回原始记忆，
  而 `retrieveMemories` 会返回字符串格式的响应，并在系统提示中融入检索到的记忆。
</Note>

### 使用记忆上下文生成文本

可使用 **OpenAI**、**Anthropic**、**Cohere** 与 **Groq** 的语言模型，配合 `generateText` 生成文本：

```ts
import { generateText } from 'ai';
import { createMem0 } from '@mem0/vercel-ai-provider';

const mem0 = createMem0();

const { text } = await generateText({
  model: mem0('gpt-4.1', { user_id: 'borat' }),
  prompt: 'Suggest me a good car to buy!',
});
```

### 带记忆的结构化消息格式

```ts
import { generateText } from 'ai';
import { createMem0 } from '@mem0/vercel-ai-provider';

const mem0 = createMem0();

const { text } = await generateText({
  model: mem0('gpt-4.1', { user_id: 'borat' }),
  messages: [
    {
      role: 'user',
      content: [
        { type: 'text', text: 'Suggest me a good car to buy.' },
        { type: 'text', text: 'Why is it better than the other cars for me?' },
      ],
    },
  ],
});
```

### 带记忆上下文的流式响应

```ts
import { streamText } from 'ai';
import { createMem0 } from '@mem0/vercel-ai-provider';

const mem0 = createMem0();

const { textStream } = streamText({
  model: mem0('gpt-4.1', {
    user_id: 'borat',
  }),
  prompt:
    'Suggest me a good car to buy! Why is it better than the other cars for me? Give options for every price range.',
});

for await (const textPart of textStream) {
  process.stdout.write(textPart);
}
```

### 使用工具调用生成响应

```ts
import { generateText } from 'ai';
import { createMem0 } from '@mem0/vercel-ai-provider';
import { z } from 'zod';

const mem0 = createMem0({
  provider: 'anthropic',
  apiKey: 'anthropic-api-key',
  mem0Config: {
    // Global User ID
    user_id: 'borat',
  },
});

const prompt = 'What the temperature in the city that I live in?';

const result = await generateText({
  model: mem0('claude-3-5-sonnet-20240620'),
  tools: {
    weather: tool({
      description: 'Get the weather in a location',
      parameters: z.object({
        location: z.string().describe('The location to get the weather for'),
      }),
      execute: async ({ location }) => ({
        location,
        temperature: 72 + Math.floor(Math.random() * 21) - 10,
      }),
    }),
  },
  prompt: prompt,
});

console.log(result);
```

### 从记忆获取来源

```ts
const { text, sources } = await generateText({
  model: mem0('gpt-4.1'),
  prompt: 'Suggest me a good car to buy!',
});

console.log(sources);
```

`streamText` 同样支持上述能力。

## 支持的 LLM 提供商

Mem0 提供商支持以下 LLM：

| 提供商    | 配置取值    |
| --------- | ----------- |
| OpenAI    | `openai`    |
| Anthropic | `anthropic` |
| Google    | `google`    |
| Groq      | `groq`      |
| Cohere    | `cohere`    |

## 最佳实践

- **用户标识**：使用稳定的唯一 `user_id` 以保持一致的记忆检索。
- **记忆清理**：定期清理不再使用的记忆数据。

<Note>
  我们还支持 `agent_id`、`app_id` 和 `run_id`。详见
  [文档](https://docs.mem0.ai/api-reference/memory/add-memories)。
</Note>
