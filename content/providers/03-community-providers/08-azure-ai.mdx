# AI SDK 的 Azure 自定义提供商

**[Quail-AI/azure-ai-provider](https://github.com/QuailAI/azure-ai-provider)** 提供与托管在 Azure 上的语言模型的非官方集成，使用 Azure 原生 API（而非标准 OpenAI 格式）。

## 语言模型

本提供商可与 Azure AI Foundry 中任意兼容 Azure-Rest AI-inference API 的模型配合。

### 已测试模型：

- DeepSeek-R1
- LLama 3.3-70B Instruct
- Cohere-command-r-08-2024

## 安装

### 安装包

```bash
npm i @quail-ai/azure-ai-provider
```

## 提供商实例

创建 Azure AI 资源并配置 endpoint URL 与 API key。在 `.env` 中加入：

```bash
AZURE_API_ENDPOINT=https://<your-resource>.services.ai.azure.com/models
AZURE_API_KEY=<your-api-key>
```

从包中导入 `createAzure` 创建提供商实例：

```ts
import { createAzure } from '@quail-ai/azure-ai-provider';

const azure = createAzure({
  endpoint: process.env.AZURE_API_ENDPOINT,
  apiKey: process.env.AZURE_API_KEY,
});
```

## 基本用法

使用 Azure 自定义提供商生成文本：

```ts
import { generateText } from 'ai';

const { text } = await generateText({
  model: azure('your-deployment-name'),
  prompt: 'Write a story about a robot.',
});
```
