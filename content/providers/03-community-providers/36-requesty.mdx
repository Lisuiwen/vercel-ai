# Requesty

[Requesty](https://requesty.ai/) 是统一 LLM 网关，可访问来自 OpenAI、Anthropic、Google、Mistral、AWS 等厂商的 300 多种大语言模型。面向 AI SDK 的 Requesty 提供商可无缝接入这些模型，并具有企业级优势：

- **通用模型访问**：一把 API 密钥访问多家提供商的 300+ 模型
- **99.99% 正常运行时间 SLA**：企业级基础设施，智能故障转移与负载均衡
- **成本优化**：按量计费，结合智能路由与提示缓存，最高可降本约 80%
- **高级安全**：提示注入检测、端到端加密与 GDPR 合规
- **实时可观测性**：内置监控、追踪与分析
- **智能路由**：自动故障转移与基于性能的路由
- **推理支持**：可配置推理力度的进阶推理能力

更多说明见 [Requesty 文档](https://docs.requesty.ai)。

## API 密钥配置

出于安全考虑，应将 API 密钥设为名为 `REQUESTY_API_KEY` 的环境变量（名称须完全一致）：

```bash
# Linux/Mac
export REQUESTY_API_KEY=your_api_key_here

# Windows Command Prompt
set REQUESTY_API_KEY=your_api_key_here

# Windows PowerShell
$env:REQUESTY_API_KEY="your_api_key_here"
```

可在 [Requesty 控制台](https://app.requesty.ai/api-keys) 获取 API 密钥。

## 提供商实例

可从 `@requesty/ai-sdk` 导入默认提供商实例 `requesty`：

```typescript
import { requesty } from '@requesty/ai-sdk';
```

也可使用 `createRequesty` 创建自定义实例：

```typescript
import { createRequesty } from '@requesty/ai-sdk';

const customRequesty = createRequesty({
  apiKey: 'YOUR_REQUESTY_API_KEY',
});
```

## 语言模型

Requesty 以统一接口支持聊天模型与补全模型：

```typescript
// Using the default provider instance
const model = requesty('openai/gpt-4o');

// Using a custom provider instance
const customModel = customRequesty('anthropic/claude-3.5-sonnet');
```

完整模型列表见 [Requesty 模型文档](https://requesty.ai/models)。

## 示例

以下示例展示如何将 Requesty 与 AI SDK 配合使用：

### `generateText`

```javascript
import { requesty } from '@requesty/ai-sdk';
import { generateText } from 'ai';

const { text } = await generateText({
  model: requesty('openai/gpt-4o'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});

console.log(text);
```

### `streamText`

```javascript
import { requesty } from '@requesty/ai-sdk';
import { streamText } from 'ai';

const result = streamText({
  model: requesty('anthropic/claude-3.5-sonnet'),
  prompt: 'Write a short story about AI.',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

### 结构化输出

```javascript
import { requesty } from '@requesty/ai-sdk';
import { generateText, Output } from 'ai';
import { z } from 'zod';

const { output } = await generateText({
  model: requesty('openai/gpt-4o'),
  output: Output.object({
    schema: z.object({
      recipe: z.object({
        name: z.string(),
        ingredients: z.array(z.string()),
        steps: z.array(z.string()),
      }),
    }),
  }),
  prompt: 'Generate a recipe for chocolate chip cookies.',
});

console.log(output.recipe);
```

## 高级功能

### 推理支持

Requesty 为支持的模型提供可配置推理力度的进阶推理：

```javascript
import { createRequesty } from '@requesty/ai-sdk';
import { generateText } from 'ai';

const requesty = createRequesty({ apiKey: process.env.REQUESTY_API_KEY });

// Using reasoning effort
const { text, reasoning } = await generateText({
  model: requesty('openai/o3-mini', {
    reasoningEffort: 'medium',
  }),
  prompt: 'Solve this complex problem step by step...',
});

console.log('Response:', text);
console.log('Reasoning:', reasoning);
```

#### 推理力度取值

- `'low'` — 最低推理强度
- `'medium'` — 中等推理强度
- `'high'` — 高推理强度
- `'max'` — 最高推理强度（Requesty 特有）
- 预算字符串（如 `"10000"`）— 为推理指定 token 预算

#### 支持的推理模型

- **OpenAI**：`openai/o3-mini`、`openai/o3`
- **Anthropic**：`anthropic/claude-sonnet-4-0` 及其他 Claude 推理模型
- **Deepseek**：全部 Deepseek 推理模型（自动推理）

### 自定义配置

使用自定义设置配置 Requesty：

```javascript
import { createRequesty } from '@requesty/ai-sdk';

const requesty = createRequesty({
  apiKey: process.env.REQUESTY_API_KEY,
  baseURL: 'https://router.requesty.ai/v1',
  headers: {
    'Custom-Header': 'custom-value',
  },
  extraBody: {
    custom_field: 'value',
  },
});
```

### 传递额外请求体参数

有三种方式向 Requesty 传递额外 body 参数：

#### 1. 通过提供商选项

```javascript
await streamText({
  model: requesty('anthropic/claude-3.5-sonnet'),
  messages: [{ role: 'user', content: 'Hello' }],
  providerOptions: {
    requesty: {
      custom_field: 'value',
      reasoning_effort: 'high',
    },
  },
});
```

#### 2. 通过模型设置

```javascript
const model = requesty('anthropic/claude-3.5-sonnet', {
  extraBody: {
    custom_field: 'value',
  },
});
```

#### 3. 通过提供商工厂

```javascript
const requesty = createRequesty({
  apiKey: process.env.REQUESTY_API_KEY,
  extraBody: {
    custom_field: 'value',
  },
});
```
