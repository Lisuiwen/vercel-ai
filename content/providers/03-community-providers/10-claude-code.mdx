# Claude Code 提供商

[ai-sdk-provider-claude-code](https://github.com/ben-vargas/ai-sdk-provider-claude-code) 允许通过官方 [Claude Agent SDK](https://www.npmjs.com/package/@anthropic-ai/claude-agent-sdk) 访问 Claude 模型；适合已通过 Claude Pro / Max 订阅、希望免管 API Key 的场景。

## 版本兼容

| Provider Version | AI SDK Version | NPM Tag     | Status      |
| ---------------- | -------------- | ----------- | ----------- |
| 3.x              | v6             | `latest`    | Stable      |
| 2.x              | v5             | `ai-sdk-v5` | Maintenance |
| 0.x              | v4             | `ai-sdk-v4` | Legacy      |

```bash
# AI SDK v6 (default)
npm install ai-sdk-provider-claude-code ai

# AI SDK v5
npm install ai-sdk-provider-claude-code@ai-sdk-v5 ai@^5.0.0

# AI SDK v4
npm install ai-sdk-provider-claude-code@ai-sdk-v4 ai@^4.0.0
```

## 提供商实例

从 `ai-sdk-provider-claude-code` 导入默认实例 `claudeCode`：

```ts
import { claudeCode } from 'ai-sdk-provider-claude-code';
```

如需自定义，可导入 `createClaudeCode`：

```ts
import { createClaudeCode } from 'ai-sdk-provider-claude-code';

const claudeCode = createClaudeCode({
  allowedTools: ['Read', 'Write', 'Edit'],
  disallowedTools: ['Bash'],
  mcpServers: {
    'my-server': {
      command: 'node',
      args: ['server.js'],
    },
  },
  permissionMode: 'default',
});
```

可选设置：

- **allowedTools** _string[]_：允许的工具列表；指定后仅这些工具可用。
- **disallowedTools** _string[]_：禁用工具列表（例如 `['Bash(rm:*)']`）。
- **mcpServers** _Record&lt;string, McpServerConfig&gt;_：MCP 服务器配置。
- **permissionMode** _'default' | 'acceptEdits' | 'bypassPermissions' | 'plan'_：工具权限处理策略。
- **maxTurns** _number_：最大对话轮数。
- **cwd** _string_：CLI 操作工作目录。
- **verbose** _boolean_：是否输出调试日志。

## 语言模型

通过提供商实例创建经由 Claude Agent SDK 调用的模型：

```ts
const model = claudeCode('sonnet');
```

支持的模型简写：

- **opus**：Claude Opus（能力最强）
- **sonnet**：Claude Sonnet（均衡）
- **haiku**：Claude Haiku（最快）

也可直接使用完整模型标识：

```ts
const model = claudeCode('claude-opus-4-5');
const model = claudeCode('claude-sonnet-4-5-20250514');
```

### 示例

```ts
import { claudeCode } from 'ai-sdk-provider-claude-code';
import { generateText } from 'ai';

const { text } = await generateText({
  model: claudeCode('sonnet'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

### 模型能力

| Model    | Image Input         | Object Generation   | Tool Usage          | Tool Streaming      |
| -------- | ------------------- | ------------------- | ------------------- | ------------------- |
| `opus`   | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |
| `sonnet` | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |
| `haiku`  | <Check size={18} /> | <Check size={18} /> | <Cross size={18} /> | <Cross size={18} /> |

<Note>
  Tool Usage 与 Tool Streaming 显示为 ❌，因为本提供商不支持向 `generateText` /
  `streamText` 传入 AI SDK 自定义工具（Zod schema）。它使用 Claude 内置工具（Bash、Edit、Read、Write
  等）与可自主执行的 MCP 服务器。对象生成使用原生结构化输出并保证符合 schema。图像输入需流式模式，且仅支持 base64 / data URL。
</Note>

## 认证

通过 Claude Code CLI 使用你已有的 Claude Pro 或 Max 订阅：

```bash
claude login
```

将在浏览器中完成登录；认证后提供商会自动使用你的订阅。

## 要求

- Node.js 18 或更高
- 已安装 Claude Code CLI（`npm install -g @anthropic-ai/claude-code`）
- Claude Pro 或 Max 订阅

更多说明见 [provider documentation](https://github.com/ben-vargas/ai-sdk-provider-claude-code)。
