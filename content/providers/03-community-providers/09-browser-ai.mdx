# 浏览器端 AI 模型提供商

[jakobhoeg/browser-ai](https://github.com/jakobhoeg/browser-ai) 是社区提供商，作为在浏览器中使用客户端 AI 模型的 AI SDK 基础提供商。
当前提供两类能力：其一，通过 JavaScript [Prompt API](https://github.com/webmachinelearning/prompt-api) 调用 Chrome / Edge 内置 AI 模型；其二，借助 [🤗 Transformers.js](https://github.com/huggingface/transformers.js) 与 [WebLLM](https://github.com/mlc-ai/web-llm) 使用开源浏览器内模型。

<Note>我们同时支持 AI SDK v5 与 v6。</Note>

## 提供商实例

### `@browser-ai/core`

从 `@browser-ai/core` 导入默认实例 `browserAI`：

```ts
import { browserAI } from '@browser-ai/core';

const model = browserAI();
```

可选设置：

- **temperature** _number_

  控制输出的随机度。多数模型下 `0` 接近确定性，数值越高随机性越大。

- **topK** _number_

  通过限制候选下一 token 控制多样性与连贯性。

### `@browser-ai/web-llm`

从 `@browser-ai/web-llm` 导入默认实例 `webLLM`：

```ts
import { webLLM } from '@browser-ai/web-llm';

const model = webLLM();
```

### `@browser-ai/transformers-js`

从 `@browser-ai/transformers-js` 导入默认实例 `transformersJS`：

```ts
import { transformersJS } from '@browser-ai/transformers-js';

const model = transformersJS();
```

## 语言模型

### `@browser-ai/core`

在支持 Prompt API 的浏览器中，由浏览器负责模型编排，提供商可直接工作。
例如在 Edge 上会使用 [Phi4-mini](https://learn.microsoft.com/en-us/microsoft-edge/web-platform/prompt-api#the-phi-4-mini-model)，Chrome 使用 [Gemini Nano](https://developer.chrome.com/docs/ai/prompt-api#model_download)。

### `@browser-ai/web-llm`

可使用 Llama3、Qwen3 等众多开源模型。完整列表参见 [WebLLM 文档](https://github.com/mlc-ai/web-llm)。

### `@browser-ai/transformers-js`

可使用 Hugging Face 上与 Transformers.js 兼容的大量开源模型。

### 示例

#### `@browser-ai/core`

```ts
import { streamText } from 'ai';
import { browserAI } from '@browser-ai/core';

const result = streamText({
  model: browserAI(), // will default to the specific browser model
  prompt: 'Hello, how are you',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

#### `@browser-ai/web-llm`

```ts
import { streamText } from 'ai';
import { webLLM } from '@browser-ai/web-llm';

const result = streamText({
  model: webLLM('Qwen3-0.6B-q0f16-MLC'),
  prompt: 'Hello, how are you',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

#### `@browser-ai/transformers-js`

```ts
import { streamText } from 'ai';
import { transformersJS } from '@browser-ai/transformers-js';

const result = streamText({
  model: transformersJS('HuggingFaceTB/SmolLM2-360M-Instruct'),
  prompt: 'Hello, how are you',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

更多示例与 API 参见 [documentation](https://www.browser-ai.dev/).
