# OLLM

[OLLM](https://ollm.com/) 是企业级路由器，聚合高安全性、零知识 LLM 提供商，并提供统一 API 网关访问 AI 模型，在各层保证军工级加密。面向 AI SDK 的 OLLM 提供商可无缝接入这些模型，并具有独特优势：

- **可验证隐私**：所有模型在机密计算环境中运行，安全性更高
- **通用模型访问**：一把密钥访问多家提供商的模型
- **机密计算**：全部模型在 TEE（可信执行环境）下进行硬件级加密
- **军工级安全**：栈各层端到端加密
- **简单集成**：跨模型 OpenAI 兼容 API

更多说明见 [OLLM 网站](https://ollm.com)。

## 提供商实例

使用 `createOLLM` 创建 OLLM 提供商实例：

```typescript
import { createOLLM } from '@ofoundation/ollm';

const ollm = createOLLM({
  apiKey: 'YOUR_OLLM_API_KEY',
});
```

可在 [OLLM 控制台](https://console.ollm.com/dashboard/api-keys) 获取 API 密钥。

## 语言模型

默认情况下，所有 OLLM 模型均在机密计算环境中运行。聊天模型使用 `ollm.chatModel()`：

```typescript
// Confidential computing chat models
const confidentialModel = ollm.chatModel('near/GLM-4.7');
```

完整模型列表见 [OLLM Models](https://ollm.com/models)。

## 示例

以下示例展示如何将 OLLM 与 AI SDK 配合使用：

### `generateText`

```typescript
import { createOLLM } from '@ofoundation/ollm';
import { generateText } from 'ai';

const ollm = createOLLM({
  apiKey: 'YOUR_OLLM_API_KEY',
});

const { text } = await generateText({
  model: ollm.chatModel('near/GLM-4.6'),
  prompt: 'What is OLLM?',
});

console.log(text);
```

### `streamText`

```typescript
import { createOLLM } from '@ofoundation/ollm';
import { streamText } from 'ai';

const ollm = createOLLM({
  apiKey: 'YOUR_OLLM_API_KEY',
});

const result = streamText({
  model: ollm.chatModel('near/GLM-4.6'),
  prompt: 'Write a short story about secure AI.',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

### 使用系统消息

```typescript
import { createOLLM } from '@ofoundation/ollm';
import { generateText } from 'ai';

const ollm = createOLLM({
  apiKey: 'YOUR_OLLM_API_KEY',
});

const { text } = await generateText({
  model: ollm.chatModel('near/GLM-4.6'),
  system: 'You are a helpful assistant that responds concisely.',
  prompt: 'What is TypeScript in one sentence?',
});

console.log(text);
```

## 高级功能

OLLM 提供多项面向企业级安全的高级能力：

1. **零数据留存（ZDR）**：提供商不会存储或记录你的提示与补全。

2. **机密计算**：基于 TEE 的硬件级加密，处理过程中仍保护数据。

3. **可验证隐私**：密码学证明数据在安全环境中处理。

4. **模型灵活**：无需改代码或管理多把密钥即可切换数百种模型。

5. **成本管理**：在控制台实时按模型查看用量与费用。

6. **企业支持**：面向高用量用户提供自定义 SLA 与专属支持。

7. **工具集成**：可与常用 AI 开发工具配合使用，包括：
   - Cursor
   - Windsurf
   - VS Code
   - Cline
   - Roo Code
   - Replit

更多信息与高级配置见 [OLLM 文档](https://ollm.com/docs)。
