# AI SDK 非官方社区提供商 - Inflection AI

<Note type="warning">
  该社区提供商尚未兼容 AI SDK 5。请等待提供商更新，或改用 [兼容 AI SDK 5 的提供商](/providers/ai-sdk-providers)。
</Note>

面向 [AI SDK](/docs) 的 **[非官方 Inflection AI 提供商](https://www.npmjs.com/package/inflection-ai-sdk-provider)** 为 [Inflection AI API](https://developers.inflection.ai/) 提供语言模型支持。

## 安装

位于 npm 的 [`inflection-ai-sdk-provider`](https://www.npmjs.com/package/inflection-ai-sdk-provider) 模块：

```bash
npm i inflection-ai-sdk-provider
```

## 提供商实例

从 `inflection-ai-sdk-provider` 导入默认实例 `inflection`：

```ts
import { inflection } from 'inflection-ai-sdk-provider';
```

## 示例

```ts
import { inflection } from 'inflection-ai-sdk-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: inflection('inflection_3_with_tools'),
  prompt: 'how can I make quick chicken pho?',
});
```

## 模型

支持的模型：

- `inflection_3_pi` — “为我们 Pi 体验提供动力的模型，包含背景叙事、情商、生产力与安全。在客户服务聊天机器人等场景中表现突出。”
- `inflection_3_productivity` — “针对遵循指令优化的模型，更适合需要 JSON 输出或严格遵照给定规则的任务。”
- `inflection_3_with_tools` — 该模型看似处于预览阶段，README 撰写时（1.0.0）尚无官方描述。

| Model                       | Text Generation | Streaming | Image Input | Object Generation | Tool Usage | Tool Streaming |
| --------------------------- | --------------- | --------- | ----------- | ----------------- | ---------- | -------------- |
| `inflection_3_pi`           | ✓               | ✓         | ✗           | ✗                 | ✗          | ✗              |
| `inflection_3_productivity` | ✓               | ✓         | ✗           | ✗                 | ✗          | ✗              |
| `inflection_3_with_tools`   | ✓               | ✓         | ✗           | ✗                 | ✗          | ✗              |

目前 API 除文本生成与流式文本外，其他能力支持有限。若后续有变，上表将更新，本非官方提供商也会跟进。
