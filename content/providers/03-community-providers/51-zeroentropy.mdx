# ZeroEntropy 提供商

[zeroentropy-ai/zeroentropy-ai-provider](https://github.com/zeroentropy-ai/zeroentropy-ai-provider) 是社区提供商，通过 [ZeroEntropy](https://zeroentropy.dev) 为 AI SDK 提供文本嵌入与重排序支持。

## 安装与配置

ZeroEntropy 提供商位于 `zeroentropy-ai-provider` 模块。可使用以下命令安装：

## 提供商实例

可从 `zeroentropy-ai-provider` 导入默认提供商实例 `zeroentropy`：

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';
```

如需自定义配置，可导入 `createZeroEntropy` 并创建实例：

```ts
import { createZeroEntropy } from 'zeroentropy-ai-provider';

const zeroentropy = createZeroEntropy({
  apiKey: process.env.ZEROENTROPY_API_KEY ?? '',
});
```

可使用以下可选设置自定义 ZeroEntropy 提供商实例：

- **baseURL** _string_

  ZeroEntropy API 的基础 URL。默认前缀为 `https://api.zeroentropy.dev/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。默认读取 `ZEROENTROPY_API_KEY` 环境变量。在 [ZeroEntropy 控制台](https://dashboard.zeroentropy.dev) 获取密钥。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。默认为全局 `fetch`。可用作中间件拦截请求，或为测试等场景提供自定义 fetch。

## 文本嵌入模型

使用 `.textEmbeddingModel()` 工厂方法可创建调用 [ZeroEntropy embeddings API](https://docs.zeroentropy.dev) 的模型。

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';

const embeddingModel = zeroentropy.textEmbeddingModel('zembed-1');
```

可使用 ZeroEntropy 嵌入模型配合 `embed` 或 `embedMany` 生成向量：

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';
import { embed, embedMany } from 'ai';

// Single embedding
const { embedding } = await embed({
  model: zeroentropy.textEmbeddingModel('zembed-1'),
  value: 'sunny day at the beach',
});

// Batch embeddings
const { embeddings } = await embedMany({
  model: zeroentropy.textEmbeddingModel('zembed-1'),
  values: ['first document', 'second document'],
});
```

ZeroEntropy 嵌入模型支持通过 `providerOptions.zeroentropy` 传入额外选项：

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';
import { embed } from 'ai';

const { embedding } = await embed({
  model: zeroentropy.textEmbeddingModel('zembed-1'),
  value: 'sunny day at the beach',
  providerOptions: {
    zeroentropy: {
      inputType: 'document',
      dimensions: 1280,
    },
  },
});
```

可用提供商选项：

- **inputType** _'query' | 'document'_

  输入是检索查询还是待索引文档。检索查询使用 `'query'`，待索引内容使用 `'document'`。默认为 `'document'`。

- **dimensions** _2560 | 1280 | 640 | 320 | 160 | 80 | 40_

  输出向量维度。默认为 `2560`。支持 Matryoshka 表示以灵活降维。

- **latency** _'fast' | 'slow'_

  `'fast'` 侧重亚秒级延迟；`'slow'` 侧重更高吞吐上限。

### 模型能力

| 模型       | 默认维度 | 上下文长度 | 说明                                       |
| ---------- | -------- | ---------- | ------------------------------------------ |
| `zembed-1` | 2560     | 8192       | 多语种嵌入模型，支持 Matryoshka 维度       |

## 重排序模型

使用 `.rerankingModel()` 工厂方法可创建调用 [ZeroEntropy reranking API](https://docs.zeroentropy.dev) 的模型。

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';
import { rerank } from 'ai';

const result = await rerank({
  model: zeroentropy.rerankingModel('zerank-2'),
  query: 'talk about rain',
  documents: ['sunny day at the beach', 'rainy day in the city'],
  topN: 2,
});
```

ZeroEntropy 重排序模型支持通过 `providerOptions.zeroentropy` 传入额外选项：

```ts
import { zeroentropy } from 'zeroentropy-ai-provider';
import { rerank } from 'ai';

const result = await rerank({
  model: zeroentropy.rerankingModel('zerank-2'),
  query: 'talk about rain',
  documents: ['sunny day at the beach', 'rainy day in the city'],
  topN: 2,
  providerOptions: {
    zeroentropy: {
      latency: 'fast',
    },
  },
});
```

可用提供商选项：

- **latency** _'fast' | 'slow'_

  `'fast'` 侧重亚秒级延迟；`'slow'` 侧重更高吞吐上限。

### 模型能力

| 模型             | 说明                                           |
| ---------------- | ---------------------------------------------- |
| `zerank-2`       | 旗舰级先进交叉编码器重排序器                   |
| `zerank-2-nano`  | 紧凑版 `zerank-2`，更低延迟场景                |
| `zerank-1`       | 上一代重排序器                                 |
| `zerank-1-small` | 轻量重排序器，更低延迟场景                     |

<Note>
  完整模型与选项列表请参阅 [ZeroEntropy 文档](https://docs.zeroentropy.dev)。
</Note>
