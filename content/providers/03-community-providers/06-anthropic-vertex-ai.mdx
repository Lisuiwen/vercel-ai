# AnthropicVertex 提供商

<Note>
  Google Vertex 上的 Anthropic 也可通过 [AI SDK Google Vertex 提供商](/providers/ai-sdk-providers/google-vertex) 使用。
</Note>

<Note type="warning">
  该社区提供商尚未兼容 AI SDK 5。请等待提供商更新，或改用 [兼容 AI SDK 5 的提供商](/providers/ai-sdk-providers)。
</Note>

[nalaso/anthropic-vertex-ai](https://github.com/nalaso/anthropic-vertex-ai) 是社区提供商，经由 Vertex AI 使用 Anthropic 模型，为 AI SDK 提供语言模型支持。

AnthropicVertex 提供商位于 `anthropic-vertex-ai` 模块。可使用：

## 提供商实例

可从 `anthropic-vertex-ai` 导入默认实例 `anthropicVertex`：

```ts
import { anthropicVertex } from 'anthropic-vertex-ai';
```

如需自定义，可导入 `createAnthropicVertex` 并使用你的配置：

```ts
import { createAnthropicVertex } from 'anthropic-vertex-ai';

const anthropicVertex = createAnthropicVertex({
  region: 'us-central1',
  projectId: 'your-project-id',
  // other options
});
```

可选配置项：

- **region** _string_

  Google Vertex 区域。默认读取环境变量 `GOOGLE_VERTEX_REGION`。

- **projectId** _string_

  Google Vertex 项目 ID。默认读取环境变量 `GOOGLE_VERTEX_PROJECT_ID`。

- **googleAuth** _GoogleAuth_

  可选。由 google-auth-library 提供的认证选项。

- **baseURL** _string_

  API 调用的 URL 前缀，可用于代理等。默认为 `https://{region}-aiplatform.googleapis.com/v1`。

- **headers** _Record&lt;string,string&gt;_

  请求中附加的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 fetch，可用于拦截请求或在测试中使用替代实现。

## 语言模型

使用提供商实例创建通过 Vertex AI 调用 Anthropic API 的模型。第一个参数为模型 ID，例如 `claude-3-sonnet@20240229`：

```ts
const model = anthropicVertex('claude-3-sonnet@20240229');
```

### 示例：生成文本

可将 AnthropicVertex 语言模型与 `generateText` 配合使用：

```ts
import { anthropicVertex } from 'anthropic-vertex-ai';
import { generateText } from 'ai';

const { text } = await generateText({
  model: anthropicVertex('claude-3-sonnet@20240229'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

AnthropicVertex 语言模型也可用于 `streamText`，并支持借助 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化数据生成（参见 [AI SDK Core](/docs/ai-sdk-core)）。

## 环境变量

使用 AnthropicVertex 需配置：

- `GOOGLE_VERTEX_REGION`：Google Vertex 区域（如 `us-central1`）
- `GOOGLE_VERTEX_PROJECT_ID`：Google Cloud 项目 ID

请在环境中或项目根 `.env` 中设置上述变量。

## 认证

AnthropicVertex 使用 Google Cloud 认证。请正确配置 GCP 凭证，可使用服务账号密钥文件或默认应用凭证。

更多说明见 [Google Cloud Authentication guide](https://cloud.google.com/docs/authentication)。
