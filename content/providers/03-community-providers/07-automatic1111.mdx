# Automatic1111

[AUTOMATIC1111](https://github.com/AUTOMATIC1111/stable-diffusion-webui) 是常用的 Stable Diffusion Web 界面，提供丰富的图像生成功能。[Automatic1111 提供商](https://github.com/Ponesicek/automatic1111-provider) 可与本地部署的 AUTOMATIC1111 实例集成，并具有以下优势：

- **本地可控**：在本机 Stable Diffusion 模型上完整控制图像生成
- **无 API 费用**：不产生按请求计费
- **模型灵活**：可使用任意 Stable Diffusion checkpoint
- **隐私**：生成全过程在本地硬件完成
- **社区模型**：可使用 Civitai、HuggingFace 上的海量社区模型

更多能力说明见 [AUTOMATIC1111 文档](https://github.com/AUTOMATIC1111/stable-diffusion-webui)。

## 安装

需以 `--api` 标志运行 AUTOMATIC1111，例如：

```bash
# Windows
./webui.bat --api

# Linux/Mac
./webui.sh --api
```

Automatic1111 提供商位于 `automatic1111-provider` 模块：

```bash
# pnpm
pnpm add automatic1111-provider

# npm
npm install automatic1111-provider

# yarn
npm install automatic1111-provider
```

## 提供商实例

使用 `createAutomatic1111` 创建 Automatic1111 提供商实例：

```typescript
import { createAutomatic1111 } from 'automatic1111-provider';

const automatic1111 = createAutomatic1111({
  baseURL: 'http://127.0.0.1:7860', // Your AUTOMATIC1111 instance
});
```

## 图像模型

通过 `image()` 方法生成图像：

```typescript
// Basic image generation
const imageModel = automatic1111.image('v1-5-pruned-emaonly');

// With custom model
const sdxlModel = automatic1111.image('sd_xl_base_1.0');
```

## 示例

### 基础图像生成

```typescript
import { automatic1111 } from 'automatic1111-provider';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: automatic1111.image('v1-5-pruned-emaonly'),
  prompt: 'A beautiful sunset over mountains',
  size: '512x512',
});
```

### 高级配置

```typescript
const { images } = await generateImage({
  model: automatic1111.image('realistic-vision-v4'),
  prompt: 'Portrait of a wise old wizard with a long beard',
  n: 2,
  seed: 12345,
  providerOptions: {
    automatic1111: {
      negative_prompt: 'blurry, ugly, deformed, low quality',
      steps: 40,
      cfg_scale: 8.5,
      sampler_name: 'DPM++ SDE Karras',
      styles: ['photorealistic', 'detailed'],
      check_model_exists: true,
    },
  },
});
```

## 提供商选项

Automatic1111 提供商可用以下选项自定义图像生成：

### 可用选项

| Option               | Type       | Default     | 说明                                      |
| -------------------- | ---------- | ----------- | ---------------------------------------- |
| `negative_prompt`    | `string`   | `undefined` | 不希望在图中出现的内容描述                |
| `steps`              | `number`   | `20`        | 采样步数                                  |
| `cfg_scale`          | `number`   | `7`         | CFG（Classifier-Free Guidance）强度       |
| `sampler_name`       | `string`   | `"Euler a"` | 采样算法名称                              |
| `denoising_strength` | `number`   | `undefined` | img2img 去噪强度（0.0–1.0）              |
| `styles`             | `string[]` | `undefined` | 预设风格列表                             |
| `check_model_exists` | `boolean`  | `false`     | 生成前校验模型是否存在                    |

## 模型管理

提供商会从你的 AUTOMATIC1111 实例自动检测可用模型。使用方式为：

1. 将 `.safetensors` 或 `.ckpt` 模型文件放入 `models/Stable-diffusion/` 文件夹
2. 重启 AUTOMATIC1111，或在 Web 界面刷新模型列表
3. 在提供商中使用确切的模型文件名（不含扩展名）
