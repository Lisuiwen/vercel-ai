# SambaNova 提供商

[sambanova-ai-provider](https://github.com/sambanova/sambanova-ai-provider) 为 SambaNova API 提供语言模型支持。

API 密钥可在 [SambaNova Cloud 平台](https://cloud.sambanova.ai/apis) 获取。

## 安装与配置

SambaNova 提供商通过 `sambanova-ai-provider` 模块提供。可使用以下命令安装：

### 环境变量

在 `.env` 中创建 `SAMBANOVA_API_KEY` 变量。

## 提供商实例

可从 `sambanova-ai-provider` 导入默认提供商实例 `sambanova`：

```ts
import { sambanova } from 'sambanova-ai-provider';
```

如需自定义配置，可导入 `createSambaNova` 并创建实例：

```ts
import { createSambaNova } from 'sambanova-ai-provider';

const sambanova = createSambaNova({
  // Optional settings
});
```

可使用以下可选设置自定义 SambaNova 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀，例如代理。
  默认前缀为 `https://api.sambanova.ai/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。
  默认读取 `SAMBANOVA_API_KEY` 环境变量。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作中间件拦截请求，
  或为测试等场景提供自定义 fetch。

## 模型

可在提供商实例上使用 [SambaNova 模型](https://docs.sambanova.ai/cloud/docs/get-started/supported-models)。
第一个参数为模型 id，例如 `Meta-Llama-3.1-70B-Instruct`。

```ts
const model = sambanova('Meta-Llama-3.1-70B-Instruct');
```

## 已测模型与能力

该提供商支持文本生成与流式输出，并可处理图像输入。

至少已在以下功能上测试（使用 `/chat/completion` 端点）：

## 示例用法

使用 SambaNova 提供商进行文本生成的基础示例。

```ts
import { createSambaNova } from 'sambanova-ai-provider';
import { generateText } from 'ai';

const sambanova = createSambaNova({
  apiKey: 'YOUR_API_KEY',
});

const model = sambanova('Meta-Llama-3.1-70B-Instruct');

const { text } = await generateText({
  model,
  prompt: 'Hello, nice to meet you.',
});

console.log(text);
```

输出文本大致如下：

```
Hello. Nice to meet you too. Is there something I can help you with or would you like to chat?
```

## 拦截 Fetch 请求

该提供商支持 [拦截 Fetch 请求](/examples/providers/intercepting-fetch-requests)。

### 示例

```ts
import { createSambaNova } from 'sambanova-ai-provider';
import { generateText } from 'ai';

const sambanovaProvider = createSambaNova({
  apiKey: 'YOUR_API_KEY',
  fetch: async (url, options) => {
    console.log('URL', url);
    console.log('Headers', JSON.stringify(options.headers, null, 2));
    console.log(`Body ${JSON.stringify(JSON.parse(options.body), null, 2)}`);
    return await fetch(url, options);
  },
});

const model = sambanovaProvider('Meta-Llama-3.1-70B-Instruct');

const { text } = await generateText({
  model,
  prompt: 'Hello, nice to meet you.',
});
```

输出类似：

```bash
URL https://api.sambanova.ai/v1/chat/completions
Headers {
  "Content-Type": "application/json",
  "Authorization": "Bearer YOUR_API_KEY"
}
Body {
  "model": "Meta-Llama-3.1-70B-Instruct",
  "temperature": 0,
  "messages": [
    {
      "role": "user",
      "content": "Hello, nice to meet you."
    }
  ]
}
```
