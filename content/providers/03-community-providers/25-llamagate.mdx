# LlamaGate

[LlamaGate](https://llamagate.dev) 是兼容 OpenAI 的 API 网关，提供 26+ 款开源 LLM 访问，价格实惠，适合独立开发者与初创团队低成本使用 Llama、Qwen、DeepSeek、Mistral 等模型。

- **26+ 开源模型**：Llama、Mistral、DeepSeek R1、Qwen 等
- **OpenAI 兼容 API**：可替换现有 OpenAI 集成
- **有竞争力定价**：每百万 token 约 $0.02–$0.55
- **视觉模型**：Qwen VL、LLaVA 等多模态
- **推理模型**：DeepSeek R1 用于复杂问题
- **代码模型**：CodeLlama、DeepSeek Coder、Qwen Coder
- **嵌入模型**：Nomic Embed Text、Qwen 3 Embedding

更多说明见 [LlamaGate Documentation](https://llamagate.dev/docs)。

## 提供商实例

使用 `createLlamaGate` 创建实例：

```typescript
import { createLlamaGate } from '@llamagate/ai-sdk-provider';

const llamagate = createLlamaGate({
  apiKey: 'YOUR_LLAMAGATE_API_KEY',
});
```

API Key 见 [LlamaGate Dashboard](https://llamagate.dev/dashboard)。

也可使用默认实例（读取环境变量 `LLAMAGATE_API_KEY`）：

```typescript
import { llamagate } from '@llamagate/ai-sdk-provider';
```

## 语言模型

通过 `llamagate()` 或 `llamagate.chatModel()` 获取对话模型：

```typescript
// Default usage
const model = llamagate('llama-3.1-8b');

// Explicit chat model
const chatModel = llamagate.chatModel('qwen3-8b');
```

## 嵌入模型

文本嵌入使用 `llamagate.textEmbeddingModel()`：

```typescript
const embeddingModel = llamagate.textEmbeddingModel('nomic-embed-text');
```

### 可用嵌入模型

| Model ID              | 说明                | 上下文   |
| --------------------- | ------------------- | ------- |
| `nomic-embed-text`    | Nomic Embed Text    | 8K      |
| `embeddinggemma-300m` | EmbeddingGemma 300M | 2K      |
| `qwen3-embedding-8b`  | Qwen 3 Embedding 8B | 40K     |

## 示例

### `generateText`

```typescript
import { createLlamaGate } from '@llamagate/ai-sdk-provider';
import { generateText } from 'ai';

const llamagate = createLlamaGate({
  apiKey: 'YOUR_LLAMAGATE_API_KEY',
});

const { text } = await generateText({
  model: llamagate('llama-3.1-8b'),
  prompt: 'Explain quantum computing in simple terms.',
});

console.log(text);
```

### `streamText`

```typescript
import { createLlamaGate } from '@llamagate/ai-sdk-provider';
import { streamText } from 'ai';

const llamagate = createLlamaGate({
  apiKey: 'YOUR_LLAMAGATE_API_KEY',
});

const result = streamText({
  model: llamagate('qwen3-8b'),
  prompt: 'Write a short story about a robot.',
});

for await (const chunk of result) {
  console.log(chunk);
}
```

### `embed`

```typescript
import { createLlamaGate } from '@llamagate/ai-sdk-provider';
import { embed } from 'ai';

const llamagate = createLlamaGate({
  apiKey: 'YOUR_LLAMAGATE_API_KEY',
});

const { embedding } = await embed({
  model: llamagate.textEmbeddingModel('nomic-embed-text'),
  value: 'The quick brown fox jumps over the lazy dog.',
});

console.log(embedding);
```

### Vision

```typescript
import { createLlamaGate } from '@llamagate/ai-sdk-provider';
import { generateText } from 'ai';

const llamagate = createLlamaGate({
  apiKey: 'YOUR_LLAMAGATE_API_KEY',
});

const { text } = await generateText({
  model: llamagate('qwen3-vl-8b'),
  messages: [
    {
      role: 'user',
      content: [
        { type: 'text', text: 'What is in this image?' },
        {
          type: 'file',
          mediaType: 'image',
          data: new URL('https://example.com/image.jpg'),
        },
      ],
    },
  ],
});

console.log(text);
```
