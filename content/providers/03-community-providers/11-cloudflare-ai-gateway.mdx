# Cloudflare AI Gateway

Cloudflare AI Gateway 提供商用于将 Cloudflare AI Gateway 与 Vercel AI SDK 集成，通过统一接口访问多家提供商的模型，并支持失败时自动回退以提高可用性。

## 特性

- **运行时无关**：可与 Node.js、Edge Runtime 及 AI SDK 支持的其他 JS 运行时配合。
- **自动回退**：某个模型失败时自动尝试队列中的下一个，增强韧性。
- **多提供商**：支持 OpenAI、Anthropic、DeepSeek、Google AI Studio、Grok、Mistral、Perplexity AI、Replicate、Groq 等来源的模型。
- **网关集成**：利用 Cloudflare AI Gateway 的请求管理、缓存与限速能力。
- **配置简化**：支持 API Key 或 Cloudflare AI binding 的简单配置。

## 提供商实例

使用 `createAiGateway` 创建 `aigateway` 实例，可通过 API Key 或 Cloudflare AI binding 认证。

### API Key 认证

```typescript
import { createAiGateway } from 'ai-gateway-provider';

const aigateway = createAiGateway({
  accountId: 'your-cloudflare-account-id',
  gateway: 'your-gateway-name',
  apiKey: 'your-cloudflare-api-key', // Only required if your gateway has authentication enabled
  options: {
    skipCache: true, // Optional request-level settings
  },
});
```

### Cloudflare AI Binding

仅适用于 Cloudflare Workers 内。

在 `wrangler.toml` 中配置 AI binding：

```bash
[AI]
binding = "AI"
```

在 Worker 中使用 binding 创建实例：

```typescript
import { createAiGateway } from 'ai-gateway-provider';

const aigateway = createAiGateway({
  binding: env.AI.gateway('my-gateway'),
  options: {
    skipCache: true, // Optional request-level settings
  },
});
```

## 语言模型

向 `aigateway` 传入模型数组；将按顺序尝试，失败则回退到下一个。

```typescript
import { createAiGateway } from 'ai-gateway-provider';
import { createOpenAI } from '@ai-sdk/openai';
import { createAnthropic } from '@ai-sdk/anthropic';

const aigateway = createAiGateway({
  accountId: 'your-cloudflare-account-id',
  gateway: 'your-gateway-name',
  apiKey: 'your-cloudflare-api-key',
});

const openai = createOpenAI({ apiKey: 'openai-api-key' });
const anthropic = createAnthropic({ apiKey: 'anthropic-api-key' });

const model = aigateway([
  anthropic('claude-haiku-4-5'), // Primary model
  openai('gpt-4o-mini'), // Fallback model
]);
```

### 请求选项

可按请求调整 Cloudflare AI Gateway：

- `cacheKey`：自定义缓存键。
- `cacheTtl`：缓存 TTL（秒）。
- `skipCache`：跳过缓存。
- `metadata`：请求元数据。
- `collectLog`：是否收集日志。
- `eventId`：自定义事件 ID。
- `requestTimeoutMs`：请求超时（毫秒）。
- `retries`：
  - `maxAttempts`：重试次数（1–5）。
  - `retryDelayMs`：重试间隔。
  - `backoff`：退避策略（`constant`、`linear`、`exponential`）。

示例：

```typescript
const aigateway = createAiGateway({
  accountId: 'your-cloudflare-account-id',
  gateway: 'your-gateway-name',
  apiKey: 'your-cloudflare-api-key',
  options: {
    cacheTtl: 3600, // Cache for 1 hour
    metadata: { userId: 'user123' },
    retries: {
      maxAttempts: 3,
      retryDelayMs: 1000,
      backoff: 'exponential',
    },
  },
});
```

## 示例

### `generateText`

使用 Cloudflare AI Gateway 提供商进行非流式生成：

```typescript
import { createAiGateway } from 'ai-gateway-provider';
import { createOpenAI } from '@ai-sdk/openai';
import { generateText } from 'ai';

const aigateway = createAiGateway({
  accountId: 'your-cloudflare-account-id',
  gateway: 'your-gateway-name',
  apiKey: 'your-cloudflare-api-key',
});

const openai = createOpenAI({ apiKey: 'openai-api-key' });

const { text } = await generateText({
  model: aigateway([openai('gpt-4o-mini')]),
  prompt: 'Write a greeting.',
});

console.log(text); // Output: "Hello"
```

### `streamText`

流式输出：

```typescript
import { createAiGateway } from 'ai-gateway-provider';
import { createOpenAI } from '@ai-sdk/openai';
import { streamText } from 'ai';

const aigateway = createAiGateway({
  accountId: 'your-cloudflare-account-id',
  gateway: 'your-gateway-name',
  apiKey: 'your-cloudflare-api-key',
});

const openai = createOpenAI({ apiKey: 'openai-api-key' });

const result = await streamText({
  model: aigateway([openai('gpt-4o-mini')]),
  prompt: 'Write a multi-part greeting.',
});

let accumulatedText = '';
for await (const chunk of result.textStream) {
  accumulatedText += chunk;
}

console.log(accumulatedText); // Output: "Hello world!"
```

## 错误处理

提供商可能抛出：

- `AiGatewayUnauthorizedError`：启用认证时 API Key 无效或缺失。
- `AiGatewayDoesNotExist`：指定的 Cloudflare AI Gateway 不存在。
