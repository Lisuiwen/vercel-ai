# Firemoon 提供商

[Firemoon Studio](https://firemoon.studio/) 是专注高质量图像与视频生成模型的 AI 平台，提供微调与前沿模型。

Firemoon 的 AI SDK 提供商让你用一致、简洁的 API 使用这些模型：

- **图像生成**：当前以图像为主，视频模型能力处于早期支持。
- **模型多样**：单一提供商即可访问多族模型。
- **提供商选项**：需要更细控制时可传入模型专属参数。

## 提供商实例

使用 API Key 创建 Firemoon 提供商实例：

```ts
import { createFiremoon } from '@firemoon/ai-provider';

const firemoon = createFiremoon({
  apiKey: process.env.FIREMOON_API_KEY,
});
```

API Key 可在 [Firemoon Dashboard](https://firemoon.studio/keys) 获取。

## 图像生成

通过 `image()` 方法使用多种图像生成模型：

```ts
import { generateImage } from 'ai';

const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'An orangered moon with a dark background',
  size: 'square_hd',
});

console.log(result.images[0]);
```

### 可用模型

- `flux/dev` — 快速、高质量图像生成
- `flux-2-dev/edit` — 基于 Flux 2 的快速高质量编辑
- `nano-banana` — 快速高质量图像生成
- `nano-banana/edit` — Gemini 高质量编辑，亦称 Gemini 2.5 Flash Image
- `nano-banana-pro/edit` — Gemini 3 Pro 高质量编辑
- `firemoon-studio/memphis-style` — Memphis 风格图像生成

全部模型见 [Firemoon Studio Models](https://firemoon.studio/models)。

### 参数

#### 尺寸（Size）

使用 `size` 指定图像尺寸：

```ts
const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'A beautiful landscape with an orangered moon in the background',
  size: 'square_hd', // or 'landscape_16_9', etc.
});
```

或使用宽高比映射：

```ts
const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'A beautiful landscape with an orangered moon in the background',
  aspectRatio: '16:9', // maps to landscape_16_9
});
```

#### 随机种子（Seed）

指定 `seed` 以提升可复现性：

```ts
const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'A beautiful landscape with an orangered moon in the background',
  seed: 12345,
});
```

#### 自定义参数

可传入 Firemoon 模型专属附加参数：

```ts
const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'A beautiful landscape with an orangered moon in the background',
  providerOptions: {
    firemoon: {
      // custom parameters
    },
  },
});
```

#### 微调模型

将模型 ID 传给 `image()` 以使用微调模型：

```ts
const result = await generateImage({
  model: firemoon.image('firemoon-studio/memphis-style'),
  prompt: 'a man smiling at the camera',
  size: 'landscape_16_9',
  providerOptions: {
    firemoon: {
      lora_scale: 0.6,
      num_images: 1,
      image_size: 'landscape_4_3',
      output_format: 'jpeg',
      guidance_scale: 3.5,
      num_inference_steps: 28,
      enable_safety_checker: true,
    },
  },
});
```

## 快速开始示例

### `generateImage`

```ts
import { createFiremoon } from '@firemoon/ai-provider';
import { generateImage } from 'ai';

const firemoon = createFiremoon({
  apiKey: process.env.FIREMOON_API_KEY,
});

const result = await generateImage({
  model: firemoon.image('flux/dev'),
  prompt: 'A beautiful landscape with an orangered moon in the background',
  aspectRatio: '16:9',
});

console.log(result.images[0]);
```

## 高级特性

1. **宽高比映射**：用 `aspectRatio` 使用预设，而无需硬编码尺寸。
2. **可复现输出**：提供 `seed` 使结果更稳定。
3. **模型专属选项**：通过 `providerOptions.firemoon` 透传 Firemoon 参数。

## 错误处理

API 相关错误会抛出 `APICallError`：

```ts
import { APICallError } from 'ai';

try {
  const result = await generateImage({
    model: firemoon.image('flux/dev'),
    prompt: 'A beautiful landscape with an orangered moon in the background',
  });
} catch (error) {
  if (error instanceof APICallError) {
    console.error('API Error:', error.message);
    console.error('Status:', error.statusCode);
  }
}
```
