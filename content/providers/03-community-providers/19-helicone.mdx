# Helicone

[Helicone AI Gateway](https://helicone.ai/) 可让你访问数百款 AI 模型，并通过可观测性平台直接集成追踪与监控。

- **统一模型访问**：一把 API Key 访问 Anthropic、Google、Meta 等多家主流提供商的数百款模型。
- **智能选路**：优先命中更便宜的路由，并在提供商故障或限流时回退。
- **简化追踪**：默认通过 Helicone 可观测性监控 LLM 表现与排障，含 OpenTelemetry 的日志、指标与链路支持。
- **性能与成本**：缓存响应以降低成本与延迟。
- **提示词管理**：在 Helicone 内管理提示词版本与 Playground，减少对工程师改动的依赖。

更多能力见 [Helicone Documentation](https://helicone.ai/docs)。

## 快速开始

使用 `createHelicone` 创建提供商实例，然后查询任意模型。

```typescript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await generateText({
  model: helicone('claude-4.5-haiku'),
  prompt: 'Write a haiku about artificial intelligence',
});

console.log(result.text);
```

API Key 可在 [Helicone Dashboard](https://us.helicone.ai/settings/api-keys) 获取。

## 示例

### `generateText`

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const { text } = await generateText({
  model: helicone('gemini-2.5-flash-lite'),
  prompt: 'What is Helicone?',
});

console.log(text);
```

### `streamText`

```javascript
const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await streamText({
  model: helicone('deepseek-v3.1-terminus'),
  prompt: 'Write a short story about a robot learning to paint',
  maxOutputTokens: 300,
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}

console.log('\n\nStream completed!');
```

## 高级能力

Helicone 提供多项高级能力以增强 AI 应用：

1. **模型灵活**：在数百款模型间切换而无需大幅改代码或管理多把 Key。
2. **成本管控**：通过 Helicone LLM 可观测面板实时掌握各模型成本。
3. **可观测性**：在面板中查看请求分析与日志。
4. **提示词管理**：在控制台管理提示词与版本。
5. **缓存**：缓存响应以降本提速。
6. **持续更新**：新模型与新能力可用时自动获得访问。

更多信息与高级配置见 [Helicone Documentation](https://docs.helicone.ai)。
