# AI/ML API 提供商

[AI/ML API](https://aimlapi.com/?utm_source=aimlapi-vercel-ai&utm_medium=github&utm_campaign=integration) 提供商通过兼容 OpenAI 的 API，提供超过 300 款 AI 模型的访问。

### API Key

请设置环境变量 `AIMLAPI_API_KEY`：

```bash
export AIMLAPI_API_KEY="sk-..."
```

## 提供商实例

可导入默认实例 `aimlapi`：

```ts
import { aimlapi } from '@ai-ml.api/aimlapi-vercel-ai';
```

## 语言模型

使用 `aimlapi` 创建文本生成模型并与 `generateText` 配合使用：

```ts
import { aimlapi } from '@ai-ml.api/aimlapi-vercel-ai';
import { generateText } from 'ai';

const { text } = await generateText({
  model: aimlapi('gpt-4o'),
  system: 'You are a friendly assistant!',
  prompt: 'Why is the sky blue?',
});
```

## 图像生成

对图像模型调用 `doGenerate` 生成图片：

```ts
import { aimlapi } from '@ai-ml.api/aimlapi-vercel-ai';

const model = aimlapi.imageModel('flux-pro');

const res = await model.doGenerate({
  prompt: 'a red balloon floating over snowy mountains, cinematic',
  n: 1,
  aspectRatio: '16:9',
  seed: 42,
  size: '1024x768',
  providerOptions: {},
});

console.log(`✅ Generated image url: ${res.images[0]}`);
```

## 向量嵌入

AI/ML API 还支持嵌入模型：

```ts
import { aimlapi } from '@ai-ml.api/aimlapi-vercel-ai';
import { embed } from 'ai';

const { embedding } = await embed({
  model: aimlapi.embeddingModel('text-embedding-3-large'),
  value: 'sunny day at the beach',
});
```

更多信息与完整模型列表见 [AI/ML API dashboard](https://aimlapi.com/app?utm_source=aimlapi-vercel-ai&utm_medium=github&utm_campaign=integration) 与 [AI/ML API documentation](https://docs.aimlapi.com/?utm_source=aimlapi-vercel-ai&utm_medium=github&utm_campaign=integration)。
