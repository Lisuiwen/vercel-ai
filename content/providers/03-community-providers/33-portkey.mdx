# Portkey 提供商

[Portkey](https://portkey.ai/?utm_source=vercel&utm_medium=docs&utm_campaign=integration) 原生集成 AI SDK，帮助应用具备生产级稳定性与可靠性。导入 Portkey 的 Vercel 包并在 Vercel AI 应用中作为提供商使用，即可启用 Portkey 的各项能力：

- 面向全部请求的端到端**可观测性**与**链路追踪**
- 跨 **250+ LLM** 的互操作
- 内置 **50+** 先进护栏（guardrails）
- 简单且语义化的**缓存**，节省成本与时间
- 条件路由、降级、负载均衡、自动重试等
- 基于用户反馈的持续改进

更多说明见 [面向 AI SDK 的 Portkey 文档](https://docs.portkey.ai/docs/integrations/libraries/vercel)

## 提供商实例

使用 `createPortkey` 创建 Portkey 提供商实例：

```typescript
import { createPortkey } from '@portkey-ai/vercel-provider';

const portkeyConfig = {
  provider: 'openai', //enter provider of choice
  api_key: 'OPENAI_API_KEY', //enter the respective provider's api key
  override_params: {
    model: 'gpt-5', //choose from 250+ LLMs
  },
};

const portkey = createPortkey({
  apiKey: 'YOUR_PORTKEY_API_KEY',
  config: portkeyConfig,
});
```

可在 [Portkey 控制台](https://app.portkey.ai) 查找 API 密钥。

## 语言模型

Portkey 支持聊天模型与补全模型。聊天模型使用 `portkey.chatModel()`，补全模型使用 `portkey.completionModel()`：

```typescript
const chatModel = portkey.chatModel('');
const completionModel = portkey.completionModel('');
```

说明：若已在 `portkeyConfig` 中定义模型，可将模型名参数传为空字符串。

## 示例

可将 Portkey 语言模型与 `generateText` 或 `streamText` 配合使用：

### `generateText`

```javascript
import { createPortkey } from '@portkey-ai/vercel-provider';
import { generateText } from 'ai';

const portkey = createPortkey({
  apiKey: 'YOUR_PORTKEY_API_KEY',
  config: portkeyConfig,
});

const { text } = await generateText({
  model: portkey.chatModel(''),
  prompt: 'What is Portkey?',
});

console.log(text);
```

### `streamText`

```javascript
import { createPortkey } from '@portkey-ai/vercel-provider';
import { streamText } from 'ai';

const portkey = createPortkey({
  apiKey: 'YOUR_PORTKEY_API_KEY',
  config: portkeyConfig,
});

const result = streamText({
  model: portkey.completionModel(''),
  prompt: 'Invent a new holiday and describe its traditions.',
});

for await (const chunk of result) {
  console.log(chunk);
}
```

说明：

- Portkey 支持在 AI SDK 中使用 `Tool`
- 配合 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化输出目前不支持。

## 高级功能

Portkey 还提供多项高级能力：

1. **互操作性**：在配置中更改提供商与模型名即可在 250+ 模型间切换。

2. **可观测性**：为所有请求提供分析与日志。

3. **可靠性**：缓存、降级、负载均衡与条件路由。

4. **护栏**：通过输入/输出检查实时约束 LLM 行为。

5. **安全与合规**：预算上限与细粒度角色权限。

详细说明与高级配置请参阅 [Portkey 文档](https://docs.portkey.ai/docs/integrations/libraries/vercel)。
