# Soniox 提供商

[Soniox provider](https://github.com/soniox/vercel-ai-sdk-provider) 将 Soniox 转写模型与 Vercel AI SDK 集成。
更多信息见 [Soniox 文档](https://soniox.com/docs)。

## 认证

在环境中设置 `SONIOX_API_KEY`，或在创建提供商时传入 `apiKey`。
在 [Soniox 控制台](https://console.soniox.com) 获取 API 密钥。

## 示例

```ts
import { soniox } from '@soniox/vercel-ai-sdk-provider';
import { experimental_transcribe as transcribe } from 'ai';

const { text } = await transcribe({
  model: soniox.transcription('stt-async-v3'),
  audio: new URL('https://soniox.com/media/examples/coffee_shop.mp3'),
});
```

## 提供商选项

使用 `createSoniox` 自定义提供商实例：

```ts
import { createSoniox } from '@soniox/vercel-ai-sdk-provider';

const soniox = createSoniox({
  apiKey: process.env.SONIOX_API_KEY,
  apiBaseUrl: 'https://api.soniox.com',
});
```

选项：

- `apiKey`：覆盖 `SONIOX_API_KEY`。
- `apiBaseUrl`：自定义 API 基础 URL。区域端点列表见 [此处](https://soniox.com/docs/stt/data-residency#regional-endpoints)。
- `headers`：额外请求头。
- `fetch`：自定义 fetch 实现。
- `pollingIntervalMs`：转写轮询间隔（毫秒）。默认 1000ms。

## 转写选项

单次请求的选项通过 `providerOptions` 传入：

```ts
const { text } = await transcribe({
  model: soniox.transcription('stt-async-v3'),
  audio,
  providerOptions: {
    soniox: {
      languageHints: ['en', 'es'],
      enableLanguageIdentification: true,
      enableSpeakerDiarization: true,
      context: {
        terms: ['Soniox', 'Vercel'],
      },
    },
  },
});
```

可用选项：

- `languageHints` — ISO 语言代码数组，用于偏置识别
- `languageHintsStrict` — 为 true 时更依赖语言提示（注意：并非所有模型均支持）
- `enableLanguageIdentification` — 自动检测口语语言
- `enableSpeakerDiarization` — 识别并分离不同说话人
- `context` — 额外上下文以提升准确度
- `clientReferenceId` — 可选客户端自定义引用 ID
- `webhookUrl` — 转写完成通知的 Webhook URL
- `webhookAuthHeaderName` — Webhook 认证头名称
- `webhookAuthHeaderValue` — Webhook 认证头值
- `translation` — 翻译配置

详见 [Soniox API 参考](https://soniox.com/docs/stt/api-reference/transcriptions/create_transcription)。

## 语言提示

Soniox 可自动检测并转写 [**60+ 种语言**](https://soniox.com/docs/stt/concepts/supported-languages)。若已知音频中可能出现的语言，可提供 `languageHints`，通过偏置模型提升准确度。

语言提示**不会限制**识别范围——仅**偏置**向指定语言，仍可在音频中存在其他语言时被检测到。

```ts
const { text } = await transcribe({
  model: soniox.transcription('stt-async-v3'),
  audio,
  providerOptions: {
    soniox: {
      languageHints: ['en', 'es'], // ISO language codes
    },
  },
});
```

更多说明见 [Soniox 语言提示文档](https://soniox.com/docs/stt/concepts/language-hints)。

## 上下文

提供自定义上下文以提升转写与翻译准确度。上下文帮助模型理解领域、识别重要术语并应用自定义词汇。

`context` 对象支持四个可选段落：

```ts
const { text } = await transcribe({
  model: soniox.transcription('stt-async-v3'),
  audio,
  providerOptions: {
    soniox: {
      context: {
        // Structured key-value information (domain, topic, intent, etc.)
        general: [
          { key: 'domain', value: 'Healthcare' },
          { key: 'topic', value: 'Diabetes management consultation' },
          { key: 'doctor', value: 'Dr. Martha Smith' },
        ],
        // Longer free-form background text or related documents
        text: 'The patient has a history of...',
        // Domain-specific or uncommon words
        terms: ['Celebrex', 'Zyrtec', 'Xanax'],
        // Custom translations for ambiguous terms
        translationTerms: [
          { source: 'Mr. Smith', target: 'Sr. Smith' },
          { source: 'MRI', target: 'RM' },
        ],
      },
    },
  },
});
```

更多说明见 [Soniox 上下文文档](https://soniox.com/docs/stt/concepts/context)。
