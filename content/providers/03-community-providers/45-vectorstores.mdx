# vectorstores 提供商

[vectorstores provider](https://www.vectorstores.org/integration/vercel/) 将 [vectorstores](https://www.vectorstores.org/) 与 AI SDK 集成，支持在向量数据库中检索与存储数据的 AI 工作流。

## 安装与配置

vectorstores 提供商位于 `@vectorstores/vercel` 模块。可使用以下命令安装：

## 文档索引

使用 vectorstores 提供商前，需要先为文档定义索引。索引将文档嵌入存储在向量数据库中，从而可对内容进行语义搜索。

一种简便方式是使用 `@vectorstores/core` 包中的 `VectorStoreIndex.fromDocuments`。该函数会自动将文档切分为更小片段、嵌入并存入向量数据库。

要对文档嵌入，需要嵌入模型。可使用 `vercelEmbedding` 函数（见[工具函数](#工具函数)）包装任意 AI SDK 嵌入模型。

建立索引后，可使用检索器基于与查询的相似度查找最相关文档。

## 工具函数

vectorstores 提供商提供两个主要工具函数，用于集成 AI SDK 与 vectorstores。

### vercelEmbedding

`vercelEmbedding` 将任意 AI SDK 嵌入模型适配为 vectorstores 可用格式，从而在 vectorstores 生态中使用 OpenAI、Cohere 等嵌入提供商。

以下示例展示如何用 `vercelEmbedding` 创建索引：

```ts
import { openai } from '@ai-sdk/openai';
import { vercelEmbedding } from '@vectorstores/vercel';
import { VectorStoreIndex } from '@vectorstores/core';

const documents = [new Document({ text: 'This is a large document.' })];

const index = await VectorStoreIndex.fromDocuments(documents, {
  embedFunc: vercelEmbedding(openai.embedding('text-embedding-3-small')),
});
```

### vercelTool

`vercelTool` 将 vectorstores 检索器包装为 AI SDK 工具，使智能体能够自主访问向量数据库中的数据。

```ts
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { vercelTool } from '@vectorstores/vercel';

const { text } = await generateText({
  model: openai('gpt-5-mini'),
  tools: {
    search: vercelTool({
      retriever: index.asRetriever(),
      description: 'Search the knowledge base for pricing information',
    }),
  },
  prompt: 'What is the price of a burrito?',
});
```

#### 配置选项

| 选项               | 类型          | 必填 | 说明                                                         |
| ------------------ | ------------- | ---- | ------------------------------------------------------------ |
| `retriever`        | BaseRetriever | 是   | 要包装的 vectorstores 检索器实例                           |
| `description`      | string        | 是   | 帮助 LLM 理解何时及如何使用该工具的引导文案                   |
| `noResultsMessage` | string        | 否   | 无匹配文档时的自定义回退文案（默认："No results found in documents."） |

## 使用场景

### 智能体 RAG

就向量数据库中的知识库提问。LLM 自主决定是否通过工具查询向量数据库，检索相关内容并融入回答。

[查看示例 →](https://github.com/marcusschiesser/vectorstores/blob/main/examples/vercel/agentic-rag.ts)

### 智能体记忆系统

结合用于写入记忆的向量数据库工具与用于读取记忆的向量数据库工具，在多轮对话间存储与检索用户相关信息。

[查看示例 →](https://github.com/marcusschiesser/vectorstores/blob/main/examples/vercel/agent-memory.ts)
