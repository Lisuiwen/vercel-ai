# Spark 提供商

<Note type="warning">
  该社区提供商尚未兼容 AI SDK 5。请等待提供商更新，或考虑使用 [兼容 AI SDK 5 的提供商](/providers/ai-sdk-providers)。
</Note>

**[Spark provider](https://github.com/klren0312/spark-ai-provider)** 为 Spark API 提供语言模型支持，可使用 lite、generalv3、pro-128k、generalv3.5、max-32k、4.0Ultra 等模型。

## 安装与配置

Spark 提供商位于 `spark-ai-provider` 模块。可使用以下命令安装：

```bash
npm i spark-ai-provider
```

## 提供商实例

从 `spark-ai-provider` 导入 `createSparkProvider` 以创建提供商实例：

```ts
import { createSparkProvider } from 'spark-ai-provider';
```

## 示例

```ts
import { createSparkProvider } from './index.mjs';
import { generateText } from 'ai';
const spark = createSparkProvider({
  apiKey: '',
});
const { text } = await generateText({
  model: spark('lite'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```
