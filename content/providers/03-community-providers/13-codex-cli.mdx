# Codex CLI 提供商

[ai-sdk-provider-codex-cli](https://github.com/ben-vargas/ai-sdk-provider-codex-cli) 社区提供商可通过 [Codex CLI](https://github.com/openai/codex) 使用 OpenAI 的 GPT-5 系列模型；适合已订阅 ChatGPT Plus/Pro，或使用 API Key 认证的场景。

## 版本兼容

| Provider Version | AI SDK Version | NPM Tag     | Status      |
| ---------------- | -------------- | ----------- | ----------- |
| 1.x              | v6             | `latest`    | Stable      |
| 0.x              | v5             | `ai-sdk-v5` | Maintenance |

```bash
# AI SDK v6 (default)
npm install ai-sdk-provider-codex-cli ai

# AI SDK v5
npm install ai-sdk-provider-codex-cli@ai-sdk-v5 ai@^5.0.0
```

## 提供商实例

从 `ai-sdk-provider-codex-cli` 导入默认实例 `codexCli`：

```ts
import { codexCli } from 'ai-sdk-provider-codex-cli';
```

如需自定义，可导入 `createCodexCli` 并为所有模型提供默认设置：

```ts
import { createCodexCli } from 'ai-sdk-provider-codex-cli';

const codexCli = createCodexCli({
  defaultSettings: {
    reasoningEffort: 'medium',
    approvalMode: 'on-failure',
    sandboxMode: 'workspace-write',
    verbose: true,
  },
});
```

也可按模型传入设置：

```ts
const model = codexCli('gpt-5.1-codex', {
  reasoningEffort: 'high',
  approvalMode: 'on-failure',
  sandboxMode: 'workspace-write',
});
```

模型设置：

- **reasoningEffort** _'none' | 'minimal' | 'low' | 'medium' | 'high' | 'xhigh'_ — 推理深度。
- **approvalMode** _'untrusted' | 'on-failure' | 'on-request' | 'never'_ — 工具审批策略。
- **sandboxMode** _'read-only' | 'workspace-write' | 'danger-full-access'_ — 沙箱限制。
- **mcpServers** _Record&lt;string, McpServerConfig&gt;_ — MCP 服务器配置。
- **verbose** _boolean_ — 是否输出冗长日志。
- **logger** _Logger | false_ — 自定义日志或关闭日志。

## 语言模型

使用提供商实例创建通过 Codex CLI 调用 GPT-5 的模型：

```ts
const model = codexCli('gpt-5.2-codex');
```

**当前代模型：**

- **gpt-5.3-codex**：最新 agentic 编程模型
- **gpt-5.2**：最新通用模型
- **gpt-5.1-codex-max**：旗舰深度推理模型（支持 `xhigh` reasoning）
- **gpt-5.1-codex-mini**：轻量更快变体

**仍支持的旧型号：**

- **gpt-5.1**：通用
- **gpt-5.1-codex**：Codex 变体
- **gpt-5**：上一代
- **gpt-5-codex**：上一代 Codex
- **gpt-5-codex-mini**：上一代轻量 Codex

### 示例

```ts
import { codexCli } from 'ai-sdk-provider-codex-cli';
import { generateText } from 'ai';

const { text } = await generateText({
  model: codexCli('gpt-5.2-codex'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

### 推理配置

```ts
const model = codexCli('gpt-5.1-codex-max', {
  reasoningEffort: 'high', // 'none' | 'minimal' | 'low' | 'medium' | 'high' | 'xhigh'
  reasoningSummary: 'detailed',
});
```

<Note>
  `xhigh` 推理强度适用于 `gpt-5.1-codex-max` 及支持该能力的新模型家族（含你的 Codex CLI
  版本所支持的 GPT-5.2 变体等）。
</Note>

## 认证

通过 Codex CLI 使用现有 ChatGPT Plus/Pro 订阅：

```bash
npm install -g @openai/codex
codex  # Follow the interactive authentication setup
```

亦可设置环境变量 `OPENAI_API_KEY` 使用 OpenAI API Key。

## 要求

- Node.js 18 或更高
- 全局安装 Codex CLI（JSON 支持需 v0.42.0+，推荐 v0.60.0+ 以使用最新模型）
- ChatGPT Plus/Pro 订阅或 OpenAI API Key

详见 [provider documentation](https://github.com/ben-vargas/ai-sdk-provider-codex-cli)。
