# Crosshatch 提供商

<Note type="warning">
  该社区提供商尚未兼容 AI SDK 5。请等待提供商更新，或改用 [兼容 AI SDK 5 的提供商](/providers/ai-sdk-providers)。
</Note>

[Crosshatch](https://crosshatch.io) 提供商支持在主流语言模型上进行安全推理，并对用户授权的共享数据进行权限访问，从而在完整用户上下文中给出个性化响应。

生成的语言模型对象可与 `generateText`、`streamText` 配合使用，并支持借助 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化数据生成。

## 安装

Crosshatch 提供商通过 `@crosshatch/ai-provider` 模块提供：

  <Tab>
    <Snippet text="pnpm add @crosshatch/ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @crosshatch/ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @crosshatch/ai-provider" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @crosshatch/ai-provider" dark />
  </Tab>
</Tabs>

[Crosshatch](https://crosshatch.io/) 提供商支持其全部可用模型，例如 OpenAI GPT 与 Anthropic Claude。还支持用于控制 Crosshatch 自定义数据集成行为的查询接口。本提供商封装底层提供者（[@ai-sdk/openai](/providers/ai-sdk-providers/openai)、[@ai-sdk/anthropic](/providers/ai-sdk-providers/openai)）。

### 凭证

Crosshatch 使用用户专属 token 认证，从而获得对个性化推理的权限访问。

可在 [Crosshatch 开发者控制台](https://platform.crosshatch.io/) 获取合成与测试用户 token。

生产环境用户 token 需通过 [Link SDK](https://www.npmjs.com/package/@crosshatch/link) 与 Crosshatch 开发者 client id 进行发放与访问。

## 提供商实例

使用 `createCrosshatch` 创建 Crosshatch 提供商实例：

```ts
import createCrosshatch from '@crosshatch/ai-provider';
```

## 语言模型

可使用提供商实例创建 [Crosshatch 模型](https://docs.crosshatch.io/endpoints/ai#supported-model-providers)。

```ts
import { createCrosshatch } from '@crosshatch/ai-provider';
const crosshatch = createCrosshatch();
```

第一个参数为目标模型 ID；第二个参数指定用户认证 token、所需上下文与模型参数。
Crosshatch 可基于授权用户上下文与你偏好的语言模型生成文本。

### 示例：带上下文的文本生成

本示例使用 `gpt-4o-mini` 生成文本。

```ts
import { generateText } from 'ai';
import createCrosshatch from '@crosshatch/ai-provider':
const crosshatch = createCrosshatch();

const { text } = await generateText({
  model: crosshatch.languageModel("gpt-4o-mini", {
    token: 'YOUR_ACCESS_TOKEN',
    replace: {
      restaurants: {
        select: ["entity_name", "entity_city", "entity_region"],
        from: "personalTimeline",
        where: [
          { field: "event", op: "=", value: "confirmed" },
          { field: "entity_subtype2", op: "=", value: "RESTAURANTS" }
        ],
        groupby: ["entity_name", "entity_city", "entity_region"],
        orderby: "count DESC",
        limit: 5
      }
    }
  }),
  system: `The user recently ate at these restaurants: {restaurants}`,
  messages: [{role: "user", content: "Where should I stay in Paris?"}]
});
```

### 示例：基于上下文的条目推荐

使用 Crosshatch 根据近期购买对条目重新排序。

```ts
import { streamText, Output } from 'ai';
import createCrosshatch from `@crosshatch/ai-provider`
const crosshatch = createCrosshatch();

const itemSummaries = [...]; // list of items
const ids = (itemSummaries?.map(({ itemId }) => itemId) ?? []) as string[];

const { elementStream } = streamText({
  output: Output.array({
    element: jsonSchema<{ id: string; reason: string }>({
      type: "object",
      properties: {
        id: { type: "string", enum: ids },
        reason: { type: "string", description: "Explain your ranking." },
      },
    }),
  }),
  model: crosshatch.languageModel("gpt-4o-mini", {
    token,
    replace: {
      "orders": {
        select: ["originalTimestamp", "entity_name", "order_total", "order_summary"],
        from: "personalTimeline",
        where: [{ field: "event", op: "=", value: "purchased" }],
        orderBy: [{ field: "originalTimestamp", dir: "desc" }],
        limit: 5,
      },
    },
  }),
  system: `Rerank the following items based on alignment with users recent purchases {orders}`,
  messages: [{role: "user", content: "Heres a list of item: ${JSON.stringify(itemSummaries)"},],
})
```
