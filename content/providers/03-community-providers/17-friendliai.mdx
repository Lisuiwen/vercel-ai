# FriendliAI 提供商

[FriendliAI](https://friendli.ai/) 提供商同时支持通过 [Friendli Serverless Endpoints](https://friendli.ai/products/serverless-endpoints) 使用开源 LLM，以及通过 [Dedicated Endpoints](https://friendli.ai/products/dedicated-endpoints) 使用自定义模型。

生成的语言模型对象可与 `generateText`、`streamText` 配合，并支持借助 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化数据生成。

### 凭证

模型访问所需 token 可在 [Friendli suite](https://suite.friendli.ai/) 获取。

使用本提供商需在环境变量中设置个人访问令牌 `FRIENDLI_TOKEN`。

```bash
export FRIENDLI_TOKEN="YOUR_FRIENDLI_TOKEN"
```

详见 [FriendliAI documentation](https://friendli.ai/docs/guides/personal_access_tokens)。

## 提供商实例

从 `@friendliai/ai-provider` 导入默认实例 `friendliai`：

```ts
import { friendli } from '@friendliai/ai-provider';
```

## 语言模型

使用提供商实例创建 [FriendliAI 模型](https://friendli.ai/docs/guides/serverless_endpoints/text_generation#model-supports)；第一个参数为模型 ID，例如 `meta-llama-3.1-8b-instruct`。

```ts
const model = friendli('meta-llama-3.1-8b-instruct');
```

### 示例：文本生成

与 `generateText` 配合使用：

```ts
import { friendli } from '@friendliai/ai-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: friendli('meta-llama-3.1-8b-instruct'),
  prompt: 'What is the meaning of life?',
});

console.log(text);
```

### 示例：推理（Reasoning）

FriendliAI 通过 `<think>` 标签在生成文本中暴露 `deepseek-r1` 的思考过程。
可使用 `extractReasoningMiddleware` 提取并在结果的 `reasoning` 字段中暴露：

```ts
import { friendli } from '@friendliai/ai-provider';
import { wrapLanguageModel, extractReasoningMiddleware } from 'ai';

const enhancedModel = wrapLanguageModel({
  model: friendli('deepseek-r1'),
  middleware: extractReasoningMiddleware({ tagName: 'think' }),
});

const { text, reasoning } = await generateText({
  model: enhancedModel,
  prompt: 'Explain quantum entanglement.',
});
```

### 示例：结构化输出（正则）

正则选项可通过模式约束输出格式，适用于：

- 需要 CSV 等特定格式
- 将输出限定为韩语、日语等字符集

可与 `generateText`、`streamText` 配合。

深入用法见博文 [Structured Output LLM Agents](https://friendli.ai/blog/structured-output-llm-agents)。

```ts highlight="6"
import { friendli } from '@friendliai/ai-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: friendli('meta-llama-3.1-8b-instruct', {
    regex: new RegExp('[\n ,.?!0-9\uac00-\ud7af]*'),
  }),
  prompt: 'Who is the first king of the Joseon Dynasty?',
});

console.log(text);
```

### 示例：结构化输出（JSON）

结构化输出是一种引导生成方式：以 JSON Schema 作为语法约束，输出始终符合 schema。

```ts
import { friendli } from '@friendliai/ai-provider';
import { generateText, Output } from 'ai';
import { z } from 'zod';

const { output } = await generateText({
  model: friendli('meta-llama-3.3-70b-instruct'),
  output: Output.object({
    schemaName: 'CalendarEvent',
    schema: z.object({
      name: z.string(),
      date: z.string(),
      participants: z.array(z.string()),
    }),
  }),
  system: 'Extract the event information.',
  prompt: 'Alice and Bob are going to a science fair on Friday.',
});

console.log(output);
```

### 示例：内置工具

<Note type="warning">Built-in tools 当前为 Beta。</Note>

使用 `@friendliai/ai-provider` 时，可通过 `tools` 选项使用 [built-in tools](https://friendli.ai/docs/guides/serverless_endpoints/tools/built_in_tools)。

内置工具让模型得以调用工具以生成更好答案，例如 `web:search` 可回答时事问题。

```ts highlight="1,5,6,7"
import { friendli } from '@friendliai/ai-provider';
import { streamText } from 'ai';

const result = streamText({
  model: friendli('meta-llama-3.3-70b-instruct', {
    tools: [{ type: 'web:search' }, { type: 'math:calculator' }],
  }),
  prompt:
    'Find the current USD to CAD exchange rate and calculate how much $5,000 USD would be in Canadian dollars.',
});

for await (const textPart of result.textStream) {
  console.log(textPart);
}
```

### 示例：Dedicated Endpoints 文本生成

通过专用 endpoint 使用自定义模型时，将 endpoint id（如 `zbimjgovmlcb`）作为模型 ID：

```ts
import { friendli } from '@friendliai/ai-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: friendli('YOUR_ENDPOINT_ID'),
  prompt: 'What is the meaning of life?',
});

console.log(text);
```

以下代码可强制发往 dedicated endpoints（默认会自动检测）。

```ts highlight="5,6,7"
import { friendli } from '@friendliai/ai-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: friendli("YOUR_ENDPOINT_ID", {
    endpoint: "dedicated",
  });
  prompt: 'What is the meaning of life?',
});

console.log(text);
```

FriendliAI 语言模型也可用于 `streamText`，并支持借助 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化生成（参见 [AI SDK Core](/docs/ai-sdk-core)）。

### OpenAI 兼容

API 兼容 OpenAI，亦可用 `@ai-sdk/openai`。

```ts
import { createOpenAI } from '@ai-sdk/openai';

const friendli = createOpenAI({
  baseURL: 'https://api.friendli.ai/serverless/v1',
  apiKey: process.env.FRIENDLI_TOKEN,
});
```

若使用 dedicated endpoints：

```ts
import { createOpenAI } from '@ai-sdk/openai';

const friendli = createOpenAI({
  baseURL: 'https://api.friendli.ai/dedicated/v1',
  apiKey: process.env.FRIENDLI_TOKEN,
});
```
