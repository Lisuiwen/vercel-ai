# Letta 提供商

[Letta AI SDK provider](https://github.com/letta-ai/vercel-ai-sdk-provider) 让你将 Letta agent 与 AI SDK 配合使用。

特性包括：

- 持久化与长期记忆
- 可访问 agent 级与模型级推理消息并带来源标注
- 支持自定义 agent 配置工具与 MCP（Model Context Protocol）
- Agent 管理文件系统，基于工具的文件访问
- 内置 Letta 与 AI SDK 消息格式互转工具
- [Letta Send Message API 的各项能力](https://docs.letta.com/api-reference/agents/messages/create-stream)，例如 [长时间运行的 Agent](https://docs.letta.com/guides/agents/long-running)

## 支持的模型

完整列表见 [Letta documentation](https://docs.letta.com/connecting-model-providers/supported-models)。若模型不在列表中，可尝试为 Letta 配置 [OpenAI proxy](https://docs.letta.com/guides/server/providers/openai-proxy)。

## 提供商实例

从 `@letta-ai/vercel-ai-sdk-provider` 导入 `lettaCloud` 或 `lettaLocal`：

```ts
// For cloud users
import { lettaCloud } from '@letta-ai/vercel-ai-sdk-provider';

// For self-hosted users
import { lettaLocal } from '@letta-ai/vercel-ai-sdk-provider';

// Create a custom Letta provider
import { createLetta } from '@letta-ai/vercel-ai-sdk-provider';

const letta = createLetta({
  baseUrl: '<your-base-url>',
  token: '<your-access-token>',
});
```

## 基本用法

在 [Letta dashboard](https://app.letta.com/api-keys) 获取 API key。

```bash
# .env
LETTA_API_KEY=your-letta-api-key
```

```typescript
import { lettaCloud } from '@letta-ai/vercel-ai-sdk-provider';
import { generateText } from 'ai';

const result = await generateText({
  model: lettaCloud(), // Model configuration (LLM, temperature, etc.) is managed through your Letta agent
  providerOptions: {
    letta: {
      agent: { id: 'your-agent-id' },
    },
  },
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});

console.log(result.text);
```

## 高级用法

### 提供商选项

通过 `providerOptions.letta` 配置 Letta 专有设置：

```typescript
import { lettaCloud } from '@letta-ai/vercel-ai-sdk-provider';
import { streamText } from 'ai';

const result = streamText({
  model: lettaCloud(),
  providerOptions: {
    letta: {
      agent: {
        id: 'your-agent-id',
        maxSteps: 100,
        includePings: true,
        streamTokens: true,
      },
      timeoutInSeconds: 300,
    },
  },
  prompt: 'Tell me a story about a robot learning to paint',
});
```

#### Agent 配置

配置创建消息时的 agent 参数，对流式与非流式均生效。

可用选项：

- `id`（`string`，必填）：Letta agent ID
- `maxSteps`（`number`）：最大执行步数
- `includePings`（`boolean`）：流中是否包含 ping 消息
- `streamTokens`（`boolean`）：是否按 token 流式输出
- `background`（`boolean`）：长时间任务是否后台执行
- 更多参数见 [Letta API documentation](https://docs.letta.com/api-reference/agents/messages/create-stream)

#### 超时配置

**类型：** `number`

**默认：** `1000`

agent 响应的最长等待时间（秒）。对长时运行 agent 或复杂推理链尤为重要。

```typescript
const result = streamText({
  model: lettaCloud(),
  providerOptions: {
    letta: {
      agent: { id: 'your-agent-id' },
      timeoutInSeconds: 300, // Wait up to 5 minutes
    },
  },
  prompt: 'Process this complex task...',
});
```

## 自定义工具与 MCP

Letta agent 支持通过提供商执行工具的自定义工具与 MCP 服务器。在 Letta agent 上 [完成配置](https://docs.letta.com/guides/agents/custom-tools) 后，可用 `letta.tool()` 在请求中包含，工具执行由 Letta 负责。

```typescript
import { streamText } from 'ai';
import { z } from 'zod';
import { lettaCloud } from '@letta-ai/vercel-ai-sdk-provider';

// Use with streaming
const result = streamText({
  model: lettaCloud(),
  tools: {
    web_search: lettaCloud.tool('web_search'),
    memory_insert: lettaCloud.tool('memory_insert'),
    memory_replace: lettaCloud.tool('memory_replace'),
    core_memory_append: lettaCloud.tool('core_memory_append'),
    my_custom_tool: lettaCloud.tool('my_custom_tool'),
    // Optionally provide description and schema (placeholders only - execution handled by Letta)
    typed_query: lettaCloud.tool('typed_query', {
      description: 'Query with typed parameters',
      inputSchema: z.object({
        query: z.string(),
      }),
    }),
  },
  providerOptions: {
    letta: {
      agent: { id: agentId },
    },
  },
  prompt: 'Tell me a story about a robot learning to paint',
});
```

## 使用其他 Letta Client 函数

`vercel-ai-sdk-provider` 扩展了 [@letta-ai/letta-client](https://www.npmjs.com/package/@letta-ai/letta-client)，可通过 `lettaCloud.client`、`lettaLocal.client` 或自定义 `letta.client` 直接调用操作：

```ts
// with Letta Cloud
import { lettaCloud } from '@letta-ai/vercel-ai-sdk-provider';

lettaCloud.client.agents.list();

// with Letta Local
import { lettaLocal } from '@letta-ai/vercel-ai-sdk-provider';

lettaLocal.client.agents.list();
```
