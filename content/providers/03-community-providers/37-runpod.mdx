# Runpod

官方 [Runpod](https://runpod.io) 提供商为公有与私有端点提供语言模型与图像生成支持。

## 安装与配置

Runpod 提供商位于 `@runpod/ai-sdk-provider` 模块。可使用以下命令安装：

## 提供商实例

可从 `@runpod/ai-sdk-provider` 导入默认提供商实例 `runpod`：

```ts
import { runpod } from '@runpod/ai-sdk-provider';
```

如需自定义配置，可导入 `createRunpod` 并创建实例：

```ts
import { createRunpod } from '@runpod/ai-sdk-provider';

const runpod = createRunpod({
  apiKey: 'your-api-key', // optional, defaults to RUNPOD_API_KEY environment variable
  baseURL: 'custom-url', // optional, for custom endpoints
  headers: {
    /* custom headers */
  }, // optional
});
```

可使用以下可选设置自定义 Runpod 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀，例如代理或自定义端点。
  支持 vLLM 部署、SGLang 服务器及任意 OpenAI 兼容 API。
  默认前缀为 `https://api.runpod.ai/v2`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。
  默认读取 `RUNPOD_API_KEY` 环境变量。
  可在 [Runpod 控制台](https://console.runpod.io/user/settings) 的「API Keys」下获取。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  可用作中间件拦截请求，
  或为测试等场景提供自定义 fetch。

## 语言模型

使用提供商实例创建语言模型。第一个参数为模型 ID：

```ts
import { runpod } from '@runpod/ai-sdk-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: runpod('qwen/qwen3-32b-awq'),
  prompt: 'What is the capital of Germany?',
});
```

**返回值：**

- `text` — 生成的文本字符串
- `finishReason` — 生成停止原因（`'stop'`、`'length'` 等）
- `usage` — Token 用量（提示、补全、合计）

### 流式输出

```ts
import { runpod } from '@runpod/ai-sdk-provider';
import { streamText } from 'ai';

const { textStream } = await streamText({
  model: runpod('qwen/qwen3-32b-awq'),
  prompt:
    'Write a short poem about artificial intelligence in exactly 4 lines.',
  temperature: 0.7,
});

for await (const delta of textStream) {
  process.stdout.write(delta);
}
```

### 模型能力

| 模型 ID               | 说明                                                         | 流式                | 结构化对象生成      | 工具使用            | 推理说明           |
| --------------------- | ------------------------------------------------------------ | ------------------- | ------------------- | ------------------- | ------------------ |
| `qwen/qwen3-32b-awq`  | 320 亿多语种模型，推理能力强                               | <Check size={18} /> | <Cross size={18} /> | <Check size={18} /> | 标准推理事件       |
| `openai/gpt-oss-120b` | 1200 亿参数开源 GPT 模型                                     | <Check size={18} /> | <Cross size={18} /> | <Check size={18} /> | 标准推理事件       |

**说明：** 最新完整模型列表见 [Runpod 公有端点参考](https://docs.runpod.io/hub/public-endpoint-reference)。

### 聊天对话

```ts
const { text } = await generateText({
  model: runpod('qwen/qwen3-32b-awq'),
  messages: [
    { role: 'system', content: 'You are a helpful assistant.' },
    { role: 'user', content: 'What is the capital of France?' },
  ],
});
```

### 工具调用

```ts
import { generateText, tool } from 'ai';
import { z } from 'zod';

const { text, toolCalls } = await generateText({
  model: runpod('openai/gpt-oss-120b'),
  prompt: 'What is the weather like in San Francisco?',
  tools: {
    getWeather: tool({
      description: 'Get weather information for a city',
      inputSchema: z.object({
        city: z.string().describe('The city name'),
      }),
      execute: async ({ city }) => {
        return `The weather in ${city} is sunny.`;
      },
    }),
  },
});
```

**额外返回值：**

- `toolCalls` — 模型发起的工具调用数组
- `toolResults` — 已执行工具的结果

### 结构化输出

本提供商所包含的两个模型不支持使用 [`Output`](/docs/reference/ai-sdk-core/output) 强制结构化输出。

仍可通过指示模型返回 JSON 并自行校验的方式获得结构化数据。

```ts
import { runpod } from '@runpod/ai-sdk-provider';
import { generateText } from 'ai';
import { z } from 'zod';

const RecipeSchema = z.object({
  name: z.string(),
  ingredients: z.array(z.string()),
  steps: z.array(z.string()),
});

const { text } = await generateText({
  model: runpod('qwen/qwen3-32b-awq'),
  messages: [
    {
      role: 'system',
      content:
        'return ONLY valid JSON matching { name: string; ingredients: string[]; steps: string[] }',
    },
    { role: 'user', content: 'generate a lasagna recipe.' },
  ],
  temperature: 0,
});

const parsed = JSON.parse(text);
const result = RecipeSchema.safeParse(parsed);

if (!result.success) {
  // handle invalid JSON shape
}

console.log(result.success ? result.data : parsed);
```

## 图像模型

使用 `.imageModel()` 工厂方法创建 Runpod 图像模型。

### 基础用法

```ts
import { runpod } from '@runpod/ai-sdk-provider';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: runpod.imageModel('qwen/qwen-image'),
  prompt: 'A serene mountain landscape at sunset',
  aspectRatio: '4:3',
});

// Save to filesystem
import { writeFileSync } from 'fs';
writeFileSync('landscape.jpg', image.uint8Array);
```

**返回值：**

- `image.uint8Array` — 图像二进制（便于处理与保存）
- `image.base64` — Base64 字符串（便于 Web 展示）
- `image.mediaType` — MIME 类型（`'image/jpeg'` 或 `'image/png'`）
- `warnings` — 与不支持参数相关的警告数组

### 模型能力

| 模型 ID                                | 说明                     | 支持的宽高比                         |
| -------------------------------------- | ------------------------ | ------------------------------------ |
| `bytedance/seedream-3.0`               | 高级文生图模型           | 1:1, 4:3, 3:4                       |
| `bytedance/seedream-4.0`               | 文生图（v4）             | 1:1（支持 1024、2048、4096）         |
| `bytedance/seedream-4.0-edit`         | 图像编辑（v4，多图）     | 1:1（支持 1024、1536、2048、4096）   |
| `black-forest-labs/flux-1-schnell`     | 快速出图（4 步）         | 1:1, 4:3, 3:4                       |
| `black-forest-labs/flux-1-dev`         | 高质量文生图             | 1:1, 4:3, 3:4                       |
| `black-forest-labs/flux-1-kontext-dev` | 上下文感知文生图        | 1:1, 4:3, 3:4                       |
| `qwen/qwen-image`                      | 文生图                   | 1:1, 4:3, 3:4                       |
| `qwen/qwen-image-edit`                 | 图像编辑（提示引导）     | 1:1, 4:3, 3:4                       |

**说明**：提供商对图像参数采用严格校验。不支持的宽高比（如 `16:9`、`9:16`、`3:2`、`2:3`）将抛出 `InvalidArgumentError`，并提示可用的替代方案。

最新图像模型列表见 [Runpod 公有端点参考](https://docs.runpod.io/hub/public-endpoint-reference)。

### 高级参数

```ts
const { image } = await generateImage({
  model: runpod.imageModel('bytedance/seedream-3.0'),
  prompt: 'A sunset over mountains',
  size: '1328x1328',
  seed: 42,
  providerOptions: {
    runpod: {
      negative_prompt: 'blurry, low quality',
      enable_safety_checker: true,
    },
  },
});
```

#### 修改图像


### 高级配置

```ts
// Full control over generation parameters
const { image } = await generateImage({
  model: runpod.imageModel('black-forest-labs/flux-1-dev'),
  prompt: 'A majestic dragon breathing fire in a medieval castle',
  size: '1328x1328',
  seed: 42, // For reproducible results
  providerOptions: {
    runpod: {
      negative_prompt: 'blurry, low quality, distorted, ugly, bad anatomy',
      enable_safety_checker: true,
      num_inference_steps: 50, // Higher quality (default: 28)
      guidance: 3.5, // Stronger prompt adherence (default: 2)
      output_format: 'png', // High quality format
      // Polling settings for long generations
      maxPollAttempts: 30,
      pollIntervalMillis: 4000,
    },
  },
});

// Fast generation with minimal steps
const { image } = await generateImage({
  model: runpod.imageModel('black-forest-labs/flux-1-schnell'),
  prompt: 'A simple red apple',
  aspectRatio: '1:1',
  providerOptions: {
    runpod: {
      num_inference_steps: 2, // Even faster (default: 4)
      guidance: 10, // Higher guidance for simple prompts
      output_format: 'jpg', // Smaller file size
    },
  },
});
```

### 提供商选项

Runpod 图像模型通过 `providerOptions.runpod` 支持灵活的提供商选项：

| 选项                    | 类型      | 默认值  | 说明                                                         |
| ----------------------- | --------- | ------- | ------------------------------------------------------------ |
| `negative_prompt`       | `string`  | `""`    | 描述图像中不希望出现的内容                                   |
| `enable_safety_checker` | `boolean` | `true`  | 是否启用内容安全过滤                                         |
| `image`                 | `string`  | -       | 输入图像：URL 或 base64 data URI（Flux Kontext 模型必填）      |
| `num_inference_steps`   | `number`  | Auto    | 去噪步数（Flux：schnell 为 4，其余默认为 28）                |
| `guidance`              | `number`  | Auto    | 提示遵循强度（Flux：schnell 为 7，其余默认为 2）             |
| `output_format`         | `string`  | `"png"` | 输出格式（`"png"` 或 `"jpg"`）                               |
| `maxPollAttempts`       | `number`  | `60`    | 异步生成的最大轮询次数                                       |
| `pollIntervalMillis`    | `number`  | `5000`  | 轮询间隔（毫秒，默认 5 秒）                                  |
