# React Native Apple 提供商

[@react-native-ai/apple](https://github.com/callstackincubator/ai/tree/main/packages/apple-llm) 是社区提供商，将 Apple 设备端 AI 能力带入 React Native 与 Expo 应用。可在设备上完整运行 AI SDK，利用 iOS 26+ 可用的 Apple Intelligence 基础模型，通过 Apple 原生 AI 框架提供文本生成、嵌入、转写与语音合成。

## 安装与配置

Apple 提供商位于 `@react-native-ai/apple` 模块。可使用以下命令安装：

### 前置条件

使用 Apple 提供商前需要：

- **React Native 或 Expo 应用**：本提供商仅适用于 React Native 与 Expo。配置步骤见 [Expo 快速入门](/docs/getting-started/expo)
- **iOS 26+**：Apple Intelligence 基础模型与核心功能要求

### 提供商实例

可从 `@react-native-ai/apple` 导入默认提供商实例 `apple`：

```ts
import { apple } from '@react-native-ai/apple';
```

### 可用性检查

在使用 Apple AI 功能前，可检查当前设备是否支持：

```ts
if (!apple.isAvailable()) {
  // Handle fallback logic for unsupported devices
}
```

## 语言模型

Apple 通过 Apple Foundation Models 提供设备端语言模型，适用于已启用 Apple Intelligence 且系统为 iOS 26+ 的设备。

### 文本生成

使用 Apple 设备端语言模型生成文本：

```ts
import { apple } from '@react-native-ai/apple';
import { generateText } from 'ai';

const { text } = await generateText({
  model: apple(),
  prompt: 'Explain quantum computing in simple terms',
});
```

### 流式文本生成

实时生成文本：

```ts
import { apple } from '@react-native-ai/apple';
import { streamText } from 'ai';

const result = streamText({
  model: apple(),
  prompt: 'Write a short story about space exploration',
});

for await (const chunk of result.textStream) {
  console.log(chunk);
}
```

### 结构化输出

使用 Zod schema 生成结构化数据：

```ts
import { apple } from '@react-native-ai/apple';
import { generateText, Output } from 'ai';
import { z } from 'zod';

const result = await generateText({
  model: apple(),
  output: Output.object({
    schema: z.object({
      recipe: z.string(),
      ingredients: z.array(z.string()),
      cookingTime: z.string(),
    }),
  }),
  prompt: 'Create a recipe for chocolate chip cookies',
});
```

### 模型配置

配置生成参数：

```ts
const { text } = await generateText({
  model: apple(),
  prompt: 'Generate creative content',
  temperature: 0.8, // Controls randomness (0-1)
  maxTokens: 150, // Maximum tokens to generate
  topP: 0.9, // Nucleus sampling threshold
  topK: 40, // Top-K sampling parameter
});
```

### 工具调用

Apple 提供商支持工具调用；工具由 Apple Intelligence 执行，而非 AI SDK。必须在使用 `createAppleProvider` 时预先注册工具后，方可在生成调用中使用。

```ts
import { createAppleProvider } from '@react-native-ai/apple';
import { generateText, tool } from 'ai';
import { z } from 'zod';

const getWeather = tool({
  description: 'Get current weather information',
  parameters: z.object({
    city: z.string().describe('The city name'),
  }),
  execute: async ({ city }) => {
    return `Weather in ${city}: Sunny, 25°C`;
  },
});

// Create a provider with all available tools
const apple = createAppleProvider({
  availableTools: {
    getWeather,
  },
});

// Use the provider with selected tools
const result = await generateText({
  model: apple(),
  prompt: 'What is the weather like in San Francisco?',
  tools: { getWeather },
});
```

<Note>
  由于工具由 Apple Intelligence 执行而非 AI SDK，
  不支持 `stopWhen`、`onStepStart`、`onStepFinish` 等多步特性。
</Note>

## 文本嵌入

Apple 使用 `NLContextualEmbedding` 提供多语言文本嵌入，要求 iOS 17+。

```ts
import { apple } from '@react-native-ai/apple';
import { embed } from 'ai';

const { embedding } = await embed({
  model: apple.embeddingModel(),
  value: 'Hello world',
});
```

## 音频转写

Apple 使用 `SpeechAnalyzer` 与 `SpeechTranscriber` 提供语音转文字，要求 iOS 26+。

```ts
import { apple } from '@react-native-ai/apple';
import { experimental_transcribe } from 'ai';

const response = await experimental_transcribe({
  model: apple.transcriptionModel(),
  audio: audioBuffer,
});

console.log(response.text);
```

## 语音合成

Apple 使用 `AVSpeechSynthesizer` 提供文本转语音，iOS 13+ 可用；iOS 17+ 具备增强能力。

### 基础语音生成

将文本转为语音：

```ts
import { apple } from '@react-native-ai/apple';
import { experimental_generateSpeech } from 'ai';

const response = await experimental_generateSpeech({
  model: apple.speechModel(),
  text: 'Hello from Apple on-device speech!',
  language: 'en-US',
});
```

### 音色选择

可通过 `voice` 选项传入标识符以指定合成音色。

```ts
const response = await experimental_generateSpeech({
  model: apple.speechModel(),
  text: 'Custom voice example',
  voice: 'com.apple.ttsbundle.Samantha-compact',
});
```

可用音色可通过 `getVoices` 查询：

```ts
import { AppleSpeech } from '@react-native-ai/apple';

const voices = await AppleSpeech.getVoices();
console.log(voices);
```

## 平台要求

不同 Apple AI 功能对 iOS 版本要求不同：

| 功能       | 最低 iOS 版本 | 其他要求                 |
| ---------- | ------------- | ------------------------ |
| 文本生成   | iOS 26+       | 已启用 Apple Intelligence 的设备 |
| 文本嵌入   | iOS 17+       | —                        |
| 音频转写   | iOS 26+       | 需下载语言资源           |
| 语音合成   | iOS 13+       | Personal Voice 需 iOS 17+ |

<Note>
  Apple Intelligence 目前仅在部分设备上可用。设备兼容性请以 Apple 官方文档为准。
</Note>
