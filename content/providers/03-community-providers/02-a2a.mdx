# A2A

[dracoblue/a2a-ai-provider](https://github.com/dracoblue/a2a-ai-provider) 是一个社区提供商，可将符合 [A2A 协议](https://a2aproject.github.io/A2A/specification/) 的智能体与 [AI SDK](https://ai-sdk.dev/) 一起使用。开发者可通过基于 HTTP 的标准化 JSON-RPC 接口进行流式传输、发送和接收文本、工具调用与制品。

<Note type="warning">
  `a2a-ai-provider` 包仍在持续开发中。
</Note>

该提供商支持（通过官方 a2a-js SDK [@a2a-js/sdk](https://github.com/a2aproject/a2a-js)）：

- **流式文本响应**：通过 `sendSubscribe` 与 SSE
- **文件与制品上传**：上传到 A2A 服务器
- **多模态消息**：支持文本与文件部分
- **完整 JSON-RPC 2.0 合规**：面向兼容 A2A 的 LLM 智能体

在 [A2A 项目站点](https://a2aproject.github.io/A2A/)了解更多。

## 提供商实例

要为 A2A 服务器创建提供商实例：

```ts
import { a2a } from 'a2a-ai-provider';
```

## 示例

现在可以像这样将提供商与 AI SDK 一起使用：

### `generateText`

```ts
import { a2a } from 'a2a-ai-provider';
import { generateText } from 'ai';

const result = await generateText({
  model: a2a('https://your-a2a-server.example.com'),
  prompt: 'What is love?',
});

console.log(result.text);
```

### `streamText`

```ts
import { a2a } from 'a2a-ai-provider';
import { streamText } from 'ai';

const chatId = 'unique-chat-id'; // for each conversation to keep history in a2a server

const streamResult = streamText({
  model: a2a('https://your-a2a-server.example.com'),
  prompt: 'What is love?',
  providerOptions: {
    a2a: {
      contextId: chatId,
    },
  },
});

await streamResult.consumeStream();

console.log(await streamResult.content);
```

## 功能

- **文本流式传输**：从 A2A 服务器按 token 流式输出
- **文件上传**：将文件作为提示的一部分发送
- **制品处理**：在流式或最终结果中接收文件制品
