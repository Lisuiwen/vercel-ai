# Qwen 提供商

<Note type="warning">
  该社区提供商尚未兼容 AI SDK 5。请等待提供商更新，或考虑使用 [兼容 AI SDK 5 的提供商](/providers/ai-sdk-providers)。
</Note>

[younis-ahmed/qwen-ai-provider](https://github.com/younis-ahmed/qwen-ai-provider) 是社区提供商，通过 [Qwen](https://www.alibabacloud.com/en/solutions/generative-ai/qwen) 为 AI SDK 提供语言模型支持。

## 安装与配置

Qwen 提供商位于 `qwen-ai-provider` 模块。可使用以下命令安装：

  <Tab>
    <Snippet text="yarn add qwen-ai-provider" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add qwen-ai-provider" dark />
  </Tab>
</Tabs>

## 提供商实例

可从 `qwen-ai-provider` 导入默认提供商实例 `qwen`：

```ts
import { qwen } from 'qwen-ai-provider';
```

如需自定义配置，可导入 `createQwen` 并创建实例：

```ts
import { createQwen } from 'qwen-ai-provider';

const qwen = createQwen({
  // optional settings, e.g.
  // baseURL: 'https://qwen/api/v1',
});
```

可使用以下可选设置自定义 Qwen 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀，例如代理。
  默认前缀为 `https://dashscope-intl.aliyuncs.com/compatible-mode/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。
  默认读取 `DASHSCOPE_API_KEY` 环境变量。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作中间件拦截请求，
  或为测试等场景提供自定义 fetch。

## 语言模型

使用提供商实例可创建调用 [Qwen 聊天 API](https://www.alibabacloud.com/help/en/model-studio/developer-reference/use-qwen-by-calling-api) 的模型。
第一个参数为模型 id，例如 `qwen-plus`。
部分 Qwen 聊天模型支持工具调用。

```ts
const model = qwen('qwen-plus');
```

### 示例

可使用 Qwen 语言模型配合 `generateText` 生成文本：

```ts
import { qwen } from 'qwen-ai-provider';
import { generateText } from 'ai';

const { text } = await generateText({
  model: qwen('qwen-plus'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

Qwen 语言模型也可用于 `streamText`，并支持配合 [`Output`](/docs/reference/ai-sdk-core/output) 的结构化数据生成
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

## 嵌入模型

使用 `.embeddingModel()` 工厂方法可创建调用 [Qwen embeddings API](https://www.alibabacloud.com/help/en/model-studio/getting-started/models#cff6607866tsg) 的模型。

```ts
const model = qwen.embeddingModel('text-embedding-v3');
```

### 模型能力

| 模型               | 默认维度 | 最大行数 | 每行最大 token |
| ------------------ | -------- | -------- | -------------- |
| `text-embedding-v3`| 1024     | 6        | 8,192          |
