# ACP（Agent Client Protocol）

[ACP（Agent Client Protocol）](https://agentclientprotocol.com/) 是一种开放协议，可在 AI 智能体与客户端应用之间实现顺畅通信。

ACP 提供商通过 Agent Client Protocol 与 ACP 智能体（如 Claude Code、Gemini CLI、Codex CLI 以及**[更多](https://agentclientprotocol.com/overview/agents)**）通信，并通过 `LanguageModel` 接口对外暴露，便于你使用 ACP 智能体构建 Web 应用与 Node.js 服务。

## 主要功能

- **多种智能体支持**：可与 Claude Code、Gemini CLI、Codex CLI 及其他兼容 ACP 的智能体配合使用
- **MCP 服务器集成**：接入 MCP（Model Context Protocol）服务器以增强智能体能力
- **工具执行**：智能体可执行工具并通过 AI SDK 接口回报结果
- **进程管理**：自动拉起并管理智能体进程生命周期

在 [Agent Client Protocol 文档](https://agentclientprotocol.com)了解更多。

## 提供商实例

使用 `createACPProvider` 创建 ACP 提供商实例：

```typescript
import { createACPProvider } from '@mcpc-tech/acp-ai-provider';

const provider = createACPProvider({
  command: 'gemini',
  args: ['--experimental-acp'],
  session: {
    cwd: process.cwd(),
    mcpServers: [],
  },
});
```

### 配置选项

提供商接受以下配置：

- **command** _string_（必填）

  用于启动 ACP 智能体的命令（例如 `'gemini'`、`'claude-code-acp'`、`'codex-acp'`）。

- **args** _string[]_（可选）

  传递给命令的参数（例如 `['--experimental-acp']`）。

- **env** _Record&lt;string, string&gt;_（可选）

  智能体进程的环境变量。

- **session** _ACPSessionConfig_（必填）

  会话配置，包括：
  - `cwd`：智能体的工作目录
  - `mcpServers`：MCP 服务器配置数组，用于向智能体提供工具

- **authMethodId** _string_（可选）

  若 ACP 智能体需要认证，可指定认证方式 ID。

## 语言模型

ACP 提供商暴露单一语言模型，表示所配置的 ACP 智能体：

```typescript
const model = provider.languageModel();
```

> **说明**：当前无法选择具体模型。详见[限制](#limitations)。

## 示例

### 文本生成

```javascript
import { createACPProvider } from '@mcpc-tech/acp-ai-provider';
import { generateText } from 'ai';

const provider = createACPProvider({
  command: 'gemini',
  args: ['--experimental-acp'],
  session: {
    cwd: process.cwd(),
    mcpServers: [],
  },
});

const { text } = await generateText({
  model: provider.languageModel(),
  prompt: 'What is the Agent Client Protocol?',
});

console.log(text);
```

### 流式文本

```javascript
import { createACPProvider } from '@mcpc-tech/acp-ai-provider';
import { streamText } from 'ai';

const provider = createACPProvider({
  command: 'claude-code-acp',
  session: {
    cwd: process.cwd(),
    mcpServers: [],
  },
});

const { textStream } = streamText({
  model: provider.languageModel(),
  prompt: 'Write a simple Hello World program',
});

for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```

### 配合 MCP 服务器（工具）

工具通过 MCP（Model Context Protocol）服务器提供给 ACP 智能体，而非通过 AI SDK 的 `tools` 参数：

```javascript
import { createACPProvider } from '@mcpc-tech/acp-ai-provider';
import { generateText } from 'ai';

const provider = createACPProvider({
  command: 'gemini',
  args: ['--experimental-acp'],
  session: {
    cwd: process.cwd(),
    mcpServers: [
      {
        type: 'stdio',
        name: 'filesystem',
        command: 'npx',
        args: ['-y', '@modelcontextprotocol/server-filesystem', '/tmp'],
      },
    ],
  },
});

const result = await generateText({
  model: provider.languageModel(),
  prompt: 'List files in /tmp',
});
```

## 使用工具

ACP 提供商通过提供商定义的工具处理工具执行。工具由 ACP 智能体调用并执行，结果通过 AI SDK 的流式接口回传。

要流式传输工具调用，将提供商工具传给 AI SDK：

```javascript
const result = await generateText({
  model: provider.languageModel(),
  prompt: 'List files in /tmp',
  tools: provider.tools,
});
```

工具调用结构如下：

```typescript
{
  toolCallId: string; // Unique ID of the tool call
  toolName: string; // Name of the tool being called
  args: Record<string, unknown>; // Input arguments
}
```

## 高级功能

### 多种智能体支持

ACP 提供商可与多种兼容 ACP 的智能体配合：

- **Gemini CLI**：使用 `command: 'gemini'` 与 `args: ['--experimental-acp']`
- **Claude Code**：使用 `command: 'claude-code-acp'`
- **Codex CLI**：使用 `command: 'codex-acp'`
- **[以及更多……](https://agentclientprotocol.com/overview/agents)**：完整列表见官方 ACP 智能体页面

### 自定义认证

部分智能体需要认证，可指定认证方式 ID：

```typescript
const provider = createACPProvider({
  command: 'gemini',
  args: ['--experimental-acp'],
  authMethodId: 'gemini-api-key',
  session: {
    cwd: process.cwd(),
    mcpServers: [],
  },
});
```

### 进程环境变量

向智能体进程传入环境变量：

```typescript
const provider = createACPProvider({
  command: 'gemini',
  args: ['--experimental-acp'],
  env: {
    GEMINI_API_KEY: process.env.GEMINI_API_KEY,
    DEBUG: 'true',
  },
  session: {
    cwd: process.cwd(),
    mcpServers: [],
  },
});
```

## 限制

- **工具定义**：工具须通过会话配置中的 MCP 服务器提供，不能仅通过 AI SDK 的 `tools` 参数
- **进程生命周期**：每个语言模型实例会拉起新的智能体进程，请注意清理
- **仅 Node.js**：当前需要具备子进程能力的 Node.js 环境
- **文件操作**：通过 ACP 客户端接口提供基础文件操作支持
- **模型选择**：暂不支持选择模型。该功能进展见 https://github.com/agentclientprotocol/agent-client-protocol/pull/182
