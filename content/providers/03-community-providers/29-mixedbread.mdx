# Mixedbread 提供商

[patelvivekdev/mixedbread-ai-provider](https://github.com/patelvivekdev/mixedbread-ai-provider) 是社区提供商，通过 [Mixedbread](https://www.mixedbread.ai/) 为 AI SDK 提供嵌入（Embedding）支持。

## 安装与配置

Mixedbread 提供商位于 `mixedbread-ai-provider` 模块。可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add mixedbread-ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install mixedbread-ai-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add mixedbread-ai-provider" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add mixedbread-ai-provider" dark />
  </Tab>
</Tabs>

## 提供商实例

可从 `mixedbread-ai-provider` 导入默认提供商实例 `mixedbread`：

```ts
import { mixedbread } from 'mixedbread-ai-provider';
```

如需自定义配置，可导入 `createMixedbread` 并按需创建实例：

```ts
import { createMixedbread } from 'mixedbread-ai-provider';

const mixedbread = createMixedbread({
  // custom settings
});
```

可使用以下可选设置自定义 Mixedbread 提供商实例：

- **baseURL** _string_

  Mixedbread API 的基础 URL。默认前缀为 `https://api.mixedbread.com/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API 密钥。默认读取 `MIXEDBREAD_API_KEY` 环境变量。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。默认为全局 `fetch`。可用作中间件拦截请求，或为测试等场景提供自定义 fetch。

## 文本嵌入模型

可使用 `.embeddingModel()` 工厂方法创建调用 [Mixedbread embeddings API](https://www.mixedbread.com/api-reference/endpoints/embeddings) 的模型。

```ts
import { mixedbread } from 'mixedbread-ai-provider';

const embeddingModel = mixedbread.embeddingModel(
  'mixedbread-ai/mxbai-embed-large-v1',
);
```

可使用 Mixedbread 嵌入模型配合 `embed` 函数生成向量：

```ts
import { mixedbread } from 'mixedbread-ai-provider';
import { embed } from 'ai';

const { embedding } = await embed({
  model: mixedbread.embeddingModel('mixedbread-ai/mxbai-embed-large-v1'),
  value: 'sunny day at the beach',
});
```

Mixedbread 嵌入模型支持通过 `providerOptions.mixedbread` 传入额外提供商选项：

```ts
import { mixedbread } from 'mixedbread-ai-provider';
import { embed } from 'ai';

const { embedding } = await embed({
  model: mixedbread.embeddingModel('mixedbread-ai/mxbai-embed-large-v1'),
  value: 'sunny day at the beach',
  providerOptions: {
    mixedbread: {
      prompt: 'Generate embeddings for text',
      normalized: true,
      dimensions: 512,
      encodingFormat: 'float16',
    },
  },
});
```

可用的提供商选项如下：

- **prompt** _string_

  可选提示，为模型提供上下文。详情请参阅模型文档。长度为 1–256 个字符的字符串。

- **normalized** _boolean_

  是否对嵌入向量做归一化。

- **dimensions** _number_

  输出向量期望的维度数。默认为模型的最大值。取值范围为 1 至该模型的最大输出维度。仅适用于基于 Matryoshka 的模型。

- **encodingFormat** _'float' | 'float16' | 'binary' | 'ubinary' | 'int8' | 'uint8' | 'base64'_

### 模型能力

| 模型                              | 上下文长度 | 维度 | 自定义维度          |
| --------------------------------- | ---------- | ---- | ------------------- |
| `mxbai-embed-large-v1`            | 512        | 1024 | <Check size={18} /> |
| `mxbai-embed-2d-large-v1`         | 512        | 1024 | <Check size={18} /> |
| `deepset-mxbai-embed-de-large-v1` | 512        | 1024 | <Check size={18} /> |
| `mxbai-embed-xsmall-v1`           | 4096       | 384  | <Cross size={18} /> |

<Note>
  上表列出常用模型。完整列表请参阅 [Mixedbread
  文档](https://www.mixedbread.com/docs/models/embedding)。
</Note>
