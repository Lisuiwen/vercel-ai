# Hindsight

[Hindsight](https://hindsight.vectorize.io) 是面向 AI 智能体的持久记忆服务。[`@vectorize-io/hindsight-ai-sdk`](https://www.npmjs.com/package/@vectorize-io/hindsight-ai-sdk) 包提供五个兼容 AI SDK 的工具，使智能体在多轮对话间具备长期记忆。

功能包括：

- 五种记忆工具：`retain`、`recall`、`reflect`、`getMentalModel`、`getDocument`
- 可与 `generateText`、`streamText`、`ToolLoopAgent` 配合使用
- 基础设施选项（预算、标签、异步模式）在创建工具时配置——语义层面的取舍交给模型
- 通过 `bankId` 实现多用户记忆隔离
- 完整 TypeScript 支持

## 安装与配置

Hindsight 可使用 Docker 本地运行，或作为云服务使用。

### 自托管（Docker）

```bash
export OPENAI_API_KEY=your-key
docker run --rm -it -p 8888:8888 -p 9999:9999 \
  -e HINDSIGHT_API_LLM_API_KEY=$OPENAI_API_KEY \
  -v $HOME/.hindsight-docker:/home/hindsight/.pg0 \
  ghcr.io/vectorize-io/hindsight:latest
```

API 地址为 `http://localhost:8888`，Web UI 为 `http://localhost:9999`。

### 云服务

注册后在 [Hindsight 控制台](https://ui.hindsight.vectorize.io/signup) 获取 API URL。

## 创建工具

初始化 `HindsightClient`，将其与标识记忆库的 `bankId`（通常为用户 ID）一并传给 `createHindsightTools`：

```ts
import { HindsightClient } from '@vectorize-io/hindsight-client';
import { createHindsightTools } from '@vectorize-io/hindsight-ai-sdk';

const client = new HindsightClient({ baseUrl: process.env.HINDSIGHT_API_URL });

const tools = createHindsightTools({
  client,
  bankId: 'user-123',
});
```

## 基础用法

### generateText

```ts
import { HindsightClient } from '@vectorize-io/hindsight-client';
import { createHindsightTools } from '@vectorize-io/hindsight-ai-sdk';
import { generateText, isStepCount } from 'ai';
import { openai } from '@ai-sdk/openai';

const client = new HindsightClient({ baseUrl: process.env.HINDSIGHT_API_URL });
const tools = createHindsightTools({ client, bankId: 'user-123' });

const { text } = await generateText({
  model: openai('gpt-4o'),
  tools,
  stopWhen: isStepCount(5),
  system: 'You are a helpful assistant with long-term memory.',
  prompt: 'Remember that I prefer dark mode and large fonts.',
});
```

### ToolLoopAgent

```ts
import { ToolLoopAgent } from 'ai';
import { openai } from '@ai-sdk/openai';
import { HindsightClient } from '@vectorize-io/hindsight-client';
import { createHindsightTools } from '@vectorize-io/hindsight-ai-sdk';

const client = new HindsightClient({ baseUrl: process.env.HINDSIGHT_API_URL });

const agent = new ToolLoopAgent({
  model: openai('gpt-4o'),
  tools: createHindsightTools({ client, bankId: 'user-123' }),
  instructions: 'You are a helpful assistant with long-term memory.',
});

const result = await agent.generate({
  prompt: 'Remember that my favorite editor is Neovim',
});
```

## 多用户记忆

在多用户应用中，应在请求处理器内创建工具，以便每次请求使用已认证用户对应的 `bankId`：

```ts
// app/api/chat/route.ts
import { streamText, isStepCount, convertToModelMessages } from 'ai';
import { openai } from '@ai-sdk/openai';
import { HindsightClient } from '@vectorize-io/hindsight-client';
import { createHindsightTools } from '@vectorize-io/hindsight-ai-sdk';

const hindsightClient = new HindsightClient({
  baseUrl: process.env.HINDSIGHT_API_URL,
});

export async function POST(req: Request) {
  const { messages, userId } = await req.json();

  const tools = createHindsightTools({
    client: hindsightClient,
    bankId: userId,
  });

  return streamText({
    model: openai('gpt-4o'),
    tools,
    stopWhen: isStepCount(5),
    system: 'You are a helpful assistant with long-term memory.',
    messages: await convertToModelMessages(messages),
  }).toUIMessageStreamResponse();
}
```

`HindsightClient` 实例共享（在模块顶层创建一次），而 `createHindsightTools` 在每个请求内按当前用户 ID 调用。

## 配置

基础设施选项在创建工具时配置，由应用控制成本、标签与性能，而将「记什么、搜什么」等语义决策交给模型。

```ts
const tools = createHindsightTools({
  client,
  bankId: userId,
  retain: {
    async: true,
    tags: ['env:prod', 'app:support'],
    metadata: { version: '2.0' },
  },
  recall: {
    budget: 'high',
    types: ['experience', 'world'],
    maxTokens: 2048,
    includeEntities: true,
  },
  reflect: {
    budget: 'mid',
  },
});
```

### `retain` 选项

| 参数          | 类型                     | 默认值   | 说明                               |
| ------------- | ------------------------ | -------- | ---------------------------------- |
| `async`       | `boolean`                | `false`  | 即发即忘的写入模式                 |
| `tags`        | `string[]`               | —        | 应用于所有保留记忆的标签           |
| `metadata`    | `Record<string, string>` | —        | 应用于所有保留记忆的元数据         |
| `description` | `string`                 | 内置     | 覆盖默认工具描述                   |

### `recall` 选项

| 参数              | 类型                                           | 默认值     | 说明                         |
| ----------------- | ---------------------------------------------- | ---------- | ---------------------------- |
| `budget`          | `'low' \| 'mid' \| 'high'`                     | `'mid'`    | 检索深度与延迟权衡           |
| `types`           | `('world' \| 'experience' \| 'observation')[]` | 全部       | 将结果限制为指定事实类型     |
| `maxTokens`       | `number`                                       | API 默认   | 返回的最大总 token 数        |
| `includeEntities` | `boolean`                                      | `false`    | 在结果中包含实体观察         |
| `includeChunks`   | `boolean`                                      | `false`    | 在结果中包含原始源片段       |
| `description`     | `string`                                       | 内置       | 覆盖默认工具描述             |

### `reflect` 选项

| 参数          | 类型                       | 默认值     | 说明                     |
| ------------- | -------------------------- | ---------- | ------------------------ |
| `budget`      | `'low' \| 'mid' \| 'high'` | `'mid'`    | 合成深度与延迟权衡       |
| `maxTokens`   | `number`                   | API 默认   | 最大响应 token 数        |
| `description` | `string`                   | 内置       | 覆盖默认工具描述         |

## 记忆工具

| 工具             | 说明                         |
| ---------------- | ---------------------------- |
| `retain`         | 将信息存入记忆库             |
| `recall`         | 使用查询搜索记忆             |
| `reflect`        | 从已存记忆中合成洞见         |
| `getMentalModel` | 检索结构化知识模型           |
| `getDocument`    | 按标识符检索已存储文档       |
