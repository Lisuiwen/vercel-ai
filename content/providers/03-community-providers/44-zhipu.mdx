# 智谱 AI（Z.AI）提供商

[Zhipu AI Provider](https://github.com/Xiang-CH/zhipu-ai-provider) 是面向 [AI SDK](/) 的社区提供商，可与 [智谱AI](https://www.zhipuai.cn/) 在 [bigmodel.cn](https://bigmodel.cn/) 或 [z.ai](https://docs.z.ai/) 提供的 **GLM** 与嵌入模型无缝集成。

在 `.env` 或环境中配置 API 密钥。

```bash
ZHIPU_API_KEY=<your-api-key>
```

## 提供商实例

可从 `zhipu-ai-provider` 导入默认提供商实例 `zhipu`（自动从环境变量 `ZHIPU_API_KEY` 读取密钥）：

```ts
import { zhipu } from 'zhipu-ai-provider';
```

也可使用 `createZhipu` 创建自定义实例（若使用非中国区 [Z.AI](https://docs.z.ai/guides) 基础设施）：

```ts
import { createZhipu } from 'zhipu-ai-provider';

const zhipu = createZhipu({
  baseURL: 'https://api.z.ai/api/paas/v4',
  apiKey: 'your-api-key',
});
```

可使用以下可选设置自定义智谱提供商实例：

- **baseURL**：_string_
  - 为 API 调用使用不同的 URL 前缀，例如代理。默认前缀为 `https://open.bigmodel.cn/api/paas/v4`。

- **apiKey**：_string_
  - 智谱 [BigModel 平台](https://bigmodel.cn/) 的 API 密钥。若未提供，提供商将尝试从环境变量 `ZHIPU_API_KEY` 读取。

- **headers**：_Record\<string, string\>_
  - 请求中要包含的自定义标头。

## 示例

```ts
import { zhipu } from 'zhipu-ai-provider';

const { text } = await generateText({
  model: zhipu('glm-4-plus'),
  prompt: 'Why is the sky blue?',
});

console.log(result);
```
