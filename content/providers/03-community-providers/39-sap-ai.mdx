# SAP AI Core 提供商

[jerome-benoit/sap-ai-provider](https://github.com/jerome-benoit/sap-ai-provider) 是基于官方 **@sap-ai-sdk/orchestration** 与 **@sap-ai-sdk/foundation-models** 的 SAP AI Core 社区提供商。

该仓库发布两个 npm 包：

- `@jerome-benoit/sap-ai-provider` — 面向 AI SDK 5.x 与 6.x 的 Language Model V4（推荐）
- `@jerome-benoit/sap-ai-provider-v2` — 面向 AI SDK 5.x 的 Language Model V2

## 功能

两种提供商均提供：

- **双 API 支持**：可在 Orchestration API（数据脱敏、内容过滤、文档 grounding、翻译）与 Foundation Models API（`logprobs`、`seed`、`dataSources`）之间选择
- **工具调用**与**多模态输入**（图像）
- **流式支持**，用于实时文本生成
- **文本嵌入**，用于 RAG 与语义检索

## 提供商实例

可从 `@jerome-benoit/sap-ai-provider` 导入默认提供商实例 `sapai`：

```ts
import { sapai } from '@jerome-benoit/sap-ai-provider';
```

如需自定义配置，可导入 `createSAPAIProvider` 并创建实例：

```ts
import { createSAPAIProvider } from '@jerome-benoit/sap-ai-provider';

const sapai = createSAPAIProvider({
  resourceGroup: 'default',
  api: 'orchestration', // or 'foundation-models'
});
```

可使用以下可选设置自定义 SAP AI 提供商实例：

- **resourceGroup** _string_

  SAP AI Core 资源组。默认为 `'default'`。

- **deploymentId** _string_

  指定部署 ID。若未提供，SDK 会自动解析部署。

- **api** _'orchestration' | 'foundation-models'_

  使用的 API。默认为 `'orchestration'`。可通过 `providerOptions` 按模型或按调用覆盖。

- **name** _string_

  用作 `providerOptions` 键名的提供商名称。默认为 `'sap-ai'`。

- **defaultSettings** _object_

  默认模型设置，包含 `modelParams`、`masking`、`filtering`、`grounding`、`translation`。

## 语言模型

使用提供商实例可创建调用 SAP AI Core API 的模型。第一个参数为模型 id。命名遵循带厂商前缀的 SAP AI Core 约定：

```ts
const model = sapai('gpt-4.1');
const claudeModel = sapai('anthropic--claude-3.5-sonnet');
const geminiModel = sapai('gemini-2.0-flash');
```

<Note>
  模型可用性取决于 SAP AI Core 租户配置与区域。
</Note>

## 嵌入模型

使用 `.embeddingModel()` 工厂方法可创建调用 SAP AI Core 嵌入 API 的模型：

```ts
const model = sapai.embeddingModel('text-embedding-3-small');
```
