# Gemini CLI 提供商

[ai-sdk-provider-gemini-cli](https://github.com/ben-vargas/ai-sdk-provider-gemini-cli) 社区提供商通过 [@google/gemini-cli-core](https://www.npmjs.com/package/@google/gemini-cli-core) 使用 Google Gemini 模型；适合已有 Gemini Code Assist 订阅或使用 API Key 认证的场景。

## 版本兼容

| Provider Version | AI SDK Version | NPM Tag     | Status      |
| ---------------- | -------------- | ----------- | ----------- |
| 2.x              | v6             | `latest`    | Stable      |
| 1.x              | v5             | `ai-sdk-v5` | Maintenance |
| 0.x              | v4             | `ai-sdk-v4` | Legacy      |

```bash
# AI SDK v6 (default)
npm install ai-sdk-provider-gemini-cli ai

# AI SDK v5
npm install ai-sdk-provider-gemini-cli@ai-sdk-v5 ai@^5.0.0

# AI SDK v4
npm install ai-sdk-provider-gemini-cli@ai-sdk-v4 ai@^4.0.0
```

## 提供商实例

导入 `createGeminiProvider`，按认证方式创建提供商实例：

```ts
import { createGeminiProvider } from 'ai-sdk-provider-gemini-cli';

// OAuth authentication (default if authType omitted)
const gemini = createGeminiProvider({ authType: 'oauth-personal' });

// API key authentication
const gemini = createGeminiProvider({
  authType: 'api-key', // or 'gemini-api-key'
  apiKey: process.env.GEMINI_API_KEY,
});

// Vertex AI authentication
const gemini = createGeminiProvider({
  authType: 'vertex-ai',
  vertexAI: {
    projectId: 'my-project',
    location: 'us-central1',
  },
});

// Google Auth Library
const gemini = createGeminiProvider({
  authType: 'google-auth-library',
  googleAuth: myGoogleAuthInstance,
});
```

认证选项：

- **authType** _'oauth' | 'oauth-personal' | 'api-key' | 'gemini-api-key' | 'vertex-ai' | 'google-auth-library'_ — 可选，默认 `'oauth-personal'`。
- **apiKey** _string_ — `'api-key'` / `'gemini-api-key'` 时必填。
- **vertexAI** _\{ projectId, location \}_ — `'vertex-ai'` 时必填。
- **googleAuth** _GoogleAuth_ — `'google-auth-library'` 时必填。
- **cacheDir** _string_ — OAuth 凭证缓存目录（可选）。
- **proxy** _string_ — HTTP/HTTPS 代理 URL。

## 语言模型

通过提供商实例创建经由 CLI 调用 Gemini 的模型：

```ts
const model = gemini('gemini-2.5-pro');
```

支持的模型：

- **gemini-3.1-pro-preview**：增强推理的最新模型（支持 `thinkingLevel`）
- **gemini-3-flash-preview**：快速 Gemini 3 模型（支持 `thinkingLevel`）
- **gemini-3.1-flash-lite-preview**：增强推理的最新轻量预览（支持 `thinkingLevel`）
- **gemini-2.5-pro**：生产可用，输出上限 64K tokens（支持 `thinkingBudget`）
- **gemini-2.5-flash**：快速高效，输出上限 64K tokens（支持 `thinkingBudget`）

### 示例

```ts
import { createGeminiProvider } from 'ai-sdk-provider-gemini-cli';
import { generateText } from 'ai';

const gemini = createGeminiProvider({
  authType: 'oauth-personal',
});

const { text } = await generateText({
  model: gemini('gemini-2.5-pro'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

### 模型设置

```ts
const model = gemini('gemini-3.1-pro-preview', {
  temperature: 0.7,
  topP: 0.95,
  topK: 40,
  maxOutputTokens: 8192,
  thinkingConfig: {
    thinkingLevel: 'medium', // 'low' | 'medium' | 'high' | 'minimal'
  },
  verbose: true, // Enable debug logging
  logger: customLogger, // Custom logger (or false to disable)
});
```

## 认证

### OAuth 认证（推荐）

全局安装并登录 Gemini CLI：

```bash
npm install -g @google/gemini-cli
gemini  # Follow the interactive authentication setup
```

随后在代码中使用 `authType: 'oauth-personal'`。

### API Key 认证

1. 在 [Google AI Studio](https://aistudio.google.com/apikey) 生成 API Key。
2. `export GEMINI_API_KEY="YOUR_API_KEY"`。
3. 使用 `authType: 'api-key'` 并传入密钥。

## 要求

- Node.js 20 或更高
- OAuth 场景需全局安装 Gemini CLI
- 有效的 Google 账号或 Gemini API Key

详见 [provider documentation](https://github.com/ben-vargas/ai-sdk-provider-gemini-cli)。
