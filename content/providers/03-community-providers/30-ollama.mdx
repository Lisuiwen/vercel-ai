# Ollama 提供商

AI SDK 通过两个社区提供商支持 [Ollama](https://ollama.com/)：

- [nordwestt/ollama-ai-provider-v2](https://github.com/nordwestt/ollama-ai-provider-v2) — 直连 HTTP API
- [ai-sdk-ollama](https://github.com/jagreehal/ai-sdk-ollama) — 基于官方 Ollama JavaScript 客户端

二者均为 AI SDK 提供语言模型支持，实现方式与功能集合不同。

## 如何选择提供商

AI SDK 生态中有多种 Ollama 提供商，各自适用于不同场景：

### 简单文本生成

[nordwestt/ollama-ai-provider-v2](https://github.com/nordwestt/ollama-ai-provider-v2) 通过直连 HTTP API 访问 Ollama 模型，适合基础文本生成与快速上手。

### 高级特性与工具可靠性

在需要以下能力时，推荐使用 [`ai-sdk-ollama`](https://github.com/jagreehal/ai-sdk-ollama)（作者 jagreehal）：

- **可靠的工具调用**，保证完整响应（缓解常见的空响应问题）
- **联网搜索**，基于 [Ollama 新版联网搜索 API](https://docs.ollama.com/web-search) 获取最新信息
- **跨环境支持**，自动检测 Node.js 与浏览器
- **高级 Ollama 功能**，如 `mirostat`、`repeat_penalty`、`num_ctx` 等细粒度控制
- **更高可靠性**，官方客户端内置错误处理与重试

关键技术优势：

- 基于官方 [`Ollama`](https://www.npmjs.com/package/ollama) JavaScript 客户端库
- 同时支持 CommonJS 与 ESM
- 完整 TypeScript 支持与类型安全的 Ollama 专属选项

两种提供商均实现 AI SDK 规范，并提供良好的 TypeScript 支持。请根据项目复杂度与功能需求选择。

## 提供商实例

可从 `ollama-ai-provider-v2` 导入默认提供商实例 `ollama`：

```ts
import { ollama } from 'ollama-ai-provider-v2';
```

如需自定义配置，可导入 `createOllama` 并创建实例：

```ts
import { createOllama } from 'ollama-ai-provider-v2';

const ollama = createOllama({
  // optional settings, e.g.
  baseURL: 'https://api.ollama.com',
});
```

可使用以下可选设置自定义 Ollama 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀，例如经由代理。
  默认前缀为 `http://localhost:11434/api`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义标头。

## 语言模型

使用提供商实例可创建调用 [Ollama Chat Completion API](https://github.com/ollama/ollama/blob/main/docs/api.md#generate-a-chat-completion) 的模型。
第一个参数为模型 id，例如 `phi3`。部分模型具备多模态能力。

```ts
const model = ollama('phi3');
```

更多模型见 [Ollama Library](https://ollama.com/library) 主页。

### 模型能力

该提供商可使用 qwen3 等混合推理模型，并支持在消息间切换推理行为。

```ts
import { ollama } from 'ollama-ai-provider-v2';
import { generateText } from 'ai';

const { text } = await generateText({
  model: ollama('qwen3:4b'),
  providerOptions: { ollama: { think: true } },
  prompt:
    'Write a vegetarian lasagna recipe for 4 people, but really think about it',
});
```

## 嵌入模型

使用 `.embeddingModel()` 工厂方法可创建调用 [Ollama embeddings API](https://github.com/ollama/ollama/blob/main/docs/api.md#generate-embeddings) 的模型。

```ts
const model = ollama.embeddingModel('nomic-embed-text');

const { embeddings } = await embedMany({
  model: model,
  values: ['sunny day at the beach', 'rainy afternoon in the city'],
});

console.log(
  `cosine similarity: ${cosineSimilarity(embeddings[0], embeddings[1])}`,
);
```
