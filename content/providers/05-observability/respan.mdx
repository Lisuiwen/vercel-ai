# Respan 可观测性

[Respan](https://www.respan.ai/)（原 Keywords AI）是面向 LLM 工程的平台，涵盖可观测、评估与网关路由。Respan 与 AI SDK 集成可提供：

- 每次 AI SDK 调用的 tracing（输入、输出、token 用量、成本、延迟）
- 工具调用与结构化输出捕获
- 可选网关：单一端点背后是 250+ 模型

## 设置

AI SDK 支持通过 OpenTelemetry 进行 tracing。使用 `@respan/instrumentation-vercel` 中的 `VercelAIInstrumentor`，traces 会自动导出到 Respan。

### 安装

```bash
npm install ai @ai-sdk/openai @respan/respan @respan/instrumentation-vercel
```

### 配置环境变量

```bash filename=".env"
RESPAN_API_KEY="your-respan-api-key"
```

在 [platform.respan.ai](https://platform.respan.ai/platform/api/api-keys) 获取 API key。

### 初始化 Respan

<Tabs items={["Next.js", "Serverless / Node"]}>

<Tab>

在 Next.js 项目根目录创建 `instrumentation.ts`：

```ts filename="instrumentation.ts"
import { Respan } from '@respan/respan';
import { VercelAIInstrumentor } from '@respan/instrumentation-vercel';

export async function register() {
  const respan = new Respan({
    apiKey: process.env.RESPAN_API_KEY,
    instrumentations: [new VercelAIInstrumentor()],
  });
  await respan.initialize();
}
```

随后在 `next.config.ts` 中将 SDK 包标为服务端外部依赖：

```ts filename="next.config.ts"
import type { NextConfig } from 'next';

const nextConfig: NextConfig = {
  serverExternalPackages: ['@respan/respan', '@respan/instrumentation-vercel'],
};

export default nextConfig;
```

</Tab>

<Tab>

在处理函数或入口文件顶部初始化 Respan：

```ts
import { Respan } from '@respan/respan';
import { VercelAIInstrumentor } from '@respan/instrumentation-vercel';

const respan = new Respan({
  apiKey: process.env.RESPAN_API_KEY,
  instrumentations: [new VercelAIInstrumentor()],

await respan.initialize();
```

### 启用 telemetry 生成文本

```ts
import { openai } from '@ai-sdk/openai';
import { generateText } from 'ai';

const result = await generateText({
  model: openai('gpt-4o-mini'),
  prompt: 'Tell me a joke about AI',
  experimental_telemetry: { isEnabled: true },
});
```

Span 会出现在 [Respan traces 页面](https://platform.respan.ai/platform/traces)，包含输入、输出、token 用量与成本。

## 网关路由（可选）

要将 Respan 作为跨厂商的统一网关，可将任意 AI SDK provider 指向 `https://api.respan.ai/api`，并使用 `RESPAN_API_KEY`。各厂商密钥在 Respan 中管理；运行时只需 Respan key。

```ts
import { createOpenAI } from '@ai-sdk/openai';
import { generateText } from 'ai';

const provider = createOpenAI({
  apiKey: process.env.RESPAN_API_KEY!,
  baseURL: 'https://api.respan.ai/api',
});

const result = await generateText({
  model: provider('gpt-4.1-nano'),
  prompt: 'Tell me a joke about AI',
  experimental_telemetry: { isEnabled: true },
});
```

同一网关端点在配合各自的 AI SDK provider（`createAnthropic`、`createGoogleGenerativeAI`）及对应 base path（`/api/anthropic`、`/api/google/gemini`）时可接受 Anthropic 与 Google 模型。
