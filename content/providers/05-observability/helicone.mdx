# Helicone 可观测性

[Helicone](https://helicone.ai) 是开源 LLM 可观测平台，帮助你监控、分析与优化 AI 应用。内置可观测能力会自动追踪每次请求，无需额外埋点即可全面了解性能、成本、用户行为与模型使用情况。

## 设置

Helicone provider 位于 `@helicone/ai-sdk-provider` 包中，安装方式：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @helicone/ai-sdk-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @helicone/ai-sdk-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @helicone/ai-sdk-provider" dark />
  </Tab>
  <Tab>
    <Snippet text="bun add @helicone/ai-sdk-provider" dark />
  </Tab>
</Tabs>

接入 Helicone：

1. 在 [helicone.ai](https://helicone.ai) 创建账户
2. 在 [Helicone Dashboard](https://us.helicone.ai/settings/api-keys) 获取 API key
3. 将 API key 设为环境变量：
   ```bash filename=".env"
   HELICONE_API_KEY=your-helicone-api-key
   ```
4. 在应用中使用 Helicone：

   ```javascript
   import { createHelicone } from '@helicone/ai-sdk-provider';
   import { generateText } from 'ai';

   const helicone = createHelicone({
     apiKey: process.env.HELICONE_API_KEY,
   });

   // Use the provider with any supported model: https://helicone.ai/models
   const result = await generateText({
     model: helicone('claude-4.5-haiku'),
     prompt: 'Hello world',
   });

   console.log(result.text);
   ```

完成！请求已通过 Helicone 记录与监控，可观测能力自动生效。

## 核心可观测能力

Helicone 无需额外埋点即可为 AI 应用提供全面可观测：

**自动请求追踪**

- 每次请求自动记录，含完整请求/响应数据
- 实时追踪延迟、token、成本与模型表现
- 无需 OpenTelemetry 或额外配置

**分析面板**

- 汇总所有 AI 请求的指标：成本、延迟、token 用量与错误率
- 按用户、会话、模型或自定义属性筛选
- 发现瓶颈并优化模型选择

**用户与会话分析**

- 追踪单用户行为与使用模式
- 通过会话追踪监控对话流
- 分析参与度与功能采用情况

**成本监控**

- 按请求、用户、功能或模型实时统计成本
- 预算告警与成本优化洞察
- 对比不同模型与提供商的成本

**调试与排障**

- 每次调用记录完整请求/响应
- 带上下文的错误追踪
- 快速检索与筛选以定位问题

## 可观测配置

### 用户追踪

追踪单个用户行为并分析跨应用的使用模式，有助于识别活跃用户、观察每用户成本等：

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await generateText({
  model: helicone('gpt-4o-mini', {
    extraBody: {
      helicone: {
        userId: 'user@example.com',
      },
    },
  }),
  prompt: 'Hello world',
});
```

**可追踪内容：**

- 每用户总请求数
- 每用户成本
- 每用户平均延迟
- 按用户分段的最常见场景

[→ 进一步了解 User Metrics](https://docs.helicone.ai/features/advanced-usage/user-metrics)

### 自定义属性

添加结构化元数据，按功能、环境或任意自定义维度拆分与分析请求，在分析面板中获得更强筛选与洞察：

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await generateText({
  model: helicone('gpt-4o-mini', {
    extraBody: {
      helicone: {
        properties: {
          feature: 'translation',
          source: 'mobile-app',
          language: 'French',
          environment: 'production',
        },
      },
    },
  }),
  prompt: 'Translate this text to French',
});
```

**自定义属性适用场景：**

- 跨功能或环境对比表现
- 按产品区域或客户等级追踪成本
- 识别驱动 AI 用量的功能
- 通过标签区分实验以做 A/B 测试

[→ 进一步了解 Custom Properties](https://docs.helicone.ai/features/advanced-usage/custom-properties)

### 会话追踪

将相关请求归组为会话，分析对话流与多轮交互，有助于理解用户路径并调试复杂会话：

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await generateText({
  model: helicone('gpt-4o-mini', {
    extraBody: {
      helicone: {
        sessionId: 'convo-123',
        sessionName: 'Travel Planning',
        sessionPath: '/chats/travel',
      },
    },
  }),
  prompt: 'Tell me more about that',
});
```

**会话追踪的收益：**

- 在单一时间线查看完整对话历史
- 计算单会话/对话总成本
- 度量会话时长与消息数量
- 发现多轮对话中的流失节点
- 通过回放整条对话排查问题

[→ 进一步了解 Sessions](https://docs.helicone.ai/features/sessions)

## 高级可观测能力

### 标签与组织

为分析面板中的请求添加标签以便组织与筛选：

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { generateText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await generateText({
  model: helicone('gpt-4o-mini', {
    extraBody: {
      helicone: {
        tags: ['customer-support', 'urgent'],
        properties: {
          ticketId: 'TICKET-789',
          priority: 'high',
          department: 'support',
        },
      },
    },
  }),
  prompt: 'Help resolve this customer issue',
});
```

**标签带来的洞察：**

- 按标签筛选与分组请求
- 跨类别追踪表现
- 发现带标签请求中的模式
- 围绕特定标签构建自定义面板

[→ 进一步了解 Helicone Features](https://docs.helicone.ai)

### 流式响应追踪

对流式响应进行完整可观测，含首 token 时间与总流式耗时：

```javascript
import { createHelicone } from '@helicone/ai-sdk-provider';
import { streamText } from 'ai';

const helicone = createHelicone({
  apiKey: process.env.HELICONE_API_KEY,
});

const result = await streamText({
  model: helicone('gpt-4o-mini', {
    extraBody: {
      helicone: {
        userId: 'user@example.com',
        sessionId: 'stream-session-123',
        tags: ['streaming', 'content-generation'],
      },
    },
  }),
  prompt: 'Write a short story about AI',
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}
```

**流式场景追踪的指标：**

- Time to first token（TTFT）
- 总流式时长
- Tokens per second
- 对流仍记录完整请求/响应
- 实时应用的用户体验指标
- 对流式响应同样附带会话、用户、标签等元数据
