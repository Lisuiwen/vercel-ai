# Arize AX 可观测性

[Arize AX](https://arize.com/docs/ax) 是面向智能体与复杂 AI 系统的企业级可观测、评估与实验平台，帮助团队严谨地开发与改进真实场景中的 AI 应用。

<Note>
  本指南也可在 [Arize AX 文档](https://arize.com/docs/ax/integrations/ts-js-agent-frameworks/vercel) 中找到。
</Note>

## 设置

Arize AX 提供一流的 OpenTelemetry 集成，可在 Next.js 与 Node.js 环境中直接与 AI SDK 配合使用。

<Note>
  Arize AX 提供
  [OpenInferenceSimpleSpanProcessor](https://github.com/Arize-ai/openinference/blob/main/js/packages/openinference-vercel/src/OpenInferenceSpanProcessor.ts#L32)
  与
  [OpenInferenceBatchSpanProcessor](https://github.com/Arize-ai/openinference/blob/main/js/packages/openinference-vercel/src/OpenInferenceSpanProcessor.ts#L86)。
  下文所有示例均可搭配 simple 或 batch 处理器使用。关于 simple / batch span processor 的更多信息，请参阅
  [文档](https://arize.com/docs/ax/observe/tracing/configure/batch-vs-simple-span-processor#batch-vs-simple-span-processor)。
</Note>

### Next.js

在 Next.js 应用中，将 OpenInference span 处理器之一与 `@vercel/otel` 的 `registerOtel` 配合使用。

首先，安装 AI SDK、OpenTelemetry 与 OpenInference 所需依赖：

```bash
npm install ai @ai-sdk/openai @ai-sdk/otel @vercel/otel @arizeai/openinference-vercel @opentelemetry/exporter-trace-otlp-proto
```

然后，在 `instrumentation.ts` 中加入以下内容，包含 AI SDK 遥测集成注册：

```typescript filename="instrumentation"
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { registerOTel } from '@vercel/otel';
import {
  isOpenInferenceSpan,
  OpenInferenceSimpleSpanProcessor,
} from '@arizeai/openinference-vercel';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-proto';

registerTelemetry(new LegacyOpenTelemetry());

export function register() {
  registerOTel({
    attributes: {
      model_id: 'my-ai-app',
      model_version: '1.0.0',
    },
    spanProcessors: [
      new OpenInferenceSimpleSpanProcessor({
        exporter: new OTLPTraceExporter({
          url: 'https://otlp.arize.com/v1/traces',
          headers: {
            space_id: process.env.ARIZE_SPACE_ID,
            api_key: process.env.ARIZE_API_KEY,
          },
        }),
        // Optionally add a span filter to only include AI related spans
        spanFilter: isOpenInferenceSpan,
      }),
    ],
  });
}
```

Span 将出现在 Arize AX 中，对应上面 `model_id` 字段指定的项目。

注册集成后，所有 AI SDK 调用会自动发出遥测：

```typescript
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';

const result = await generateText({
  model: openai('gpt-5-mini'),
  prompt: 'Please write a haiku.',
});
```

### Node.js

在 Node.js 中可以使用 `NodeSDK` 或 `NodeTraceProvider`。

#### NodeSDK

首先，安装 AI SDK、OpenTelemetry 与 OpenInference 所需依赖：

```bash
npm install ai @ai-sdk/openai @ai-sdk/otel @opentelemetry/sdk-node @arizeai/openinference-vercel @opentelemetry/exporter-trace-otlp-proto @opentelemetry/resources
```

然后，在 instrumentation.ts 中加入以下内容：

```typescript
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import {
  isOpenInferenceSpan,
  OpenInferenceSimpleSpanProcessor,
} from '@arizeai/openinference-vercel';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-proto';
import { resourceFromAttributes } from '@opentelemetry/resources';
import { NodeSDK } from '@opentelemetry/sdk-node';

const sdk = new NodeSDK({
  resource: resourceFromAttributes({
    model_id: 'my-ai-app',
    model_version: '1.0.0',
  }),
  spanProcessors: [
    new OpenInferenceSimpleSpanProcessor({
      exporter: new OTLPTraceExporter({
        url: 'https://otlp.arize.com/v1/traces',
        headers: {
          space_id: process.env.ARIZE_SPACE_ID,
          api_key: process.env.ARIZE_API_KEY,
        },
      }),
      spanFilter: isOpenInferenceSpan,
    }),
  ],
});

sdk.start();
registerTelemetry(new LegacyOpenTelemetry());
```

Span 将出现在 Arize AX 中，对应上面 `model_id` 字段指定的项目。

注册集成后，所有 AI SDK 调用会自动发出遥测：

```typescript
const result = await generateText({
  model: openai('gpt-5-mini'),
  prompt: 'Please write a haiku.',
});
```

#### NodeTraceProvider

首先，安装 AI SDK、OpenTelemetry 与 OpenInference 所需依赖：

```bash
npm install ai @ai-sdk/openai @ai-sdk/otel @opentelemetry/sdk-trace-node @arizeai/openinference-vercel @opentelemetry/exporter-trace-otlp-proto @opentelemetry/resources
```

然后，在 instrumentation.ts 中加入以下内容：

```typescript
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import {
  isOpenInferenceSpan,
  OpenInferenceSimpleSpanProcessor,
} from '@arizeai/openinference-vercel';
import { OTLPTraceExporter } from '@opentelemetry/exporter-trace-otlp-proto';
import { resourceFromAttributes } from '@opentelemetry/resources';
import { NodeTracerProvider } from '@opentelemetry/sdk-trace-node';

const provider = new NodeTracerProvider({
  resource: resourceFromAttributes({
    model_id: 'my-ai-app',
    model_version: '1.0.0',
  }),
  spanProcessors: [
    new OpenInferenceSimpleSpanProcessor({
      exporter: new OTLPTraceExporter({
        url: 'https://otlp.arize.com/v1/traces',
        headers: {
          space_id: process.env.ARIZE_SPACE_ID,
          api_key: process.env.ARIZE_API_KEY,
        },
      }),
      spanFilter: isOpenInferenceSpan,
    }),
  ],
});
provider.register();
registerTelemetry(new LegacyOpenTelemetry());
```

Span 将出现在 Arize AX 中，对应上面 `model_id` 字段指定的项目。

注册集成后，所有 AI SDK 调用会自动发出遥测：

```typescript
const result = await generateText({
  model: openai('gpt-5-mini'),
  prompt: 'Please write a haiku.',
});
```

## 资源

将 span 发送到 Arize AX 项目后，可进一步了解其他功能：

- 在 [prompt playground](https://arize.com/docs/ax/prompts/prompt-playground) 中重跑 span，迭代并比较提示词与参数
- 将 span 加入 [datasets](https://arize.com/docs/ax/develop/datasets)，用于评估与开发工作流
- 对入库 span 持续运行 [online evaluations](https://arize.com/docs/ax/evaluate/online-evals)，了解应用表现

AX 提供用于管理数据集与评估的 [TypeScript client](https://www.npmjs.com/package/@arizeai/ax-client)。
