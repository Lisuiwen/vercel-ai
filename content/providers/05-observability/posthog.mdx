# PostHog LLM Analytics

[PostHog](https://posthog.com/) 是开源的产品分析平台。借助 PostHog LLM Analytics 集成，你可以在统一的产品分析视角下追踪 LLM 用量，理解 AI 功能如何影响用户行为。PostHog 提供：

- [LLM analytics](https://posthog.com/docs/llm-analytics/start-here) —— 按模型与用户的成本、延迟与 token 用量
- [Product analytics](https://posthog.com/docs/product-analytics) —— 将 AI 用量与用户行为及业务指标关联
- [Experiments](https://posthog.com/docs/experiments) —— 对 prompts、模型与 AI 功能做 A/B 测试

## 设置

PostHog 通过 [OpenTelemetry](https://opentelemetry.io/docs/) 支持 [AI SDK telemetry data](/docs/ai-sdk-core/telemetry)。`@posthog/ai` 包中的 `PostHogTraceExporter` 将 `gen_ai.*` span 发往 PostHog 的 OTLP 接入端点，并自动转换为 `$ai_generation` 事件。

Next.js 通过 [instrumentation hook](https://nextjs.org/docs/app/building-your-application/optimizing/open-telemetry) 内置支持 OpenTelemetry instrumentation。

**步骤 1。** 安装依赖

```bash
npm install @posthog/ai @opentelemetry/sdk-node @opentelemetry/resources @ai-sdk/otel
```

**步骤 2。** 在项目根目录创建 `instrumentation.ts`，在 OTel 设置旁注册 AI SDK 遥测集成：

```ts filename="instrumentation.ts"
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { NodeSDK } from '@opentelemetry/sdk-node';
import { resourceFromAttributes } from '@opentelemetry/resources';
import { PostHogTraceExporter } from '@posthog/ai/otel';

registerTelemetry(new LegacyOpenTelemetry());

export function register() {
  const sdk = new NodeSDK({
    resource: resourceFromAttributes({
      'service.name': 'my-nextjs-app',
    }),
    traceExporter: new PostHogTraceExporter({
      apiKey: process.env.POSTHOG_API_KEY!,
      host: 'https://us.i.posthog.com', // use https://eu.i.posthog.com for EU
    }),
  });
  sdk.start();
}
```

**步骤 3。** 在路由处理函数或 Server Actions 中使用 AI SDK 并启用 telemetry

```ts highlight="6-11"
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';

const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Tell me a fun fact about hedgehogs.',
  telemetry: {
    functionId: 'my-ai-function',
    metadata: {
      posthog_distinct_id: 'user_123', // optional: links events to a PostHog user
    },
  },
});
```

```ts filename="tracing.ts"
import { NodeSDK } from '@opentelemetry/sdk-node';
import { resourceFromAttributes } from '@opentelemetry/resources';
import { PostHogTraceExporter } from '@posthog/ai/otel';
import { registerTelemetry } from 'ai';

  resource: resourceFromAttributes({
    'service.name': 'my-ai-app',
  }),
  traceExporter: new PostHogTraceExporter({
    apiKey: '<your-posthog-project-api-key>',
    host: 'https://us.i.posthog.com', // use https://eu.i.posthog.com for EU
  }),
});
sdk.start();
registerTelemetry(new LegacyOpenTelemetry());
```

**步骤 3。** 使用 AI SDK。注册集成后会自动采集 telemetry：

```ts highlight="6-12"

  },

console.log(result.text);

await sdk.shutdown();
```

## 配置

### 将事件关联到 PostHog 用户

在 `metadata` 中传入 `posthog_distinct_id`，将 LLM 事件与 PostHog 中的特定用户关联。若省略，事件将匿名采集。

```ts highlight="6"
const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Write a short story about a cat.',
  telemetry: {
    metadata: {
      posthog_distinct_id: 'user_123',
    },
  },
});
```

### 隐私控制

将 `recordInputs` 与 `recordOutputs` 设为 `false` 可禁用输入与输出记录：

```ts highlight="5-6"
const result = await generateText({
  model: openai('gpt-4o'),
  prompt: 'Write a short story about a cat.',
  telemetry: {
    recordInputs: false,
    recordOutputs: false,
  },
});
```
