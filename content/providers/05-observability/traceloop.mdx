# Traceloop

[Traceloop](https://www.traceloop.com/) 是用于构建可靠 AI 应用的研发平台。
接入 AI SDK 后，可使用 Traceloop 追踪、监控并实验 LLM 提供商、prompts 与流程。

## 设置

Traceloop 通过 [OpenTelemetry](https://opentelemetry.io/docs/) 支持 [AI SDK telemetry data](/docs/ai-sdk-core/telemetry)。
请在 https://app.traceloop.com 注册并获取 API Key。

### Next.js

要在 Next.js 应用中通过 AI SDK 向 Traceloop 发送 telemetry，在应用的 `.env` 中设置这些环境变量：

```bash
OTEL_EXPORTER_OTLP_ENDPOINT=https://api.traceloop.com
OTEL_EXPORTER_OTLP_HEADERS="Authorization=Bearer <Your API Key>"
```

安装 `@ai-sdk/otel`，并在 `instrumentation.ts` 注册 `LegacyOpenTelemetry`：

```typescript filename="instrumentation.ts"
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';

registerTelemetry(new LegacyOpenTelemetry());
```

随后可在支持的 AI SDK 函数调用上使用 `telemetry` 选项启用遥测：

```typescript highlight="7-12"
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';

const result = await generateText({
  model: openai('gpt-4o-mini'),
  prompt: 'What is 2 + 2?',
  telemetry: {
    metadata: {
      query: 'weather',
      location: 'San Francisco',
    },
  },
});
```
