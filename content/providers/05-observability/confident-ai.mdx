# Confident AI 可观测性

[Confident AI](https://confident-ai.com/) 是面向团队的 LLM 可观测与评估平台，可在开发与生产环境中构建可靠的 AI 应用。

`deepeval-ts` 包与 AI SDK 集成，提供追踪、在线评估与会话分析。

## 设置

启用追踪：安装 `deepeval-ts`、配置 API key，并在应用启动时全局注册 Confident AI 集成。

### 1. 安装 deepeval-ts

```bash
npm install deepeval-ts @ai-sdk/otel
```

### 2. 设置环境变量

在 [Confident AI](https://app.confident-ai.com) 注册或登录以获取 API key，并设为环境变量：

```bash
CONFIDENT_API_KEY="YOUR-PROJECT-API-KEY"
```

### 3. 注册 OpenTelemetry 集成

在应用启动时全局注册 `LegacyOpenTelemetry`，使 AI SDK 发出 Confident AI 可采集的 OpenTelemetry span：

```typescript
import { registerTelemetry } from 'ai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

configureAiSdkTracing();
registerTelemetry(new LegacyOpenTelemetry());
```

## 追踪你的应用

现在可将 `tracer` 传入任意 AI SDK 调用的 `telemetry` 字段，在 Confident AI 平台查看 traces。

以下是使用 Confident AI tracer 追踪多种 AI SDK 函数的示例：

### 生成文本

```typescript
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing();

const { text } = await generateText({
  model: openai('gpt-4o'),
  prompt: 'What are LLMs?',
  telemetry: {
    integrations: new LegacyOpenTelemetry({ tracer }),
  },
});
```

### 流式文本

```typescript
import { streamText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing();

const result = streamText({
  model: openai('gpt-4o'),
  prompt: 'Invent a new holiday and describe its traditions.',
  telemetry: {
    integrations: new LegacyOpenTelemetry({ tracer }),
  },
});

for await (const textPart of result.textStream) {
  console.log(textPart);
}
```

### 带工具调用的文本生成

```typescript
import { generateText, tool, isStepCount } from 'ai';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing();

const result = await generateText({
  model: openai('gpt-4o'),
  tools: {
    weather: tool({
      description: 'Get the weather in a location',
      inputSchema: z.object({
        location: z.string().describe('The location to get the weather for'),
      }),
      execute: async ({ location }) => ({
        location,
        temperature: 72 + Math.floor(Math.random() * 21) - 10,
      }),
    }),
  },
  stopWhen: isStepCount(5),
  prompt: 'What is the weather in San Francisco?',
  telemetry: {
    integrations: new LegacyOpenTelemetry({ tracer }),
  },
});
```

### 生成结构化输出

```typescript
import { generateObject } from 'ai';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing();

const { object } = await generateObject({
  model: openai('gpt-4o'),
  schema: z.object({
    recipe: z.object({
      name: z.string(),
      ingredients: z.array(z.object({ name: z.string(), amount: z.string() })),
      steps: z.array(z.string()),
    }),
  }),
  prompt: 'Generate a lasagna recipe.',
  telemetry: {
    integrations: new LegacyOpenTelemetry({ tracer }),
  },
});
```

## 配置

可向 `configureAiSdkTracing` 传入选项以自定义 trace 分组与评估行为，例如：

- 将相关 traces 分组（如聊天会话）
- 将提示词版本与 traces 关联
- 在 span 与 trace 级别启用在线评估

### 设置 Trace 属性

可传入 `name`、`threadId`、`userId`、`environment` 等属性，便于检索与筛选 traces。

```typescript
import { registerTelemetry, generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing({
  name: 'AI SDK Confident AI Tracing',
  threadId: 'thread-123',
  userId: 'user-456',
  environment: 'production',
});

registerTelemetry(new LegacyOpenTelemetry({ tracer }));

const { text } = await generateText({
  model: openai('gpt-4o'),
  prompt: 'How do you make the best coffee?',
});
```

### 记录托管提示词（Managed Prompts）

若使用 Confident AI Prompt Management，可将 traces 关联到特定提示词版本。向 `configureAiSdkTracing` 传入 `Prompt` 对象，即可在运行时绑定所用提示词版本。

```typescript
import { registerTelemetry, generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing, Prompt } from 'deepeval-ts';

const prompt = new Prompt({ alias: 'my-prompt-alias' });
await prompt.pull();

const tracer = configureAiSdkTracing({
  confidentPrompt: prompt,
});
registerTelemetry(new LegacyOpenTelemetry({ tracer }));

const { text } = await generateText({
  model: openai('gpt-4o'),
  prompt: 'How do you make the best coffee?',
});
```

记录提示词有助于监控线上运行的提示及其长期表现：

![Confident AI 上的提示词可观测性](https://confident-docs.s3.us-east-1.amazonaws.com/llm-tracing:prompt-logging.png)

<Note>
  在传给 `configureAiSdkTracing` 之前务必 **`pull` 提示词**。若不先 pull，Confident AI 上将看不到提示词版本。
</Note>

## 在线评估（Online Evaluations）

Confident AI 支持通过项目中定义的 metric collection 对 traces 自动进行在线评估。启用步骤：

1. 在 Confident AI 平台 [创建 metric collection](https://www.confident-ai.com/docs/metrics/metric-collections)
2. 在 `configureAiSdkTracing` 选项中传入 metric collection 名称
3. 可为 trace、LLM span、tool span 等不同层级指定不同的 metric collection

将 metric collection 挂载到 traces 的示例：

```typescript
import { registerTelemetry, generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { LegacyOpenTelemetry } from '@ai-sdk/otel';
import { configureAiSdkTracing } from 'deepeval-ts';

const tracer = configureAiSdkTracing({
  metricCollection: 'my-trace-metrics',
  llmMetricCollection: 'my-llm-metrics',
  toolMetricCollection: 'my-tool-metrics',
});
registerTelemetry(new LegacyOpenTelemetry({ tracer }));

const { text } = await generateText({
  model: openai('gpt-4o'),
  prompt: 'How do you make the best coffee?',
});
```

此后所有入库 traces 将自动评估，结果可在 Confident AI Observatory 中与 traces 一同查看。
