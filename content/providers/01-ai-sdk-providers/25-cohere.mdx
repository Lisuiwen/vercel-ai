# Cohere 提供商

[Cohere](https://cohere.com/) 提供商为 Cohere Chat API 提供语言模型与嵌入模型支持。

## 安装

Cohere 提供商位于 `@ai-sdk/cohere` 模块中，可使用以下命令安装：

  <Tab>

  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/cohere" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/cohere" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/cohere" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/cohere` 导入默认的 `cohere` 提供商实例：

```ts
import { cohere } from '@ai-sdk/cohere';
```

如需自定义配置，可导入 `createCohere` 并以你的设置创建提供商实例：

```ts
import { createCohere } from '@ai-sdk/cohere';

const cohere = createCohere({
  // custom settings
});
```

可使用以下可选设置自定义 Cohere 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀（例如经由代理）。
  默认前缀为 `https://api.cohere.com/v2`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `COHERE_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义请求头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可作为中间件拦截请求，
  或在测试中提供自定义 `fetch` 实现。

- **generateId** _() => string_

  为每个请求生成唯一 ID 的可选函数。
  默认为 SDK 内置 ID 生成器。

## 语言模型

可使用提供商实例创建调用 [Cohere Chat API](https://docs.cohere.com/v2/docs/chat-api) 的模型。
首个参数为模型 ID，例如 `command-r-plus`。
部分 Cohere Chat 模型支持工具调用。

```ts
const model = cohere('command-r-plus');
```

### 示例

可使用 Cohere 语言模型与 `generateText` 生成文本：

```ts
import { cohere } from '@ai-sdk/cohere';
import { generateText } from 'ai';

const { text } = await generateText({
  model: cohere('command-r-plus'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

Cohere 语言模型也可用于 `streamText`，
并支持通过 [`Output`](/docs/reference/ai-sdk-core/output) 生成结构化数据
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

#### 图像输入

具备视觉能力的模型（如 `command-a-vision-07-2025`）可将图像作为消息内容的一部分。图像可传入原始字节、base64 字符串或 URL：

```ts
import { cohere } from '@ai-sdk/cohere';
import { generateText } from 'ai';
import { readFileSync } from 'node:fs';

const { text } = await generateText({
  model: cohere('command-a-vision-07-2025'),
  messages: [
    {
      role: 'user',
      content: [
        { type: 'text', text: 'Describe the image in detail.' },
        {
          type: 'file',
          mediaType: 'image',
          data: readFileSync('./data/comic-cat.png'),
        },
      ],
    },
  ],
});
```

可在文件片段上使用 `cohere` 提供商选项，将 [图像输入细节](https://docs.cohere.com/docs/image-inputs) 设为 `high`、`low` 或 `auto`：

```ts highlight="2,16-18"
import { cohere } from '@ai-sdk/cohere';
import { generateText } from 'ai';

const { text } = await generateText({
  model: cohere('command-a-vision-07-2025'),
  messages: [
    {
      role: 'user',
      content: [
        { type: 'text', text: 'Describe the image in detail.' },
        {
          type: 'file',
          mediaType: 'image',
          data: 'https://github.com/vercel/ai/blob/main/examples/ai-functions/data/comic-cat.png?raw=true',

          // Cohere specific options - image detail:
          providerOptions: {
            cohere: { detail: 'high' },
          },
        },
      ],
    },
  ],
});
```

#### 推理

Cohere 通过 `command-a-reasoning-08-2025` 模型引入推理能力。更多信息见 https://docs.cohere.com/docs/reasoning。

```ts
import { cohere, type CohereLanguageModelChatOptions } from '@ai-sdk/cohere';
import { generateText } from 'ai';

async function main() {
  const { text, reasoning } = await generateText({
    model: cohere('command-a-reasoning-08-2025'),
    prompt:
      "Alice has 3 brothers and she also has 2 sisters. How many sisters does Alice's brother have?",
    // optional: reasoning options
    providerOptions: {
      cohere: {
        thinking: {
          type: 'enabled',
          tokenBudget: 100,
        },
      } satisfies CohereLanguageModelChatOptions,
    },
  });

  console.log(reasoning);
  console.log(text);
}

main().catch(console.error);
```

## 嵌入模型

可使用 `.embedding()` 工厂方法创建调用 [Cohere Embed API](https://docs.cohere.com/v2/reference/embed) 的模型。

```ts
const model = cohere.embedding('embed-english-v3.0');
```

可使用 Cohere 嵌入模型与 `embed` 生成向量：

```ts
import { cohere, type CohereEmbeddingModelOptions } from '@ai-sdk/cohere';
import { embed } from 'ai';

const { embedding } = await embed({
  model: cohere.embedding('embed-english-v3.0'),
  value: 'sunny day at the beach',
  providerOptions: {
    cohere: {
      inputType: 'search_document',
    } satisfies CohereEmbeddingModelOptions,
  },
});
```

Cohere 嵌入模型还支持通过 `providerOptions.cohere` 传入的附加选项：

```ts
import { cohere, type CohereEmbeddingModelOptions } from '@ai-sdk/cohere';
import { embed } from 'ai';

const { embedding } = await embed({
  model: cohere.embedding('embed-english-v3.0'),
  value: 'sunny day at the beach',
  providerOptions: {
    cohere: {
      inputType: 'search_document',
      truncate: 'END',
    } satisfies CohereEmbeddingModelOptions,
  },
});
```

可用的提供商选项：

- **inputType** _'search_document' | 'search_query' | 'classification' | 'clustering'_

  输入类型。默认为 `search_query`。
  - `search_document`：存入向量数据库、用于检索的文档嵌入。
  - `search_query`：在向量库中检索时所使用的查询嵌入。
  - `classification`：送入文本分类器的嵌入。
  - `clustering`：送入聚类算法的嵌入。

- **truncate** _'NONE' | 'START' | 'END'_

  当输入超出最大 token 长度时的截断策略。
  默认为 `END`。
  - `NONE`：超出最大输入长度则报错。
  - `START`：从开头丢弃内容，直至剩余长度等于模型上限。
  - `END`：从末尾丢弃内容，直至剩余长度等于模型上限。

## 重排序模型

可使用 `.reranking()` 工厂方法创建调用 [Cohere Rerank API](https://docs.cohere.com/v2/reference/rerank) 的模型。

```ts
const model = cohere.reranking('rerank-v3.5');
```

可使用 Cohere 重排序模型与 `rerank` 对文档重排序：

```ts
import { cohere } from '@ai-sdk/cohere';
import { rerank } from 'ai';

const documents = [
  'sunny day at the beach',
  'rainy afternoon in the city',
  'snowy night in the mountains',
];

const { ranking } = await rerank({
  model: cohere.reranking('rerank-v3.5'),
  documents,
  query: 'talk about rain',
  topN: 2,
});

console.log(ranking);
// [
//   { originalIndex: 1, score: 0.9, document: 'rainy afternoon in the city' },
//   { originalIndex: 0, score: 0.3, document: 'sunny day at the beach' }
// ]
```

Cohere 重排序模型还支持通过 `providerOptions.cohere` 传入的附加选项：

```ts
import { cohere, type CohereRerankingModelOptions } from '@ai-sdk/cohere';
import { rerank } from 'ai';

const { ranking } = await rerank({
  model: cohere.reranking('rerank-v3.5'),
  documents: ['sunny day at the beach', 'rainy afternoon in the city'],
  query: 'talk about rain',
  providerOptions: {
    cohere: {
      maxTokensPerDoc: 1000,
      priority: 1,
    } satisfies CohereRerankingModelOptions,
  },
});
```

可用的提供商选项：

- **maxTokensPerDoc** _number_

  每个文档的最大 token 数。默认为 `4096`。

- **priority** _number_

  请求优先级。默认为 `0`。
