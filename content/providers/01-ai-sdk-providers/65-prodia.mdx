# Prodia 提供商

[Prodia](https://prodia.com/) 是面向生成式 AI 的高速推理平台，基于 FLUX 与 Stable Diffusion 模型提供快速图像生成。

## 安装

Prodia 提供商通过 `@ai-sdk/prodia` 模块提供，可使用以下命令安装：

  <Tab>
    <Snippet text="pnpm add @ai-sdk/prodia" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/prodia" dark />

    <Snippet text="yarn add @ai-sdk/prodia" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/prodia" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/prodia` 导入默认的 `prodia` 提供商实例：

```ts
import { prodia } from '@ai-sdk/prodia';
```

如需自定义配置，可导入 `createProdia` 并创建实例：

```ts
import { createProdia } from '@ai-sdk/prodia';

const prodia = createProdia({
  apiKey: 'your-api-key', // optional, defaults to PRODIA_TOKEN environment variable
  baseURL: 'custom-url', // optional
  headers: {
    /* custom headers */
  }, // optional
});
```

可使用以下可选设置自定义 Prodia 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀。
  默认前缀为 `https://inference.prodia.com/v2`。

- **apiKey** _string_

  作为 Bearer token 通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `PRODIA_TOKEN`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 图像模型

可使用 `.image()` 工厂方法创建 Prodia 图像模型。
关于在 AI SDK 中生成图像，参见 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

### 基本用法

```ts
import { writeFileSync } from 'node:fs';
import { prodia } from '@ai-sdk/prodia';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: prodia.image('inference.flux-fast.schnell.txt2img.v2'),
  prompt: 'A cat wearing an intricate robe',
});

const filename = `image-${Date.now()}.png`;
writeFileSync(filename, image.uint8Array);
console.log(`Image saved to ${filename}`);
```

### 模型能力

Prodia 为多种图像生成模型提供高速推理支持。以下为支持的模型类型：

| Model                                    | 说明                                          |
| ---------------------------------------- | --------------------------------------------- |
| `inference.flux-fast.schnell.txt2img.v2` | 用于文生图的快速 FLUX Schnell 模型 |
| `inference.flux.schnell.txt2img.v2`      | 用于文生图的 FLUX Schnell 模型      |

<Note>
  Prodia 还支持更多模型 ID。完整列表请参阅 [Prodia 文档](https://docs.prodia.com/)。
</Note>

### 图像尺寸

可通过 `size` 参数指定 `WIDTHxHEIGHT` 格式的图像尺寸：

```ts
import { prodia } from '@ai-sdk/prodia';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: prodia.image('inference.flux-fast.schnell.txt2img.v2'),
  prompt: 'A serene mountain landscape at sunset',
  size: '1024x768',
});
```

### Provider 选项

Prodia 图像模型可通过 `providerOptions.prodia` 传入额外选项：

```ts
import { prodia, type ProdiaImageModelOptions } from '@ai-sdk/prodia';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: prodia.image('inference.flux-fast.schnell.txt2img.v2'),
  prompt: 'A cat wearing an intricate robe',
  providerOptions: {
    prodia: {
      width: 1024,
      height: 768,
      steps: 4,
      stylePreset: 'cinematic',
    } satisfies ProdiaImageModelOptions,
  },
});
```

支持的提供商选项如下：

- **width** _number_ — 输出宽度（像素，256–1920）。设置后将覆盖由 `size` 推导出的宽度。
- **height** _number_ — 输出高度（像素，256–1920）。设置后将覆盖由 `size` 推导出的高度。
- **steps** _number_ — 迭代步数（1–4）。步数通常越高，质量越好。
- **stylePreset** _string_ — 为输出图像应用视觉主题。支持的预设：`3d-model`、`analog-film`、`anime`、`cinematic`、`comic-book`、`digital-art`、`enhance`、`fantasy-art`、`isometric`、`line-art`、`low-poly`、`neon-punk`、`origami`、`photographic`、`pixel-art`、`texture`、`craft-clay`。
- **loras** _string[]_ — 最多使用 3 个 LoRA 模型增强输出。
- **progressive** _boolean_ — 使用 JPEG 输出时返回渐进式 JPEG。

### 随机种子

可使用 `seed` 参数获得可复现的结果：

```ts
import { prodia } from '@ai-sdk/prodia';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: prodia.image('inference.flux-fast.schnell.txt2img.v2'),
  prompt: 'A serene mountain landscape at sunset',
  seed: 12345,
});
```

### Provider 元数据

`generateImage` 的响应在 `providerMetadata.prodia.images[]` 中包含提供商专有元数据。每张图像对象可包含：

- **jobId** _string_ — 生成任务的唯一标识符。
- **seed** _number_ — 生成使用的种子，可用于复现结果。
- **elapsed** _number_ — 生成耗时（秒）。
- **iterationsPerSecond** _number_ — 处理速度指标。
- **createdAt** _string_ — 任务创建时间戳。
- **updatedAt** _string_ — 任务最后更新时间戳。

```ts
import { prodia } from '@ai-sdk/prodia';
import { generateImage } from 'ai';

const { image, providerMetadata } = await generateImage({
  model: prodia.image('inference.flux-fast.schnell.txt2img.v2'),
  prompt: 'A serene mountain landscape at sunset',
});

// Access provider metadata
const metadata = providerMetadata?.prodia?.images?.[0];
console.log('Job ID:', metadata?.jobId);
console.log('Seed:', metadata?.seed);
console.log('Elapsed:', metadata?.elapsed);
```
