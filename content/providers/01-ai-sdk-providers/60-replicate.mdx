# Replicate 提供商

[Replicate](https://replicate.com/) 用于运行开源 AI 模型的平台，
也是常用的图像生成模型托管服务。

## 安装

Replicate 提供商位于 `@ai-sdk/replicate` 模块中，可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/replicate" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/replicate" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/replicate" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/replicate" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/replicate` 导入默认的 `replicate` 提供商实例：

```ts
import { replicate } from '@ai-sdk/replicate';
```

如需自定义配置，可导入 `createReplicate` 并以你的设置创建提供商实例：

```ts
import { createReplicate } from '@ai-sdk/replicate';

const replicate = createReplicate({
  apiToken: process.env.REPLICATE_API_TOKEN ?? '',
});
```

可使用以下可选设置自定义 Replicate 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀（例如经由代理）。
  默认前缀为 `https://api.replicate.com/v1`。

- **apiToken** _string_

  通过 `Authorization` 请求头发送的 API Token。
  默认为环境变量 `REPLICATE_API_TOKEN`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义请求头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 图像模型

可使用 `.image()` 工厂方法创建 Replicate 图像模型。
详见 AI SDK 中的 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

<Note>
  不同模型对 `size` 等参数的支持各不相同。请参阅
  [Replicate](https://replicate.com/explore) 上对应模型的文档，确认支持的选项以及可通过 `providerOptions.replicate` 传入的附加参数。
</Note>

### 基本用法

```ts
import { replicate } from '@ai-sdk/replicate';
import { generateImage } from 'ai';
import { writeFile } from 'node:fs/promises';

const { image } = await generateImage({
  model: replicate.image('black-forest-labs/flux-schnell'),
  prompt: 'The Loch Ness Monster getting a manicure',
  aspectRatio: '16:9',
});

await writeFile('image.webp', image.uint8Array);

console.log('Image saved as image.webp');
```

### 模型专属选项

```ts highlight="9-11"
import { replicate, type ReplicateImageModelOptions } from '@ai-sdk/replicate';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: replicate.image('recraft-ai/recraft-v3'),
  prompt: 'The Loch Ness Monster getting a manicure',
  size: '1365x1024',
  providerOptions: {
    replicate: {
      style: 'realistic_image',
    } satisfies ReplicateImageModelOptions,
  },
});
```

### 带版本模型

```ts
import { replicate } from '@ai-sdk/replicate';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: replicate.image(
    'bytedance/sdxl-lightning-4step:5599ed30703defd1d160a25a63321b4dec97101d98b4674bcc56e41f62f35637',
  ),
  prompt: 'The Loch Ness Monster getting a manicure',
});
```

### 图像编辑

Replicate 通过多种模型支持图像编辑。通过 `prompt.images` 传入输入图像即可变换或编辑现有图像。

#### 基本图像编辑

使用文本提示变换现有图像：

```ts
const imageBuffer = readFileSync('./input-image.png');

const { images } = await generateImage({
  model: replicate.image('black-forest-labs/flux-fill-dev'),
  prompt: {
    text: 'Turn the cat into a golden retriever dog',
    images: [imageBuffer],
  },
  providerOptions: {
    replicate: {
      guidance_scale: 7.5,
      num_inference_steps: 30,
    } satisfies ReplicateImageModelOptions,
  },
});
```

#### 使用遮罩修复

使用遮罩编辑图像的指定区域。对 FLUX Fill 模型，遮罩中的白色区域表示需要编辑的位置：

```ts
const image = readFileSync('./input-image.png');
const mask = readFileSync('./mask.png'); // White = inpaint, black = keep

const { images } = await generateImage({
  model: replicate.image('black-forest-labs/flux-fill-pro'),
  prompt: {
    text: 'A sunlit indoor lounge area with a pool containing a flamingo',
    images: [image],
    mask: mask,
  },
  providerOptions: {
    replicate: {
      guidance_scale: 7.5,
      num_inference_steps: 30,
    } satisfies ReplicateImageModelOptions,
  },
});
```

#### 多参考图生成（Flux-2）

Flux-2 模型最多支持 8 张输入参考图像，可用于风格迁移、构图与多主体生成：

```ts
import { replicate } from '@ai-sdk/replicate';
import { generateImage } from 'ai';

const reference1 = readFileSync('./style-reference.png');
const reference2 = readFileSync('./subject-reference.png');

const { images } = await generateImage({
  model: replicate.image('black-forest-labs/flux-2-pro'),
  prompt: {
    text: 'Combine the style and subjects from the reference images',
    images: [reference1, reference2],
  },
});
```

<Note>
  Flux-2 模型在内部使用不同的输入格式（`input_image`、`input_image_2` 等），由 SDK 自动处理。Flux-2 模型不支持基于遮罩的修复（mask-based inpainting）。
</Note>

<Note>
  输入图像可为 `Buffer`、`ArrayBuffer`、`Uint8Array` 或 base64 字符串。不同 Replicate 模型的参数名与能力各异，详见
  [Replicate](https://replicate.com/explore) 上的模型文档。
</Note>

### 提供商选项

图像生成常用的提供商选项：

- **maxWaitTimeInSeconds** _number_ — 同步模式下等待预测完成的最长时间（秒）。默认使用 [sync mode](https://replicate.com/docs/topics/predictions/create-a-prediction#timeout-duration)，超时 60 秒。设为正数可自定义时长（例如 `120` 表示 2 分钟）；未指定则使用默认 60 秒等待。
- **guidance_scale** _number_ — 无分类器引导的强度；值越高，输出越贴近提示词。
- **num_inference_steps** _number_ — 去噪步数；步数越多质量通常越高但越慢。
- **negative_prompt** _string_ — 负向提示，约束生成中应避免的内容。
- **output_format** _'png' | 'jpg' | 'webp'_ — 输出图像格式。
- **output_quality** _number (1-100)_ — 输出画质；仅对 jpg、webp 有效。
- **strength** _number (0-1)_ — 图生图变换强度；值越低越保留原图。

更多说明见 [Replicate models](https://replicate.com/explore)。
