# LMNT 提供商

[LMNT](https://lmnt.com/) 提供商为 LMNT 语音合成 API 提供语音模型支持。

## 安装

LMNT 提供商位于 `@ai-sdk/lmnt` 模块中，可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/lmnt" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/lmnt" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/lmnt" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/lmnt" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/lmnt` 导入默认的 `lmnt` 提供商实例：

```ts
import { lmnt } from '@ai-sdk/lmnt';
```

如需自定义配置，可导入 `createLMNT` 并创建实例：

```ts
import { createLMNT } from '@ai-sdk/lmnt';

const lmnt = createLMNT({
  // custom settings, e.g.
  fetch: customFetch,
});
```

可使用以下可选设置自定义 LMNT 提供商实例：

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `LMNT_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 语音模型

可使用 `.speech()` 工厂方法创建调用 [LMNT speech API](https://docs.lmnt.com/api-reference/speech/synthesize-speech-bytes) 的模型。

首个参数为模型 ID，例如 `aurora`。

```ts
const model = lmnt.speech('aurora');
```

`voice` 可设为 LMNT 的音色 ID，可用音色见 [LMNT 文档](https://docs.lmnt.com/api-reference/voices/list-voices)。

```ts highlight="7"
import { experimental_generateSpeech as generateSpeech } from 'ai';
import { lmnt } from '@ai-sdk/lmnt';

const result = await generateSpeech({
  model: lmnt.speech('aurora'),
  text: 'Hello, world!',
  voice: 'ava',
  language: 'en',
});
```

也可通过 `providerOptions` 传入更多提供商专有选项：

```ts highlight="10-14"
import { experimental_generateSpeech as generateSpeech } from 'ai';
import { lmnt } from '@ai-sdk/lmnt';
import { type LMNTSpeechModelOptions } from '@ai-sdk/lmnt';

const result = await generateSpeech({
  model: lmnt.speech('aurora'),
  text: 'Hello, world!',
  voice: 'ava',
  language: 'en',
  providerOptions: {
    lmnt: {
      conversational: true,
      speed: 1.2,
    } satisfies LMNTSpeechModelOptions,
  },
});
```

### Provider 选项

LMNT 提供商可通过 `providerOptions.lmnt` 接受以下选项：

- **format** _'aac' | 'mp3' | 'mulaw' | 'raw' | 'wav'_

  返回的音频格式。默认为 `'mp3'`。

- **sampleRate** _8000 | 16000 | 24000_

  音频采样率（Hz）。默认为 `24000`。

- **speed** _number_

  语速。须在 0.25 至 2 之间。默认为 `1`。

- **seed** _number_

  可选的随机种子，用于可复现生成。

- **conversational** _boolean_

  是否使用会话风格。默认为 `false`。不能与 `blizzard` 模型一起使用。

- **length** _number_

  音频最大长度（秒），最大 300。不能与 `blizzard` 模型一起使用。

- **topP** _number_

  Top-p 采样参数。须在 0 至 1 之间。默认为 `1`。

- **temperature** _number_

  采样温度。须 ≥ 0。默认为 `1`。
