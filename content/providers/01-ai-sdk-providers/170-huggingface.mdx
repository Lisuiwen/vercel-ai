# Hugging Face 提供商

[Hugging Face](https://huggingface.co/) 提供商通过 [Hugging Face Inference Providers](https://huggingface.co/docs/inference-providers/index) 提供对数千语言模型的访问，涵盖 Meta、DeepSeek、Qwen 等厂商模型。

可在 [Hugging Face 设置](https://huggingface.co/settings/tokens) 获取 API Key。

## 安装

Hugging Face 提供商通过 `@ai-sdk/huggingface` 模块提供，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/huggingface` 导入默认的 `huggingFace` 提供商实例：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
```

如需自定义配置，可导入 `createHuggingFace` 并创建实例：

```ts
import { createHuggingFace } from '@ai-sdk/huggingface';

const huggingFace = createHuggingFace({
  apiKey: process.env.HUGGINGFACE_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Hugging Face 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀，例如经由代理。
  默认前缀为 `https://router.huggingface.co/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `HUGGINGFACE_API_KEY`。可在 [Hugging Face 设置](https://huggingface.co/settings/tokens) 获取。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 语言模型

可使用提供商实例创建语言模型：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
import { generateText } from 'ai';

const { text } = await generateText({
  model: huggingFace('deepseek-ai/DeepSeek-V3-0324'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

也可使用 `.responses()` 或 `.languageModel()` 工厂方法：

```ts
const model = huggingFace.responses('deepseek-ai/DeepSeek-V3-0324');
// or
const model = huggingFace.languageModel('moonshotai/Kimi-K2-Instruct');
```

Hugging Face 语言模型可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

可在 [Hugging Face Inference Models](https://huggingface.co/inference/models) 页面浏览最新与热门模型及其能力、上下文长度、吞吐量与定价。

### Provider 选项

Hugging Face 语言模型支持通过 `providerOptions.huggingface` 传入提供商专有选项：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
import { generateText } from 'ai';

const { text } = await generateText({
  model: huggingFace('deepseek-ai/DeepSeek-R1'),
  prompt: 'Explain the theory of relativity.',
  providerOptions: {
    huggingface: {
      reasoningEffort: 'high',
      instructions: 'Respond in a clear and educational manner.',
    },
  },
});
```

可用的提供商选项：

- **metadata** _Record&lt;string, string&gt;_

  随请求附加的额外元数据。

- **instructions** _string_

  模型指令，可用于提供更多上下文或引导。

- **strictJsonSchema** _boolean_

  结构化输出是否使用严格 JSON schema 校验。默认为 `false`。

- **reasoningEffort** _string_

  控制如 DeepSeek-R1 等推理模型的推理强度；值越高推理越充分。

### 推理输出

对于 `deepseek-ai/DeepSeek-R1` 等推理模型，可控制推理强度并在响应中访问推理过程：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
import { streamText } from 'ai';

const result = streamText({
  model: huggingFace('deepseek-ai/DeepSeek-R1'),
  prompt: 'How many r letters are in the word strawberry?',
  providerOptions: {
    huggingface: {
      reasoningEffort: 'high',
    },
  },
});

for await (const part of result.fullStream) {
  if (part.type === 'reasoning') {
    console.log(`Reasoning: ${part.textDelta}`);
  } else if (part.type === 'text-delta') {
    process.stdout.write(part.textDelta);
  }
}
```

使用 `generateText` 的非流式调用中，推理内容位于响应的 `reasoning` 字段：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
import { generateText } from 'ai';

const result = await generateText({
  model: huggingFace('deepseek-ai/DeepSeek-R1'),
  prompt: 'What is 25 * 37?',
  providerOptions: {
    huggingface: {
      reasoningEffort: 'medium',
    },
  },
});

console.log('Reasoning:', result.reasoning);
console.log('Answer:', result.text);
```

### 图像输入

对于具备视觉能力的模型（如 `Qwen/Qwen2.5-VL-7B-Instruct`），可在消息内容中包含图像：

```ts
import { huggingFace } from '@ai-sdk/huggingface';
import { generateText } from 'ai';
import { readFileSync } from 'fs';

const result = await generateText({
  model: huggingFace('Qwen/Qwen2.5-VL-7B-Instruct'),
  messages: [
    {
      role: 'user',
      content: [
        { type: 'text', text: 'Describe this image in detail.' },
        {
          type: 'file',
          mediaType: 'image',
          data: readFileSync('./image.png'),
        },
      ],
    },
  ],
});
```

也可传入图像 URL：

```ts
{
  type: 'file',
  mediaType: 'image',
  data: 'https://example.com/image.png',
}
```
