# Rev.ai 提供商

[Rev.ai](https://www.rev.ai/) 提供商为 Rev.ai 转写 API 提供语言模型支持。

## 安装

Rev.ai 提供商位于 `@ai-sdk/revai` 模块中，可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/revai" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/revai" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/revai" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/revai" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/revai` 导入默认的 `revai` 提供商实例：

```ts
import { revai } from '@ai-sdk/revai';
```

如需自定义配置，可导入 `createRevai` 并创建实例：

```ts
import { createRevai } from '@ai-sdk/revai';

const revai = createRevai({
  // custom settings, e.g.
  fetch: customFetch,
});
```

可使用以下可选设置自定义 Rev.ai 提供商实例：

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `REVAI_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 转写模型

可使用 `.transcription()` 工厂方法创建调用 [Rev.ai transcription API](https://www.rev.ai/docs/api/transcription) 的模型。

首个参数为模型 ID，例如 `machine`。

```ts
const model = revai.transcription('machine');
```

也可通过 `providerOptions` 传入更多提供商专有选项。例如，若能事先确定输入语言，使用 ISO-639-1（如 `en`）有时可改善转写表现。

```ts highlight="7"
import { experimental_transcribe as transcribe } from 'ai';
import { revai } from '@ai-sdk/revai';
import { type RevaiTranscriptionModelOptions } from '@ai-sdk/revai';
import { readFile } from 'fs/promises';

const result = await transcribe({
  model: revai.transcription('machine'),
  audio: await readFile('audio.mp3'),
  providerOptions: {
    revai: { language: 'en' } satisfies RevaiTranscriptionModelOptions,
  },
});
```

可用的提供商选项如下：

- **metadata** _string_

  可选的元数据字符串，用于关联转写任务。

- **notification_config** _object_

  任务完成时 Webhook 通知配置。
  - **url** _string_ — 通知目标 URL。
  - **auth_headers** _object_ — 通知请求的可选授权头。
    - **Authorization** _string_ — Authorization 头取值。

- **delete_after_seconds** _integer_

  任务在若干秒后自动删除。

- **verbatim** _boolean_

  是否在转写中包含填充词与错误起手。

- **rush** _boolean_

  【不支持 HIPAA】是否优先排队以加快处理。仅适用于人工转写选项。

- **test_mode** _boolean_

  是否在测试模式下运行任务。默认为 `false`。

- **segments_to_transcribe** _Array_

  仅转写音频的特定片段。
  - **start** _number_ — 片段起始时间（秒）。
  - **end** _number_ — 片段结束时间（秒）。

- **speaker_names** _Array_

  为说话人分配名称。
  - **display_name** _string_ — 说话人的显示名称。

- **skip_diarization** _boolean_

  是否跳过说话人分离（diarization）。默认为 `false`。

- **skip_postprocessing** _boolean_

  是否跳过后处理步骤。仅适用于英语与西班牙语。默认为 `false`。

- **skip_punctuation** _boolean_

  是否跳过为转写添加标点。默认为 `false`。

- **remove_disfluencies** _boolean_

  是否去除语气词与不流畅片段（如 um、uh）。默认为 `false`。

- **remove_atmospherics** _boolean_

  是否移除氛围类音效标记（如 `<laugh>`、`<affirmative>`）。默认为 `false`。

- **filter_profanity** _boolean_

  是否用星号过滤粗话（首尾字符保留）。默认为 `false`。

- **speaker_channels_count** _integer_

  音频中的说话声道数。仅适用于英语、西班牙语与法语。

- **speakers_count** _integer_

  预估说话人数量。仅适用于英语、西班牙语与法语。

- **diarization_type** _string_

  分离类型。可选：`"standard"`（默认）、`"premium"`。

- **custom_vocabulary_id** _string_

  通过 Custom Vocabularies API 提交的自定义词表 ID。

- **custom_vocabularies** _Array_

  用于本次转写的自定义词表条目。

- **strict_custom_vocabulary** _boolean_

  是否严格强制执行自定义词表。

- **summarization_config** _object_

  为转写生成摘要的配置。
  - **model** _string_ — 摘要模型：`"standard"`（默认）、`"premium"`。
  - **type** _string_ — 摘要格式：`"paragraph"`（默认）、`"bullets"`。
  - **prompt** _string_ — 自定义摘要提示（与 `type` 互斥）。

- **translation_config** _object_

  转写翻译配置。
  - **target_languages** _Array_ — 目标语言，每项包含：
    - **language** _string_ — 语言代码：`"en"`、`"en-us"`、`"en-gb"`、`"ar"`、`"pt"`、`"pt-br"`、`"pt-pt"`、`"fr"`、`"fr-ca"`、`"es"`、`"es-es"`、`"es-la"`、`"it"`、`"ja"`、`"ko"`、`"de"`、`"ru"`。
  - **model** _string_ — 翻译模型：`"standard"`（默认）、`"premium"`。

- **language** _string_

  音频内容的语言（ISO 639-1）。默认为 `"en"`。

- **forced_alignment** _boolean_

  是否执行强制对齐，以提升逐词时间戳精度。
  默认为 `false`。

  当前支持的语言：
  - English (en, en-us, en-gb)
  - French (fr)
  - Italian (it)
  - German (de)
  - Spanish (es)

  注意：低价环境不可用此选项。
