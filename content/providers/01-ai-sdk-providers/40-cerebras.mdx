# Cerebras 提供商

[Cerebras](https://cerebras.ai) 提供商通过 Cerebras API 提供强大的语言模型访问能力，并利用 Wafer-Scale Engine 与 CS-3 系统实现高速推理。

可在 [Cerebras 平台](https://cloud.cerebras.ai) 获取 API Key。

## 安装

Cerebras 提供商通过 `@ai-sdk/cerebras` 模块提供，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/cerebras` 导入默认的 `cerebras` 提供商实例：

```ts
import { cerebras } from '@ai-sdk/cerebras';
```

如需自定义配置，可导入 `createCerebras` 并创建实例：

```ts
import { createCerebras } from '@ai-sdk/cerebras';

const cerebras = createCerebras({
  apiKey: process.env.CEREBRAS_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Cerebras 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀。
  默认前缀为 `https://api.cerebras.ai/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `CEREBRAS_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 语言模型

可使用提供商实例创建语言模型：

```ts
import { cerebras } from '@ai-sdk/cerebras';
import { generateText } from 'ai';

const { text } = await generateText({
  model: cerebras('llama3.1-8b'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

Cerebras 语言模型可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

可使用提供商实例创建 Cerebras 语言模型；首个参数为模型 ID，例如 `llama-3.3-70b`：

```ts
const model = cerebras('llama-3.3-70b');
```

也可使用 `.languageModel()` 与 `.chat()` 方法：

```ts
const model = cerebras.languageModel('llama-3.3-70b');
const model = cerebras.chat('llama-3.3-70b');
```

### 推理模型

Cerebras 提供多款推理模型，包括 `gpt-oss-120b`、`qwen-3-32b` 与 `zai-glm-4.7`，会在最终答复前生成中间 thinking token；推理输出通过 AI SDK 标准的 reasoning part 进行流式传输。

对于 `gpt-oss-120b`，可使用 `reasoningEffort` 提供商选项控制推理深度：

```ts
import { cerebras } from '@ai-sdk/cerebras';
import { streamText } from 'ai';

const result = streamText({
  model: cerebras('gpt-oss-120b'),
  providerOptions: {
    cerebras: {
      reasoningEffort: 'medium',
    },
  },
  prompt: 'How many "r"s are in the word "strawberry"?',
});

for await (const part of result.fullStream) {
  if (part.type === 'reasoning') {
    console.log('Reasoning:', part.text);
  } else if (part.type === 'text-delta') {
    process.stdout.write(part.textDelta);
  }
}
```

如何将推理接入聊天机器人的更多说明，参见 [AI SDK UI：Chatbot](/docs/ai-sdk-ui/chatbot#reasoning)。

### Provider 选项

Cerebras 语言模型支持以下可选的提供商选项：

- **reasoningEffort** _'low' | 'medium' | 'high'_

  控制 GPT-OSS 模型的推理深度。默认为 `'medium'`。

- **user** _string_

  代表终端用户的唯一标识符，可用于监控与滥用检测。

- **strictJsonSchema** _boolean_

  是否启用严格 JSON schema 校验。为 `true` 时模型使用约束解码以确保符合 schema。默认为 `true`。
