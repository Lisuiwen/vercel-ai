# ByteDance 提供商

[ByteDance](https://www.bytedance.com/) 提供商通过 [BytePlus ModelArk](https://docs.byteplus.com/en/docs/ModelArk/Video_Generation_API) 平台支持 Seedance 系列视频生成模型。Seedance 提供高质量的文生视频与图生视频能力，包括音画同步、首尾帧控制与多参考图生成。

## 安装

ByteDance 提供商位于 `@ai-sdk/bytedance` 模块中，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/bytedance` 导入默认的 `byteDance` 提供商实例：

```ts
import { byteDance } from '@ai-sdk/bytedance';
```

如需自定义配置，可导入 `createByteDance` 并以你的设置创建提供商实例：

```ts
import { createByteDance } from '@ai-sdk/bytedance';

const byteDance = createByteDance({
  apiKey: 'your-api-key', // optional, defaults to ARK_API_KEY environment variable
  baseURL: 'custom-url', // optional
  headers: {
    /* custom headers */
  }, // optional
});
```

可使用以下可选设置自定义 ByteDance 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀（例如经由代理）。
  默认前缀为 `https://ark.ap-southeast.bytepluses.com/api/v3`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `ARK_API_KEY`。
  可在 BytePlus 控制台 [获取 API Key](https://console.byteplus.com/ark/apiKey)。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义请求头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  可作为中间件拦截请求，
  或在测试中提供自定义 `fetch` 实现。

## 视频模型

可使用 `.video()` 工厂方法创建 ByteDance 视频模型。
详见 AI SDK 的 [generateVideo()](/docs/reference/ai-sdk-core/generate-video)。

### 文生视频

由文本提示生成视频：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('seedance-1-0-pro-250528'),
  prompt:
    'Photorealistic style: Under a clear blue sky, a vast expanse of white daisy fields stretches out. The camera gradually zooms in and fixates on a close-up of a single daisy.',
  aspectRatio: '16:9',
  duration: 5,
  providerOptions: {
    bytedance: {
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});

console.log(video.url);
```

### 图生视频

由首帧图像与可选文本提示生成视频：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('seedance-1-5-pro-251215'),
  prompt: {
    image: 'https://example.com/first-frame.png',
    text: 'The cat slowly turns its head and blinks',
  },
  duration: 5,
  providerOptions: {
    bytedance: {
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 带音频的图生视频

Seedance 1.5 Pro 支持在生成视频的同时生成同步音频：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('seedance-1-5-pro-251215'),
  prompt: {
    image: 'https://example.com/pianist.png',
    text: 'A young man sits at a piano, playing calmly. Gentle piano music plays in sync with his movements.',
  },
  duration: 5,
  providerOptions: {
    bytedance: {
      generateAudio: true,
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 首尾帧视频

在起始关键帧与结束关键帧图像之间生成平滑过渡：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('seedance-1-5-pro-251215'),
  prompt: {
    image: 'https://example.com/first-frame.jpg',
    text: 'Create a 360-degree orbiting camera shot based on this photo',
  },
  duration: 5,
  providerOptions: {
    bytedance: {
      lastFrameImage: 'https://example.com/last-frame.jpg',
      generateAudio: true,
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 多参考图图生视频

使用 Seedance 1.0 Lite I2V 模型时，可提供多幅参考图（1–4 张），模型会据此尽可能还原物体形状、颜色与纹理：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('seedance-1-0-lite-i2v-250428'),
  prompt:
    'A boy wearing glasses and a blue T-shirt from [Image 1] and a corgi dog from [Image 2], sitting on the lawn from [Image 3], in 3D cartoon style',
  aspectRatio: '16:9',
  duration: 5,
  providerOptions: {
    bytedance: {
      referenceImages: [
        'https://example.com/boy.png',
        'https://example.com/corgi.png',
        'https://example.com/lawn.png',
      ],
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 参考视频

Seedance 2.0 支持参考视频，用于引导生成视频的风格、运动或构图：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('dreamina-seedance-2-0-260128'),
  prompt:
    'First-person perspective promotional ad, using the composition and camera movement from the reference video',
  aspectRatio: '16:9',
  duration: 4,
  providerOptions: {
    bytedance: {
      referenceVideos: ['https://example.com/reference-video.mp4'],
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 参考音频

Seedance 2.0 支持参考音频，用作生成视频的背景音乐或音效：

```ts
import {
  byteDance,
  type ByteDanceVideoProviderOptions,
} from '@ai-sdk/bytedance';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: byteDance.video('dreamina-seedance-2-0-260128'),
  prompt: 'A serene mountain landscape at sunrise with gentle camera movement',
  aspectRatio: '16:9',
  duration: 4,
  providerOptions: {
    bytedance: {
      referenceAudio: ['https://example.com/background-music.mp3'],
      generateAudio: true,
      watermark: false,
    } satisfies ByteDanceVideoProviderOptions,
  },
});
```

### 视频的提供商选项

可通过 `providerOptions.bytedance` 使用以下提供商选项：

#### 生成选项

- **watermark** _boolean_

  是否为生成的视频添加水印。

- **generateAudio** _boolean_

  是否为视频生成同步音频。仅 Seedance 1.5 Pro 支持。

- **cameraFixed** _boolean_

  生成过程中是否固定镜头。

- **returnLastFrame** _boolean_

  是否返回生成视频的末帧。可用于串联多段连续视频。

- **serviceTier** _'default' | 'flex'_

  推理档位。`'default'` 为在线推理。`'flex'` 为离线推理，价格为 50%，延迟更高（响应时间可达数小时量级）。

- **draft** _boolean_

  是否启用草稿采样模式以低成本预览。仅 Seedance 1.5 Pro 支持。先生成 480p 预览以便快速迭代，再生成正式高质量版本。

#### 图像输入选项

- **lastFrameImage** _string_

  首尾帧视频中的末帧图像 URL。模型在由 `image` 提示提供的首帧与此末帧之间生成平滑过渡。Seedance 1.5 Pro、1.0 Pro 与 1.0 Lite I2V 支持。

- **referenceImages** _string[]_

  多参考图图生视频的参考图 URL 数组（1–4 张）。模型从每张图提取关键特征并在视频中还原。在提示中使用 `[Image 1]`、`[Image 2]` 等引用具体图像。Seedance 1.0 Lite I2V 支持。

#### 媒体引用选项

- **referenceVideos** _string[]_

  参考视频 URL 数组（最多 3 个，每个最长 15 秒），用于参考引导生成。模型根据参考视频引导风格、运动或构图。Seedance 2.0 支持。

- **referenceAudio** _string[]_

  参考音频 URL 数组（最多 3 个，每个最长 15 秒），用于音频引导生成。模型将参考音频用作背景音乐或同步音效。支持 data URI（如 `data:audio/wav;base64,...`）。Seedance 2.0 支持。

#### 轮询选项

- **pollIntervalMs** _number_

  处理中轮询完成状态的间隔（毫秒）。
  默认为 3000 ms。

- **pollTimeoutMs** _number_

  视频生成完成前的最长等待时间。
  默认为 300000 ms（5 分钟）。

<Note>
  视频生成是异步过程，可能需要数分钟。
  建议将 `pollTimeoutMs` 至少设为 10 分钟（600000 ms）以保证稳定。
</Note>

### 视频模型能力

| 模型                    | Model ID                            | 能力说明                                                                                                                                        |
| ----------------------- | ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Seedance 2.0            | `dreamina-seedance-2-0-260128`      | 文生视频、图生视频、参考视频（最多 3 个）、参考音频（最多 3 个）、音画同步。时长 4–15s。分辨率 480p、720p。                                      |
| Seedance 2.0 Fast       | `dreamina-seedance-2-0-fast-260128` | 文生视频、图生视频、参考视频（最多 3 个）、参考音频（最多 3 个）、音画同步；侧重速度。时长 4–15s。分辨率 480p、720p。                             |
| Seedance 1.5 Pro        | `seedance-1-5-pro-251215`           | 文生视频、图生视频（首帧）、图生视频（首尾帧）、音画同步、草稿模式。时长 4–12s。分辨率 480p、720p、1080p。                                      |
| Seedance 1.0 Pro        | `seedance-1-0-pro-250528`           | 文生视频、图生视频（首帧）、图生视频（首尾帧）。时长 2–12s。分辨率 480p、720p、1080p。                                                            |
| Seedance 1.0 Pro Fast   | `seedance-1-0-pro-fast-251015`      | 文生视频、图生视频（首帧）；侧重速度与成本。时长 2–12s。                                                                                        |
| Seedance 1.0 Lite (T2V) | `seedance-1-0-lite-t2v-250428`      | 仅文生视频。时长 2–12s。分辨率 480p、720p、1080p。                                                                                              |
| Seedance 1.0 Lite (I2V) | `seedance-1-0-lite-i2v-250428`      | 图生视频（首帧）、图生视频（首尾帧）、多参考图（1–4）。时长 2–12s。分辨率 480p、720p。                                                           |

支持的宽高比：`16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`21:9`、`adaptive`（仅图生视频）。

所有模型输出 24 fps 的 MP4 视频。

<Note>
  如需也可直接传入任意模型 ID 字符串，例如尚未列入此处的未来模型。
</Note>
