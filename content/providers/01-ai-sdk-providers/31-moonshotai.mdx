# Moonshot AI 提供商

[Moonshot AI](https://www.moonshot.ai) 提供商通过 Moonshot API 提供强大的语言模型访问能力，其中包括具备推理能力的 Kimi 系列模型。

可在 [Moonshot 平台](https://platform.moonshot.ai) 获取 API Key。

## 安装

Moonshot AI 提供商通过 `@ai-sdk/moonshotai` 模块提供，可使用以下命令安装：

  <Tab>
    <Snippet text="bun add @ai-sdk/moonshotai" dark />

## Provider 实例

可从 `@ai-sdk/moonshotai` 导入默认的 `moonshotai` 提供商实例：

```ts
import { moonshotai } from '@ai-sdk/moonshotai';
```

如需自定义配置，可导入 `createMoonshotAI` 并创建实例：

```ts
import { createMoonshotAI } from '@ai-sdk/moonshotai';

const moonshotai = createMoonshotAI({
  apiKey: process.env.MOONSHOT_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Moonshot AI 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀。
  默认前缀为 `https://api.moonshot.ai/v1`

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `MOONSHOT_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 语言模型

可使用提供商实例创建语言模型：

```ts
import { moonshotai } from '@ai-sdk/moonshotai';
import { generateText } from 'ai';

const { text } = await generateText({
  model: moonshotai('kimi-k2.5'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

也可使用 `.chatModel()` 或 `.languageModel()` 工厂方法：

```ts
const model = moonshotai.chatModel('kimi-k2.5');
// or
const model = moonshotai.languageModel('kimi-k2.5');
```

Moonshot AI 语言模型可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

### 推理模型

Moonshot AI 提供如 `kimi-k2-thinking` 等 thinking 模型，会在最终答复前生成中间推理 token；推理输出通过 AI SDK 标准的 reasoning part 进行流式传输。

```ts
import {
  moonshotai,
  type MoonshotAILanguageModelOptions,
} from '@ai-sdk/moonshotai';
import { generateText } from 'ai';

const { text, reasoningText } = await generateText({
  model: moonshotai('kimi-k2-thinking'),
  providerOptions: {
    moonshotai: {
      thinking: { type: 'enabled', budgetTokens: 2048 },
      reasoningHistory: 'interleaved',
    } satisfies MoonshotAILanguageModelOptions,
  },
  prompt: 'How many "r"s are in the word "strawberry"?',
});

console.log(reasoningText);
console.log(text);
```

如何将推理接入聊天机器人的更多说明，参见 [AI SDK UI：Chatbot](/docs/ai-sdk-ui/chatbot#reasoning)。

### Provider 选项

Moonshot AI 语言模型支持以下可选的提供商选项：

- **thinking** _object_

  针对 Kimi K2 Thinking 等 thinking/reasoning 模型的配置。
  - **type** _'enabled' | 'disabled'_

    是否启用 thinking 模式。

  - **budgetTokens** _number_

    thinking 可使用的最大 token 数（最小 1024）。

- **reasoningHistory** _'disabled' | 'interleaved' | 'preserved'_

  控制多轮对话中推理历史的处理方式：
  - `'disabled'`：从历史中移除推理内容。
  - `'interleaved'`：在同一轮中在工具调用之间穿插推理。
  - `'preserved'`：在历史中保留全部推理。
