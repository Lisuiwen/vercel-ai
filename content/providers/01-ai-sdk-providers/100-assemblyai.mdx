# AssemblyAI 提供商

[AssemblyAI](https://assemblyai.com/) 提供商为 AssemblyAI 转写 API 提供语言模型支持。

## 安装

AssemblyAI 提供商位于 `@ai-sdk/assemblyai` 模块中，可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab>
    <Snippet text="pnpm add @ai-sdk/assemblyai" dark />
  </Tab>
  <Tab>
    <Snippet text="npm install @ai-sdk/assemblyai" dark />
  </Tab>
  <Tab>
    <Snippet text="yarn add @ai-sdk/assemblyai" dark />
  </Tab>

  <Tab>
    <Snippet text="bun add @ai-sdk/assemblyai" dark />
  </Tab>
</Tabs>

## Provider 实例

可从 `@ai-sdk/assemblyai` 导入默认的 `assemblyai` 提供商实例：

```ts
import { assemblyai } from '@ai-sdk/assemblyai';
```

如需自定义配置，可导入 `createAssemblyAI` 并创建实例：

```ts
import { createAssemblyAI } from '@ai-sdk/assemblyai';

const assemblyai = createAssemblyAI({
  // custom settings, e.g.
  fetch: customFetch,
});
```

可使用以下可选设置自定义 AssemblyAI 提供商实例：

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `ASSEMBLYAI_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 转写模型

可使用 `.transcription()` 工厂方法创建调用 [AssemblyAI transcription API](https://www.assemblyai.com/docs/getting-started/transcribe-an-audio-file/typescript) 的模型。

首个参数为模型 ID，例如 `best`。

```ts
const model = assemblyai.transcription('best');
```

也可通过 `providerOptions` 传入额外选项；例如开启 `contentSafety` 可启用内容安全过滤。

```ts highlight="7"
import { experimental_transcribe as transcribe } from 'ai';
import { assemblyai } from '@ai-sdk/assemblyai';
import { type AssemblyAITranscriptionModelOptions } from '@ai-sdk/assemblyai';
import { readFile } from 'fs/promises';

const result = await transcribe({
  model: assemblyai.transcription('best'),
  audio: await readFile('audio.mp3'),
  providerOptions: {
    assemblyai: {
      contentSafety: true,
    } satisfies AssemblyAITranscriptionModelOptions,
  },
});
```

可用的提供商选项：

- **audioEndAt** _number_

  音频结束时间（毫秒）。
  可选。

- **audioStartFrom** _number_

  音频起始时间（毫秒）。
  可选。

- **autoChapters** _boolean_

  是否为转写自动生成章节。
  可选。

- **autoHighlights** _boolean_

  是否自动生成亮点摘要。
  可选。

- **boostParam** _enum_

  转写 boost 参数。
  允许：`'low'`、`'default'`、`'high'`。
  可选。

- **contentSafety** _boolean_

  是否启用内容安全过滤。
  可选。

- **contentSafetyConfidence** _number_

  内容安全阈值（25–100）。
  可选。

- **customSpelling** _array of objects_

  自定义拼写规则：每项含 `from`（字符串数组）与 `to`（字符串）。
  可选。

- **disfluencies** _boolean_

  是否保留语气词与不流畅片段（如 um、uh）。
  可选。

- **entityDetection** _boolean_

  是否进行实体检测。
  可选。

- **filterProfanity** _boolean_

  是否过滤脏话。
  可选。

- **formatText** _boolean_

  是否对文本做格式化。
  可选。

- **iabCategories** _boolean_

  是否包含 IAB 分类标签。
  可选。

- **languageCode** _string_

  音频语言代码。
  支持多种 ISO 639-1 / ISO 639-3。
  可选。

- **languageConfidenceThreshold** _number_

  语言检测置信度阈值。
  可选。

- **languageDetection** _boolean_

  是否启用语言检测。
  可选。

- **multichannel** _boolean_

  是否分声道处理多声道音频。
  可选。

- **punctuate** _boolean_

  是否添加标点。
  可选。

- **redactPii** _boolean_

  是否对个人信息（PII）做脱敏。
  可选。

- **redactPiiAudio** _boolean_

  是否在音频文件中遮盖 PII。
  可选。

- **redactPiiAudioQuality** _enum_

  脱敏音频质量。
  允许：`'mp3'`、`'wav'`。
  可选。

- **redactPiiPolicies** _array of enums_

  PII 脱敏策略类型（如 `'person_name'`、`'phone_number'` 等）。
  可选。

- **redactPiiSub** _enum_

  PII 替换方式。
  允许：`'entity_name'`、`'hash'`。
  可选。

- **sentimentAnalysis** _boolean_

  是否进行情感分析。
  可选。

- **speakerLabels** _boolean_

  是否为不同说话人打标签。
  可选。

- **speakersExpected** _number_

  预期说话人数量。
  可选。

- **speechThreshold** _number_

  语音检测阈值（0–1）。
  可选。

- **summarization** _boolean_

  是否生成转写摘要。
  可选。

- **summaryModel** _enum_

  摘要所用模型。
  允许：`'informative'`、`'conversational'`、`'catchy'`。
  可选。

- **summaryType** _enum_

  摘要类型。
  允许：`'bullets'`、`'bullets_verbose'`、`'gist'`、`'headline'`、`'paragraph'`。
  可选。

- **webhookAuthHeaderName** _string_

  Webhook 认证头名称。
  可选。

- **webhookAuthHeaderValue** _string_

  Webhook 认证头取值。
  可选。

- **webhookUrl** _string_

  Webhook 通知 URL。
  可选。

- **wordBoost** _array of strings_

  需要在转写中「加权」的词表。
  可选。
