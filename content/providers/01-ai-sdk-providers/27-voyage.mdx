# Voyage AI 提供商

[Voyage AI](https://www.voyageai.com/) 提供商为 Voyage AI API 提供嵌入与重排序模型支持。

## 安装

Voyage AI 提供商位于 `@ai-sdk/voyage` 模块中，可使用以下命令安装：

  </Tab>

## Provider 实例

可从 `@ai-sdk/voyage` 导入默认的 `voyage` 提供商实例：

```ts
import { voyage } from '@ai-sdk/voyage';
```

如需自定义配置，可导入 `createVoyage` 并创建实例：

```ts
import { createVoyage } from '@ai-sdk/voyage';

const voyage = createVoyage({
  // custom settings
});
```

可使用以下可选设置自定义 Voyage AI 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀，例如经由代理。
  默认前缀为 `https://api.voyageai.com/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `VOYAGE_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 嵌入模型

可使用 `.embedding()` 工厂方法创建调用 [Voyage AI embeddings API](https://docs.voyageai.com/reference/embeddings-api) 的模型。

```ts
const model = voyage.embedding('voyage-3.5');
```

可使用 Voyage AI 嵌入模型与 `embed` 函数生成向量：

```ts
import { voyage } from '@ai-sdk/voyage';
import { embed } from 'ai';

const { embedding } = await embed({
  model: voyage.embedding('voyage-3.5'),
  value: 'sunny day at the beach',
});
```

Voyage AI 嵌入模型支持通过 `providerOptions.voyage` 传入额外选项：

```ts
import { voyage, type VoyageEmbeddingModelOptions } from '@ai-sdk/voyage';
import { embed } from 'ai';

const { embedding } = await embed({
  model: voyage.embedding('voyage-4-large'),
  value: 'sunny day at the beach',
  providerOptions: {
    voyage: {
      inputType: 'document',
      outputDimension: 512,
    } satisfies VoyageEmbeddingModelOptions,
  },
});
```

可用的提供商选项：

- **inputType** _'query' | 'document' | null_

  指定输入类型。默认为 `null`。
  - `null`：模型直接将输入转换为数值向量。
  - `'query'`：用于在向量数据库中检索时的查询向量。
  - `'document'`：存入向量数据库、供检索使用的文档向量。

- **truncation** _boolean_

  是否截断文本以适配模型上下文长度。默认为 `true`。

- **outputDimension** _number_

  输出向量维度。
  适用于支持可调维度的模型（如 `voyage-4-large`、`voyage-4`、`voyage-4-lite`、`voyage-4-nano`、`voyage-code-3.5`、`voyage-code-3`、`voyage-3-large`、`voyage-3.5`、`voyage-3.5-lite`）。
  可选：256、512、1024、2048。

- **outputDtype** _'float' | 'int8' | 'uint8' | 'binary' | 'ubinary'_

  输出嵌入的数据类型。默认为 `'float'`。
  - `'float'`：32 位浮点。
  - `'int8'`、`'uint8'`：8 位整数。
  - `'binary'`、`'ubinary'`：按位打包的单比特量化嵌入。

## 重排序模型

可使用 `.reranking()` 工厂方法创建调用 [Voyage AI rerank API](https://docs.voyageai.com/reference/rerank-api) 的模型。

```ts
const model = voyage.reranking('rerank-2.5');
```

可使用 Voyage AI 重排序模型与 `rerank` 函数对文档重排：

```ts
import { voyage } from '@ai-sdk/voyage';
import { rerank } from 'ai';

const documents = [
  'sunny day at the beach',
  'rainy afternoon in the city',
  'snowy night in the mountains',
];

const { ranking } = await rerank({
  model: voyage.reranking('rerank-2.5'),
  documents,
  query: 'talk about rain',
  topN: 2,
});

console.log(ranking);
```

Voyage AI 重排序模型可通过 `providerOptions.voyage` 传入额外选项：

```ts
import { voyage, type VoyageRerankingModelOptions } from '@ai-sdk/voyage';
import { rerank } from 'ai';

const { ranking } = await rerank({
  model: voyage.reranking('rerank-2.5'),
  documents: ['sunny day at the beach', 'rainy afternoon in the city'],
  query: 'talk about rain',
  providerOptions: {
    voyage: {
      returnDocuments: true,
      truncation: true,
    } satisfies VoyageRerankingModelOptions,
  },
});
```

可用的提供商选项：

- **returnDocuments** _boolean_

  是否在响应中返回文档全文。默认为 `false`。

- **truncation** _boolean_

  是否截断文本以适配模型上下文长度。默认为 `true`。

### 模型能力

| Model             | 上下文长度 |
| ----------------- | ---------- |
| `rerank-2.5`      | 32,000     |
| `rerank-2.5-lite` | 32,000     |
| `rerank-2`        | 16,000     |
| `rerank-2-lite`   | 8,000      |
| `rerank-1`        | 8,000      |
| `rerank-lite-1`   | 4,000      |
