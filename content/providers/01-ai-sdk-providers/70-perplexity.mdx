# Perplexity 提供商

[Perplexity](https://sonar.perplexity.ai) 提供商可访问 Sonar API —— 将实时 Web 搜索与自然语言处理结合的语言模型。每次响应基于当前网页数据并附带详尽引用，适合研究、事实核查与获取最新资讯。

可在 [Perplexity 文档站](https://docs.perplexity.ai) 了解如何获取 API Key。

## 安装

## Provider 实例

可从 `@ai-sdk/perplexity` 导入默认的 `perplexity` 提供商实例：

```ts
import { perplexity } from '@ai-sdk/perplexity';
```

如需自定义配置，可导入 `createPerplexity` 并创建实例：

```ts
import { createPerplexity } from '@ai-sdk/perplexity';

const perplexity = createPerplexity({
  apiKey: process.env.PERPLEXITY_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Perplexity 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀。
  默认前缀为 `https://api.perplexity.ai`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `PERPLEXITY_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 语言模型

可使用提供商实例创建 Perplexity 模型：

```ts
import { perplexity } from '@ai-sdk/perplexity';
import { generateText } from 'ai';

const { text } = await generateText({
  model: perplexity('sonar-pro'),
  prompt: 'What are the latest developments in quantum computing?',
});
```

### 来源（Sources）

用于生成回复的网页会出现在结果的 `sources` 属性中：

```ts
import { perplexity } from '@ai-sdk/perplexity';
import { generateText } from 'ai';

const { text, sources } = await generateText({
  model: perplexity('sonar-pro'),
  prompt: 'What are the latest developments in quantum computing?',
});

console.log(sources);
```

### Provider 选项与元数据

Perplexity 提供商会在响应中通过 `providerMetadata` 附带额外元数据。
更多配置可通过 `providerOptions` 传入。

```ts
const result = await generateText({
  model: perplexity('sonar-pro'),
  prompt: 'What are the latest developments in quantum computing?',
  providerOptions: {
    perplexity: {
      return_images: true, // Enable image responses (Tier-2 Perplexity users only)
      search_recency_filter: 'week', // Filter search results by recency
    },
  },
});

console.log(result.providerMetadata);
// Example output:
// {
//   perplexity: {
//     usage: { citationTokens: 5286, numSearchQueries: 1 },
//     images: [
//       { imageUrl: "https://example.com/image1.jpg", originUrl: "https://elsewhere.com/page1", height: 1280, width: 720 },
//       { imageUrl: "https://example.com/image2.jpg", originUrl: "https://elsewhere.com/page2", height: 1280, width: 720 }
//     ]
//   },
// }
```

#### Provider 选项

可用的提供商专有选项：

- **return_images** _boolean_

  启用图像类响应。为 `true` 时响应可能包含相关图像。仅限 Perplexity Tier-2 及以上用户。

- **search_recency_filter** _string_

  按时效过滤搜索结果。可选：`'hour'`、`'day'`、`'week'`、`'month'`。未指定则不限时间。

<Note>
  其他 [Perplexity API 参数](https://docs.perplexity.ai/api-reference/chat-completions) 也可通过 `providerOptions.perplexity` 透传。
</Note>

#### Provider 元数据

响应元数据包括：

- `usage`：包含 `citationTokens` 与 `numSearchQueries` 的对象
- `images`：当启用 `return_images` 时为图像对象数组（仅 Tier-2 用户）。每张图包含 `imageUrl`、`originUrl`、`height`、`width`。

### PDF 支持

Perplexity 提供商支持读取 PDF。
可通过消息的 `file` 类型附带 PDF：

```ts
const result = await generateText({
  model: perplexity('sonar-pro'),
  messages: [
    {
      role: 'user',
      content: [
        {
          type: 'text',
          text: 'What is this document about?',
        },
        {
          type: 'file',
          data: fs.readFileSync('./data/ai.pdf'),
          mediaType: 'application/pdf',
          filename: 'ai.pdf', // optional
        },
      ],
    },
  ],
});
```

也可传入 PDF 的 URL：

```ts
{
  type: 'file',
  data: new URL('https://example.com/document.pdf'),
  mediaType: 'application/pdf',
  filename: 'document.pdf', // optional
}
```

模型可读取 PDF 内容并据此回答问题。
