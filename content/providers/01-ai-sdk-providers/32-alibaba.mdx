# Alibaba 提供商

[阿里云百炼](https://modelstudio.console.alibabacloud.com/) 提供通义千问（Qwen）系列模型，包括高级推理能力。

可在 [控制台](https://modelstudio.console.alibabacloud.com/) 获取 API Key。

## 安装

Alibaba 提供商位于 `@ai-sdk/alibaba` 模块中，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/alibaba` 导入默认的 `alibaba` 提供商实例：

```ts
import { alibaba } from '@ai-sdk/alibaba';
```

自定义配置时可导入 `createAlibaba`，并按你的设置创建提供商实例：

```ts
import { createAlibaba } from '@ai-sdk/alibaba';

const alibaba = createAlibaba({
  apiKey: process.env.ALIBABA_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Alibaba 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀（例如代理或区域端点）。
  默认前缀为 `https://dashscope-intl.aliyuncs.com/compatible-mode/v1`。

- **videoBaseURL** _string_

  视频生成 API 使用的前缀；视频接口使用 DashScope 原生端点（非 OpenAI 兼容端点）。
  默认前缀为 `https://dashscope-intl.aliyuncs.com`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `ALIBABA_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义请求头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

- **includeUsage** _boolean_

  是否在流式响应中包含用量信息。启用后，令牌用量包含在最后一个数据块中。
  默认为 `true`。

## 语言模型

可使用提供商实例创建语言模型：

```ts
import { alibaba } from '@ai-sdk/alibaba';
import { generateText } from 'ai';

const { text } = await generateText({
  model: alibaba('qwen-plus'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

也可使用 `.chatModel()` 或 `.languageModel()` 工厂方法：

```ts
const model = alibaba.chatModel('qwen-plus');
// or
const model = alibaba.languageModel('qwen-plus');
```

Alibaba 语言模型可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

面向 Alibaba 模型可使用以下可选提供商选项：

- **enableThinking** _boolean_

  是否为支持的模型启用思考/推理模式。启用后模型会在回复前生成推理内容。
  默认为 `false`。

- **thinkingBudget** _number_

  可生成的推理 token 数量上限。

- **parallelToolCalls** _boolean_

  是否在工具使用过程中启用并行函数调用。
  默认为 `true`。

### Thinking 模式

阿里云 Qwen 模型支持面向复杂任务的思考/推理模式：

```ts
import { alibaba, type AlibabaLanguageModelChatOptions } from '@ai-sdk/alibaba';
import { generateText } from 'ai';

const { text, reasoning } = await generateText({
  model: alibaba('qwen3-max'),
  providerOptions: {
    alibaba: {
      enableThinking: true,
      thinkingBudget: 2048,
    } satisfies AlibabaLanguageModelChatOptions,
  },
  prompt: 'How many "r"s are in the word "strawberry"?',
});

console.log('Reasoning:', reasoning);
console.log('Answer:', text);
```

对仅推理类模型（如 `qwen3-235b-a22b-thinking-2507`），思考模式默认开启。

### 工具调用

Alibaba 模型支持并行工具调用：

```ts
import { alibaba } from '@ai-sdk/alibaba';
import { generateText, tool } from 'ai';
import { z } from 'zod';

const { text } = await generateText({
  model: alibaba('qwen-plus'),
  tools: {
    weather: tool({
      description: 'Get the weather in a location',
      parameters: z.object({
        location: z.string().describe('The location to get the weather for'),
      }),
      execute: async ({ location }) => ({
        location,
        temperature: 72 + Math.floor(Math.random() * 21) - 10,
      }),
    }),
  },
  prompt: 'What is the weather in San Francisco?',
});
```

### Prompt 缓存

阿里云同时支持隐式与显式提示缓存，以降低重复提示的成本。

**隐式缓存**自动生效——提供商会在无需配置的情况下缓存适用内容。需要更细粒度控制时，可使用 **显式缓存**，通过 `cacheControl` 标记特定消息：

### 单条消息的缓存控制

```ts
import { alibaba } from '@ai-sdk/alibaba';
import { generateText } from 'ai';

const { text, usage } = await generateText({
  model: alibaba('qwen-plus'),
  messages: [
    {
      role: 'system',
      content: 'You are a helpful assistant. [... long system prompt ...]',
      providerOptions: {
        alibaba: {
          cacheControl: { type: 'ephemeral' },
        },
      },
    },
  ],
});
```

### 多段消息的缓存控制

```ts
import { alibaba } from '@ai-sdk/alibaba';
import { generateText } from 'ai';

const longDocument = '... large document content ...';

const { text, usage } = await generateText({
  model: alibaba('qwen-plus'),
  messages: [
    {
      role: 'user',
      content: [
        {
          type: 'text',
          text: 'Context: Please analyze this document.',
        },
        {
          type: 'text',
          text: longDocument,
          providerOptions: {
            alibaba: {
              cacheControl: { type: 'ephemeral' },
            },
          },
        },
      ],
    },
  ],
});
```

**说明：** 可被缓存的单块内容最少需 1,024 个 token。

## 视频模型

可使用 `.video()` 工厂方法创建调用阿里云 DashScope API 的 [Wan](https://www.alibabacloud.com/help/en/model-studio/use-video-generation) 视频模型。
详见 AI SDK 的 [generateVideo()](/docs/reference/ai-sdk-core/generate-video)。

Alibaba 支持三种视频生成模式：文生视频、图生视频（首帧）、参考图生视频。

### 文生视频

由文本提示生成视频：

```ts
import { alibaba, type AlibabaVideoModelOptions } from '@ai-sdk/alibaba';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: alibaba.video('wan2.6-t2v'),
  prompt: 'A serene mountain lake at sunset with gentle ripples on the water.',
  resolution: '1280x720',
  duration: 5,
  providerOptions: {
    alibaba: {
      promptExtend: true,
      pollTimeoutMs: 600000, // 10 minutes
    } satisfies AlibabaVideoModelOptions,
  },
});
```

### 图生视频

由首帧图像与可选文本提示生成视频：

```ts
import { alibaba, type AlibabaVideoModelOptions } from '@ai-sdk/alibaba';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: alibaba.video('wan2.6-i2v'),
  prompt: {
    image: 'https://example.com/landscape.jpg',
    text: 'Camera slowly pans across the landscape',
  },
  duration: 5,
  providerOptions: {
    alibaba: {
      pollTimeoutMs: 600000, // 10 minutes
    } satisfies AlibabaVideoModelOptions,
  },
});
```

### 参考图生视频

使用参考图像和/或视频保持角色一致性。在提示中用 `character1`、`character2` 等占位符引用角色：

```ts
import { alibaba, type AlibabaVideoModelOptions } from '@ai-sdk/alibaba';
import { experimental_generateVideo as generateVideo } from 'ai';

const { video } = await generateVideo({
  model: alibaba.video('wan2.6-r2v-flash'),
  prompt: 'character1 walks through a beautiful garden and waves at the camera',
  resolution: '1280x720',
  duration: 5,
  providerOptions: {
    alibaba: {
      referenceUrls: ['https://example.com/character-reference.jpg'],
      pollTimeoutMs: 600000, // 10 minutes
    } satisfies AlibabaVideoModelOptions,
  },
});
```

### 视频的提供商选项

可通过 `providerOptions.alibaba` 使用以下提供商选项：

- **negativePrompt** _string_

  描述生成视频中应避免出现的内容（最多 500 字符）。

- **audioUrl** _string_

  用于音视频同步的音频文件 URL（WAV/MP3，3–30 秒，最大 15MB）。

- **promptExtend** _boolean_

  是否启用提示扩展/改写以提升生成质量。默认为 `true`。

- **shotType** `'single'` | `'multi'`

  镜头类型。`'multi'` 启用多分镜叙事（仅 wan2.6 模型）。

- **watermark** _boolean_

  是否为生成的视频添加水印。默认为 `false`。

- **audio** _boolean_

  是否生成音频（适用于支持音频的 I2V、R2V 模型）。

- **referenceUrls** _string[]_

  参考图生视频模式下参考图像/视频的 URL 数组。支持 0–5 张图与 0–3 个视频，总计最多 5 个引用。

- **pollIntervalMs** _number_

  轮询任务状态的间隔（毫秒）。默认为 5000。

- **pollTimeoutMs** _number_

  视频生成的最长等待时间（毫秒）。默认为 600000（10 分钟）。

<Note>
  视频生成是异步过程，可能需要数分钟。
  建议将 `pollTimeoutMs` 至少设为 10 分钟（600000 ms）以保证稳定。
</Note>
