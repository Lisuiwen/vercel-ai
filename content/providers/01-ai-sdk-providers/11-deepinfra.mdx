# DeepInfra 提供商

[DeepInfra](https://deepinfra.com) 提供商通过 DeepInfra API 接入众多前沿模型，包括 Llama 3、Mixtral、Qwen 以及多种热门开源模型。

## 安装

DeepInfra 提供商通过 `@ai-sdk/deepinfra` 模块提供，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/deepinfra` 导入默认的 `deepInfra` 提供商实例：

```ts
import { deepInfra } from '@ai-sdk/deepinfra';
```

如需自定义配置，可导入 `createDeepInfra` 并创建实例：

```ts
import { createDeepInfra } from '@ai-sdk/deepinfra';

const deepInfra = createDeepInfra({
  apiKey: process.env.DEEPINFRA_API_KEY ?? '',
});
```

可使用以下可选设置自定义 DeepInfra 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀，例如经由代理。
  默认前缀为 `https://api.deepinfra.com/v1`。

  说明：语言模型与嵌入使用 `{baseURL}/openai` 下的 OpenAI 兼容端点；
  图像模型则使用 `{baseURL}/inference`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `DEEPINFRA_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 语言模型

可使用提供商实例创建语言模型；首个参数为模型 ID，例如：

```ts
import { deepInfra } from '@ai-sdk/deepinfra';
import { generateText } from 'ai';

const { text } = await generateText({
  model: deepInfra('meta-llama/Meta-Llama-3.1-70B-Instruct'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

DeepInfra 语言模型也可用于 `streamText`（参见 [AI SDK Core](/docs/ai-sdk-core)）。

## 图像模型

可使用 `.image()` 工厂方法创建 DeepInfra 图像模型。
关于在 AI SDK 中生成图像，参见 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

```ts
import { deepInfra } from '@ai-sdk/deepinfra';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: deepInfra.image('stabilityai/sd3.5'),
  prompt: 'A futuristic cityscape at sunset',
  aspectRatio: '16:9',
});
```

<Note>
  模型对 `size` 与 `aspectRatio` 的支持各不相同，请参阅 [DeepInfra 文生图模型页](https://deepinfra.com/models/text-to-image) 中单模型的文档与可选参数。
</Note>

### 模型特定选项

可通过 `providerOptions.deepinfra` 传入模型专有参数：

```ts
import { deepInfra } from '@ai-sdk/deepinfra';
import { generateImage } from 'ai';

const { image } = await generateImage({
  model: deepInfra.image('stabilityai/sd3.5'),
  prompt: 'A futuristic cityscape at sunset',
  aspectRatio: '16:9',
  providerOptions: {
    deepinfra: {
      num_inference_steps: 30, // Control the number of denoising steps (1-50)
    },
  },
});
```

### 图像编辑

DeepInfra 支持通过诸如 `Qwen/Qwen-Image-Edit` 的模型编辑图像；经 `prompt.images` 传入待变换或修改的图像。

#### 基础编辑

用文本指令变换既有图像：

```ts
const imageBuffer = readFileSync('./input-image.png');

const { images } = await generateImage({
  model: deepInfra.image('Qwen/Qwen-Image-Edit'),
  prompt: {
    text: 'Turn the cat into a golden retriever dog',
    images: [imageBuffer],
  },
  size: '1024x1024',
});
```

#### 遮罩（Inpaint）

借助遮罩仅编辑局部；遮罩透明区域为待编辑区域：

```ts
const image = readFileSync('./input-image.png');
const mask = readFileSync('./mask.png');

const { images } = await generateImage({
  model: deepInfra.image('Qwen/Qwen-Image-Edit'),
  prompt: {
    text: 'A sunlit indoor lounge area with a pool containing a flamingo',
    images: [image],
    mask: mask,
  },
});
```

#### 多图合成

将多张参考图融合为单张输出：

```ts
const cat = readFileSync('./cat.png');
const dog = readFileSync('./dog.png');

const { images } = await generateImage({
  model: deepInfra.image('Qwen/Qwen-Image-Edit'),
  prompt: {
    text: 'Create a scene with both animals together, playing as friends',
    images: [cat, dog],
  },
});
```

<Note>
  输入图像可使用 `Buffer`、`ArrayBuffer`、`Uint8Array` 或 base64 字符串。DeepInfra 使用位于 `https://api.deepinfra.com/v1/openai/images/edits` 的 OpenAI 兼容图像编辑 API。
</Note>

### 模型能力

对支持宽高比的模型，常见可选比例包括：
`1:1 (default), 16:9, 1:9, 3:2, 2:3, 4:5, 5:4, 9:16, 9:21`

对支持 `size` 的模型，尺寸通常需满足：

- 为 32 的倍数
- 宽、高在 256–1440 像素之间
- 默认 1024×1024

## 嵌入模型

可使用 `.embeddingModel()` 工厂方法创建 DeepInfra 嵌入模型。
关于嵌入，参见 [embed()](/docs/reference/ai-sdk-core/embed)。

```ts
import { deepInfra } from '@ai-sdk/deepinfra';
import { embed } from 'ai';

const { embedding } = await embed({
  model: deepInfra.embeddingModel('BAAI/bge-large-en-v1.5'),
  value: 'sunny day at the beach',
});
```
