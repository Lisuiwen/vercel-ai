# Fal 提供商

[Fal AI](https://fal.ai/) 面向开发者提供生成式媒体平台，具备极快的推理能力。该平台针对扩散模型运行做了性能优化，速度最高可达同类方案的约 4 倍。

## Provider 实例

可从 `@ai-sdk/fal` 导入默认的 `fal` 提供商实例：

```ts
import { fal } from '@ai-sdk/fal';
```

如需自定义配置，可导入 `createFal` 并创建实例：

```ts
import { createFal } from '@ai-sdk/fal';

const fal = createFal({
  apiKey: 'your-api-key', // optional, defaults to FAL_API_KEY environment variable, falling back to FAL_KEY
  baseURL: 'custom-url', // optional
  headers: {
    /* custom headers */
  }, // optional
});
```

可使用以下可选设置自定义 Fal 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀，例如经由代理。
  默认前缀为 `https://fal.run`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `FAL_API_KEY`，若无则回退至 `FAL_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

## 图像模型

可使用 `.image()` 工厂方法创建 Fal 图像模型。
关于在 AI SDK 中生成图像，参见 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

### 基本用法

```ts
import { fal } from '@ai-sdk/fal';
import { generateImage } from 'ai';
import fs from 'fs';

const { image, providerMetadata } = await generateImage({
  model: fal.image('fal-ai/flux/dev'),
  prompt: 'A serene mountain landscape at sunset',
});

const filename = `image-${Date.now()}.png`;
fs.writeFileSync(filename, image.uint8Array);
console.log(`Image saved to ${filename}`);
```

Fal 图像模型可能会在响应中附带与图像或请求相关的额外信息。

每张图像可能出现的部分属性示例：

```js
providerMetadata.fal.images[0].nsfw; // boolean, image is not safe for work
providerMetadata.fal.images[0].width; // number, image width
providerMetadata.fal.images[0].height; // number, image height
providerMetadata.fal.images[0].contentType; // string, mime type of the image
```

#### 图像修改（Modify）

通过文本提示变换已有图像。

```ts
await generateImage({
  model: fal.image('fal-ai/flux-pro/kontext/max'),
  prompt: {
    text: 'Put a donut next to the flour.',
    images: [
      'https://v3.fal.media/files/rabbit/rmgBxhwGYb2d3pl3x9sKf_output.png',
    ],
  },
});
```

图像可使用 base64 字符串、`Uint8Array`、`ArrayBuffer` 或 `Buffer`。
也可附带 mask：

```ts
await generateImage({
  model: fal.image('fal-ai/flux-pro/kontext/max'),
  prompt: {
    text: 'Put a donut next to the flour.',
    images: [imageBuffer],
    mask: maskBuffer,
  },
});
```

### Provider 选项

Fal 图像模型通过 `providerOptions.fal` 支持灵活的提供商选项。可为具体模型的 API 传入其支持的任意参数，常见字段包括：

- **imageUrl** — 图生图的参考图像 URL（已弃用，请改用 `prompt.images`）
- **strength** — 输出相对输入的差异程度
- **guidanceScale** — 对提示词的贴合度（范围 1–20）
- **numInferenceSteps** — 去噪步数（范围 1–50）
- **enableSafetyChecker** — 是否启用安全检查
- **outputFormat** — 输出格式：`'jpeg'` 或 `'png'`
- **syncMode** — 是否在完成后才返回响应
- **acceleration** — 生成速度：`'none'`、`'regular'` 或 `'high'`
- **safetyTolerance** — 内容安全级别（1–6，数值越小越严格）
- **useMultipleImages** — 为 true 时将多输入图转换为模型所需的 `image_urls` 数组（如 `fal-ai/flux-2/edit`）

<Note type="warning">
  **弃用说明**：snake_case 参数名（如 `image_url`、`guidance_scale`）已弃用，未来版本将移除。请改用 camelCase（例如 `imageUrl`、`guidanceScale`）。
</Note>

特定模型的参数请参阅 [Fal AI 模型文档](https://fal.ai/models)。

## 转写模型

可使用 `.transcription()` 工厂方法创建调用 [Fal transcription API](https://docs.fal.ai/guides/convert-speech-to-text) 的模型。

首个参数为模型 ID，且不含 `fal-ai/` 前缀，例如 `wizper`。

```ts
const model = fal.transcription('wizper');
```

也可通过 `providerOptions` 传入额外选项；例如传入 `batchSize` 以提高并行处理的音频分块数量。

```ts highlight="6"
import { experimental_transcribe as transcribe } from 'ai';
import { fal, type FalTranscriptionModelOptions } from '@ai-sdk/fal';
import { readFile } from 'fs/promises';

const result = await transcribe({
  model: fal.transcription('wizper'),
  audio: await readFile('audio.mp3'),
  providerOptions: {
    fal: { batchSize: 10 } satisfies FalTranscriptionModelOptions,
  },
});
```

可用的提供商选项：

- **language** _string_
  音频语言。默认为 `'en'`。若为 `null` 则自动检测。
  接受 ISO 语言代码（如 `'en'`、`'fr'`、`'zh'` 等）。
  可选。

- **diarize** _boolean_
  是否对音频做说话人分离（标注不同说话人）。
  默认为 true。
  可选。

- **chunkLevel** _string_
  返回分块粒度：`'segment'` 或 `'word'`。
  默认：`"segment"`。
  可选。

- **version** _string_
  模型版本（均为 Whisper large 家族变体）。
  默认：`"3"`。
  可选。

- **batchSize** _number_
  处理批大小。
  默认：64。
  可选。

- **numSpeakers** _number_
  音频中说话人数量；未提供则自动估计。
  可选。

## 语音模型

可使用 `.speech()` 工厂方法创建调用 Fal TTS 端点的模型。

### 基本用法

```ts
import { experimental_generateSpeech as generateSpeech } from 'ai';
import { fal } from '@ai-sdk/fal';

const result = await generateSpeech({
  model: fal.speech('fal-ai/minimax/speech-02-hd'),
  text: 'Hello from the AI SDK!',
});
```

### 模型能力

| Model                                     | 说明                                                                                                                                                               |
| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `fal-ai/minimax/voice-clone`              | 从采样音频克隆音色并由文本生成语音                                                                                                                                     |
| `fal-ai/minimax/voice-design`             | 从文本描述设计个性化音色并由文本生成语音                                                                                                                               |
| `fal-ai/dia-tts/voice-clone`              | 从采样音频克隆对白音色并由文本生成对白                                                                                                                                 |
| `fal-ai/minimax/speech-02-hd`             | 文本生成语音（多音色，高清）                                                                                                                                         |
| `fal-ai/minimax/speech-02-turbo`          | 文本快速生成语音（多音色）                                                                                                                                             |
| `fal-ai/dia-tts`                          | 从台词直接生成写实对白，并可借音频条件控制情绪；生成笑声、清嗓等自然副语言                                                                                                     |
| `resemble-ai/chatterboxhd/text-to-speech` | Resemble Chatterbox：表现力强且自然；支持独特情绪控制、短音频瞬时克隆与水印                                                                     |

### Provider 选项

视模型而定，可通过 `providerOptions.fal` 传入：

- **voice_setting** _object_
  - `voice_id` (string)：预置音色 ID
  - `speed` (number)：0.5–2.0
  - `vol` (number)：0–10
  - `pitch` (number)：-12–12
  - `emotion` (enum)：happy | sad | angry | fearful | disgusted | surprised | neutral
  - `english_normalization` (boolean)

- **audio_setting** _object_
  音频相关模型配置。

- **language_boost** _enum_
  Chinese | Chinese,Yue | English | Arabic | Russian | Spanish | French | Portuguese | German | Turkish | Dutch | Ukrainian | Vietnamese | Indonesian | Japanese | Italian | Korean | Thai | Polish | Romanian | Greek | Czech | Finnish | Hindi | auto

- **pronunciation_dict** _object_
  自定义发音词典。

模型特有参数（如 `audio_url`、`prompt`、`preview_text`、`ref_audio_url`、`ref_text`）可直接放在 `providerOptions.fal` 下并转发至 Fal API。
