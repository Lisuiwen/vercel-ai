# Together.ai 提供商

[Together.ai](https://together.ai) 通过 [Together.ai API](https://docs.together.ai/reference) 支持 200+ 开源模型。

## 安装

Together.ai 提供商位于 `@ai-sdk/togetherai` 模块中，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/togetherai` 导入默认的 `togetherai` 提供商实例：

```ts
import { togetherai } from '@ai-sdk/togetherai';
```

如需自定义配置，可导入 `createTogetherAI` 并以你的设置创建提供商实例：

```ts
import { createTogetherAI } from '@ai-sdk/togetherai';

const togetherai = createTogetherAI({
  apiKey: process.env.TOGETHER_API_KEY ?? '',
});
```

可使用以下可选设置自定义 Together.ai 提供商实例：

- **baseURL** _string_

  为 API 调用使用不同的 URL 前缀（例如经由代理）。
  默认前缀为 `https://api.together.xyz/v1`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。
  默认为环境变量 `TOGETHER_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义请求头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  默认为全局 `fetch`。
  可作为中间件拦截请求，
  或在测试中提供自定义 `fetch` 实现。

## 语言模型

可使用提供商实例创建 [Together.ai 模型](https://docs.together.ai/docs/serverless-models)。首个参数为模型 ID，例如 `google/gemma-2-9b-it`。

```ts
const model = togetherai('google/gemma-2-9b-it');
```

### 推理模型

Together.ai 在 `deepseek-ai/DeepSeek-R1` 的生成文本中用 `<think>` 标签暴露思考过程。
可使用 `extractReasoningMiddleware` 抽出该思考，并在结果的 `reasoning` 字段中提供：

```ts
import { togetherai } from '@ai-sdk/togetherai';
import { wrapLanguageModel, extractReasoningMiddleware } from 'ai';

const enhancedModel = wrapLanguageModel({
  model: togetherai('deepseek-ai/DeepSeek-R1'),
  middleware: extractReasoningMiddleware({ tagName: 'think' }),
});
```

随后可在 `generateText`、`streamText` 等函数中使用该包装后的模型。

### 示例

可使用 Together.ai 语言模型与 `generateText` 生成文本：

```ts
import { togetherai } from '@ai-sdk/togetherai';
import { generateText } from 'ai';

const { text } = await generateText({
  model: togetherai('meta-llama/Meta-Llama-3.1-8B-Instruct-Turbo'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

Together.ai 语言模型也可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

Together.ai 提供商还支持（仿照上文示例）通过 `togetherai.completionModel()` 调用 [补全模型](https://docs.together.ai/docs/serverless-models#language-models)，以及通过 `togetherai.embeddingModel()` 调用 [嵌入模型](https://docs.together.ai/docs/serverless-models#embedding-models)。

## 图像模型

可使用 `.image()` 工厂方法创建 Together.ai 图像模型。
详见 AI SDK 的 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

```ts
import { togetherai } from '@ai-sdk/togetherai';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: togetherai.image('black-forest-labs/FLUX.1-dev'),
  prompt: 'A delighted resplendent quetzal mid flight amidst raindrops',
});
```

可通过 `providerOptions` 传入可选的、面向提供商的请求参数。

```ts
import {
  togetherai,
  type TogetherAIImageModelOptions,
} from '@ai-sdk/togetherai';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: togetherai.image('black-forest-labs/FLUX.1-dev'),
  prompt: 'A delighted resplendent quetzal mid flight amidst raindrops',
  size: '512x512',
  // Optional additional provider-specific request parameters
  providerOptions: {
    togetherai: {
      steps: 40,
    } satisfies TogetherAIImageModelOptions,
  },
});
```

可用的提供商选项：

- **steps** _number_

  生成步数；更高值通常可提升画质。

- **guidance** _number_

  图像生成的引导强度（guidance scale）。

- **negative_prompt** _string_

  负向提示，约束应避免的内容。

- **disable_safety_checker** _boolean_

  是否关闭图像生成的安全检查。
  为 `true` 时，API 不会因疑似 NSFW 而拒绝图像。
  Flux Schnell Free 与 Flux Pro 模型不可用。

### 图像编辑

Together AI 通过 FLUX Kontext 模型支持图像编辑。通过 `prompt.images` 传入输入图像即可变换或编辑。

<Note>
  Together AI 不支持基于遮罩的修复（inpainting），请用描述性提示说明希望对图像作出的修改。
</Note>

#### 基本图像编辑

使用文本提示变换现有图像：

```ts
const imageBuffer = readFileSync('./input-image.png');

const { images } = await generateImage({
  model: togetherai.image('black-forest-labs/FLUX.1-kontext-pro'),
  prompt: {
    text: 'Turn the cat into a golden retriever dog',
    images: [imageBuffer],
  },
  size: '1024x1024',
  providerOptions: {
    togetherai: {
      steps: 28,
    } satisfies TogetherAIImageModelOptions,
  },
});
```

#### 使用图像 URL

也可直接传入图像 URL：

```ts
const { images } = await generateImage({
  model: togetherai.image('black-forest-labs/FLUX.1-kontext-pro'),
  prompt: {
    text: 'Make the background a lush rainforest',
    images: ['https://example.com/photo.png'],
  },
  size: '1024x1024',
  providerOptions: {
    togetherai: {
      steps: 28,
    } satisfies TogetherAIImageModelOptions,
  },
});
```

<Note>
  输入图像可为 `Buffer`、`ArrayBuffer`、`Uint8Array`、base64 字符串或 URL。Together AI 每个请求仅支持单张输入图像。
</Note>

## 嵌入模型

可使用 `.embeddingModel()` 工厂方法创建 Together.ai 嵌入模型。
详见 AI SDK 的 [embed()](/docs/reference/ai-sdk-core/embed)。

```ts
import { togetherai } from '@ai-sdk/togetherai';
import { embed } from 'ai';

const { embedding } = await embed({
  model: togetherai.embeddingModel('togethercomputer/m2-bert-80M-2k-retrieval'),
  value: 'sunny day at the beach',
});
```

## 重排序模型

可使用 `.reranking()` 工厂方法创建 Together.ai 重排序模型。
详见 AI SDK 的 [rerank()](/docs/reference/ai-sdk-core/rerank)。

```ts
import { togetherai } from '@ai-sdk/togetherai';
import { rerank } from 'ai';

const documents = [
  'sunny day at the beach',
  'rainy afternoon in the city',
  'snowy night in the mountains',
];

const { ranking } = await rerank({
  model: togetherai.reranking('mixedbread-ai/Mxbai-Rerank-Large-V2'),
  documents,
  query: 'talk about rain',
  topN: 2,
});

console.log(ranking);
// [
//   { originalIndex: 1, score: 0.9, document: 'rainy afternoon in the city' },
//   { originalIndex: 0, score: 0.3, document: 'sunny day at the beach' }
// ]
```

面向 JSON 对象的文档时，Together.ai 重排序模型还支持额外提供商选项，可指定用于打分的字段：

```ts
import {
  togetherai,
  type TogetherAIRerankingModelOptions,
} from '@ai-sdk/togetherai';
import { rerank } from 'ai';

const documents = [
  {
    from: 'Paul Doe',
    subject: 'Follow-up',
    text: 'We are happy to give you a discount of 20%.',
  },
  {
    from: 'John McGill',
    subject: 'Missing Info',
    text: 'Here is the pricing from Oracle: $5000/month',
  },
];

const { ranking } = await rerank({
  model: togetherai.reranking('mixedbread-ai/Mxbai-Rerank-Large-V2'),
  documents,
  query: 'Which pricing did we get from Oracle?',
  providerOptions: {
    togetherai: {
      rankFields: ['from', 'subject', 'text'], // Specify which fields to rank by
    } satisfies TogetherAIRerankingModelOptions,
  },
});
```

可用的提供商选项：

- **rankFields** _string[]_

  当文档为 JSON 对象时，用于打分的字段名数组。未指定则使用全部字段。

| `mixedbread-ai/Mxbai-Rerank-Large-V2` |
