# Black Forest Labs 提供商

[Black Forest Labs](https://bfl.ai/) 基于 FLUX 系列模型为开发者提供图像生成平台，具备速度快、画面质量高、支持上下文理解与编辑等优势，可实现精确且连贯的生成与修改。

## 安装

Black Forest Labs 提供商通过 `@ai-sdk/black-forest-labs` 模块提供，可使用以下命令安装：

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>

## Provider 实例

可从 `@ai-sdk/black-forest-labs` 导入默认的 `blackForestLabs` 提供商实例：

```ts
import { blackForestLabs } from '@ai-sdk/black-forest-labs';
```

如需自定义配置，可导入 `createBlackForestLabs` 并创建实例：

```ts
import { createBlackForestLabs } from '@ai-sdk/black-forest-labs';

const blackForestLabs = createBlackForestLabs({
  apiKey: 'your-api-key', // optional, defaults to BFL_API_KEY environment variable
  baseURL: 'custom-url', // optional
  headers: {
    /* custom headers */
  }, // optional
});
```

可使用以下可选设置自定义 Black Forest Labs 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀，例如切换到区域端点。
  默认前缀为 `https://api.bfl.ai/v1`。

- **apiKey** _string_

  通过 `x-key` 请求头发送的 API Key。
  默认为环境变量 `BFL_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。
  可用作拦截请求的中间件，
  或在测试中提供自定义 `fetch`。

- **pollIntervalMillis** _number_

  等待图像生成完成时轮询间隔（毫秒）。
  默认为 500ms。

- **pollTimeoutMillis** _number_

  轮询整体超时（毫秒）。
  默认为 60000ms（60 秒）。

## 图像模型

可使用 `.image()` 工厂方法创建 Black Forest Labs 图像模型。
关于在 AI SDK 中生成图像，参见 [generateImage()](/docs/reference/ai-sdk-core/generate-image)。

### 基本用法

```ts
import { writeFileSync } from 'node:fs';
import { blackForestLabs } from '@ai-sdk/black-forest-labs';
import { generateImage } from 'ai';

const { image, providerMetadata } = await generateImage({
  model: blackForestLabs.image('flux-pro-1.1'),
  prompt: 'A serene mountain landscape at sunset',
});

const filename = `image-${Date.now()}.png`;
writeFileSync(filename, image.uint8Array);
console.log(`Image saved to ${filename}`);
```

### 模型能力

Black Forest Labs 提供多款面向不同场景的模型；以下为示例。完整列表见 [Models 页面](https://bfl.ai/models)。

| Model                | 说明                                                                                                                                 |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `flux-kontext-pro`   | FLUX.1 Kontext [pro]：同时支持文本与参考图，可进行定向编辑与复杂变换                                                                     |
| `flux-kontext-max`   | FLUX.1 Kontext [max]：更强的提示遵从与排版生成                                                                                               |
| `flux-pro-1.1-ultra` | 超快、超高分辨率图像生成                                                                                                               |
| `flux-pro-1.1`       | 从文本快速生成高质量图像。                                                                                                                   |
| `flux-pro-1.0-fill`  | Inpainting：为遮罩区域补全新内容                                                                                                            |

宽高比覆盖自 3:7（竖构图）至 7:3（横构图）。

### 图像编辑

Kontext 模型支持引用参考图编辑。可通过 `prompt.images` 传入待变换、合并或编辑的图片。

#### 单图编辑

使用文本指令变换既有图像：

```ts
import {
  blackForestLabs,
  BlackForestLabsImageModelOptions,
} from '@ai-sdk/black-forest-labs';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: blackForestLabs.image('flux-kontext-pro'),
  prompt: {
    text: 'A baby elephant with a shirt that has the logo from the input image.',
    images: [
      'https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png',
    ],
  },
  providerOptions: {
    blackForestLabs: {
      width: 1024,
      height: 768,
    } satisfies BlackForestLabsImageModelOptions,
  },
});
```

#### 多参考图编辑

可使用最多 10 张参考图进行复合变换：

```ts
import { blackForestLabs } from '@ai-sdk/black-forest-labs';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: blackForestLabs.image('flux-kontext-pro'),
  prompt: {
    text: 'Combine the style of image 1 with the subject of image 2',
    images: [
      'https://example.com/style-reference.jpg',
      'https://example.com/subject-reference.jpg',
    ],
  },
});
```

<Note>
  输入图像可使用 URL 或 base64；单张最长 20MB 或最长 2000 万像素。
</Note>

#### Inpainting

`flux-pro-1.0-fill` 支持 Inpainting：通过 `prompt.images` 传原图，`prompt.mask` 传遮罩。

```ts
import { blackForestLabs } from '@ai-sdk/black-forest-labs';
import { generateImage } from 'ai';

const { images } = await generateImage({
  model: blackForestLabs.image('flux-pro-1.0-fill'),
  prompt: {
    text: 'A beautiful garden with flowers',
    images: ['https://example.com/source-image.jpg'],
    mask: 'https://example.com/mask-image.png',
  },
});
```

Mask 建议使用灰度图：白色区域为待填补区域，黑色为保留区域。

### Provider 选项

通过 `providerOptions.blackForestLabs` 传入的参数取决于模型 ID：

- **width** _number_ — 输出宽（像素，256–1920）。设定后将覆盖由 `size` 推导宽度。
- **height** _number_ — 输出高（像素，256–1920）。设定后将覆盖由 `size` 推导高度。
- **outputFormat** _string_ — 输出格式：`"jpeg"` 或 `"png"`。
- **steps** _number_ — 推理步数；更高可能提升质量但更耗时。
- **guidance** _number_ — 引导尺度；数值越高越贴近提示词。
- **imagePrompt** _string_ — 额外视觉上下文的 Base64 图像。
- **imagePromptStrength** _number_ — 图像提示影响强度（0.0–1.0）。
- **promptUpsampling** _boolean_ — 若为 true，对提示进行上采样优化。
- **raw** _boolean_ — Raw 模式，追求更自然的审美表达。
- **safetyTolerance** _number_ — 输入输出的审核强度（0 最严格，6 相对宽松）。
- **pollIntervalMillis** _number_ — 轮询间隔（默认 500ms）。
- **pollTimeoutMillis** _number_ — 轮询总超时（默认 60 秒）。
- **webhookUrl** _string_ — 异步完成通知 URL，须为有效 HTTP(S)。
- **webhookSecret** _string_ — Webhook 验签密钥，通过 `X-Webhook-Secret` 传输。

<Note>
  编辑场景的参考图像请放在 `prompt.images`，而非混入 provider options。最多支持 10 张 URL 或 base64。
</Note>

### Provider 元数据

`generateImage` 在 `providerMetadata.blackForestLabs.images[]` 中返回提供商元数据，单张图可包含：

- **seed** _number_ — 随机种子（便于复现）。
- **start_time** _number_ — 开始生成的 Unix 时间戳。
- **end_time** _number_ — 完成生成的 Unix 时间戳。
- **duration** _number_ — 耗时（秒）。
- **cost** _number_ — 本次调用成本。
- **inputMegapixels** _number_ — 输入图像百万像素总量。
- **outputMegapixels** _number_ — 输出图像百万像素总量。

```ts
import { blackForestLabs } from '@ai-sdk/black-forest-labs';
import { generateImage } from 'ai';

const { image, providerMetadata } = await generateImage({
  model: blackForestLabs.image('flux-pro-1.1'),
  prompt: 'A serene mountain landscape at sunset',
});

// Access provider metadata
const metadata = providerMetadata?.blackForestLabs?.images?.[0];
console.log('Seed:', metadata?.seed);
console.log('Cost:', metadata?.cost);
console.log('Duration:', metadata?.duration);
```

### 区域端点

默认请求发往 `https://api.bfl.ai/v1`。创建提供商时在 `baseURL` 中选择[区域端点](https://docs.bfl.ai/api_integration/integration_guidelines#regional-endpoints)：

```ts
import { createBlackForestLabs } from '@ai-sdk/black-forest-labs';

const blackForestLabs = createBlackForestLabs({
  baseURL: 'https://api.eu.bfl.ai/v1', // or https://api.us.bfl.ai/v1
});
```
