# DeepSeek 提供商

[DeepSeek](https://www.deepseek.com) 提供商通过 DeepSeek API 提供强大的语言模型访问能力。

可在 [DeepSeek 平台](https://platform.deepseek.com/api_keys) 获取 API Key。

## 安装

DeepSeek 提供商通过 `@ai-sdk/deepseek` 模块提供，可使用以下命令安装：

## Provider 实例

可从 `@ai-sdk/deepseek` 导入默认的 `deepSeek` 提供商实例：

```ts
import { deepSeek } from '@ai-sdk/deepseek';
```

如需自定义配置，可导入 `createDeepSeek` 并创建实例：

```ts
import { createDeepSeek } from '@ai-sdk/deepseek';

const deepSeek = createDeepSeek({
  apiKey: process.env.DEEPSEEK_API_KEY ?? '',
});
```

可使用以下可选设置自定义 DeepSeek 提供商实例：

- **baseURL** _string_

  为 API 请求使用不同的 URL 前缀。
  默认前缀为 `https://api.deepseek.com`。

- **apiKey** _string_

  通过 `Authorization` 请求头发送的 API Key。默认为环境变量 `DEEPSEEK_API_KEY`。

- **headers** _Record&lt;string,string&gt;_

  请求中要包含的自定义头。

- **fetch** _(input: RequestInfo, init?: RequestInit) => Promise&lt;Response&gt;_

  自定义 [fetch](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 实现。

## 语言模型

可使用提供商实例创建语言模型：

```ts
import { deepSeek } from '@ai-sdk/deepseek';
import { generateText } from 'ai';

const { text } = await generateText({
  model: deepSeek('deepseek-chat'),
  prompt: 'Write a vegetarian lasagna recipe for 4 people.',
});
```

也可使用 `.chat()` 或 `.languageModel()` 工厂方法：

```ts
const model = deepSeek.chat('deepseek-chat');
// or
const model = deepSeek.languageModel('deepseek-chat');
```

DeepSeek 语言模型可用于 `streamText`
（参见 [AI SDK Core](/docs/ai-sdk-core)）。

DeepSeek 模型还支持以下可选的提供商选项：

- `thinking` _object_

  可选。控制 thinking 模式（链式推理）。可使用 `deepseek-reasoner` 模型或通过此选项启用 thinking。
  - `type`：`'adaptive' | 'enabled' | 'disabled'` — 启用、禁用或由模型自适应决定是否思考。详见 [DeepSeek thinking 模式文档](https://api-docs.deepseek.com/guides/thinking_mode)。

- `reasoningEffort` _'low' | 'medium' | 'high' | 'xhigh' | 'max'_

  可选。控制 DeepSeek V4 推理模型的思考强度。按
  DeepSeek 文档，`low` 与 `medium` 在服务端映射为 `high`，`xhigh` 映射为 `max`，以兼容其他提供商。使用顶层 `reasoning` 设置时，`minimal` 会作为 `low` 发送，`low`、`medium`、`high`、`xhigh` 会直接传入 DeepSeek 原生的强度值。

```ts highlight="7-12"
import {
  deepSeek,
  type DeepSeekLanguageModelChatOptions,
} from '@ai-sdk/deepseek';
import { generateText } from 'ai';

const { text, reasoning } = await generateText({
  model: deepSeek('deepseek-chat'),
  prompt: 'How many "r"s are in the word "strawberry"?',
  providerOptions: {
    deepseek: {
      thinking: { type: 'enabled' },
      reasoningEffort: 'high',
    } satisfies DeepSeekLanguageModelChatOptions,
  },
});
```

### 推理

`deepseek-reasoner` 模型支持推理能力，推理过程通过流式输出暴露：

```ts
import { deepSeek } from '@ai-sdk/deepseek';
import { streamText } from 'ai';

const result = streamText({
  model: deepSeek('deepseek-reasoner'),
  prompt: 'How many "r"s are in the word "strawberry"?',
});

for await (const part of result.fullStream) {
  if (part.type === 'reasoning') {
    // This is the reasoning text
    console.log('Reasoning:', part.text);
  } else if (part.type === 'text') {
    // This is the final answer
    console.log('Answer:', part.text);
  }
}
```

如何将推理接入聊天机器人的更多说明，参见 [AI SDK UI：Chatbot](/docs/ai-sdk-ui/chatbot#reasoning)。

### 缓存 Token 用量

DeepSeek 提供磁盘上下文缓存技术，可对重复内容显著降低 token 成本。可通过响应中的 `providerMetadata` 查看缓存命中/未命中指标：

```ts
import { deepSeek } from '@ai-sdk/deepseek';
import { generateText } from 'ai';

const result = await generateText({
  model: deepSeek('deepseek-chat'),
  prompt: 'Your prompt here',
});

console.log(result.providerMetadata);
// Example output: { deepseek: { promptCacheHitTokens: 1856, promptCacheMissTokens: 5 } }
```

指标包括：

- `promptCacheHitTokens`：命中缓存的输入 token 数
- `promptCacheMissTokens`：未缓存的输入 token 数

<Note>
  关于 DeepSeek 缓存机制的更多细节，参见 [DeepSeek 缓存文档](https://api-docs.deepseek.com/guides/kv_cache#checking-cache-hit-status)。
</Note>
