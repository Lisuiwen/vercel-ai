# 流式文本

文本生成有时需要较长时间，尤其是生成较长段落时。此时将文本生成过程实时流式传输到客户端会很有用，这样客户端可在生成过程中展示文本，而不必让用户等到全部完成。

## 客户端

创建一个简单的 React 组件，从 `@ai-sdk/react` 模块导入 `useCompletion` hook。点击按钮时，`useCompletion` 会调用 `/api/completion` 端点；该端点根据输入提示生成文本并流式传输到客户端。

```tsx filename="app/page.tsx"
'use client';

import { useCompletion } from '@ai-sdk/react';

export default function Page() {
  const { completion, complete } = useCompletion({
    api: '/api/completion',
  });

  return (
    <div>
      <div
        onClick={async () => {
          await complete('Why is the sky blue?');
        }}
      >
        Generate
      </div>

      {completion}
    </div>
  );
}
```

## 服务端

为 `/api/completion` 创建路由处理器，根据输入提示生成文本。路由调用 `ai` 模块的 `streamText`，生成结果并流式返回客户端。

```typescript filename='app/api/completion/route.ts'
import { streamText } from 'ai';

export async function POST(req: Request) {
  const { prompt }: { prompt: string } = await req.json();

  const result = streamText({
    model: 'openai/gpt-4o',
    system: 'You are a helpful assistant.',
    prompt,
  });

  return result.toUIMessageStreamResponse();
}
```
