# 流式文本

<Note>
  本示例使用 React Server Components (RSC)。若需要客户端渲染与 hooks，请参阅[使用 useCompletion 的「流式文本」示例](/examples/next-pages/basics/streaming-text-generation)。
</Note>

文本生成有时需要较长时间，尤其是生成较长段落时。此时将文本生成过程实时流式传输到客户端会很有用，这样客户端可在生成过程中展示文本，而不必让用户等到全部完成。

## 客户端

创建一个简单的 React 组件：点击按钮时调用 `generate`。`generate` 会调用 `streamText` 根据输入提示生成文本。在客户端消费文本流时，使用 `@ai-sdk/rsc` 模块的 `readStreamableValue` 函数。

```tsx filename="app/page.tsx"
'use client';

import { useState } from 'react';
import { generate } from './actions';
import { readStreamableValue } from '@ai-sdk/rsc';

// Allow streaming responses up to 30 seconds
export const maxDuration = 30;

export default function Home() {
  const [generation, setGeneration] = useState<string>('');

  return (
    <div>
      <button
        onClick={async () => {
          const { output } = await generate('Why is the sky blue?');

          for await (const delta of readStreamableValue(output)) {
            setGeneration(currentGeneration => `${currentGeneration}${delta}`);
          }
        }}
      >
        Ask
      </button>

      <div>{generation}</div>
    </div>
  );
}
```

## 服务端

在服务端实现 `generate`，调用 `streamText` 根据输入提示生成文本。为将生成过程流式传输到客户端，使用 `createStreamableValue` 包装可变更值并流式发送到客户端。

使用 DevTools 可看到文本生成实时流式传输到客户端。

```typescript filename='app/actions.ts'
'use server';

import { streamText } from 'ai';
import { createStreamableValue } from '@ai-sdk/rsc';

export async function generate(input: string) {
  const stream = createStreamableValue('');

  (async () => {
    const { textStream } = streamText({
      model: 'openai/gpt-5.4',
      prompt: input,
    });

    for await (const delta of textStream) {
      stream.update(delta);
    }

    stream.done();
  })();

  return { output: stream.value };
}
```
