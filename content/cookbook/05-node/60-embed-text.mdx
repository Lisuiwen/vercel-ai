# 嵌入文本

文本嵌入是以数值形式表示文本、捕获语义含义的表示，使机器能以数学方式理解和处理语言。这些向量表示对许多 AI 应用至关重要，可支持语义搜索、文档相似度比较、内容推荐等任务。

本示例演示如何使用文本嵌入模型将文本转换为嵌入。得到的嵌入是高维向量，表示输入文本的语义。更实用的嵌入应用请参阅 [RAG 示例](/cookbook/node/retrieval-augmented-generation)，其中展示如何将嵌入用于文档检索。

```ts
import { embed } from 'ai';
import 'dotenv/config';

async function main() {
  const { embedding, usage } = await embed({
    model: 'openai/text-embedding-3-small',
    value: 'sunny day at the beach',
  });

  console.log(embedding);
  console.log(usage);
}

main().catch(console.error);
```
