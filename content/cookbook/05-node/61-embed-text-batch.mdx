# 批量嵌入文本

处理大型数据集或多段文本时，逐条生成嵌入效率较低。批量嵌入可同时将多条文本输入转换为嵌入，显著提升性能并减少 API 调用。这在处理文档、聊天消息或任何需要向量化的文本集合时尤其有用。

本示例展示如何使用 AI SDK 在单次操作中嵌入多条文本。单条文本嵌入请参阅 [嵌入文本](/cookbook/node/embed-text)；实用场景请参阅 [RAG 示例](/cookbook/node/retrieval-augmented-generation)，其中演示批量嵌入在文档检索系统中的用法。

```ts
import { embedMany } from 'ai';
import 'dotenv/config';

async function main() {
  const { embeddings, usage } = await embedMany({
    model: 'openai/text-embedding-3-small',
    values: [
      'sunny day at the beach',
      'rainy afternoon in the city',
      'snowy night in the mountains',
    ],
  });

  console.log(embeddings);
  console.log(usage);
}

main().catch(console.error);
```
