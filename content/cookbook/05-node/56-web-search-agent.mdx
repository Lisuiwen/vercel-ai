# 网络搜索 Agent

使用 AI SDK 构建网络搜索 agent 有两种方式：

1. 使用具备原生联网搜索能力的模型
2. 使用工具访问网络并返回搜索结果

各有优劣。原生搜索通常更快且无需额外搜索费用，但对搜索内容控制较少，且仅限支持该能力的模型。

使用工具则更灵活，可自定义搜索策略与参数，并用于任何支持工具调用的 LLM。会产生所用搜索 API 的额外费用，但可完全掌控搜索体验。

## 使用原生联网搜索

多种模型提供原生联网搜索（Perplexity、OpenAI、Gemini）。下面看如何在各提供商上构建网络搜索 agent。

### OpenAI Responses API

OpenAI Responses API 内置 `web_search` 工具，可通过 `openai` 提供商搜索网络并返回结果。

```ts
import { openai } from '@ai-sdk/openai';
import { generateText } from 'ai';

const { text, sources } = await generateText({
  model: 'openai/gpt-5-mini',
  prompt: 'What happened in San Francisco last week?',
  tools: {
    web_search: openai.tools.webSearch({}),
  },
});

console.log(text);
console.log(sources);
```

也可用 `ToolLoopAgent` 构建 OpenAI 网络搜索 agent。以下示例展示如何从工具调用中提取查询与来源：

```ts
import { openai } from '@ai-sdk/openai';
import { ToolLoopAgent } from 'ai';

const agent = new ToolLoopAgent({
  model: 'openai/gpt-5-mini',
  tools: {
    web_search: openai.tools.webSearch({
      searchContextSize: 'low',
    }),
  },
});

const { text, sources, toolResults } = await agent.generate({
  prompt: 'What happened in San Francisco last week?',
});

// Access sources directly
console.log('Sources:', sources);

// Or extract query and sources from tool results
for (const toolResult of toolResults) {
  if (toolResult.toolName === 'web_search') {
    const { action, sources } = toolResult.output;
    console.log('Search query:', action.query);
    console.log('Sources:', sources);
  }
}
```

### Perplexity

Perplexity 的 Sonar 模型将实时网络搜索与自然语言处理结合，每条回复基于当前网络数据并含详细引用。

```ts
import { generateText } from 'ai';

const { text, sources } = await generateText({
  model: 'perplexity/sonar-pro',
  prompt: 'What are the latest developments in quantum computing?',
});

console.log(text);
console.log(sources);
```

### Gemini

兼容的 Gemini 模型可启用 search grounding，通过 Google 搜索为模型提供最新信息。

```ts
import { google } from '@ai-sdk/google';
import { generateText } from 'ai';

const { text, sources, providerMetadata } = await generateText({
  model: 'google/gemini-2.5-flash',
  tools: {
    google_search: google.tools.googleSearch({}),
  },
  prompt:
    'List the top 5 San Francisco news from the past week.' +
    'You must include the date of each article.',
});

console.log(text);
console.log(sources);

// access the grounding metadata.
const metadata = providerMetadata?.google;
const groundingMetadata = metadata?.groundingMetadata;
const safetyRatings = metadata?.safetyRatings;
```

## 使用工具

用工具做网络搜索时，可选用与 AI SDK 直接集成的现成工具，或按需求自建工具。

与内置搜索的模型不同，网络搜索工具需多步：模型先生成一次以调用搜索工具（从上下文提取查询），再处理结果并生成回复。将 `stopWhen: isStepCount(n)` 设为大于 1 时，该多步流程会自动处理。

<Note>
  使用 `stopWhen` 可将工具结果与原始问题一并送回语言模型，使模型能根据搜索结果回答用户问题，实现 agent 联网搜索并将结果融入回复的无缝体验。
</Note>

### 使用现成工具

若不想从零实现，可选用多种与 AI SDK 直接集成的网络搜索工具。

#### Exa

<Note>
  在 [Exa Dashboard](https://dashboard.exa.ai/api-keys) 获取 API 密钥。
</Note>

首先安装 Exa 的 `webSearch` 工具：

```bash
pnpm install @exalabs/ai-sdk
```

然后导入并传入 `generateText`、`streamText` 或 agent：

```ts
import { generateText, isStepCount } from 'ai';
__PROVIDER_IMPORT__;
import { webSearch } from '@exalabs/ai-sdk';

const { text } = await generateText({
  model: __MODEL__,
  prompt: 'Tell me the latest developments in AI',
  tools: {
    webSearch: webSearch(),
  },
  stopWhen: isStepCount(3),
});

console.log(text);
```

更多配置与定制请参阅 [Exa AI SDK 文档](https://docs.exa.ai/reference/vercel)。

#### Parallel Web

<Note>在 [Parallel Platform](https://parallel.ai) 获取 API 密钥。</Note>

首先安装 Parallel Web AI SDK 工具：

```bash
pnpm install @parallel-web/ai-sdk-tools
```

然后导入并传入 `generateText`、`streamText` 或 agent。Parallel Web 提供 `searchTool`（网络搜索）与 `extractTool`（提取网页内容）：

```ts
import { generateText, isStepCount } from 'ai';
__PROVIDER_IMPORT__;
import { searchTool, extractTool } from '@parallel-web/ai-sdk-tools';

const { text } = await generateText({
  model: __MODEL__,
  prompt: 'When was Vercel Ship AI?',
  tools: {
    webSearch: searchTool,
    webExtract: extractTool,
  },
  stopWhen: isStepCount(3),
});

console.log(text);
```

#### You.com

<Note>
  在 [You.com API Keys
  页面](https://you.com/platform/api-keys) 获取 API 密钥。
</Note>

首先安装 You.com AI SDK 插件：

```bash

然后导入并传入 `generateText`、`streamText` 或 agent。You.com 提供三种工具：`youSearch`（带高级过滤的实时搜索）、`youResearch`（带引用与可配置深度的深度研究）、`youContents`（以 Markdown 或 HTML 提取网页内容）：

    research: youResearch(),

  stopWhen: isStepCount(5),
});

console.log(text);
```

更多配置请参阅 [You.com AI SDK Plugin 文档](https://github.com/youdotcom-oss/dx-toolkit/tree/main/packages/ai-sdk-plugin#readme)。

### 构建并使用自定义工具

若需更细控制，可用网页抓取与爬取 API 自建工具，自定义搜索参数、处理特定数据格式并接入专用搜索服务。

#### Exa

下面看如何用 Exa 实现搜索工具：

```bash
pnpm install exa-js
```

```ts
import { generateText, tool, isStepCount } from 'ai';
__PROVIDER_IMPORT__;
import { z } from 'zod';
import Exa from 'exa-js';

export const exa = new Exa(process.env.EXA_API_KEY);

export const webSearch = tool({
  description: 'Search the web for up-to-date information',
  inputSchema: z.object({
    query: z.string().min(1).max(100).describe('The search query'),
  }),
  execute: async ({ query }) => {
    const { results } = await exa.searchAndContents(query, {
      livecrawl: 'always',
      numResults: 3,
    });
    return results.map(result => ({
      title: result.title,
      url: result.url,
      content: result.text.slice(0, 1000), // take just the first 1000 characters
      publishedDate: result.publishedDate,
    }));
  },
});

const { text } = await generateText({
  model: __MODEL__, // can be any model that supports tools
  prompt: 'What happened in San Francisco last week?',
  tools: {
    webSearch,
  },
  stopWhen: isStepCount(5),
});
```

#### Firecrawl

[Firecrawl](https://firecrawl.dev) 提供网页抓取与爬取 API。下面看如何用 Firecrawl 构建自定义抓取工具：

```bash
pnpm install @mendable/firecrawl-js
```

```ts
import { generateText, tool, isStepCount } from 'ai';
__PROVIDER_IMPORT__;
import { z } from 'zod';
import FirecrawlApp from '@mendable/firecrawl-js';
import 'dotenv/config';

const app = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

export const webSearch = tool({
  description: 'Search the web for up-to-date information',
  inputSchema: z.object({
    urlToCrawl: z
      .string()
      .url()
      .min(1)
      .max(100)
      .describe('The URL to crawl (including http:// or https://)'),
  }),
  execute: async ({ urlToCrawl }) => {
    const crawlResponse = await app.crawlUrl(urlToCrawl, {
      limit: 1,
      scrapeOptions: {
        formats: ['markdown', 'html'],
      },
    });
    if (!crawlResponse.success) {
      throw new Error(`Failed to crawl: ${crawlResponse.error}`);
    }
    return crawlResponse.data;
  },
});

const main = async () => {
  const { text } = await generateText({
    model: __MODEL__, // can be any model that supports tools
    prompt: 'Get the latest blog post from vercel.com/blog',
    tools: {
      webSearch,
    },
    stopWhen: isStepCount(5),
  });
  console.log(text);
};

main();
```
