# 生成文本

最基础的 LLM 用例是根据提示生成文本。
例如，你可能想为问题生成回答，或总结一段文字。
可以使用 `generateText` 函数根据输入提示生成文本。

```ts file='index.ts'
import { generateText } from 'ai';

const result = await generateText({
  model: 'openai/gpt-4o',
  prompt: 'Why is the sky blue?',
});

console.log(result);
```
